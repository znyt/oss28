　　那闪的乱七八糟的灯语很快便被旁边的年轻人记录了下来，一连串或长或短的线条看的薛仁贵有些眼晕，不由好奇的问道：“这是什么？为何如此奇怪？难道长安送过来的消息就是这个？”
<br/>　　“这是密语，还需要翻译一下。”一个白大褂接过年轻人递上来的纸张，看的津津有味，好像真的能从其中看出什么东西来一般。
<br/>　　薛仁贵在一边急的火上房，可又不好说什么，这帮科学院里出来的怪人一个个脾气臭的要命，才不会管你官职到底多大，稍不顺心就算是当着皇帝的面都敢发脾气，老薛自然不想给自己找不自在。
<br/>　　不过好在那白大褂也没有让他等的太久，很快就用一支铅笔在那些线条的下面写出了一个又一个字母，然后又把那些字母组合到一起，很快写出了一句话：大唐万胜。
<br/>　　这都什么和什么？薛仁贵不管用什么样的方式都理解不了这帮人是怎么从一堆长短不一的线条里看出这样四个字的。
<br/>　　白大褂看出薛仁贵的迷惑，心情大好之下便对他解释道：“薛将军，这无线电眼下刚刚研究出来，功能还很单一，只能做到这样。你看刚刚那些信号灯所亮的时间长短代表着每一次通电时间的长短，而通过数次通电，按照之前的约定，我们便可以知道接收到的是一个什么样的字母，如此反反复复之下，接收到大量的字母之后，再将这些字母组合到一起，利用拼音之法，便可组织成一句话。”
<br/>　　薛仁贵眨眨眼睛，若有所悟。
<br/>　　所谓拼音他是知道的，甚至他自己还学过一些，不过那都是好多年以前的事情了，现在能记住的字母都不多。
<br/>　　可就算如此，他还是不明白，那些灯怎么就能代表一个字母了，什么长短又是什么意思？
<br/>　　带着一份不解，薛仁贵将自己的问题问了出来。
<br/>　　白大褂听了哈哈大笑，拿起笔在纸上画了几下，然后说道：“薛将军请看，这是一长一短两个信号，我们规定这个所代表的含义就是字母a，然后一长两短便是b，以此类推，最多四个信号便可以将所有字母全都囊括其中。”
<br/>　　这下，薛仁贵算是听的明白了一些，敢情这东西并不能直接说一句话，而是要把一句话翻译成拼音，再把拼音翻译成拼音，最后才通过不断开合开关达到发送信息的目的。
<br/>　　而接收的一方也只是知道了另一边开合开关时间的长短，不过却可以根据提前的约定，将开合开关的规律换成拼音字母，再将这些字母组织到一起，弄成一句话。
<br/>　　天可怜见，这也太麻烦了，而且这种简单的通讯方式，敌人只要知道的拼音的含义，岂不是也能将这些信息复原，从而知道很多机密？
<br/>　　带着深深的担忧，薛仁贵将这个问题问了出来。
<br/>　　结果那白大褂再次笑了起来，摇头说道：“薛将军，您完全不必担心这个，因为就眼下来说，能够发送和接收这些消息还需要这台设备，如果没有这个，便是大罗神仙来了，也搞不清楚我们传递的信息到底是什么。”
<br/>　　好尴尬，一着急竟然忘了这一点，薛仁贵揉了揉脸，想了想问道：“那现在我们可以发信息给长安么？”
<br/>　　“可以试试，现在波段已经定下了，短时间内不会再改变。”白大褂很想炫耀一下那个叫无线电台的大家伙，薛仁贵有消息要发，这正是他巴不得的。
<br/>　　“嗯，那你帮我发个信息，就说这边天气突变，需要一批手套，大概十万副左右，看看那边如何回复。”薛仁贵想了想，决定从实际出发，把眼下需要解决的问题解决一个。
<br/>　　“没问题！“白大褂十分痛快的答应着，拿起笔飞快的将薛仁贵的话记录下来，然后交给一边的那个年轻人。
<br/>　　很快，薛仁贵就发现那个年轻人拿着一张已经变成长短不一线条的白纸走到了一个开关的跟前，开始拼了命的折腾起那个开关，一会儿拉开，一会儿合上，就好像那开关与他有仇一般。
<br/>　　而随着那青年不断折腾那个开关，那些被白大褂称之为信号灯的灯泡又开始闪了起来，一明一暗闪的人眼晕。
<br/>　　又过了大概不到一刻钟的时间，青年拍拍手走了回来，往纸上签了一个名字，然后递给薛仁贵：“将军，麻烦你签字。”
<br/>　　“什么意思？”薛仁贵愣愣问道。
<br/>　　“哦，是这样的，我们这里每发出一份信息都要有记录和存档，以备将来查找，刚刚他已经将将军你的要求发到了长安，所以需要将军您签字确认一下。”白大褂对薛仁贵解释道。
<br/>　　老薛神情麻木的接过青年递上来的纸，在上面草草签下自己的名字，然后呆呆问道：“你们的意思是，我的那个要求已经发出去了，长安那边已经收到了？”
<br/>　　“对，如果不出意外的话，现在正在送往兵部的路上。”白大褂点头不无得意的说道。
<br/>　　“这，这怎么可能，这，这也太快了吧？”薛仁贵实在无法接受这个现实，想他以前在大非川驻扎的时候，往长安送一次信怎么也要半月之久，后来到了吐蕃，送一次信要多半个月，而现在送出一封信虽然有了火车，但同样也需要小半个月。
<br/>　　可是，就在刚刚，这个带着眼镜的白大褂竟然告诉自己，自己在一刻钟之前提出的要求已经到长安了，这，这让薛仁贵几不敢相信自己的耳朵。
<br/>　　以他的能力竟然完全理解不上去，怎么可能有如此快的信息传递方式，那个巨大的箱子里面到底装了些什么东西，怎么可能，怎么可能……。
<br/>　　“薛将军，这叫科学，您别看他在那边不断开合开关，可是他每一次开合，都会有一段电波以每秒六十万里的速度发往长安，按照我们现在的这个距离，可以说我们这边发送的消息长安那边可以同步接收。”
