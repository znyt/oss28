　　林无影手中的短刀较为特殊，呈透明之状，薄如蝉翼。
<br/>　　其一刀斩出，刀气犹如一道细线一般，不易被人察觉，但同样锋锐无比，威力不凡。
<br/>　　咻！
<br/>　　细线般的刀气，快如闪电，裹挟凌厉的杀机，向苏莫迎面斩去。
<br/>　　凌厉的刀气将虚空平滑的切开，仿佛整片空间都一分为二。
<br/>　　这一刀颇为不弱，然而，这一刀完全不能对苏莫造成丝毫的伤害。
<br/>　　只见苏莫手中长剑一挥，剑光闪过，细线般的刀气瞬间便被斩灭。
<br/>　　“你败了！”
<br/>　　苏莫淡漠的说了一句，另一只手掌张开，一把便抓在了飞来的林无影的脖颈上。
<br/>　　林无影并无反抗之力，事实上从他一刀斩出之后，便开始控制不住体内的玄力了。
<br/>　　他体内的玄力严重絮乱，疯狂的倾斜而出，根本不受他的控制。
<br/>　　而此刻，他体外裹挟的那股无形的气流，也被苏莫吞噬了。
<br/>　　无形的气流入体，苏莫顿时心中微震，这无形的气流，居然就是战魂。
<br/>　　“原来如此！”苏莫心中暗喜，原来这气流便是能让对方隐藏身形的战魂，果然玄妙。
<br/>　　战魂被吞噬，林无影顿时不再挣扎，彻底石化了。
<br/>　　因为战魂中蕴含他的神魂，战魂被吞噬，身体便也只是一具皮囊而已，彻底死亡了。
<br/>　　少倾，苏莫收起林无影手中的短刀，和对方的储物戒，手臂微微用力一甩。
<br/>　　林无影的尸体，顿时飞射而出，划过一道优美的弧线，砸在了远处一座小岛上。
<br/>　　嘭！
<br/>　　林无影的尸体砸在了尤天韩的面前，溅起一片尘土。
<br/>　　静！
<br/>　　死一般的寂静！
<br/>　　所有人均是怔怔的看着这一幕，林无影依旧败了，即便神出鬼没，行走于黑暗之中，身形隐藏不出，依旧被苏莫打败了！
<br/>　　苏莫，这是同阶无敌了吗？
<br/>　　众人心惊不已。
<br/>　　不过，大多数人均是有些疑惑，不太明白林无影为何会暴露出来，又为何会突然就死了，因为苏莫好像并未下杀手吧！
<br/>　　只有少量的内府弟子，或者是武尊境的导师人物，才看出了门道，个个面现惊疑之色。
<br/>　　适才苏莫展现的力量，他们虽然没有切身感觉到，但也能看出那是吞噬的力量。
<br/>　　苏莫动用吞噬的力量，让得林无影暴露了出来，连战魂好像都被吞噬了！
<br/>　　这种吞噬力量，不少人都非常熟悉，因为神武学府的神子，就拥有这种恐怖的力量。
<br/>　　而学府的神子，拥有太古武魂榜排名第九十三的吞灵战魂，难道苏莫也拥有吞灵战魂不成？
<br/>　　一众内府弟子和导师心中疑惑，这应该不太可能吧！
<br/>　　太古武魂榜前一百的战魂，世所罕见，万亿人之中也很难出现一位，苏莫怎么可能拥有！
<br/>　　苏莫应该是有其它吞噬类的战魂吧！
<br/>　　比如太古武魂榜排名第三百九十八名的七彩吞天蟒战魂！
<br/>　　当然，这也不绝对，因为苏莫好像是有剑之战魂，或许是身上有能吞噬的宝物也说不定！
<br/>　　众人心中暗暗猜测了起来，越发觉得苏莫隐藏的很深。
<br/>　　尤其是木璃导师，她的柳眉微微皱起，心中对苏莫亦是充满了好奇，她感觉完全看不透苏莫。
<br/>　　“尤天韩，你们还有人敢战吗？”苏莫淡笑着问道，吞噬了林无影的战魂，他心中大爽。
<br/>　　这个特殊的战魂，用处可是非常的大，简直是杀人逃命的利器。
<br/>　　所以，他不惜暴露吞噬之力，也要将这种战魂吞噬。
<br/>　　反正他也没有什么顾忌，这是在神武学府，又不是在巫族额天冥圣城，吞噬战魂就算暴露了也无妨。
<br/>　　因为别人不可能知道他吞噬战魂的种种逆天能力，最多知道他吞噬战魂就是一般的吞噬而已。
<br/>　　尤天韩的脸色，阴沉的仿佛要跌出水来了，他眼眸盯着脚下林无影的尸体，双手都忍不住颤抖了起来。
<br/>　　他是彻底怒了，接连陨落两名天才，这是对他们太史世家派系巨大的打击。
<br/>　　再派人上场？
<br/>　　他们太史世家派系，在武帝境四重境界之中，已经没有更厉害的天才了。
<br/>　　尤天韩真想一掌将苏莫镇杀，但考虑到此地有不少的导师，以及苏莫背后的火元圣师，他极力的压制着心中的杀机。
<br/>　　其他太史世家派系的人，脸色并不比尤天韩好看，个个眸中杀机爆射，双拳捏的咯咯作响。
<br/>　　一众观战的普通弟子，以及其它派系的人，见苏莫杀死林无影后，居然还敢邀战，顿时心惊不已。
<br/>　　苏莫，这是要和太史世家派系死磕到底了啊！
<br/>　　没有人说话，所有人的目光，均是紧紧的凝视着尤天韩等太史世家派系的人，想要看看他们如何接招！
<br/>　　嘭！
<br/>　　就在此时，尤天韩的身后，一名高高壮壮的青年轻轻踏出一步，走到了前方。
<br/>　　“苏莫，我是武帝境五重修为，比你略高一重，可敢一战？”高壮青年目光如刀，直视苏莫的双眼，沉声问道。
<br/>　　高壮青年话音一落，十几座岛屿上顿时一片哗然。
<br/>　　“是滕金山，滕金山忍不住要出手了！”
<br/>　　“这下有好戏看了！”
<br/>　　“苏莫应该不敢应战吧！毕竟修为不同！”
<br/>　　“……”
<br/>　　众人议论纷纷，声音吵杂不已。
<br/>　　神武学府之中，虽然天才众多，但天才也是有档次之分的。

<br/>　　而滕金山，绝对算是第三个档次的天才。
<br/>　　至于林无影和雷灿，只能算是第四个档次的天才。
<br/>　　滕金山无论是天赋还是越级战斗力，都要超越雷灿和林无影一个档次，完全不可同日而语。
<br/>　　而且，滕金山还比苏莫高一重的修为，所以不少人猜测苏莫不敢应战。
<br/>　　滕金山战力无双，同阶之中未曾一败。
<br/>　　最主要的是，其拥有逆天的防御力，就算是实力高于他的人，也破不了他的防御。
<br/>　　这是因为滕金山有特殊体质，乃是万中无一的金刚之体，霸道无双，万法不破。
<br/>　　这种金刚之体，在太古异体榜上的排名，高达二百二十八名。
<br/>　　能在一千种特殊体质之中，排名如此之高，便可见这种金刚之体的恐怖。
