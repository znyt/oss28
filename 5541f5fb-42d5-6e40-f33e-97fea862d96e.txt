　　　　女子听着，开口道：“这两种说法都有点玄乎，我都不信”
<br/>　　男子笑的说：“我到是希望是第一种传说，三阶生灵的女妖，化为人形，真的好想见识一下”
<br/>　　“哼，我看是女鬼还差不多吗，这笛声凄凉，到很适合女鬼的出场”女子吃醋似的说。
<br/>　　“嘻嘻，真与假的情况如何，恐怕连我们的院长都说不清楚”
<br/>　　暗处的涂小安听的一阵心惊，暗道：“没想到这个学院竟然有这等古怪之地，镇压三阶生灵？那么大手笔吗，有机会我还真想下水探个究竟”

<br/>　　“你知道吗，现在外院听说群蛇出没，遍地是蛇，都快变成蛇窟了呢”
<br/>　　女子道：“关我什么事，反正内院无蛇就好了”
<br/>　　男的笑了：“我到希望内院也有蛇，这样就有蛇羹吃了”
<br/>　　涂小安在听几句，摇了摇头，不准备再继续听下去，自那对情侣旁边无声无息的绕了过去。

<br/>　　时不时有野兽的嘶吼声传出。
<br/>　　涂小安冷淡无波的环视一圈，最后目光锁定到一个数过去第五洞前。
<br/>　　第五个山洞，特殊蛇类气息浓郁，想必猫精就藏匿与此了。
<br/>　　它跑什么呀，看来是不相信自己。
<br/>　　涂小安思忖一下，直接朝着光线昏暗的山洞进去，一入山洞，有一股阴风呼啸，让人毛骨悚然。

<br/>　　忽然，里面有一大股的黑色蝙蝠吱吱吱的群体飞了出来，足足上百只，但是这些蝙蝠好像是受到了什么惊吓，直接无视涂小安，逃命似得朝着洞外飞了出去。
<br/>　　旋即，洞深处，一股残暴的蛇类气息席卷，阴冷无比。
<br/>　　这猫精看来又是发疯了吗。
<br/>　　涂小安现在知道，海棠的蛇类气息散发越浓郁，它的神志也就越不清楚。
<br/>　　蛇王石...
<br/>　　涂小安猜测海棠应该是将蛇王石吞进了肚子里，才会产生变异，成了猫不猫，蛇不蛇的存在。

<br/>　　这蟒蛇条纹鲜艳，品种不错，是觉醒生物，但是却残忍的被猫精当成了食物。

<br/>　　怪不得那么多山洞，它选择躲进这洞中，原来这洞中藏着一条蟒蛇，应该是古学院圈养的。

<br/>　　“嗷...”猫精一声惊天动地的咆哮，蛇类的气息铺天盖地的席卷出去。
<br/>　　整个山洞剧烈的震动起来，碎石尘土簌簌而落。
<br/>　　涂小安瞬间感觉到一股强风气浪席卷，连忙浑身一震，席卷而来的气浪立刻溃散。
<br/>　　看这猫精如此样子，涂小安就知道它已经失去理智，而且它身上的蛇类气息越来越强大，按照这种趋势，恐怕没多久，这古学院也要被群蛇笼罩。
<br/>　　这蛇王石，到底是蛇王的什么器官，能有那么大的威力。
<br/>　　难不成是三阶生灵的内在金丹所化？
<br/>　　要是这样，那可是大宝贝，对于涂小安这种蛇类来说简直是如同至宝。
<br/>　　“海棠，冷静”
<br/>　　涂小安一声大喝，可回复他的却是猫精凶煞的咆哮声，一双眼眸宛如一片血海，充满了杀意。
<br/>　　很显然，海棠将涂小安也当成了食物，想一口吞了。
<br/>　　就在这时，它动了，动如奔雷，其势滔滔，张开一口大嘴，露出一排寒光凛然的青白獠牙，骤然朝着涂小安冲去。

<br/>　　“该死，看来又要费上一番手脚”
<br/>　　涂小安原地不动，眸若冷电，一手握拳，整个拳头顿时火焰沸腾，周边的温度骤然上升。

<br/>　　涂小安淡淡一语，拳头一松，火焰顿消。

<br/>　　涂小安蹙眉，这才是耐揍呀，刚才这一拳，他可没有收力。
