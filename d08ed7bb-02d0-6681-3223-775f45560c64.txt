　　苏莫闻听火元圣师之言，见对方皱眉，心中顿时一惊。
<br/>　　他心中急思，若是自己不愿意留在此地炼器，会不会得罪对方？
<br/>　　不过，他是真的不能留在此地炼器，他要追求极致的武道，尽快的提升实力。
<br/>　　炼器，会耗费他大量的时间。
<br/>　　虽然以前他炼器，没有消耗多少的时间，但随着炼器等级的提高，炼器难度会越来越大，自然会越来越消耗时间。
<br/>　　所以，即便会得罪这位火元圣师，他也要拒绝了。
<br/>　　“前辈，晚辈不想在此地学习炼器！”苏莫沉声说道，他之所以敢拒绝，也是有些底气的。
<br/>　　对方总不可能，因为他不学习炼器，而迁怒于他吧！这样未免也太小气了，不符合对方强者的身份。
<br/>　　苏莫此言一出，火元圣师和白衣女子，均是神情微怔。
<br/>　　火元圣师面现意外之色，因为他从来没有想过，有人会拒绝他教授炼器之术。
<br/>　　虽然他没有说要收苏莫为徒，但他让苏莫留在此地，便说明了会教授炼器大道。
<br/>　　而苏莫，却拒绝了。
<br/>　　白衣女子更是满脸愕然，居然真的不愿意！
<br/>　　她心中不禁无语，火元圣师是她的导师，更是古周星最高级别的炼器师，就算在整个古灵星河之内，都是最巅峰的圣级炼器师之一。
<br/>　　整个古灵星河之内，无数的炼器师想拜火元圣师为师，学习炼气大道。
<br/>　　然而，火元圣师并不收徒，最多就是指点一些炼器天赋极佳的神武学府弟子。
<br/>　　但就是如此，都有大量的年轻炼器师，加入神武学府，并且极力想得到火元圣师的指点。
<br/>　　但是，能得到火元圣师指点的炼器师，千不足一，少之又少。
<br/>　　可以说，只要火元圣师一句话，就能有无数的炼器师蜂拥而至，愿意跟随他学习炼器。
<br/>　　但现在，火元圣师亲自开口，让苏莫留在此地学习炼器之术，苏莫居然拒绝了！
<br/>　　白衣女子无语至极，她身为外院的导师，更是火元圣师的记名弟子，能得到火元圣师指点的机会都不多。
<br/>　　她实在想不明白，这个苏莫是傻还是呆！
<br/>　　如此天大的好事，居然就这么果断的拒绝了！
<br/>　　火元圣师眉头紧皱，少顷之后，淡淡的问道：“既然你不愿意在此跟随老夫学习炼器，为何要来的此地！”
<br/>　　“晚辈想加入神武学府，成为神武学府的弟子！”苏莫沉声说道。
<br/>　　火元圣师闻言沉默了，他很看重苏莫，他能清晰的感觉到，苏莫的精神力天赋远超同阶武者数倍不止，乃是天生的炼丹师和炼器师之才。
<br/>　　本来，他想先将苏莫留在此地，指导一段时日。
<br/>　　若是苏莫真能符合他的要求，他甚至可以将苏莫收为亲传弟子，悉心教导，传授一生所学。
<br/>　　但是，让他郁闷的是，这个天生奇才，居然只想成为神武学府的普通弟子，而不想成为他的弟子！
<br/>　　这真是让他心中苦笑不已。
<br/>　　难道此子不明白，能得到他指导的机会，是多么的珍贵吗？
<br/>　　不过，既然苏莫不愿随他学习炼器，火元圣师堂堂的圣级炼器师，自然不可能强求。
<br/>　　叹了口气，火元圣师挥了挥手，道：“木璃，送他出去吧！”
<br/>　　“是！导师！”白衣女子点了点头。
<br/>　　苏莫见此，顿时心中一急，对方还没说同不同意他加入神武学府呢！
<br/>　　“前辈，晚辈能成为神武学府的弟子吗？”苏莫急忙问道，满脸露出一丝忐忑之色。
<br/>　　“可以，你去外院吧！木璃为你安排！”火元圣师略一沉吟，便微微颌首，他准备再给苏莫一次机会，说不定过几日之后，苏莫就想通了。
<br/>　　言罢，火元圣师便再次闭上了双眼。
<br/>　　“多谢前辈！”苏莫闻言顿时心中一喜，立刻向对方抱拳道谢。
<br/>　　他只要能留在神武学府便好，在这神武学府中，他就能得到更高级的功法武技，和更好的培养。
<br/>　　“走吧！”白衣女子木璃招呼苏莫一声，便向外走了出去。
<br/>　　随即，苏莫便跟随着木璃离开了宫殿，而后飞出了山峰。
<br/>　　很快，两人便回到了之前刚来到这片星空时，所处的地方。
<br/>　　之前，苏莫被这内府的景象震惊，没有仔细注意出入口，这时他才注意到，此地虚空之中，有一个浩大的漩涡。
<br/>　　唰！
<br/>　　木璃身形不停，瞬间便飞进了漩涡之中，苏莫也随之跟了进去。
<br/>　　外界，白玉石门之前，光芒一闪，木璃和苏莫的身形便出现在了外界。
<br/>　　而后，木璃也不言语，身形不停，急速向远处飞掠而去，苏莫立刻跟上。
<br/>　　不多时，木璃飞到了外院的几座建筑之前，这几座建筑有阁楼、有宫殿、也有院落。
<br/>　　“这几座居所，均是无人居住之地，你可以随意选择一处居住，也可以在附近自己建造！”木璃淡淡的说道。
<br/>　　“嗯！”苏莫闻言点了点头，同时他也是明白了，为何这外院的建筑如此杂乱无章了，原来外院弟子可以随意建造。
<br/>　　“明日.我会将身份牌为你送来！”木璃再次开口，言罢，便转身准备离去。
<br/>　　“等等！”苏莫见此突然开口，喊住了对方。
<br/>　　“何事？”木璃转头问道，面无丝毫表情。
<br/>　　“多谢！”苏莫向对方抱拳，真诚的感谢道，对方为他安排了一切，他自然要感谢一句。
<br/>　　木璃闻言，面色微微一怔，随即她略一沉吟，淡淡的道：“我没见过你这么蠢的人！”
<br/>　　言罢，木璃身形一闪，如同一道白色幻影，急速离去。
<br/>　　呃~~
<br/>　　苏莫愣住了，望着对方离去的背影，他满脸愕然之色。
<br/>　　自己很蠢吗？
<br/>　　自己哪里蠢了？
<br/>　　这女人有病吧！
<br/>　　苏莫心中诽腹不已，随即他摇了摇头，便不再多想，开始挑选自己的住所。
<br/>　　既然有现成的住处，他当然不会自己再费力建造了。
<br/>　　很快，苏莫便挑选了一座精美的三层阁楼，作为自己以后的住所。
<br/>　　这座阁楼内空间不小，一楼是一个大厅，二楼有数个房间，足够他使用了。
<br/>　　阁楼可能很长时间无人居住了，其内落满了灰尘，苏莫随意的打扫了一番，便居住了下来。
