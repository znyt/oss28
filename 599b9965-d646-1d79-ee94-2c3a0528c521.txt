　　“快杀了萧逸那逆贼。”
<br/>　　宗主等人看到天际中那人影的面容后，脸色一变，当即大喝一声。
<br/>　　宗主、血谷主等四人，四位天炎卫队长。
<br/>　　八人联手而上，直取萧逸而去。
<br/>　　萧逸立于原地，没有动作，只是淡笑一声。
<br/>　　天际那人影，一个瞬间，便已来到他身旁。
<br/>　　手指轻扬，一道剑气打出。
<br/>　　宗主等八人，口中猛地吐出大口腥血，瞬间被剑气轰飞。
<br/>　　轰飞的速度，甚至比他们攻向萧逸的速度还要快得多。
<br/>　　待得八人站稳身子后，看向来人，脸色难看到极点。
<br/>　　“易疯子。”宗主等人，咬牙切齿地说着。
<br/>　　“易天行。”远处的段云，脸色变得极其凝重。
<br/>　　“天行。”只有大长老，脸上喜意十足。
<br/>　　看来，果然如传言所说那般，大长老与易老确实交情不错。
<br/>　　萧逸看向来人，也微微吃惊，“这就是易老的真正实力吗？”
<br/>　　来人的出现，瞬间成为了在场所有人的焦点。
<br/>　　不错，来人正是易老。
<br/>　　“走吧。”易老桀骜的目光，扫视了在场所有人一眼。
<br/>　　随后看向萧逸，冷漠地说了一声。
<br/>　　“想走？你妄想！”宗主等人，冷喝一声。
<br/>　　“易疯子，你好大的胆子，不仅自己回来了，竟还让你徒弟大闹王都。”
<br/>　　“哼，果然是有其师，必有其徒。”
<br/>　　“当年你血洗血雾谷，在王都大肆杀戮，弄得天怒人怨。”
<br/>　　“今日，你徒弟也是如此。”
<br/>　　“杀了我王都如此多强者，罪行滔天，现在拍拍屁股就想走人？”
<br/>　　易老的面容，很是淡漠，仿佛不带任何生气。
<br/>　　“是我现在带他走，还是我杀光你们，再带他走。”
<br/>　　“自己选一个。”
<br/>　　“大言不惭。”血谷主冷笑一声。
<br/>　　“今日王都强者齐聚，你还妄想安然带人走？”
<br/>　　“真当我们一众强者是纸糊的吗？”
<br/>　　“难道不是吗？”易老淡漠地说了一声。
<br/>　　“你…”血谷主等人，脸色发黑。
<br/>　　“走吧。”易老没有理会血谷主等人，只是淡漠地再次对萧逸说了一句。
<br/>　　萧逸摇了摇头，“白墨寒尚未死，小子不会走。”
<br/>　　“你与他，并无直接仇怨，你杀他做什么？”易老仍旧淡漠。
<br/>　　萧逸皱眉道，“欺师灭祖，不该杀？”
<br/>　　“那也是我去杀，与你何干？”易老冷漠地反问一声。
<br/>　　“我想不明白，为何你总爱多管闲事。”
<br/>　　“更想不明白，你哪来的资格管我的事。”
<br/>　　“另外，你我之间，何时有的师徒之名？”
<br/>　　此言一出，周遭之人，纷纷侧目。
<br/>　　在他们眼中，易老看向萧逸的目光，无比淡漠。
<br/>　　惊愕之余，甚至发出了讥笑。
<br/>　　“原来易疯子并没有承认萧逸这个徒弟吗？”
<br/>　　“难道，这只是萧逸一厢情愿？”
<br/>　　远处，白墨寒冷笑道，“哈哈哈哈。”
<br/>　　“我就说嘛，老东西你当年好歹是裂天剑宗第一天才。”
<br/>　　“我这种绝世天骄，入你门下，倒是正常。”
<br/>　　“萧逸那种控火兽废物，竟也能得你青睐？”
<br/>　　“啧啧，今日这场闹剧，竟只是他一厢情愿。”
<br/>　　“可笑，可笑，真真可笑。”
<br/>　　一瞬间，萧逸仿佛成了在场所有人眼中的傻瓜。
<br/>　　一声声讥讽传来。
<br/>　　萧逸却脸色不变。
<br/>　　看着易老那冷漠的表情，不禁微微一笑。
<br/>　　“好不容易等到你现身，没想到，竟是等来这样一些话。”
<br/>　　萧逸淡淡地说着。
<br/>　　易老淡漠道，“我只是说实话罢了，怎么，伤了你的自尊？”
<br/>　　萧逸蓦地一笑。
<br/>　　这一笑，在所有人眼中，似乎是‘可怜’而‘凄凉’的一笑。
<br/>　　竭心竭力，为报师仇，却只是换来这样的一些轻蔑和呵斥。
<br/>　　任何人都难以承受吧。
<br/>　　孰料，下一秒，萧逸却摇了摇头。
<br/>　　“伤我的自尊？易老，你与我许久不见了，怎地说话永远这般没有新意？”
<br/>　　“你什么意思？”易老皱了皱眉头。
<br/>　　周遭所有强者，没有出言打扰。
<br/>　　在他们眼中，似乎更乐于看到原本嚣张至极的萧逸，沦为一个‘小丑’。
<br/>　　萧逸没有理会周遭的嘲笑，只是直视着易老。
<br/>　　“既然易老认为小子在多管闲事，一厢情愿。”
<br/>　　“那易老你的多管闲事，你的一厢情愿，又算什么？”
<br/>　　“你在说什么？”易老冷漠的神情中，似有不满和怒气。
<br/>　　萧逸摇了摇头，轻声道，“易老不必骗我，你更瞒不过我。”
<br/>　　“一年了，从我初入王都，不，准确来说，是我初在王都现身时。”
<br/>　　“易老便一直在暗中跟着。”
<br/>　　“我在酒楼过夜，你跟着；我与大长老聊及你的事，你在暗处听着。”
<br/>　　“我进入裂天剑宗参加考核，你跟着。”
<br/>　　“我被诸多宗门长老刁难，你在暗处看着。”
<br/>　　“一年了，包括我这一个多月在王都四处流浪，再到今日的大战伊始，易老一直跟着。”
<br/>　　“你，一直都在！”
<br/>　　萧逸说到最后，嘴角咧过一丝会心笑意。
<br/>　　“你…”易老脸色一变。
<br/>　　萧逸笑笑，手中一道剑气猛地出现于手中。
<br/>　　这道剑气，不属于萧逸，而是散发着易老的气息。
<br/>　　正是易老当初给予他的那道剑气。
<br/>　　“易老该知道，小子向来谨慎，无论何时何地，都不会有半分懈怠。”
<br/>　　“若有人跟踪我，就算实力再强，我也能有所察觉。”
<br/>　　“至于这道剑气，我一直不曾用。”
<br/>　　“只因，这是易老留给我的，唯一一样与你气息相连之物。”
<br/>　　“凭此，每次我都能感觉到你的存在。”
<br/>　　“你一直都知道？”易老脸上的淡漠，早已全无，只是重重地叹了口气。
<br/>　　“知道。”萧逸点了点头。
<br/>　　“只是，以易老的实力，若不愿现身出来，我也没办法。”
<br/>　　“当然了，在查清所有事之前，小子也没那脸面让您现身。”
<br/>　　“当年之事，你不愿说，我便自己查。”
<br/>　　“现在查清楚了，自然让您出来了。”
<br/>　　萧逸得意地笑着。
<br/>　　“臭小子。”易老同样会心一笑，但下一秒，却呵斥一声。
<br/>　　“弄这般动静，只为逼我出来，你真是胆大包天。”
<br/>　　“呵呵。”萧逸笑笑，“易老真以为，小子是个冲动之辈？”
<br/>　　“我敢弄这般动静，自然有全身而退的把握。”
<br/>　　“待我将那人杀了，我们一起走。”
<br/>　　说罢，萧逸身影一闪，手中之剑，凭空而现。
<br/>　　仗剑朝白墨寒而去。
<br/>　　“休想得逞。”四位天炎卫队长，冷喝一声，手中再次凝聚一股恐怖火焰。
<br/>　　萧逸不屑一笑，“真以为你们之前那些火焰能伤我？”
<br/>　　“在我面前控火，你们还不够格。”
<br/>　　“散。”
<br/>　　话音刚落，四位天炎卫队长脸色大骇，手中火焰，竟是凭空消散。
