　　“本尊要三滴祖巫精血，来交换突破巫神之法门。”万劫魔尊那淡淡的声音在盘古殿前响起。
<br/>　　“好，三滴精血，我么可以给。”
<br/>　　帝江等人沉默一会儿后，说道。他们自是知道，三滴祖巫精血换突破巫神的法门，是他们赚了。
<br/>　　但是，要用谁的精血呢？这是一个问题。这里有十二祖巫，他却要用三滴精血，这就意味着，有三人要放血了，这对这三人很不公平，搞不好会发生乱斗。
<br/>　　还好，万劫魔尊给出了限制，只听他慢慢说道：
<br/>　　“本尊要的三滴祖巫精血，乃是后土祖巫一滴，烛九阴祖巫的一滴，帝江祖巫的一滴。”
<br/>　　万劫魔尊的要求自然是很有讲究的，后土乃土之祖巫，虽然不怎样，但她还有一重身份，日后能够身化轮回之人，又岂会简单。烛九阴乃时间祖巫，血脉之中，蕴含的自然是时间之道。帝江乃空间祖巫，空间之道元始虽然领悟了，但并不是那么深，如今多了一些参考，自然是极为美妙的。
<br/>　　“可以。”三人沉默了片刻，沉声道，“但你那突破巫神的法门一定要是真的。”
<br/>　　“爽快。”万劫魔尊笑道：
<br/>　　“放心，本尊还不屑于说假话。既然汝等如此爽快，那么，本尊就再免费给你们提个醒。你等若是有时间，还是多参悟一下盘古殿中的盘古法相吧，说不定你们还能从中领悟出点什么。”
<br/>　　“道友此言何意？”后土问道，他们之前就有所感触，只是一直不明所以。
<br/>　　“没什么意思。”
<br/>　　众人无奈，只好放弃。之后，后土、烛九阴、帝江三人将自己的精血放出三滴。顿时，天空仿佛变得沉重起来，三滴精血闪烁着淡青色的不朽神光，各种玄奥神光，尽显其中。
<br/>　　“喏，这是突破巫神的法门。”
<br/>　　确认无误后，万劫魔尊便直接将法门分成十二分，打入每个祖巫脑海之中。之后，收了三滴精血。
<br/>　　这巫神突破法门，是元始根据自己的《玄黄不灭》炼体功法改变了一些，再加上盘古传承中的一些信息，根据祖巫的情况，创出来的。原理很简单，就是不断的挖掘祖巫血脉中的大道法则。
<br/>　　盘古的大道，又岂会简单？通过不断地挖掘血脉，之后用血脉之中的大道法则淬炼肉身，这可比他们之前用地煞之气来磨练肉身好多了。毕竟，地煞之气用多了还会在自己身体之中留下祸患，这需要他们不断用天地领悟来滋养身体。
<br/>　　而若是用他们炫迈之中的法则之力来淬炼肉体，不仅使他们的血脉更加精纯了，而且以后还不用怕煞气影响自己的神志。祖巫的好战粗狂虽然有天生的血脉影响，但与用煞气炼体还是分不开的。
<br/>　　用法则之力炼体到准圣巅峰后，就需要将用血脉之中的法则在体内开辟法则空间，之后汇聚法则之海，不断积累，直至法则之海凝结成晶，才能突破巫神之境。
<br/>　　十二祖巫都非凡人，自然清楚这份法门的重要性，在确定无误之后，万劫魔尊便直接带着精血闪身离开了。
<br/>　　十二祖巫也需要尽快商量执掌洪荒大地的事情，自己自然不会留在此地打扰人家。
<br/>　　“大哥，此人你怎么看？”在万劫魔尊走后，烛九阴对着帝江问道。
<br/>　　“此人不简单啊。不说其他的，就算他的实力，就并不逊色我们之中任何一人，还有那神秘的巫神突破法门，此人不可招惹。”帝江深叹一口气，缓缓道。
<br/>　　“是啊，此人颇为神秘。”时间祖巫烛九阴也默默道。
<br/>　　“算了，只要此人不与我们为恶，我们也不必多管，我们还是来讨论一下，怎么分配部落之事？”
<br/>　　“好。”
<br/>　　当下。十二祖巫聚在一起，商议怎样分配部落之事。经过一番商议，最终决定分成十二个大部落，十二个大部落中又有若干个小部落。这些大部落分散到洪荒东、南、北大地之中，遥相呼应。
<br/>　　同时，他们也决定留下刑天、蚩尤等数十个极为厉害的大巫，命其留下守护盘古殿，不得外出。
<br/>　　之后，一个个祖巫带着各自的部落，离开了盘古殿，朝着洪荒大地出发。这些部落，没过多久便分布在东方、南方、北方大地上，紧接着，众位祖巫立刻下令，准备祭祀盘古神尊的事宜。
<br/>　　而这个时候，万劫魔尊并没有返回昆仑山，而是来到西方大地，找到了一座名山，名为太华山。太华山，也就是日后的西岳华山。《山海经卷二西山经》中记载，太华之山，削成而四方，其高五千仞，其广十里，鸟兽莫居。
<br/>　　于是，这一日，太华山有了主人，他用了很少的时间，整顿了太华山方圆数千万里的实力，在太华山上立了一方势力，名曰“劫魔殿”。
<br/>　　就这样，一个庞大的势力悄无声息地在西方大地之上屹立起来。
<br/>　　万劫魔尊建起劫魔殿后，并没有大肆扩张势力，而是赐下手下修炼功法后，便闭关不出了。
<br/>　　万劫魔尊修炼的是《万劫魔典》，在这大劫之中，万劫魔典最好提升实力。只要有争斗，就会有劫气产生，各种劫气，他来之不拒，全部吸收。
<br/>　　在他的紫府之中，形成了一个漆黑的小世界，这个小世界在缓缓旋转，源源不断地吸收着洪荒之中的各种劫气，然后形成灰色大陆，形成阴暗天空，形成各种具有魔气的事物。
<br/>　　随着吸收的劫煞之气的增多，大陆在扩张，天在变高，地在变厚，他自身的实力也自然随之在增加。
<br/>　　万劫魔尊开始研究起三滴祖巫精血中蕴含的玄妙。空间祖巫帝江的精血自然不用多说，其中蕴含着盘古对空间法则的感悟，元始自身便会空间法则，但两者的研究方向不同，两者相互结合，给了他很大的启发，于是乎，他对空间一道的理解，更加精进了。
<br/>　　时间祖巫烛九阴的精血，其中的时间之道，也是非常厉害的法则。有这么一句话，来形容排名前五的大道法则：
<br/>　　空间为王，时间为尊，命运不出，因果称王！
<br/>　　由此可见，时间法则的厉害。时间贯穿于古今未来，四季轮回，时空交织，纪元更替，乃是除了命运、因果之外，最为神秘的法则。时间之力，也最是难以掌控。
<br/>　　岁月如刀，刀斩天骄！
<br/>　　岁月杀人，无声无息，让人防不胜防。如今，有了时间祖巫的精血，再加上三生中蕴含的时间、命运等法则，两者相互印证，对时间一道的参悟，便会更加容易了。
<br/>　　最后是，后土祖巫的精血，其中浓郁的土之法则并没有让元始意外。要说后土祖巫精血之中没有土之法则那才叫奇怪呢。魔尊要她的精血并不是为了土之法则，而是为了研究后土到底有何特殊之处，她为何能够以区区祖巫之躯身化轮回？
<br/>　　但万劫魔尊并没有报太大希望，因为他知道，仅凭一滴祖巫精血，就想要查清其中的秘密，这不太可能。但他仍没有放弃，运用自己的神痕道眼，元神之力，心力等各种神秘的力量来探查这一滴精血。
<br/>　　时间过去很久，他还是没有任何收获。但，就在他快要放弃之际，终于发现了一丝不同寻常之处。
<br/>　　“这……这是——？”
