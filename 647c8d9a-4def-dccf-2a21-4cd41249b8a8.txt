　　随着实力的不断提升，苏莫御剑飞行的速度，也比以前提升了很大一截。
<br/>　　他只用了三日有余的时间，便接近了帝玄山脉。
<br/>　　远远的，距离帝玄山脉还有两三万里之时，苏莫催动幽冥魔瞳武魂，向帝玄山脉方向望去。
<br/>　　放眼望去，整个帝玄山脉俱是黑压压的人群，漫山遍野，无穷无尽。
<br/>　　这些人简直多到爆炸了，如同一片黑色的森林，覆盖了整个帝玄山脉。
<br/>　　尤其是帝玄宫的遗址之处，天上地下，铺天盖地全是武者，仿佛整个苍穹世界之人，全部来到了帝玄山脉。
<br/>　　而在帝玄山脉之下的帝灵城，简直要被挤爆了，本来因为帝玄宫被灭，已经变得很是萧条的帝灵城，现在则是人口爆炸，城中无尽的人.流，如同滚滚长龙。
<br/>　　距十日的约战之期，还有一日有余的时间，苏莫率一沉吟，便身形下沉，落进了下方的一处小湖之中。
<br/>　　进入湖底，苏莫立刻便进入了虚界神图中。
<br/>　　神图空间中，他取出炼器熔炉，以及所有炼器材料，便准备炼器。
<br/>　　炼制本命灵剑的材料，钱副阁主以及傀儡人已经为他备齐了，而且量非常的足。
<br/>　　而通神玉、血纹籽这两种外界最难寻的材料，他身上都有。
<br/>　　不过，通神玉苏莫虽然有很多，但血纹籽他却是很有限。
<br/>　　当初他在神武国之内，得到了二十四颗血纹籽，炼制了就把本命灵剑，已经消耗了九颗，如今还剩下十五颗。
<br/>　　“炼制十五柄本命灵剑，一柄都不能失败！”苏莫深深的吸了口气。
<br/>　　本命铸剑术之中，有很多种本命灵剑阵，但无一例外，威力越是强大的剑阵，所需的灵剑便越多。
<br/>　　苏莫以前只有九柄本命灵剑，所以只能施展威力不算多强的九绝剑阵，但在九绝剑阵之上，还有十二柄灵剑组成的‘十二天煞剑阵’，十六柄灵剑组成的‘天衍剑阵’等等。
<br/>　　而二十四柄本命灵剑，也能组成一种剑阵，此阵名为‘二十四星宿剑阵’。
<br/>　　这二十四星宿剑阵，按照本命铸剑术上的记载，威力强大绝伦，最少比九绝剑阵强大十几倍。
<br/>　　所以，苏莫若想修炼这种剑阵，这一次炼制本命灵剑，他一次也不能失败。
<br/>　　好在，他有过九次成功的经验，对于本命灵剑的炼制，早已得心应手。
<br/>　　苏莫手掌一挥，分拣好的炼器材料便飞进了炼器熔炉之中。
<br/>　　手掌按在炼器熔炉的火口处，催动火属性神丹，狂暴的火属性玄力立刻汹涌而出，化为熊熊的火焰。
<br/>　　苏莫修炼的火属性功法是帝级上品，虽然他这一段时间内，主要是以提升修为境界为主，但随着修为的不断提升，他五种属性的帝级功法，造诣也在逐渐加深，早已不是当初可比。
<br/>　　所以，他的火属性玄力，比起当初不仅威力更大，而且更为纯净凝练，自然也就更适合炼器。
<br/>　　熊熊火焰，不断的煅烧着炼器材料。
<br/>　　炼化、铸形、淬火，一切都有条不絮。
<br/>　　几乎没有任何的悬念，第一柄本命灵剑，苏莫一个时辰之后便炼制成功了，比以往速度更快几分。
<br/>　　而后，苏莫速度不停，继续炼制本命灵剑。
<br/>　　第二柄！
<br/>　　第三柄！
<br/>　　第十柄！
<br/>　　第十一柄！
<br/>　　第十二柄！
<br/>　　用了十个时辰的时间，苏莫一口气炼制出了十二柄本命灵剑，无一失败。
<br/>　　随即，苏莫深吸了口气，暂时停止了炼制，盘膝打坐。
<br/>　　他的神情稍稍有些萎靡，因为炼制本命灵剑，需要加入他的精血，十二柄灵剑炼制下来，消耗了不少精血，让他感觉到了一丝疲累。
<br/>　　休息了一个时辰之后，恢复了一番，苏莫继续炼制，不到三个时辰的时间，最后三柄本命灵剑，便全部炼制成功了。
<br/>　　他身上的血纹籽，也已经全部消耗完了，以后再也难以再炼制了。
<br/>　　至此，十五柄崭新的本命灵剑，彻底炼成了，再加上苏莫原有的九柄，他共有二十四柄本命灵剑。
<br/>　　二十四柄本命灵剑，将能组成二十四星宿大阵。
<br/>　　“希望此阵的威力，不会让我失望！”
<br/>　　苏莫深吸口气，虽然他神情稍有些疲惫，但一双眼眸却是神采奕奕。
<br/>　　因为，他又有了底牌，或许这个底牌能够逆转战局。
<br/>　　苏莫没有再休息，立刻便开始演练二十四星宿剑阵。
<br/>　　他的时间已经不多了，约战的日期，便是今日。
<br/>　　所以，他必须要在最短的时间内，熟悉此剑阵。
<br/>　　二十四星宿剑阵，属于小星宿剑阵，其上还有大星宿剑阵，名为二十八星宿剑阵。
<br/>　　不过，虽然是小星宿剑阵，威力也能甩九绝剑阵十条街。
<br/>　　此剑阵，每一柄本命灵剑对应一种星宿，攻击之时仿佛能借助星宿之力，威力奇大。
<br/>　　不过，此剑阵由于复杂不少，练习的难度自然也更大。
<br/>　　时间有限，苏莫不求能精深此剑阵，只要能发挥出其一部分威力便可。
<br/>　　咻咻咻！
<br/>　　心神一动，二十四柄本命灵剑如同道道闪电，激.射虚空，瞬间便布置成了二十四星宿剑阵，笼罩了方圆百余里。
<br/>　　而后在苏莫的控制之下，一柄柄本命灵剑不断的在剑阵中穿梭，裹挟滔天杀伐之气，其上更是蕴含玄妙莫测的力量。
<br/>　　苏莫对于此剑阵毫不熟悉，运用起来非常的生涩，威力并没有多么恐怖。
<br/>　　但是，随着他一遍又一遍的演练，他越发的熟练，剑阵的威力也越来越大。
<br/>　　一次性超控二十四柄本命灵剑，非常的消耗精神力，对于精神力的控制，亦是要求极高。
<br/>　　但或许因为苏莫是穿越者的缘故，天生精神力强大绝伦，这点对他来说没有丝毫的问题。
<br/>　　苏莫仿佛不知道疲倦一般，一边又一遍的演练二十四星宿剑阵，漫天剑光闪烁，霎时骇人。
<br/>　　直到几个时辰之后，他才停止了练习，将二十四柄本命灵剑，尽数收进了识海之中。
<br/>　　“黄泉魔宗，巫森！来吧！”
<br/>　　苏莫目光如电，低声自喃，少倾之后，他身形一闪，便离开了虚界神图。
<br/>　　而后，苏莫飞出了小湖，身形挺拔如松，虚空踏步，大步向不远处的帝玄山脉而去。
