<br/>    进入九幽寒泉之中，陈明靠着手中的夜明珠看清周围，不过都是清幽一片，周围几乎都是水。

<br/>        陈明确实感受到了寒气，不过却是没有感受到寒冷，水系功法，他可是修行了不少。相反，现在他还蛮舒服的。

<br/>        陈明看向身旁的卓青瑶，靠着血气隔开周围的水，不让这水临身，“怎么样，冷不冷？”

<br/>        卓青瑶道：“有那么一点点冷，师父，你不是修行过水系妖族功法吗，你直接化形成妖族本体吧，水系妖族本体不消耗任何法力就可以隔绝这寒气，然后我坐在你的身上，就一点也不需要消耗血气了。”

<br/>        陈明想了想，自己修行的妖族水系功法，可是老龟的功法啊！

<br/>        难不成，我要在这里变成一只王八？

<br/>        画美不看。
<br/>    
<br/>        陈明的眉头跳了跳，“我拒绝！”
<br/>    
<br/>        卓青瑶扎着一双水盈盈的大眼睛看向陈明，如同在无声的说话。
<br/>    
<br/>        “你撒娇也没用！我拒绝！”

<br/>        卓青瑶伸手拉住陈明的袖子，“师父父，你最好了！”

<br/>        “师父，你就这么对待人家吗？”

<br/>        “师父，你既然收我为徒，就要对人家负责！”

<br/>        这真的是卓青瑶？果然，凌仙，李素衣，陈灵鱼三个小家伙不在，卓青瑶完全不需要摆大师姐的架子啊！

<br/>        周围已经有着不少和紫霞仙子一起来看了过来，陈明看到他们诡异的目光，不能继续下去了，继续下去这些人只怕会觉得自己对卓青瑶做了什么超越师徒伦理的事，“好吧好吧，我变还不行吗？”
<br/>    
<br/>        一阵水花散开，一只一丈长的黑色乌龟出现在卓青瑶的面前，周围的水自然而然的隔开，根本近不了陈明的身，没办法，这就是种族天赋。
<br/>    
<br/>        卓青瑶单手按住陈明的龟壳，轻轻一跃跳到陈明的背上，然后跪坐在陈明的背上，陈明忽然察觉到有一点不对的地方，为啥，我感觉得到背后有两瓣软软的，嗯，那是什么？
<br/>    
<br/>        陈明扭头一看，那是卓青瑶的挺翘的两瓣柔软。
<br/>    
<br/>        陈明莫名老脸一红，索性变成了黑色乌龟，压根看不出来，“师父，怎么了？”
<br/>    
<br/>        陈明道：“没事，你坐稳了！”
<br/>    
<br/>        控制着水流，让四只爪子和尾巴进入水中，陈明开始畅快地遨游在九幽寒泉之中，别说，虽然是变王八，但是跑的还真是快！
<br/>    
<br/>        紫霞仙子前进了一阵，视察了一番队伍中的人，立马发现了问题，那个道士哪儿去了？看向卓青瑶，乖乖的跪坐在一只王八的身上，特别是这只王八还在水里游的飞快，不禁眼前一亮，问道：“你哪儿弄得一只王八当坐骑，对了，你师父呢？”

<br/>        卓青瑶伸手指了指屁股下面的陈明，陈明张口道：“怎么了，前面有什么宝物吗？”

<br/>        紫霞仙子伸手捂住小嘴，前面那个看起来仙风道骨的年轻人，竟然是一个妖族，而且本体是一只王八？“还真是不敢相信，王八化形居然能够化形的这么仙风道骨！”
