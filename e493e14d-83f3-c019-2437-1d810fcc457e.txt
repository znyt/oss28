　　听到慕清霄的名字，杨长老眼神一亮，上下打量了他一番。
<br/>　　“你的事迹可是在老生中传开了，拥有巨额火能点，身边还跟着个貌美如花的小姑娘，估计许多老生正惦记着你呢。”
<br/>　　“多亏了夏管事，暂时没有老生找我的麻烦。”
<br/>　　慕清霄微微一笑，其实根本不当回事。
<br/>　　在他看来，不过是一群乳臭未干的小鬼罢了，当然，有夏管事替他挡下许多麻烦，倒也令他清静许多。
<br/>　　“那个暴脾气，也对，你住在他的管辖区域，老生暂时也不敢为难你们，若是没事的话，以后不要来中央区域，这里很危险。”
<br/>　　说罢，杨长老就转身离去，没走几步，脚步顿住，道：“以后在塔内遇上困难，可以随时来找老夫。”
<br/>　　望着杨长老背影消失在视线中，慕清霄微微一笑，若非他在猎捕赛上的表现，估计后者看都不会看他一眼吧。
<br/>　　这就是现实，实力为尊，弱者就只有陪衬的份，强者，或者有潜力的学员，学院都会稍加照顾。
<br/>　　...
<br/>　　...
<br/>　　离开天焚炼气塔后，慕清霄与小医仙就在迦南学院中随便逛了逛，随后找了个饭店，饱餐一顿后，就回到住处。
<br/>　　站在窗前，望着碧空如洗的蓝天，慕清霄不禁陷入沉思。
<br/>　　开塔的时间是早晨八点，闭塔的时间是晚上八点多，期间十二个小时，他必须在这一段时间内，悄无声息的潜伏进去，但......
<br/>　　就在这时，慕清霄的袖口中，忽然探出一个可爱的小脑袋。
<br/>　　同时，脑海中响起一道悦耳妩媚的清脆声音。
<br/>　　“先前的炽热温度，迦南学院居然将‘陨落心炎’封印在塔内，给学生提供修炼的环境，真是一群疯子！”
<br/>　　“既然他们敢这样做，就肯定考虑过后果，而且陨落心炎现在被封印，有多位守塔长老镇压，倒是谨慎。”
<br/>　　说着，慕清霄又道：“而且，学院里面几个老家伙，实力倒是挺厉害，哪怕陨落心炎爆发，也能及时镇压，不过封印却在变弱，估计离冲破封印不远了。”
<br/>　　陨落心炎早已成灵，对于迦南学院的做法，估计很愤怒吧？
<br/>　　换做是谁，被封印住，当成工具来使用，估计都会愤怒，否则，它也不会长期的冲击封印。
<br/>　　利用陨落心炎的特效，给学员提供修炼环境，确实是疯狂的做法，能想出这种办法的，绝对不是普通人。
<br/>　　而且，还成功了，且带来了无与伦比的好处。
<br/>　　长年累月下来，迦南学院出来的学员，正是因为这种压抑，难以忍受的锻炼，使他们心性更加坚毅，才能在这个强者为尊的世界中，站在更高的位置。
<br/>　　而这一切，都是迦南学院带给他们的，只要学院要求，估计这些强者，九CD会相助。
<br/>　　美杜莎沉默，她虽心性高傲，却也不是蠢货，迦南学院底蕴深厚，她自然知晓，否则，当初早就闯进来了。
<br/>　　“你打算什么时候动手，若是被发现，很可能就无法离开这里......”
<br/>　　“凭守塔的长老，还留不住我，不过塔底有一层封印，破开封印的话，肯定会惊动学院的长老，事情有点难办啊。”
<br/>　　慕清霄也有些头疼，他实力虽高，却也不会自负到，抗衡整个迦南学院。
<br/>　　数十名斗皇强者，在他眼里虽不算什么，但明里暗里的斗宗强者，最起码有四位，还有藏书阁中的两位老者。
<br/>　　那两位可是斗宗巅峰，说是半步斗尊也不为过，根本不是他现在能对付的。
<br/>　　就在慕清霄思绪万千之际，脑海中再度响起悦耳的声音。
<br/>　　“若是没有迦南学院的阻挠，你有几成把握，吞噬陨落心炎？”
<br/>　　“十成把握。”
<br/>　　自信一笑，若是这点自信都没有，他又怎会来迦南学院，在神火的威压下，异火就只有瑟瑟发抖的份。
<br/>　　“那好，由我来破开封印，到时候将吸引他们的注意力，你下去吞噬异火。”
<br/>　　“不行。”
<br/>　　慕清霄剑眉一皱，立马拒绝。
<br/>　　美杜莎已经算是自己的女人，迦南学院中强者无数，要在破坏封印后遁走，几率微乎其微，怎么可能让她冒险。
<br/>　　此时，七彩吞天蟒小尾巴摆动间，一道璀璨的七色光芒，自吞天蟒的体内涌出。
<br/>　　只见吞天蟒在七色光芒下，急速的蠕动起来，旋即，一名身穿红色衣裙，雍容华贵的绝世女子出现在面前。
<br/>　　妖艳绝伦，魅惑天成，美人如玉，倾国倾城。
<br/>　　哪怕不是第一次见到，慕清霄眼中都闪过一丝惊艳。
<br/>　　那犹如上天杰作般的脸颊，找不出一丝的瑕疵，仅仅是站在那里，便给人一种冷艳的气质，淡紫色的眸子，犹如世上最美丽的宝石。
<br/>　　在慕清霄的印象中，唯有一名女子略胜她一筹，只是佳人却在天边，无迹可寻。
<br/>　　修长玉葱指捋过散在额前的一缕青丝，美杜莎美眸凝视着他，道：“这是现在唯一的办法，难道你有更好的办法吗？”
<br/>　　慕清霄竟无言以对，他确实没有更好的办法，若是他自己冲破封印，迦南学院根本不会给他吞噬陨落心炎的机会。
<br/>　　他想要躲过这些强者的眼睛，进入塔底深处，就必须有人吸引住他们的注意力。
<br/>　　陨落心炎虽自己在冲击封印，但想要完全冲破封印，没有三五年，是完全不可能的。
<br/>　　“你......是不是太小看本王了。”
<br/>　　见慕清霄露出纠结的目光，美杜莎语气平淡的说道，旋即，一股极强的气息，自她雍容的身躯中爆发而出。
<br/>　　一星斗宗！
<br/>　　“别忘了，本王比普通的斗宗强的多，迦南学院中，现在能压制我的强者，也只有内院几位长老。”
<br/>　　闻言，慕清霄沉吟起来，美杜莎能够化作吞天蟒战斗，实力确实比普通斗宗强的多。
<br/>　　藏书阁中的老者估计不会出手，至于外院院长，应该也不会管内院的事情，能压制她的还真不多。
<br/>　　叹息一声，慕清霄心情有些复杂，终究是自己的实力太弱了，上前一步，伸出手轻轻的抚摸着她绝美的脸颊。
<br/>　　“答应我，不要乱来，引开他们后就遁走，七彩吞天蟒与我签定了灵魂契约，等我办完事就去找你。”
