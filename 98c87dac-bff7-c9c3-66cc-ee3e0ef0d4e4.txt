　　黄衣青年显然是听到了苏莫和谢彬的谈话，所以非常不屑，嘲笑了一句便大步离开。
<br/>　　“站住！”苏莫闻言顿时冷喝一声，他可不是被人骂了，而忍气吞声的主。
<br/>　　黄衣青年闻言，停下了脚步，面带淡笑的回头看向苏莫。
<br/>　　这名黄衣青年相貌堂堂，身材略胖，但不会给人肥胖之感，反而显得较为壮硕。
<br/>　　其一双眼眸极为闪亮，仿佛天生星辰，眸中噙着若有若无的笑意。
<br/>　　“阁下此言未免的侮辱人了吧？”苏莫打量着对方，冷冷的问道，此人修为不弱，高达武帝境七重境界，但他丝毫不惧。
<br/>　　“侮辱人吗？”黄衣青年闻言一笑，笑眯眯的道：“你连真圣界和真圣之气都不知道，难道还不是土包子？真不知道你是怎么修炼到武帝境的？”
<br/>　　言罢，黄衣青年转身离去，大步的走到拍卖会场的门前，排队进场。
<br/>　　“找虐吧！”苏莫眸中冷光一闪，对方嚣张的模样，分明是想要找虐啊！
<br/>　　“苏兄！莫要冲动，此人实力很强，我们还是先参加拍卖会要紧！”谢彬急忙说道。
<br/>　　苏莫闻言点了点头，的确，现在并不是和他人起冲突的时候。
<br/>　　“走，我们进拍卖场！”
<br/>　　随即，两人也走向拍卖场的大门，排队进入拍卖会场。
<br/>　　他和谢彬都是武帝境的修为，只能进武帝境的拍卖会场，这让苏莫有些无奈，不知武帝境的会场，有没有高等级的宝物。
<br/>　　进入拍卖会场，需要交纳一块下品灵晶，价格颇为昂贵。
<br/>　　进入会场以后，苏莫耳中立刻传来了震天的喧嚣声，整个会场中，此刻已经聚集了五、六十万人。
<br/>　　会场的中央，是一座高达数十丈的拍卖台，修建的晶莹剔透，美轮美奂。
<br/>　　东、南、西三个方向，环形的坐席一排又一排，延绵无际，人山人海。
<br/>　　而在北方，是一座巨大的宫殿，宫殿的大门前，白玉阶梯长达千丈，直通中央拍卖台。
<br/>　　“人真是多！”苏莫面露惊叹之色，此刻的拍卖会场之中，不仅是人族之人，还有不少异族之人。
<br/>　　比如，有几位巨灵族之人，身高达到了三、四丈，坐在哪里就如同一座小楼一般，一个人就占据了好几张坐位。
<br/>　　随后，苏莫和谢彬东部方向，找了两张靠后的坐位坐了下来。
<br/>　　“好多厉害的天才！”
<br/>　　谢彬眸光环视四周，面带兴奋之色，随即，他指向远处，一名正在闭目养神的华服青年，道：“苏兄你看，那人叫石惊生，乃是三神宗的妖孽天才！”
<br/>　　“还有那一位女子，是妖族冰蚕族的天之骄女！”谢彬又指向另一边的一名美貌的绿衣女子。
<br/>　　“还有那边那位巨灵族之人……！”
<br/>　　“还有那位飞灵族的天才……！”
<br/>　　谢彬口中滔滔不绝，目光不断的转动，向苏莫介绍一位又一位天才。
<br/>　　苏莫目光转动，扫视了一番谢彬介绍的这些人，的确是个个不凡，这些人修为都是极高，大多都是武帝境八、九重境界。
<br/>　　“嗯？”正在环顾的苏莫，眸光顿时一凝，落在了一名俊朗的白衣青年的身上。
<br/>　　尤天韩！
<br/>　　苏莫双眸微眯，没想到会在这里碰到对方，真是冤家路窄。
<br/>　　仿佛是感受到了苏莫的目光，尤天韩眸光转动，向苏莫望了过来，两人的目光瞬间对视在一起。
<br/>　　不过，让苏莫有些疑惑的是，对方的目光中并没有杀意，反而露出幸灾乐祸的笑意。
<br/>　　而后，尤天韩微微一笑，便不再关注苏莫。
<br/>　　“奇怪！”苏莫低声自喃，他杀了太史世家派系这么多人，尤天韩居然面带笑意，这是什么情况？
<br/>　　“苏兄，你以后不再招惹太史世家派系的人，想必他们也不敢对付你！”谢彬也发现了尤天韩，低声说道。
<br/>　　“或许吧！”苏莫点了点头，但他心中却知道，双方之间的恩怨，怕是不会这么简单的就结束了。
<br/>　　叹了口气，苏莫便也不再关注尤天韩，静心等待拍卖会的开始。
<br/>　　就在此时，他的目光扫过拍卖场的入口，顿时心中一震，他见到了一个意料之外的人。
<br/>　　只见一名黑衣青年大步走进了拍卖场，这名青年眉清目秀、唇红齿白，一头长毛垂落到臀下，给人阴柔狠厉之感。
<br/>　　此人不是别人，正是巫族巫人王！
<br/>　　此时的巫人王，面无表情，没有了往日的张狂，整个人给人一种极度阴霾之感。
<br/>　　苏莫眉头顿时一皱，巫人王居然来到了古周星！
<br/>　　古周星距离天冥星极远，又是异族势力非常薄弱之地，对方怎么会来到这里？
<br/>　　难道对方只是单纯的来参加拍卖会？
<br/>　　这应该不太可能，大荣城的拍卖会虽然规模宏大，但古灵星河这么大，其它星辰肯定也有大的拍卖会。
<br/>　　对方不可能为了大荣城的拍卖会，而耗费这么长时间，从天冥星来到古周星。
<br/>　　苏莫心中升起了不好的预感，心中暗凛，难道对方是来找他的？
<br/>　　若真是如此的话，说明巫族已经知道了他在古周星，很可能已经查到了神武学府。
<br/>　　“这下糟了！”苏莫脸色凝重了下来，若真是如此的话，他说不得又要逃了，逃离古灵星河，去往其它星河。
<br/>　　但是，现如今逃走的话，危险性则是极大。
<br/>　　怕是巫族已经查到了神武学府，肯定会严防他逃走，说不定已经安插好了人手，严防他逃离古周星。
<br/>　　苏莫心中沉思了起来，但又觉得好像不对，若是巫族查到了他的踪迹，估计第一时间就会对他出手，巫人王也不会这么大摇大摆的现身吧！
<br/>　　这不是明摆着让他得知情况，暗中逃离吗？
<br/>　　难道这只是巧合？
<br/>　　巫人王只是路过此地，听闻拍卖会的情况，所以来参加拍卖会，与他完全无关。
<br/>　　苏莫心中急思，也是有些糊涂了，但是有一点能肯定的是，巫族定然是快要查到神武学府了。
<br/>　　现在，他面临两个选择，一是逃离古周星，二是就留在神武学府。
<br/>　　但是，这两条路都不好走，均是危险重重。
<br/>　　逃离的话，他说不定已经被巫族强者盯上了，很快就会被击杀，但留在神武学府，学府也不一定能保住他，这让苏莫惆怅不已！
<br/>　　一身黑衣的巫人王，走进拍卖会场以后，随意的打量了一眼四周，便在南方坐席找了个位置坐了下来，一脸的冷漠，仿佛对任何人任何事都毫不关心。
