　　　　与民同乐。
<br/>　　很多人受了涂小安秘制的蛇王水好处，百病全消，自然举家高兴。
<br/>　　涂小安也高兴非常，做好事的感觉是幸福的，而且幸福之余，还有回报。
<br/>　　信仰之力：12050000。
<br/>　　不过区区一个下午，他系统之中的信仰之力直接暴涨到一千多万，这是一个什么概念呀。
<br/>　　他简直就是一下子成为了千万富翁。
<br/>　　他甚至可以去系统商城购买神兽的惊天血脉了。
<br/>　　庙前，依然人山人海，人头窜动，但凡领到蛇王水跟雕像的，都如负重托的送了一口气，生怕轮到自己，连渣渣都没了。
<br/>　　三小时后。
<br/>　　夕阳之后，晚17时许。
<br/>　　人实在太多，几十万民众，不可能等待大家全部领完，只能先行压后，为什么呢...
<br/>　　因为蛇王雕像要入庙堂正位，岂能一直在庙外放着。
<br/>　　现场开始维持秩序，雕像入庙前，还有一件事情大事要做，那么便是群蛇放生佘山。
<br/>　　一个巨大的蛇筐之内，里面有万蛇等待放生，一条条颜色各异，品种各异的毒蛇丝丝惊叫，好像也非常的兴奋。
<br/>　　因为它们马上要自由了，回到自己熟悉的大自然。
<br/>　　毒蛇放生，是蛇王节的一大特色，就宛如古代新皇登基，大赦天下一样。
<br/>　　佘山百蛇出没，毒蛇品种众多，归根结底要托了这个蛇王节的福。
<br/>　　数百年流传下来的节日，每一年，大家都会为蛇王献上自己最珍贵的蛇品种，很多毒蛇都不是出自佘山，都是全国各地的毒蛇，乃至于全世界各地的毒蛇。
<br/>　　每一年，在这一天，总有很多珍贵的蛇品种放生佘山，就好比如今蛇筐中就有一条通体黑色的毒蛇，他叫墨西哥黑王蛇，就是那个中年的胖大叔特别献上。
<br/>　　还被李清照一方采访。
<br/>　　胖大叔就是为了让佘山在多一种蛇品种，也是为了白镇人民做贡献。
<br/>　　绝大部分的人，都尽量献上各国的名蛇，在古代科技不发达，放生也就是一种形式，放的也都是一些普通的毒蛇，当地的蛇种。
<br/>　　可随着现代，这种仪式升级了，为了向蛇王表达自己最大的虔诚，都是费尽心思的寻找佘山没有的毒蛇。
<br/>　　如今蛇筐中万蛇出没，那画面是思极恐，一条一条毒蛇相互缭绕的场景让人头皮发麻。
<br/>　　你可以在里面随随便便都能找到不少外国的蛇品种，等这些外国的蛇品种进入佘山，休养生息，繁衍后代，佘山自然多了一种蛇的种类。
<br/>　　所谓的百蛇出没，所谓的佘山，都是经过人为的保护跟培养，不然世世代代那么多捕蛇人已蛇为生，佘山的蛇，早就给捕光了。
<br/>　　今天蛇筐中的毒蛇数量也格外的夸张惊人，可谓是打破了数百年来的记录。
<br/>　　早就有专门的人来记录蛇的数量，记录到蛇王仪式开始截止，如今蛇筐中的毒蛇达到了23758条，不同的蛇品种达到了369种。
<br/>　　这是何等的夸张呀，恐怕早就可以申请吉尼斯国家记录了。
<br/>　　将近二万四千条毒蛇，每一条都不是普通毒蛇，绝大部分都是罕见蛇种，另外最让人震惊的是蛇品种，369种。
<br/>　　往年，放生毒蛇都破一万条都非常的可怕了，蛇品种就更别说了，能有五十种不同的就非常不错了。
<br/>　　还有一点不得不说，这二万多条毒蛇，产生的经济价值绝对是一个天文数字，不可想象。
<br/>　　但白镇人民不在乎，甚至是财大气粗，他们因为蛇王而自豪，心甘情愿献上最好的蛇放生，同时也是为了让佘山拥有更多的毒蛇。
<br/>　　这369种蛇品种入住佘山，会给佘山带来新气象，更也是带来了财富。
<br/>　　白镇的经济就是靠着蛇产业而发达。
<br/>　　万蛇要放生，不断有人拿着手机拍摄，谁见过那么多条毒蛇呀，密密麻麻，叠成叠，不同的条纹颜色，不同的品种，赫然勾画成一幅七彩绚丽的群蛇油画。
<br/>　　这样的蛇场景，虽然很让看了人头皮发麻，浑身鸡皮疙瘩，但依然阻止不了民众的热情。
<br/>　　二万多条毒蛇啊，怎能不留下一个纪念，以后还可以出去吹牛：“劳资见过的蛇，比你身上的毛还多，你信不”
<br/>　　当然，这不过是恶趣味，但也间接的说明了白镇的蛇王节有多么的隆重跟不凡了。
<br/>　　万蛇放生，规模浩大，这些蛇要是在城市里放出，那么绝对是一场灾难浩劫，不知道要有多少人死在毒蛇的手中。
<br/>　　况且绝大部分都是非常剧毒的毒蛇，杀伤力很大，轻易就能要了一个人的性命。
<br/>　　往常，大家都是用篮筐，几十个一起装着毒蛇抬到佘山放生，但是今年的毒蛇贡献的太多，光放生都不是一件简单的事情。
<br/>　　只见道袍男子李顺浑身一颤，好像是得到了什么指令，走到了装着二万多条毒蛇的蛇筐面前，但接下来他的举动将众人吓的不轻。
<br/>　　他在开筐放蛇。
<br/>　　什么，就在这里直接放蛇？
<br/>　　无人不震惊，二万多条毒蛇涌现出来，那是何等凶煞的场景，根本不是人为的可以控制。
<br/>　　李顺是疯了吗。
<br/>　　不少人一想到那个万蛇涌动的画面就满是惊悸，毛发着了魔一样地冰冷地直立起来，茫然不知所措的脑子像一张白纸。
<br/>　　这样的举动太具备危险性，一个控制不好，不知道有多少人要死在毒蛇的蛇口下。
<br/>　　现场可是有几十万的民众呀。
<br/>　　虽然这是白镇，已蛇为生，但也正因为如此，大家都很清楚毒蛇的秉性。
<br/>　　没有人能一下子控制二万多条毒蛇，就算是白镇最优秀的捕蛇人都不可以，李顺这种举动可谓是陷大家与危险之中。
<br/>　　嚓...
<br/>　　蛇篮被打开了，顿时间嘶嘶惊叫声不绝于耳，里面的毒蛇像是找到了宣泄的出口，排山倒海的从入口涌现了出来。
<br/>　　霎时间，所有人不断的后腿，脸色骤然苍白下来。
<br/>　　万蛇涌动，层出不穷的游出来，炎热的空气都被毒蛇身上阴冷的气息所秒杀。
<br/>　　“李顺，你在做什么，为什么要这样的放蛇”
<br/>　　很多人咆哮的质问道袍男子，这普天同庆的好日子，难道要出人命不成。
