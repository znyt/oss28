　　　　猴儿山谷的天现在是黑的，地面反而白茫茫一片。
<br/>　　两道电火光柱带着磅礴无匹的力量压着那条巍峨巨蟒抬不起头来，明明五十米庞然大物，此时如同大海上可怜的纸船，苦苦的支撑。
<br/>　　在毁天灭地的天劫之下，任你是条龙，也要给盘着。
<br/>　　要你死，你就要死。
<br/>　　涂小安整个都不好，内心只想草泥马，为什么自己渡一个天劫，都跟西天取经一样。
<br/>　　若是每个精怪渡劫都如他这般艰难，他相信这个世界上不会存在任何一头三阶妖灵。
<br/>　　蛇躯体表外雷电跟火焰两种元素力量形成了最强防御力，但在这样摧枯拉朽的天劫力量下，哪怕是这种超自然的力量也如同虚构，脆弱如白纸。
<br/>　　只闻呯...的一声，在外围的火焰之力能量全部消耗殆尽，骤然溃散掉，而雷电属性力量还在苦苦支撑护体，帮助涂小安抵消这股混合体的天雷天火。
<br/>　　但一分钟之后，呯...雷电之力也不复存在了，这条巍峨巨蟒彻底暴露在天劫摧枯拉朽的毁灭力量下。
<br/>　　涂小安不断的咳出蛇血，他的蛇血一滴成珠，殷虹妖艳，落地而不散，每一滴都可救死扶伤。
<br/>　　大股大股的蛇血染红了山谷废墟般的地面。
<br/>　　好在，好在这两道混合天雷天火的力量也被消磨的差不多了，白茫茫一片的视觉效果消失，涂小安心头一松大口喘着粗气，熬过去了，总算再次熬过去了。
<br/>　　体内四种元素属性力量全部消耗殆尽，已经无法在使用了。
<br/>　　活下来了，有惊无险，仅仅只剩下六道天雷天火了。
<br/>　　在熬过去，他的天劫也总算度过去了，不过接下来的渡劫，他就全凭肉身力量去抗。
<br/>　　而如今每一道天雷天火的力量都不是之前的那些可比，甚至完全没有可比性。
<br/>　　无数关注这场天劫的精怪妖灵看到蛇影重新出现，彻底麻木了，这条蛇精还活着，还杵着？
<br/>　　刚才两道电火光柱有多强，它们在方圆几十里外，乃至于几百里之外，都浑身瑟瑟发抖，诚惶诚恐。
<br/>　　几乎就以为是世界末日降临了一般，几乎以为是天塌下来了一般。
<br/>　　很多生物莫名被感动了，多坚强的一条蛇，与天争命，不屈不挠。
<br/>　　能活谁想死，在死亡的面前，哪怕是一只弱小到尘埃里的蚂蚁，也能被激发潜力，撼动大象。
<br/>　　关注这场天劫的生物已经开始暗暗给这条蛇精加油鼓劲，仅仅就剩下六道了，挺过去，这条蛇将超凡脱俗，甚至要凌驾众妖灵之上。
<br/>　　必死天劫都能闯过，他的潜力可想是有多可怕。
<br/>　　可接下来的一幕，就让无数生物三观尽毁，替渡劫的蛇精感到绝望，感到悲悯。
<br/>　　三朵劫云花顾盼生姿，含苞待放，它们悄然动了，三朵相互的环绕在一起，附近电闪雷鸣，惊雷不绝，诡异的事情发生了，三朵变成了一朵，雍容华贵，繁花似锦，整朵花闪烁丝丝电光，天威不凡。
<br/>　　里面仿佛藏着一条电龙迫不及待的冲天而起。
<br/>　　另一片天，三颗天外陨石同样诡异的融合成了一颗，漫天火势横流，变成更加庞大的陨石，三股天火合而为一，就如同火山要蓄势待发的爆炸。
<br/>　　上空的天变化莫测，触目惊心。
<br/>　　有一股颠倒六道轮回的力量在酝酿。
<br/>　　三朵劫云花化为一朵，三颗天外陨石融为一颗，看的所有生物都怀疑人生。
<br/>　　很显然，这场必死天劫也在捍卫自己的天威，睥睨之下，不容许任何生物可以活下来。
<br/>　　又是之前的那一幕，化为一朵的劫云花飞到了融合成一颗的天外陨石上面，闪烁霹雳的光，彻底植入。
<br/>　　轰...
<br/>　　这颗天外陨石里面所蕴含的力量或许足可将一座城市都给天道毁灭了。
<br/>　　大...大...大...！
<br/>　　劫云花的植入，让这颗天外陨石瞬间暴涨，将上空整片天都给遮天蔽日了。
<br/>　　天昏地暗，一种颠覆六道轮回的磅礴力量彻底露出了自己峥嵘的一面，地下的涂小安巍峨的蛇躯隐隐发抖，被压抑的喘不过气。
<br/>　　举目望天，日月无光。
<br/>　　澎...
<br/>　　遮天蔽日的天外陨石火山爆发了，顷刻间，涂小安什么都看不到了，好像自己再次处在了混沌之中，在被天道毁灭。
<br/>　　这颗究极混合的天劫一爆发，天地间任何生物都成了睁眼瞎，白茫茫一片，谁都看不到任何画面。
<br/>　　哪怕是山巅之上的白妖王也是如此，这股天劫力量让它都诚惶诚恐。
<br/>　　天劫力量之磅礴，之强，万物皆灭，涂小安已经感觉不到什么疼痛了，只是隐约感觉自己的蛇躯本体风雨飘渺了起来，若隐若现，体形开始不断的在缩小。
<br/>　　渡劫渡到这种地步，涂小安第一次感觉到了绝望，感觉到自己的渺小，无能为力，无法抵挡。
<br/>　　而且也根本不可能抵挡，每一缕空气都蕴含摧枯拉朽，毁灭他的力量。
<br/>　　涂小安真的绝望了，死亡已经悄然降临，他满心不甘，劳资不过证道渡劫，又尼玛的不是羽化飞升，要不要这样费尽心机的毁灭自己？
<br/>　　层出不穷的天劫手段，简直特么的是游戏按键一样，QWER，技能轮番的释放。
<br/>　　这属于终极大招，就问你死不死。
<br/>　　这种天劫，别说精怪层次的渡，哪怕是妖王层次的来渡劫，也要死的渣渣都不剩下。
<br/>　　必死天劫一出，一切皆是虚无。
<br/>　　此时此刻，涂小安才真正明白什么才叫做必死天劫。
<br/>　　这时，涂小安自己的蛇躯本体不断的开始彻底缩小，从五十米巍峨变小。
<br/>　　这种缩小，是非常的可怕了，宛如世界末日，从大到小，从有到无。
<br/>　　天道毁灭，无情而可怕，五十米，四十五米，四十米...
<br/>　　巍峨的身躯在这股摧枯拉朽的天劫力量下，已肉眼可见的速度缩小，直至将他彻底摧毁干净。
<br/>　　“不行，我涂小安怎能死，这场天劫我一定要渡过去”
<br/>　　“系统，快给劳资出来，快帮帮我”
<br/>　　涂小安现在一下子醒悟，为什么之前的系统奖励是他渡劫结束才赋予。
<br/>　　敢情系统都不看好他。
<br/>　　“证道渡劫，宿主必须依靠自己的力量活下来，唯有这样，你才能成为真正的蛇王，脱胎换骨，超凡脱俗，统领天下群蛇”
<br/>　　系统的声音带着一股机械般的冰冷。
