　　神秘人的灵魂一颤，一瞬间仿佛回到了母亲的怀抱，似乎这世间的纷扰，一切与他无关。
<br/>　　随即神秘人心中一惊，神魂急忙从那股奇怪的感觉中挣脱出来，心中亦升起一股惊骇之意，以自己的意志，竟然被人迷惑了。
<br/>　　“圣人手段，果然厉害。”神秘人定睛一看，只见在那江山社稷图上，此时出现一道美丽的女子身影，他摇了摇头，道：
<br/>　　“我自问心志还算坚定，却不料被娘娘一句话便差点迷失了自我。”
<br/>　　女娲很美，这个美不是说颜值，当然，她的颜值也是顶尖的，她的美在于那股独特的气质。那是一种博爱苍生的气质，无论是人族还是妖族，她都是怜悯的态度对待，久而久之，便形成了独特的气质，令人无法阻挡。
<br/>　　“人皇的心智，更令人钦佩。”
<br/>　　女娲淡淡道，一句话便道出神秘人的身份，正是纣王。她看了一眼陆压的尸体，女娲脸上多了一股无奈：“只是人皇的手段，未免过于歹毒了吧。”
<br/>　　“陆压先冒犯于朕，朕出手将其击杀，又有何不对？”纣王也不再隐瞒身份，露出了本来的面目，淡淡道。
<br/>　　“成汤气数已尽，周兴商亡，乃是天数使然，人皇纵然惊才绝艳，也难敌天数。不过，你如今杀了他，希望将来你不惧他找你麻烦吧。”女娲淡淡说了一句，之后目光瞥了陆压的尸体一眼。
<br/>　　纣王神色一凝，道：“娘娘什么意思，这陆压已经被朕所杀，将来更不可能是朕的对手，又何来找朕麻烦一说？”
<br/>　　女娲淡淡看了他一眼，道：“陆压师弟又岂是那般容易死的。”
<br/>　　说完，她收起陆压的尸体与江山社稷图，不等纣王询问，身形便已消失不见。
<br/>　　“陆压师弟？女娲你竟然称呼陆压为师弟，这陆压难道还有其他身份不成？难道刚刚那陆压只是一个分身，那真正的陆压如今又在何处？”
<br/>　　原地，纣王皱眉不已，他在思索女娲临走时所说的话。这时，他才想起原来所知的传说中，这陆压好像在洪荒身份十分神秘，有的说他是金乌十太子，有的是说陆压为先天离火之精，有的书是西昆仑度厄散仙，有的说是女娲、鸿钧与混鲲老祖的师弟，各种传说都有。最神秘的便是“先有盘古后有天，陆压道君更在前”这句古老传言。
<br/>　　“算了，将来之事，谁有说得准呢，陆压又如何，若是敢来找朕麻烦，朕让你变死鸦！”纣王神色一冷，之后收起东皇钟，便向着朝歌飞去。
<br/>　　汜水关前，三霄姐妹与三仙岛的菡芝仙子，彩云仙子来到商营之中，见到赵公明的惨样，双目失明，气血枯败，只差最后一口气，便命丧黄泉了。三霄顿时大怒，在关前摆下九曲黄河大阵，浊浪排空，黄沙漫天。神阵凶威滔天，大杀四方神仙，神威大展。
<br/>　　哪咤、杨戬、雷震子等三代弟子，尽皆被云霄用混元金斗丢入九曲黄河大阵之中。甚至就连阐教的清虚道德真君都被混元金斗削去了顶上三花，落为天仙。
<br/>　　后来，孔宣出手，五色之光惊天动地，无物不刷，混元金斗在厉害，三霄的修为也没达到准圣，如何与准圣境的孔宣对抗，五色神光刷去混元金斗与金蛟剪。
<br/>　　没了法宝的三霄，实力便降了一半，又如何与阐教众仙大战。仅仅数个回合，便被阐教众仙镇压了。
<br/>　　阐教虽然有些胜之不武，但这是战场，本就是你死我活，又何来的公平？量劫之中，既然身入其中，便要做好死亡的准备。
<br/>　　碧游宫中，众多弟子环绕坐下，通天教主在榻上讲道，忽然手中拂尘失手落下，砸入地面，童子连忙上前，将之捡起，递与教主。通天教主没有去接，反而闭目不语，似乎是极为疲惫。多宝道人、金灵圣母目露悲哀之色，龟灵圣母、无当圣母满脸愤恨，其余众多弟子不知情况，都不敢出声。
<br/>　　九曲黄河阵已破，哪咤等弟子也被救出，只是他们的修为如今已被削去，成了凡仙。这时，突然从天而降数道造化之光，落入杨戬、哪咤、雷震子、清虚道德真君等人身上，仅仅数个呼吸，数人的修为便尽皆恢复至原先的修为，他们只是修为倒退，境界还在，只需补足能量，修为自然恢复。
<br/>　　“多谢老师（师祖）！”清虚道德真君等人连忙朝着天上跪拜，口中大呼道。
<br/>　　“尔等今日被削去道果，落下神通，需回去洞府之中静坐参悟，感悟其中至理，若是一日能够返本回元，则境界更胜往昔。”
<br/>　　说罢，元始天尊的声音便再没有传出。众弟子领命，不敢多语。
<br/>　　燃灯、申公豹、与广成子等阐教众仙继续商议攻破成汤军队之事。
<br/>　　汜水关前，擂鼓惊天，马声烈烈，军队的声音依旧雄浑，却透露着一股无限悲凉之意。
<br/>　　战车之上，武王亲自击鼓，下方黄沙滚滚，周军四十万，商军五十万，战将千百，铁甲锵锵作响，车鼓交错，飞箭如雨，万千战士大战。
<br/>　　此战自白天杀到黑夜，再从黑夜杀到日出，又杀到日落，再杀到日出，商军虽然人数众多，但周军之中有杨戬、哪咤、金吒、木吒、韦护、雷震子、苏护等西岐大将，各种兵器法宝满天飞，如龙入大海，虎出山林，，一击便是死亡一大片，势不可挡！
<br/>　　闻太师固然勇猛，但身边良将却少之又少，如何能够抵挡住周军的凶猛，而似菡芝、彩云、吉立、余庆等仙人，此时也已经战死，太师无奈，只好向着燕山方向退去。
<br/>　　而商朝的仙人，如吕布、宋缺、寇仲、李元霸等人，却被阐教众仙缠住，一时间脱不开身，或者是被战死。商朝中仙人固然潜力惊人，奈何来到洪荒的时间太短，不足以让他们成长起来。
<br/>　　太师等人退到燕山，又行了七八里，见到一座高峰，前有一石碑，高达数十丈，宽七八丈，只是被大雪覆盖，不见字迹。
<br/>　　太师到了石碑下，挥袖一拂，长风吹过，积雪一扫而尽，露出三个大字：
<br/>　　绝龙岭！
<br/>　　太师见了，急忙拉住麒麟，后退十余步，太师面色惨淡，默默无语，良久，才道：“我当日自碧游宫下山，我师金灵圣母曾说道，我这一生，决不能遇到‘绝’字，否则必死！难道真的天意难违！”
<br/>　　太师目露悲色，转身看着身后的大军，道：“我六十万大军伐周，如今只剩下二十万，又逢大雪封山，难道今日我要将大军葬送于此？”
<br/>　　此时，周军在申公豹等人的领导下，再次杀来，将商军围在这绝龙岭外，双方大军再次大战起来，眼见商军越来越少，太师闻仲目眦尽裂，披发奋战，死不投降。
<br/>　　千万里外，上皇峰头，金灵圣母遥望西岐方向，目中之中，隐隐还有泪光，她身躯不动，任由山风吹过，衣袍猎猎作响。
<br/>　　多宝道人手持拂尘，立于身旁，黯然神伤，默默不语，良久叹息一声。
