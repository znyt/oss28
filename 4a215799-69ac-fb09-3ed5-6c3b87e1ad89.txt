　　“这尸傀门和血尸堂的尸体一攻一守，太烦人了。看来得朝组织内求援了。”
<br/>　　钟大叔立即回到了战门的一个据点，把这件事情说明了一二。
<br/>　　“我战门现在大不如人力都在白骨山附近陷阱，等待曙光基地那些家伙的自投罗网，可没有多余的人手给你调派过去。”
<br/>　　一个战门外门长老皱着眉头道。
<br/>　　“长老，这尸傀门血尸堂欺我太甚，杀了我这么多成员，难道就任由他们嚣张吗？？？”
<br/>　　钟大叔压制着怒气道。
<br/>　　“可是……”
<br/>　　这长老还要不同意，但一边一个长老开口道：“王长老，曙光基地那群人，就是小绵羊罢了，他们虽然人多武器多还有坦克大炮，但一旦进入我们封锁的范围，就全部死翘翘，我们根本不需要布置这么多人手在这里白瞪眼，留下几个人看守各大活人祭坛就足够了，反倒是这尸傀门和血尸堂，最近的事情我也听闻过，三番五次偷袭我们成员，这可是对我们战门的挑衅，日后若是让其余组织知道了我战门堂堂十大组织居然被他们欺负，岂不是毫无颜面？”
<br/>　　“这样的话，那就给你二十人。”
<br/>　　王长老略一犹豫就点头同意了。
<br/>　　“多谢两位长老！”
<br/>　　钟大叔要了人立即回去继续干。
<br/>　　而此时。
<br/>　　在距离白骨山大概二十里地外。
<br/>　　曙光基地的先遣部队气势汹汹的来了。
<br/>　　坦克大炮步兵战车应有尽有。
<br/>　　这是一股精锐，他们来这里的目的就是为了歼灭白骨山的怪物群和丧尸群，化解对曙光基地的潜在威胁。不能全部歼灭，那也要想办法把这群怪物群和丧尸群给吸引到其余地方去。
<br/>　　陆勇是这一支队伍的最高指挥官，中将军衔，战斗大队的副队长。
<br/>　　在他身边则是副参谋长少将军衔，也是陆勇的左右手。
<br/>　　“首长，最新伤亡报告出来了，我们损失了一辆主战坦克以及一百七十七个士兵！！”
<br/>　　一人报告道。
<br/>　　他们从基地出来一路上自然遇到了怪物和丧尸的疯狂进攻，虽然火力足够猛，但也难免损伤。
<br/>　　“周兵王，你对这一次白骨山的歼灭任务有什么看法。”
<br/>　　陆勇看着一个魁梧的男人。
<br/>　　此人就是曙光基地第一高手，兵王周华，九阶的强者。
<br/>　　“不看好，顶多消灭五分之一，剩下的，要想尽办法吸引到其余地方去。所以我们最好是找个地方先停下来观察一下具体情况。然后再考虑作战安排。”周华道。
<br/>　　“你说的有道理。谋而后动。”
<br/>　　陆勇点了点头，他也是这样认为的。
<br/>　　一边的副参谋长扫了眼周华，眼中露出一抹异色。
<br/>　　“周兵王，你这就是长他人志气灭自己威风，我们这么多坦克这么多全副武装的军人，就算丧尸有十万那又如何。”
<br/>　　周华摇了摇头：“丧尸倒是其次，主要是怪物群体，这些怪物其中不乏可以免疫火炮攻击的。这样冒失冲过去不是明智的选择。”
<br/>　　“可是……”
<br/>　　副参还要说什么被陆勇打断了。
<br/>　　“周兵王说的有道理，就在前面这个什么八仙镇停下，然后派侦查小队出去看看情报，这个八仙镇距离白骨山也没多远了。”
<br/>　　陆勇道。
<br/>　　副参眼中露出一抹异色，旋即不再说话。
<br/>　　一刻钟后。
<br/>　　副参离开了队伍。
<br/>　　在他面前出现了一个人。
<br/>　　“情况有变，陆勇要将部队驻扎在八仙镇。”
<br/>　　“八仙镇？嘿嘿，无妨，那也是在我们的攻击范围之中，这个陆勇还有点脑子，没傻乎乎的冲过去。”此人嘿嘿一笑。
<br/>　　“干得不错，这一次若是能够完成任务，那么我战门肯定不会亏待你的。”
<br/>　　“多谢。”副参立即满脸笑容，“不过迟则生变，干脆你们现在就攻击吧，先把他们困在其中再说。陆勇把周华也带来了。”
<br/>　　“周华？那个曙光基地第一强者？九阶的实力也的确是个人物，外门成员你说的也是，我现在就通报上去。”
<br/>　　一刻钟后。
<br/>　　先遣部队抵达八仙镇。
<br/>　　然而下一刻。大地颤抖。
<br/>　　旋即一道光芒冲天而起直接把整个先遣部队笼罩其中。
<br/>　　几乎在这一瞬间，所有的坦克枪械全部报废。
<br/>　　“搞什么。我的枪坏了。”
<br/>　　“坦克也出问题了。”
<br/>　　“这是什么东西。”
<br/>　　坦克兵纷纷从坦克内钻了出来，一些步兵也都纳闷的看着头顶上的光芒就好像是北极光一样的。
<br/>　　陆勇和周华也是第一时间发现了不对劲的情况，不过他们也根本不知道发生了什么。
<br/>　　副参一脸的冷笑。
<br/>　　“陆勇，周华，要不了多久，你就要和你的几千部下葬身于此咯！”
<br/>　　不过他心中虽然冷笑，但脸上却同样表现出惊慌。
<br/>　　……
<br/>　　“根据一个傀儡的报告，先遣部队最高指挥官决定在八仙镇驻扎，为了以防万一，我已经让组织内的异能者启动了，先遣部队已经全部被笼罩其中，那些坦克火炮甚至是他们的枪械武器全部失效。当然了，手榴弹还是可以使用的，不过那玩意，我想他们应该会留着当光荣弹，哈哈！！”一个战门成员兴奋道。
<br/>　　“很好，什么时候可以开展歼灭计划。”
<br/>　　王长老微微点头。
<br/>　　“需要一点时间，因为之前被抽调了一些人手去对付尸傀门和血尸堂，大概半个小时重新调配人手。”

<br/>　　一群战门成员都跟着戏谑大笑起来。
