　　虚幻的空间通道之中，苏莫急速而行。
<br/>　　没过多久，他便飞出了空间通道，来到了一片广阔的空间之中。
<br/>　　抬眼望去，正是真圣界之内，巨大的真圣碑，就在眼前。
<br/>　　缓缓走进真圣碑，苏莫手掌按在了真圣碑上，随后便被吸进了真圣碑之中。
<br/>　　真圣碑空间之中，一片明亮，寂静无声。
<br/>　　前方，穿过一条空间通道，就是那巨大的九彩旗杆，亘古长存。
<br/>　　苏莫深吸口气，再次捏起了法诀，召唤传送之力。
<br/>　　这传送的原理苏莫不太清楚，但他也能猜测的到，那巨大的真圣碑就是传送阵，提供传送的力量。
<br/>　　法诀捏出，他周身的空间立刻凝固了起来，而后裹挟着他的身体，飞快的传送于虚无之中。
<br/>　　传送到万界山，还需要不段的时间，苏莫缓缓盘坐了下来，闭目调息。
<br/>　　神魂的分化，精神力的减弱，让他显得很是虚弱，实力估计都下降了两三成左右。
<br/>　　苏莫也没有恢复的方法，只能等待精神力自行的恢复了。
<br/>　　时间在传送中悄然流逝，一日又一日，约莫八、九日之后，传送停止了。
<br/>　　苏莫走出虫洞，出现在了庞大的万界山上。
<br/>　　星光璀璨，各种奇花异草，展现在了苏莫的眼中。
<br/>　　目光扫视四方，此刻的万界山上不再是空无一人，而是有少量同门之人的身影，下方那处屹立着界主雕像的广场上，也有少量的武者。
<br/>　　另外，苏莫他走出虫洞之时，有两名俊朗的年轻男子，进入了其它的虫洞之中。
<br/>　　两名年轻男子远远的瞥了苏莫一眼，凌厉的目光让的苏莫心中微震，身躯一僵。
<br/>　　“好恐怖的实力！”
<br/>　　苏莫暗暗心惊，这两人应该也是万界山的弟子，不知道来自哪个星河，起码也是中阶武尊！
<br/>　　“不知道真贤师兄在不在？”苏莫呢喃一声，他并没有去寻找对方，免得对方询问他任务的事情。
<br/>　　万界山给他下达的任务，只能让分身去做了，而且不知何年何月才有机会完成。
<br/>　　随即，苏莫便在万界上的上方区域转悠了起来，寻找通向明虚星河的虫洞，尽快的离开万界山。
<br/>　　对于万界山这个势力，苏莫心中完全没有底，太不同寻常了，虽然山上见不到几个人，但这个势力的实力绝对的恐怖。
<br/>　　所以，他虽然加入了万界山，但并不想有太多的交集。
<br/>　　万界山的上方区域，一直到山巅的位置，全部都是虫洞，密密麻麻，数以万计。
<br/>　　每个虫洞之上，都刻有古朴的字迹，表明了通往何地。
<br/>　　“太海星河……！”
<br/>　　“冰洲星河……！”
<br/>　　“苍冥星河……！”
<br/>　　“明虚星河……！”
<br/>　　因为通向明虚星河的虫洞，距离通向古灵星河的虫洞极近，只是片刻之间，苏莫便找到了。
<br/>　　而后，他身形一闪，直接进入了虫洞之中。
<br/>　　随即，空间传送开始了，苏莫的身形被一片空间之力包裹，便消失在了虫洞之中。
<br/>　　传送之中，苏莫闭目而坐，安心的修炼，他也不清楚到达明虚星河要多久，但想必不会比古灵星河快。
<br/>　　……
<br/>　　在苏莫前往明虚星河的途中，分身尤天韩到了途山镇，和苍穹神宫的一众高层言明了情况。
<br/>　　随后，尤天韩分身将身上所有的灵石灵晶留给了苍穹神宫，便返回了神武学府。
<br/>　　而苏莫的另一位分身，已经完全变成了苏莫之前变化的模样，化身成了上官昊，返回到了赤龙营。
<br/>　　只不过，赤龙营中的人都对‘上官昊’不熟悉，就算有觉得奇怪之人，也没有深究。
<br/>　　然后，苏莫分身到赤龙营几日之后，古天意来了，来到了赤龙营。
<br/>　　石殿中。
<br/>　　上官昊和古天意相对而坐。
<br/>　　“你是上官兄？”古天意望着眼前的苏莫分身，皱眉问道，因为此刻眼前的人，与苏莫的气息差别太大了，不仅完全不一样，还弱了很多。
<br/>　　“当然，古兄，我修炼了一种改变气息的秘法，就算是武圣境的强者，也不一定能识破我的身份！”分身笑着说道。
<br/>　　“是吗？还有这种玄妙的秘法？”古天意显然不太相信，眸中闪过惊疑之色，他感觉眼前的人，好像不是苏莫。
<br/>　　“古兄，我也是不得不谨慎，万一被人识破了身份，巫族之人不会放过我！”
<br/>　　分身摇头叹道，言罢，他话锋一转，沉声问道：“对了，古兄，我两位朋友救出来了吗？”
<br/>　　古天意没有立刻回答，眸光直视分身，心中惊疑不定，他感觉眼前的人不是苏莫，但对方的语气、神态等等，都与苏莫一般无二。
<br/>　　而且，他和苏莫之间的事情，对方都知道的清楚，不然不会问出聂乾和韩盈之事。
<br/>　　这让古天意心中疑惑不已！
<br/>　　“救回来了！”
<br/>　　古天意点了点头，沉声道：“聂乾和韩盈现在已经彻底的脱离了神武学府，我将他们安排在了皇宫中，以后就在皇宫中担职，安全和日后的前途都无忧了。”
<br/>　　分身闻言，眸中精光微闪，果然，对方不会轻易让聂乾和韩盈离开。
<br/>　　而是将两人安排在了皇宫中，从而牵制祝他，将他束缚在皇祖圣朝。
<br/>　　而在表面上，对方不仅救出两人，还给两人在皇宫中安排之位，实乃大恩大德。
<br/>　　“真是多谢古兄了！”分身不动声色，立刻向古天意抱拳道谢。
<br/>　　“上官兄不必客气，你我之间，何需说感谢二字！”
<br/>　　古天意摆了摆手，旋即笑问道：“对了，上官兄，听说你已经成了二十九小队的队长？”
<br/>　　“嗯，我击败了原队长赖羽！”分身点头道。
<br/>　　“很好，再接再厉，将来取代你目前的中队长祝英！”古天意微笑道。
<br/>　　“上官兄放心心，不会让你失望的！”分身点了点头，心中却是暗想，他现在的实力可是和本体有巨大的差距，已经没有了击败武尊的实力。
<br/>　　不过，他可以修炼圣级上品功法混天魔功，从而增强战力。
<br/>　　随后，古天意又和苏莫分身闲聊了片刻，便告辞离去。
<br/>　　走出赤龙营，古天意的脸色阴沉了下来，眸中光芒犀利。
<br/>　　“诡异！”
<br/>　　古天意心中暗道，他感觉这个上官昊，很可能不是苏莫，一切都显得非常的诡异。
<br/>　　不过，不管苏莫耍什么花招，只要还在乎聂乾和韩盈，就能为他所用。
<br/>　　时间会证明，此人到底是不是苏莫。
