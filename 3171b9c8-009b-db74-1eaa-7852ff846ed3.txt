　　飞长云高瘦的身形伫立在战台上，目光望向真圣碑，打量着八百个名字，少倾之后，朗声道：“周盛！”
<br/>　　随着飞长云话音一落，真圣碑上‘周盛’二字同样青光大放。
<br/>　　不远处，一名紫衣青年闻言，顿时面色一苦，他虽然也是真圣榜上的天才，但自知和飞长云相比，怕是差距不小。
<br/>　　而后，紫袍青年的身形，不受控制的飞到了平台上。
<br/>　　被人挑战，无法拒绝，只能应战。
<br/>　　众人对于飞长云挑战周盛并没有多少意外，因为来参加考验的人，目标都是真圣碑留名，获得真圣之气，自然不会挑战太强的人。
<br/>　　只不过，等到战斗的不断展开，弱者会被逐渐淘汰，一两轮之后，将没有弱者，全部是强强对决的场面。
<br/>　　当然了，周盛并不是弱者，能被排上真圣榜第二百名，便可见实力之强，只是相对于飞长云来说，算是较弱的人而已。
<br/>　　巨大的青色平台上，飞长云和周盛眸光对视，战意澎湃。
<br/>　　周盛虽然没有战胜的把握，但也会尽力一战，不会有丝毫的退却。
<br/>　　唰！
<br/>　　飞长云没有浪费时间，背后双翅一扇，瞬间闪掠而出，如同一道划破天际的光线，速度快到不可思议。
<br/>　　“什么？”周盛见此顿时一惊，对方的速度几乎快到让他无法锁定，面对此种局面，他只能暗暗准备，以静制动。
<br/>　　刺耳的破空声不断的响起，飞长云仿佛化成了无形的幻影，围绕着周盛四周不断的飞舞。
<br/>　　咻！
<br/>　　一道犀利的刀气，突然从上空劈下，速度如风似电。
<br/>　　周盛见此，二话不说，凌空就是一拳，强大的拳劲如同爆炸的火山，直冲天穹。
<br/>　　然而，拳劲还没击中犀利的刀气，后方和侧方再次有两道刀气向周盛袭杀而来。
<br/>　　“怎么可能？”周盛大惊失色，对方的速度实在太快了，不仅移动速度快，让他无法锁定，攻击速度同样快到极致。
<br/>　　毫不犹豫，周盛身形一闪，向另一侧急速躲避。
<br/>　　咻！
<br/>　　就在此时，一道更加恐怖的刀气，突然从前方而来，瞬间就到了周盛的面前。
<br/>　　这一刀更加强大，刀气横斩虚空，无边的锋芒让人全身冰寒。
<br/>　　电光火石之间，周盛根本来不及凝聚强大的招式，只能双拳向前一挡。
<br/>　　轰！
<br/>　　刀气斩在周盛的双拳之上，强大的威力顿时将周盛斩飞了出去，倒飞数千里，直接飞出了战台。
<br/>　　飞出了战台的周盛，面色大变，他败了，只要被击出战台，便是败了。
<br/>　　瞬息之间，周盛的身形消失无踪，被传送了出去。
<br/>　　与此同时，真圣碑上的‘周盛’二字，也消失无踪。
<br/>　　青色平台上，飞长云的身形浮现，挺拔如松，面色淡淡。
<br/>　　众人心惊不已，这真圣榜的排名，果然不虚啊，飞长云虽然平常低调，但实力真不是盖的。
<br/>　　这种速度，估计没有几个人能抵挡的住吧！
<br/>　　“恐怖！”苏莫眸中灼灼，此人的速度堪称恐怖，就算是他都锁定不住对方的身形，只能看到一道模糊的影子，急速的闪烁。
<br/>　　不过，这种恐怖的速度，相对于别人来说或许无法应付，但在他面前，对方想快都快不起来。
<br/>　　飞长云胜利之后，他的名字再次暗淡了下来，他便立刻飞下了战台。
<br/>　　而后真圣碑上，又有一个名字光芒大方，第二个挑战的人出现了。
<br/>　　没过多久，第二场战斗便又开始了，战斗双方是苏莫没有听过的天才。
<br/>　　随后的时间里，一场又一场的战斗毫不停歇，战败的人均是立刻遭到淘汰。
<br/>　　“这第三关，估计起码要用三、四天的时间！”一边观看战斗，苏莫一边暗想。
<br/>　　有些对战的人，实力相差无几，一场战斗都要斗个几百上千招，真是太慢了。
<br/>　　时间流逝，很快变过去了三个时辰，青色的平台上经过了数十场战斗，已经有数十人被淘汰。
<br/>　　不过，一直没有人挑战苏莫，虽然很多人认为苏莫名不副实，但大家都知道苏莫是巫族的目标，自然不会再去挑战他。
<br/>　　苏莫也不急，反正无论是他先挑战巫族之人，还是巫族的人先挑战他，都没有任何差别。
<br/>　　“怎么这么慢！”
<br/>　　然而苏莫不及，巫人王却是有些等不及了，已经三个时辰了，迟迟轮不到他，让他焦急不已。
<br/>　　少倾之后，又一场战斗结束之后，巨摩的名字亮了起来。
<br/>　　“哈哈！”
<br/>　　巨摩一声狂笑，而后一步踏出，庞大的身躯落在青色的平台上，让得平台都微微一震，传出一声闷响。
<br/>　　见巨摩上场，不少人又紧张了起来，可以说巨摩是任何人都不想碰到的对手，即便是苏莫、姜神夜、飞长云等人，都不想和其交手。
<br/>　　这并不是怕了对方，而是在真圣界这种地方，每个人都是想获得真圣之气，不想多无谓的苦战。
<br/>　　巨摩高大的身躯伫立在青石平台上，目光望向真圣碑不断的扫视众多的名字，不多时便落在了苏莫的名字上。
<br/>　　众人见此心中惊讶，巨摩这是要挑战苏莫吗？
<br/>　　苏莫的脸色沉了下来，他只想擒拿巫族的天才，可不想和巨摩战斗，毫无意义。
<br/>　　巫人王等巫族的天才见此，亦是脸色微变，这巨摩若是挑战苏莫，可以说苏莫必败了。
<br/>　　苏莫一旦败了，就会败传送出真圣界，他们就没有办法动手了。
<br/>　　虽说大祭司在外面，苏莫不可能逃走，但他们可是信誓旦旦向大祭司保证过，必杀苏莫。
<br/>　　若是最终还让大祭司出手，他们的脸上也是无光。
<br/>　　“巨摩，苏莫是我们的对手！”
<br/>　　就在此刻，一阵呼喊声响起，有三名巫族之人同时开口大喝。
<br/>　　这三人之中，其中一人正是巫人王，另外两人也是巫族之中一等一的天才，实力比巫人王还要强，均是名列真圣榜的天才。
<br/>　　至于巫族第一天才巫殇，虽然眉头微皱，但是没并有开口。
<br/>　　“哦？是你们的对手！”
<br/>　　巨摩闻言一笑，随即转头打量巫人王等三人，眸中露出戏虐之色。
<br/>　　“不错，你还是挑战别人吧！”一名瘦弱的巫族青年朗声说道。
