　　　　“哼...”
<br/>　　蛇精仿佛看破了涂小安的想法，一条蛇尾看似轻轻的拍了一下地面，猛然间，地面剧烈震动起来，随后，只听“轰”的一声巨响，地面层层被掀开，十几条不知向外延伸了多少米的裂缝触目惊心。
<br/>　　涂小安表情难看起来，蛇精葬身雷海下，还能有这般力量。
<br/>　　看来它确实还有实力，真尼玛的变态。
<br/>　　可为什么它明明还有实力，干嘛跟自己认怂，说不打了。
<br/>　　“小土蛇，我们来一场交易吧”蛇精异常认真的看着金银蛇。
<br/>　　涂小安蹙眉道：“你想说什么”
<br/>　　它B不会跟自己耍诈吧。
<br/>　　这等生物，智商比人还要出色，而且涂小安看的出来，这条巴蛇不知道活了多久。
<br/>　　绝对不是他可比。
<br/>　　“你只要答应帮本仙做一件事情，那么之前发生的所有事情，本仙都可以既往不咎”
<br/>　　涂小安闻言，露出了古怪之色，刚才这条蛇差点被自己的雷电劈成半身不遂，转眼要跟自己做交易，还让自己帮它做一件事情，想什么呢。
<br/>　　“如果本仙没有料错，你的雷电之力已经消耗殆尽，你的手段也仅限如此，你跟本仙的境界差距太大，你是不可能打败我的”蛇精霸气的说着：“而本仙要想跟你拼命，你招架不住，只要你答应帮我做一件事情，本仙掉头离开你白镇的地界，永不侵犯”
<br/>　　涂小安听着，思忖着，淡淡的开口：“你要我帮你做什么，先说清楚”
<br/>　　蛇精微微一喜，道：“什么事情到时候我自然会告诉你，时机成熟本仙会来找你”
<br/>　　“滚蛋，要嘛一五一十说清楚，要嘛我们重新打过”
<br/>　　涂小安又不是傻子，什么都不说，自己答应他个球，真的继续打，它又有何惧。
<br/>　　要是他的等级也是二十七级，这条上古蛇种算什么，早就碾压了。
<br/>　　蛇精怒目圆瞪，深深的吞了一口气，像是压制自己的暴脾气，发出沙哑的声音：“跟你说也无妨，本仙天赋异禀，迟早会突破三阶，三阶为生灵，凌驾普通生物之上，超凡脱俗”
<br/>　　“只是天地法则是不容许生物随便突破三阶，到时候想突破，就要承受天雷天火的浩劫洗礼，这叫渡劫，如果抗过去，变成为三阶生灵，要是抗不过去，就只能灰飞烟灭”
<br/>　　蛇精继续说着：“本仙拥有水系属性能力，到不怕天火，独独天雷让我没有把握，你掌控雷电，我希望你帮我抗几道天雷”
<br/>　　涂小安听明白之后诧异不已，原来这条蛇精打着如此好的主意。
<br/>　　这渡劫还能组团的？
<br/>　　渡劫这件事情很好理解，凡有修仙大成者，亦或逆天妖魔现世，乃至人间有不公之事，云雷震震，轰天霹雳，在这个时候总是成为铁面无私，判断善恶的天道所在。
<br/>　　也就是天地法则的制衡。
<br/>　　不管是人修炼，还是妖精修炼，达到了一定的地步，就要承受天劫的洗礼。
<br/>　　都说建国之后，动物不准成妖。
<br/>　　二阶生物为精怪，三阶为妖怪，当今时代在进步，称呼也变的更加的与时俱进，叫为三阶生灵。
<br/>　　凌驾普通生物之上。
<br/>　　目前为止，涂小安没有碰见过三阶生灵，也不知道三阶生灵有何等的强大，但能承受天雷天火的浩劫而活下来，一定是非常可怕的存在。
<br/>　　估计跟神话中的妖怪也不会相差太大。
<br/>　　这蛇精选择跟涂小安做交易，就是看上了他掌控雷电的能力。
<br/>　　只是天雷天火何等可怕，躲都来不及，谁还凑上去找死。
<br/>　　旋即涂小安鄙夷的看着蛇精，皮笑肉不笑，一副你当我傻的表情。
<br/>　　蛇精也明白，这种事情恐怕天底下没有那个傻子愿意干，立刻解释：“你先别急着拒绝，这件事情对你本身也是有好处的”
<br/>　　“好处？”涂小安古怪的看着蛇精，听它接着忽悠。
<br/>　　蛇精道：“你掌控雷电之力，如果去抗天雷，说不定还可以吸收天雷的煌煌天威，让你的雷电之力变的更加的强大，威力倍增”
<br/>　　嗯？
<br/>　　涂小安蛇眸一亮，这句话好像不完全算忽悠，他的雷电之力也是可以进化的，至于如何进化，自然是吸收更加强大的雷电力量。
<br/>　　天地法则降下来的天雷，自然不是他的雷电可比，根本不是在一个档次上。
<br/>　　要是他的雷电释放出来，具备天威浩荡，那么将今非昔比，威力不可预测。
<br/>　　这等涂小安有点心动起来，要知道释放的雷电威力越强，也代表他的实力越强。
<br/>　　这蛇精到现在还活蹦乱跳，归根结底，不是它耐抗，而是涂小安的雷电之力威力太弱。
<br/>　　否则它也不会邀请涂小安帮它对抗天雷了。
<br/>　　另外涂小安也想看看所谓的天劫是什么样子，曾经只出现在修真小说中。
<br/>　　还有就是，他迟早有一天也要面对天劫。
<br/>　　知己知彼更加有把握。
<br/>　　蛇精见金银蛇沉默，生怕他不答应，继续劝说：“本仙的三阶天劫起码还要几年，到时候你肯定会变的比现在更强，如果本仙度过天劫，等你到时候渡劫需要帮忙，本仙也可以帮你，所以这笔交易，你绝对不吃亏”
<br/>　　几年？
<br/>　　涂小安眨了眨眼睛，那估计他渡劫都要抢在这条蛇精的面前了。
<br/>　　要知道徐小安化为家蛇才数月时间而已，现在就已经十七级了，三十级，乃为三阶。
<br/>　　“我渡劫，你也帮我？”涂小安憋着笑的看着蛇精。
<br/>　　蛇精立刻道：“本仙一言既出驷马难追，到时候帮你对抗天火，不过天劫也不是什么虾兵蟹将都有资格体验的。”
<br/>　　这句话说出来牛气哄哄，天劫虽然可怕，杀人于无形，但是一般人想有这个资格都不可能，你必须修炼到那一步。
<br/>　　天底下，多少生物，数以千万亿计，几个能享受这种VIP待遇。
<br/>　　“好，成交”
<br/>　　涂小安突然笃定一声，暗暗发笑，只怕你要先帮我抗天火在先喽。
<br/>　　当然，这句话，他不会说出来。
<br/>　　蛇精怔怔的愣了一下，涂小安突然一下答应了，反倒让他感觉怪怪的。
<br/>　　答应是答应了，但涂小安俨然的看着蛇精：“你要记住，我们的交易前提是你不准在找舒瑶跟他男朋友陈响的麻烦，从此不准出现，如果你违背了，那么不要说交易，我跟你就是不死不休”
<br/>　　涂小安这时还不忘记自己的系统任务，舒瑶的麻烦点就在蛇精身上，它不出现，麻烦点就消失了。
<br/>　　“嘎嘎，没问题”蛇精虽然有点舍不得舒瑶美女，但关于自己渡劫，它也不得不忍痛答应，突破三阶的天劫，是所有修炼生物的第一门槛。
<br/>　　只有度过雷劫火劫才能洗去凡身，超凡脱俗，成为三阶生灵，凌驾普通生物之上。
<br/>　　最关键是，成为了三阶生灵，具备化形的能力，这是所有精怪梦寐以求的，到时候潜入都市，什么美女没有。
