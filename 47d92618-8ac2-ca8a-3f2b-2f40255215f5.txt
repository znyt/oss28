　　“找到了！”
<br/>　　元始经过不断的计算推演，终于以伏羲身上那段因果，找到了伏羲之前所在世界的道标。
<br/>　　“不过，位置比较远，要走到那里，还不知需多少年？”
<br/>　　元始皱了皱眉道，他抬头遥遥看向远方混沌之中，在那个方向，有一方世界，也正是他的目标所在。
<br/>　　“洪荒如今正处在三皇五帝时期，吾还需坐镇昆仑，此时不能走开。”元始眉头一皱，对此颇为不满意。
<br/>　　“超我原始之身乃是洪荒圣人，与天道联系密切，更何况，还需要随时防备着鸿钧道祖，也不能离开，‘道我’融合了灵神界，‘逝我’坐镇人族，也不能离开，恶尸万劫魔尊统御魔族，亦不能离开，三生大帝坐镇地府，参悟六道轮回，六道乃是重中之重，亦无法离开，那么，便只剩下善尸与执尸了。”
<br/>　　元始微微沉吟，在心中思索着是该让善尸永恒岛主出世，还是执尸出世？
<br/>　　一番思索，元始还是决定让执尸去那方世界，因为无论是善尸还是执尸，如今都被困在准圣大圆满，不得突破，而他们想要突破，除非是有大造化或者大机缘。善尸在永恒岛，消失了终究会引起别人的注意，特别是圣人的注意，而只有执尸，从来没有在别人面前出现过，之前一直在元始印中参悟三千大道，就算消失了，别人也不会知道。
<br/>　　决定好，元始一挥手，便召唤出一个身披红袍，头顶混沌钟的身影，他不苟言笑，与元始有七八分相似，正是元始的执尸。
<br/>　　“见过道友！”执尸刚一出来，便对着元始拜道。
<br/>　　“嗯。”
<br/>　　元始微微颔首，开口淡言道：“想必你已知晓了吧。”
<br/>　　“自是知道，贫道此去定然会找到另一界。”执尸微微点头，说道。
<br/>　　“好，既然如此，那便祝道友一路顺风！”
<br/>　　说着，元始将一道灵光打入执尸体内，那正是伏羲所在世界的道标，有了它，执尸才不会迷失在混沌之中，能够找到那方世界。
<br/>　　“多谢道友，吾去也！”
<br/>　　说着执尸身形一动，便淡淡消散于虚空之中，完全不见了踪影，却是已经离开了。
<br/>　　执尸离开之后，元始并不担忧他的安危。执尸已经是准圣圆满了，更有混沌钟这等开天之宝在手，就算是遇到一般混元强者，也能够对战一二，没有什么好担忧的，更何况，混沌无垠，说不定什么也遇不到呢。
<br/>　　他微微沉思，便将心思重新放回到昆仑山上，心念一动，他便知道，洪荒已经过去了百年。昆仑山上的小轩辕也长成了大轩辕，一身修为深不可测，已经是大乙金仙高手了。果然不愧是天地主角，气运汇聚啊。
<br/>　　别人想要修炼到太乙金仙，无不是用了数万年乃是数十万年时间，而公孙轩辕却只用了区区百年时间。这就是量劫主角，天地灵气汇聚，气运所钟之人，做什么事情都很顺。
<br/>　　在这百年时间内，公孙轩辕除了随着广成子修炼，还经常到昆仑山上设置的一些秘境之中试炼。如当初元始收徒时设置的六欲惑心阵，百级求道路，三千问道台。当然，除此之外，元始还设置了一些其他的任务，让平时弟子们前去试炼。
<br/>　　在昆仑山的周围，密布着不下于百个秘境，全是元始开辟的，里面有着各种各样的机缘，或者阻碍，只要任务完成，便能获得奖励，毕竟，元始可不会养一些懒人。
<br/>　　公孙轩辕这些年便是在秘境之中度过的。当然，仙神楼与三界楼，他也去过，毕竟那里面有好多秘术，都是值得他学习的。不过，进去里面终究是要进行考验的，毕竟，道不可亲传！
<br/>　　元始心念一动，便来到一座小山之上。山间随处都冲斥着凌厉的剑气，树木之间，山石之间，甚至就连溪流之间，都有一股不可磨灭的剑意。
<br/>　　山崖之上，有着一道道杂乱不堪的剑痕，剑痕之中，有的是毫无任何气息，而有的则是剑意惊天。从左到右，剑痕之中的剑意逐渐增强，到了最右边，甚至连一个生命都没有，因为那里的剑意实在是太强了，生物无法在此生存。
<br/>　　元始默默的看了一眼这些剑痕，他仿佛能够看到一个剑客在不断成长着，从稚嫩到成熟，再到巅峰，不断进步，不断成长。他眼光一扫，石壁上便能够反射出昔日的一道道场景……
<br/>　　若是有剑客能够在此参悟，绝对能够让他实力大增，剑道修为飞速进步。
<br/>　　元始在此站立一刻钟，见天边红日东升，一轮赤色的旭日缓缓透出山头，一缕晨光打破夜的寂静，微风轻抚，疏影摇曳，元始的内心便如同这早晨的山林，寂静而幽深，空旷而充满希望。
<br/>　　突然，一个身影出现在山头之上，他背负一柄木剑，缓缓走上山头。他剑眉挺翘，目光犀利，黑色的发丝在早晨的微风中浮动，如同一柄柄利剑在飞扬，他全身上下无不散发着一股凌厉的剑气，在他周边一丈范围内，全部是剑气，他周围的环境都被剑气同化了。
<br/>　　可见，这是一个剑道高手。
<br/>　　他，便是公孙轩辕。
<br/>　　公孙轩辕在这百年时间内，一有时间，便会到这山头之上练剑，四周的一切，都仿佛被他同化了，化为剑域，这还是他平时不经意间释放出来的剑意导致的，若是他刻意去改变，说不得此处早已成为一片剑域坟墓了。
<br/>　　“徒孙公孙轩辕拜见师祖！”
<br/>　　突然，轩辕看见了元始，他身上剑意一收，瞬间又恢复了一个普通人的模样，向着元始恭敬道
<br/>　　。这绝对是到了能够将自己的剑意收放自如的地步，同时这也是一个成功的剑客必备的手段。
<br/>　　“嗯。”
<br/>　　元始看着公孙轩辕微微点头，淡淡道：“你来昆仑也有百年了吧。”
<br/>　　公孙轩辕廷听闻此言，微微一愣，不知道元始想要干什么，不过，他还是如实回答道：“九十九年单九月，距百年还差三月。”
<br/>　　“是啊，九十九年了，人生有多少个九十九啊。你自到了昆仑山，你学到了些什么？”元始感慨一声，向着公孙轩辕问道。
<br/>　　“徒孙学了《九转玄功》，前五转有成，又修炼了《葬剑诀》、《乱剑诀》，刚刚入门不久，更兼修《神元九转》前三转，神海成域！还学了一些其他厉害的剑术，以及一些炼药之术！”公孙轩辕恭敬道。
<br/>　　元始点头道：“你也有诸多本领在身，与刚上山之时相比，也算是脱胎换骨了，到了如今这个份上，你留在我昆仑玉虚，也已是进步不大。你接下来需要的是在红尘中磨练，早日突破。
<br/>　　人族如今正处多事之秋，诸侯蠢蠢欲动，你一出生便十分不平凡，吾便知道你是身具大使命之人，想必与这有关。你如今也是时候回人族了。”
<br/>　　“徒孙谨记！”公孙轩辕心中恍然。
<br/>　　原来……
<br/>　　这是要吩咐自己下山啊，不过，人族处在纷乱之中，这事……倒是要好好计较一番。
<br/>　　“如今你也到了下山的时候了，三个月后，正好一百年，你便下山去吧。”元始看着公孙轩辕，淡淡道：
<br/>　　“在这最后的三个月中，吾便再与你两大好处。”
