　　第161章
<br/>　　幽州王给齐德强的军令是三天之内攻破蛤蟆寨，砍下程大雷的脑袋。而齐德强用了多长时间内……十天。
<br/>　　十天时间，他没和程大雷打上一仗。
<br/>　　程大雷定下的战略和薛半川一摸一样，只有两个字：不打。
<br/>　　管你在外面闹成什么样，是假装撤兵，示弱诱敌之类的，我都坚持不出来，夜袭什么的，程大雷坚决不搞。
<br/>　　而山林之内，投石机等大规模攻城器械施展不开。而蛤蟆寨的城门又忒小了一些，齐德强带了一万人，每次攻城真正用得上的兵丁不过几千人。
<br/>　　靠几千人打不破这座蛤蟆寨。
<br/>　　两军休战时，程大雷就会站在城门上，冲齐德强喊话。
<br/>　　“老齐啊，还记得我们秉烛达旦，促膝夜谈的时候么，你要去远方，我要去找我的姑娘。”
<br/>　　“你我一见如故，虽非亲兄弟但胜似亲兄弟，没想到今日兄弟之间，兵戎相见。”
<br/>　　“老齐，我新做了一首诗，是歌颂我们兄弟感情的，我给你读读……”
<br/>　　“还记得我们曾鸿雁传书，信中你说杨龙停酒囊饭袋，你若做幽州王必然比他做得更好。“
<br/>　　……
<br/>　　齐德强终于感受到杨龙停被谣言支配的恐惧，谣言这种东西，他伤不着你，但他恶心你。尤其是最后一句，完全是诛心之论。这说法若是被幽州王知道，先不提他信不信，心中肯定会有芥蒂，日后自己的前程就算毁了。而若禁止士兵传播，就显得自己好像做贼心虚似的。
<br/>　　程大雷绝不是一个强大的对手，但绝对是个恶心的对手。
<br/>　　好吧，你既然不出来，我也不和你打，就让大兵扎在青牛山，护送运粮车队。
<br/>　　但若让一万大军守在此地，实在有些不值得。齐德强无奈向幽州王求助，承认自己打不破蛤蟆寨的事实。
<br/>　　杨龙停让他带一部分人回来，留下一部分人保护运粮队伍。
<br/>　　可如此，程大雷却来了精神，夜袭、放火、陷阱……仗着对周围环境的熟悉，程大雷带着一队人抢劫运粮队伍。能抢就抢，抢不走就烧……而且绝不和你正面厮杀，只有一碰到大股部队，立刻逃回山寨，连影子都不剩下。
<br/>　　最后幽州王运十车粮，至少损失三车，盘算下来，还不如一开始就遵守蛤蟆寨的规矩，给程大雷两成粮食呢。
<br/>　　这边展开与程大雷捉迷藏般的丛林战，而杨龙停的主战场还是在黑石城。
<br/>　　攻城的战斗也不顺利。
<br/>　　地道、夜袭、刺客潜入……攻城能用到的法子都想了，的确给薛半川带来巨大的压力，但没有取到真正的效果。
<br/>　　缺粮始终是杨龙停的一块心病，程大雷握着粮道，想给你粮就休息几天不抢粮了，想不给你粮就加班加点干活。
<br/>　　喉咙被人捏着什么感觉，杨龙停现在就是什么感觉。于是，他在攻打黑石城的时候，始终无法全力以赴。
<br/>　　杨龙停感觉自己陷入一个泥沼中。半生戎马，他遇到过很多敌人，强大的，残忍的，狡猾的……但绝没有那个敌人和程大雷与薛半川一样令人恶心。
<br/>　　二人的套路，该不会是一个师父教出来的吧，或者他们之间有某种合作。
<br/>　　杨龙停已经有灭掉程大雷的打算，先杀蛤蟆，再斩薛四。可等他刚刚拔营，从没露过面的薛半川就从城里杀出来了。
<br/>　　好了，终于出来了，打吧！终于能酣畅淋漓的杀一场了。
<br/>　　等他整顿后队形，准备好冲锋的时候，薛半川又回去了，城门关得紧紧的。
<br/>　　那……继续拔营，薛半川又出来了。准备打的时候，他又回去了。
<br/>　　现在，杨龙停确信，程大雷和薛半川一定有某种合作，不然不会配合得如此默契。
<br/>　　战争是一件很复杂的事情，不是大手一挥，大军杀到，双方兵力一对比，谁输谁赢就清楚明白。战争没这么简单。坐飞机还有晚点的时候，随便耽误两三个小时，更何况是一支上万人的军队，任何人发生一丁点情况，就足以酝酿成一场毁灭全军的意外。
<br/>　　这场兴师动众的战争，最后打了多久呢……两个月。
<br/>　　两个月后，杨龙停这边疲惫不堪，人心惶惶，薛半川那里何尝不是如此。围城两个月，城里已快断粮了，能不能再撑下去，谁也不知道。
<br/>　　战争拼到最后，拼得就是谁更能扛。杨龙停要是扛不住了撤兵，薛半川就一战扬名，势力足以覆盖周边几座城池。薛半川要死扛不住了，稍微松懈杨龙停就可以破城。
<br/>　　至于程大雷……程大雷过得还不错。
<br/>　　他时不时打劫杨龙停的粮队，这两个月来收入颇丰，蛤蟆寨的山贼整体来说都胖了一圈。
<br/>　　这几日下起雨来，雨水淅淅沥沥下个不停，路上坎坷难行。程大雷也就一直没有出活（抢劫）。到这一日天还下着小雨，程大雷带着人在外面转悠，看今天能不能劫一次。
<br/>　　忽然，就见从黑石城方向过来一拨拨的残兵。
<br/>　　“怎么，薛半川把杨龙停熬败了。”
<br/>　　这一点出乎程大雷预料，他赶紧带人撤回山寨。如果真和自己预料一样，兵败的杨龙停说不得会把怨气倾泻在自己身上。
<br/>　　他带人站在城门上往官道上望，就见一拨拨败兵经过。这些人也没有攻打蛤蟆寨的意思，完全就是一群丧家犬，每个人都丢盔弃甲，脸上写满恐惧，看他们都样子，就像是有恶魔在身后追赶一般。
<br/>　　“这是怎么了，败得这么惨！”程大雷皱紧眉头。
<br/>　　“大当家，大当家！”
<br/>　　从官道上急匆匆都跑过来一匹马，马上驮着黄三元和赵子龙二人，自从上次去了黑石城后，二人就一直没回来。这次回来，赵子龙浑身浴血，黄三元也没有好到那里去。
<br/>　　程大雷赶紧打开城门放他们进山，刚进山寨，马就跌倒在地，口吐白沫不止。
<br/>　　“你们是怎么了？”程大雷道。
<br/>　　“大当家，大事不好了！”黄三元急促呼吸，身体瑟瑟发抖：“戎族，戎族杀过来了！”
<br/>　　“什么！”程大雷一惊。
<br/>　　幽州兵军临城下，苦战两月不死不休，极北之地，北蛮部一万儿郎南下。
<br/>　　然后，一日破幽州兵，半日破黑石城。
<br/>　　然后，屠城。
<br/>　　十日不封刀。
