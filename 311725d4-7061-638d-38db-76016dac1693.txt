　　智天使站在原地想了想，嘴角勾起一抹冰冷至极的微笑，随后很肯定的说道：“会！不管有没有那颗子弹，我都会引爆地铁站那颗地雷。只有混乱了，才能让我们有更多的机会，不是吗？更何况，他们既然已经把我们定义成了恐怖分子，那么我们自然是要满足他们的定义。”
<br/>　　智天使和影天使两个人离开了，只剩下吴云一个人留在房间内，笔记本电脑上的视频还在继续播放着，但是视频中K国国防部长所说的话，一句都没有传进吴云的耳朵中。
<br/>　　吴云不认为自己是个好人，如果是好人他就不会加入“暗月”这样一个组织。他杀过人，杀过很多很多人，老人、小孩、男人、女人，他都杀过，什么手段都使用过；可是如智天使这般，随意按下遥控器，让上百人葬身爆炸中，他还是感觉到心惊，智天使刚才在回答自己问题的时候，是那么的风轻云淡，是那么的……随意，好像那些死在地铁站中的人，在他眼中根本就不是人，只是一些微不足道的蝼蚁。
<br/>　　昏暗的房间中，只有笔记本电脑屏幕的光亮，吴云站在房间里突然笑出声，他一只手捂着自己的脸，大笑着，整个人都好像陷入了一种癫狂之中。
<br/>　　良久，吴云才止住了笑，黑暗中他的眼睛闪过残忍、冷酷、杀戮、狰狞这所有代表着人最阴暗一面的神采，盯着这如墨一般，仿佛随时都会他卷入，让他深陷其中的黑暗，冷声说道：“果然，我还是太仁慈了吗？暗天使是黑暗的象征，不需要仁慈！”
<br/>　　走廊转角处，隐在黑暗中的智天使看着吴云所在的房间，刚才发生在房间里的一幕并没有瞒过智天使和影天使，或者，更准确的说，从一开始智天使和影天使就一直关注着房间中吴云的反应。
<br/>　　“没有让我失望。”智天使嘴角噙着一抹冷笑，说道：“这才是我想看到的暗天使。”
<br/>　　“如果他的表现没有达到你的要求，你会怎么做？”影天使突然开口对智天使问道。
<br/>　　“杀了他，换一个合适的人。”智天使回答的时候几乎没有多余的思考，说道：“盯着暗天使的人有很多，对他有所不满的人也有很多，要想找出一个合适的人来接任暗天使的位置，不是很容易吗？”
<br/>　　“只能说，这家伙父亲的运气不错，不然他早就死了。”
<br/>　　影天使看了眼站在房间里的吴云，轻轻摇头。
<br/>　　……
<br/>　　距离影天使定下的对决时间还有22个小时，不，准确的说，是21个小时零39分钟。
<br/>　　而就在这时一则出现在网络上的视频，却是一石激起千层浪，令得K国原本已经平静下去的局势又是变得动荡起来。
<br/>　　这段视频一共有五分钟左右，视频中的人同样在经过了处理后，隐藏在阴影中看不到他的真实面目，但是罗昊在看到视频的第一眼就认出了视频中的人是谁。
<br/>　　智天使！
<br/>　　不会有错，绝对就是这个家伙，罗昊跟他交过手，也多次领教过他的阴谋诡计，心狠手辣，所以即便智天使整个人都隐在黑影中，罗昊还是一眼就把他给认了出来。
<br/>　　经过了技术处理之后，智天使的声音变得有些低沉，“两天前发生在地铁站的爆炸案的确是我做的。”
<br/>　　智天使这句话一出，马路上、公交车上、出租车上，拿着手机观看视频的人都是倒吸一口冷气。
<br/>　　随即，智天使话锋一转，继续说道：“但是事实的真相确实如国防部长先生所说的那样吗？我只能说，那位国防部长先生真的是个优秀的演员。”
<br/>　　K国国防部，国防部长看着笔记本电脑中的视频直播，几乎要把牙齿给咬碎掉，抓起办公桌上的电话机，拨了一个号码，吼道：“立刻给我中断视频播放！”
<br/>　　“对、对不起，他们那边有人阻扰，我们的技术人员没办法中断视频播放。而、而且这个视频不仅是在我们国内播放，在国外很多网站上也出现了这个视频，播放量已经超过了八千万，我们阻止不了了。”
<br/>　　吧嗒……
<br/>　　国防部长手里的电话掉落在办公桌上，他靠在椅子中，双眼无神的盯着自己桌子上的笔记本电脑，喃喃的说道：“完了……”
<br/>　　智天使的声明视频还在继续。
<br/>　　视频中智天使冷笑一声，说道：“地铁站爆炸，的确是我对K国政府的报复，但是实际情况却是跟国防部长说的有很大出入。我这里有一份当时监控所拍摄下来的影像，我想你们看完这段录像，不需要我再多说什么，就能明白，那个在新闻招待会上义正言辞的家伙，是个多么龌龊、卑劣的骗子！”
<br/>　　视频画面一转，变成了爆炸发生那天晚上，写字楼外马路上的画面。从视频的画面角度判断，这段录像应该是通过隐藏在暗处的监控探头拍摄下来的，而这个监控探头显然没有被人发现，要不然，它也不会存在。
<br/>　　录像清楚的记录下了当时的情况，不仅有画面还有声音，包括罗昊勒令肇事狙击手待命，但是肇事狙击手固执己见，开枪射击，最后导致爆炸发生的全部过程。
<br/>　　所有观看到这段视频的人哗然不已，他们没想到事情的真相会是这样的。马路上拿着手机正在观看视频的行人相互之间交头接耳，窃窃私语，他们之间可能原来并不认识，但现在却成了同一战线的“战友”！
<br/>　　指挥中心，慕容复看着投影仪放大在投影布上的视频，说道：“虽然我对智天使这个阴险狡诈的家伙很不喜欢，但是他现在做的这件事情却是让我感到很爽。你们是不知道，那天晚上那个狙击手有多臭屁，感觉他就是狙神，一副天老大他老二的样子，那嚣张的样子，让我恨不得抽他几个耳光。”
<br/>　　“如果你们以为事情这样就结束了？那你们就太天真了。”
<br/>　　监控录像播放完后，智天使又重新出现在视频中。
