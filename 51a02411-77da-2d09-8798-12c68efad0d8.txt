        苏漫雪又呜咽道：“爷爷说，有人想对夜沉不利，而我是夜沉的软肋……”
<br/>
<br/>        “真是这样？！”童筱颖凝了凝眉。
<br/>
<br/>        “筱颖，我好羡慕你……找了一个温柔体贴的老公！”苏漫雪可怜兮兮地看着童筱颖，随即转开了话题。
<br/>
<br/>        她一点都不想让童筱颖知道，是傅老太爷发现她冒充她，因此而惹怒了傅老太爷，才被傅老太爷禁足的。
<br/>
<br/>        而每次提及傅昼景，童筱颖的脸上总是不由地露出了甜甜的笑容。
<br/>
<br/>        “筱颖，我们不要绝交，好不好？我就你这么一个真心相待的闺蜜，以前我错了，我们重新开始，重拾友谊吧！人都有犯错的时候，你也冷落了我这么久，我也受到了应有的惩罚……”苏漫雪重新拉起童筱颖的手，信誓旦旦地说。
<br/>
<br/>        童筱颖看着苏漫雪，只见她眼眸里还有泪光晃动。
<br/>
<br/>        就算不念友情，也要念大学三年的同窗之情吧！
<br/>
<br/>        然而，童筱颖却冷言冷语道：“我来这里，不是为了跟你叙旧，而是为了昼景。”
<br/>
<br/>        苏漫雪眼珠子一转，起身给童筱颖和自己一人泡了一杯果茶，并在落地窗前的玻璃圆桌前坐下，接着说道：“筱颖，来这里坐坐。我们边喝茶，边说话。”
<br/>
<br/>        童筱颖走过去，坐在了苏漫雪的对面。
<br/>
<br/>        苏漫雪将果茶递给了童筱颖，然后自顾自地端起自己面前的杯子，喝了一口，说道：“其实，冷爷爷他一直在暗中算计你家昼景……”
<br/>
<br/>        “爷爷算计昼景？”童筱颖难以置信地看着苏漫雪。
<br/>
<br/>        苏漫雪微微点头，并迎了迎手，示意童筱颖先喝口茶解解渴，她再接着说。
<br/>
<br/>        童筱颖若有所思起来，毫无防备地端着果茶抿了一口。
<br/>
<br/>        “其实，我住在傅家大庄园里的时候，也是当时无意间偷听了冷爷爷打电话的内容才得知的。冷爷爷说，随昼景去。最好帮他把他的事情闹得越大越好，到时候，冷爷爷好坐收渔翁之利。其实，我也不太清楚是个什么事情。只是隐约间，听到是个什么项目的工程。那个项目应该有上百亿吧！”苏漫雪努着嘴，弱弱地说道。
<br/>
<br/>        事实上，苏漫雪只是凭着自己偷听到的片面之词，故意在这件事情上添油加醋罢了。
<br/>
<br/>        至于真相到底是什么，她无所谓，只要能糊弄得住童筱颖就行了。
<br/>
<br/>        而苏漫雪所说的这件事情，正巧与童筱颖昨晚在傅夜沉那儿听到的事情吻合了。
<br/>
<br/>        而且，她跪在爷爷傅炎廷的房门外，跪了一宿，也没能打动爷爷傅炎廷。
<br/>
<br/>        到底是昼景自己的野心让他犯了错，还是被爷爷傅炎廷给算计了。
<br/>
<br/>        这件事情最后的结果，都是昼景一个人承担。
<br/>
<br/>        “你要跟我说的，就只是这件事情？”童筱颖还以为她苏漫雪知道了什么重要的事情，可以救昼景，可到头来，却只是一个对她来说几乎没有用处的消息。
<br/>
<br/>        “当然还有就是……我觉得你可能需要睡一觉！”苏漫雪勾唇一笑。
<br/>
<br/>        童筱颖忽然觉得有些头晕脑胀，眼前的苏漫雪，在她自己的视野里慢慢地出现了重影。
<br/>
<br/>        “你……”童筱颖身心一怔。
<br/>
<br/>        “睡吧！筱颖！”苏漫雪突然用跟她童筱颖一模一样的嗓音，温柔地说道。
<br/>
<br/>        苏漫雪的话，就像催眠曲一样，催得童筱颖闭上了眼睛。
<br/>
<br/>        童筱颖似乎做了一个梦，梦到了傅昼景，他帅气的脸庞朝她凑了过来，一吻落下，可转眼间，视野里的面孔却变成了另一个男人。
<br/>
<br/>        而这个男人……
<br/>
<br/>        竟然是冷、夜、沉！
<br/>
<br/>        “啊——”童筱颖一声呓语，腾地坐起身来。
<br/>
<br/>        屋内一片漆黑，伸手不见五指。
<br/>
<br/>        “叩叩叩——”
<br/>
<br/>        “谁？”童筱颖几乎是条件反射般地质问。
<br/>
<br/>        “苏小姐，现在已经晚上七点了，您是不打算吃晚饭了吗？”门外传来一个中年男子的声音。
<br/>
<br/>        童筱颖惊愕，慌忙从床上下来，她摸着黑，在墙壁上摸索了半天，才摸到开关。
<br/>
<br/>        灯亮的一瞬间，童筱颖下意识地闭了闭眼，待视野适应环境后，她才发现自己竟然还在苏漫雪的房间里，然而，却不见了苏漫雪的身影。
<br/>
<br/>        童筱颖穿上鞋，走过去拉开了房门，看到出现在房门口的中年男子，冥思了一下，才想起这个男人是苏漫雪家的管家刘叔。
<br/>
<br/>        “刘叔，漫雪她不在房间里。”童筱颖接着说道。
<br/>
<br/>        刘管家冷笑：“苏小姐，您又和我开什么玩笑呢？”
<br/>
<br/>        童筱颖下意识地回过头去看了看，房间里确实没有苏漫雪啊！为什么这个刘叔开口闭口就是一个“苏小姐”。
