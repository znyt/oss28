　　他们是标准的三流强者。
<br/>　　芮冷玉看他们聊完了，迫不及待地问叶少阳：“少阳，那个叫陈悦的，真的跟你是同年同月同日生？”
<br/>　　“啊，没错，反正身份证上是这么写的。”
<br/>　　道风一听，问道：‘什么人？”
<br/>　　因为与主线无关紧要，叶少阳之前并没提到陈悦，当下只好把有关陈悦的情况说了一遍。
<br/>　　失忆，与叶少阳莫名遇到，同年同月同日生，气海内有大量神秘的戾气……
<br/>　　道风心中咯噔了一下，该来的，还是来了啊。
<br/>　　他波澜不惊的心中，罕见地翻起了浪花，故作平静地对叶少阳说道：“你盯着她，不要让她走脱了，另外也防着她，先弄清她的身份，到那时候，你把她带到我这里来……”
<br/>　　“什么意思。”
<br/>　　“没有什么，你照做就可以了。”道风闭口不再谈这件事。
<br/>　　叶少阳满心疑惑，但道风就是不开口，也是没办法。
<br/>　　“行了，你可以走了，我还要静养，对了，你出去告诉林三生，给我准备一些石膏水……”
<br/>　　“你要做豆腐吗？”
<br/>　　道风不想跟他开玩笑，道：“再找一个木人，跟我一般大小，雷劈过的枣木最好，还有一些材料，我写在纸上了，你让他按照上面准备就行。”
<br/>　　叶少阳从他手中接过了那张纸，细看的时候，道风说道：“还有，少阳，你要小心阴司，你现在不再是他们保护的对象，甚至，他们会去对付你。”
<br/>　　叶少阳惊了一下说道：“我听雨晴说，之前那个我冒充我的家伙，就是被地藏菩萨派来的几个人追杀了，还是从地狱里找的，魑魅魍魉四兄弟。地藏菩萨为什么突然对我下手。”
<br/>　　道风沉吟道：“第一，是为了山海印，地藏菩萨不想如此神通的法宝留在人间，第二……他在试探。”
<br/>　　“试探什么？”叶少阳追问。
<br/>　　“试探大帝这边的底线，在你身上的底线，由此来判断你的真实身份。”
<br/>　　“真实身份？”叶少阳眉头皱起来，“我就是叶少阳啊，我还有什么身份？”
<br/>　　“这件事，你不要再问，否则对你并没好处，你好生修炼，先把实力提上去再说。”
<br/>　　叶少阳想开口，被道风用手势止住，说道：“我知道你好奇，但给我点时间，我答应你，只要时机稍微成熟，一定告诉你真相。”
<br/>　　道风都这么说了，叶少阳再着急也是没办法，点点头，道：“你就彻底在这不出去了？”
<br/>　　“等我修为大成，我再出去，现在只要露面，立刻就会被杀。”
<br/>　　叶少阳还想说什么，道风道：“好了，我这没什么事了，你们去溜达一会吧，我要调息了。瓜瓜你来给我护法。”
<br/>　　“遵命！”瓜瓜冲叶少阳眨了眨眼睛，跳到道风身边坐着。
<br/>　　叶少阳当然明白，这是道风特意给自己和芮冷玉独处的时间，于是拉着芮冷玉走到外面去。
<br/>　　“去哪儿？”芮冷玉问他。
<br/>　　“随便走走，这里你比我熟，你带路吧。”
<br/>　　“好吧，我带你去后面走走，那里风景不错。”
<br/>　　道观后面的山上，长满了桃花，漫山遍野，花比较偏白，跟人间的有些不同，香味昂然。两人在花丛中间散步。
<br/>　　“为啥是你在这照顾他，为啥不是陈露或者宫梓？”叶少阳忍不住问道。
<br/>　　芮冷玉抿嘴一笑，“你是吃醋了吗？”
<br/>　　“额，才没有。”叶少阳摸了摸鼻子。
<br/>　　芮冷玉上前挽住他的胳膊，跟他一起走，说道：“我只知道，因为我跟现在的他一样，都是不能露面的，否则会被多方追杀，还有一个原因……他没告诉我，我也不知道。宫梓他们，因为要留在风之谷指挥战争，当然不能来这里。”
<br/>　　说完冲叶少阳笑了笑，“你就放心吧，道风再帅，也不是我的菜，更不要说他是师兄。”
<br/>　　“我不是这个意思……”叶少阳有点尴尬。
<br/>　　“好啦。”
<br/>　　两人沿着山路下山，走到山谷中，这时候已经出了结界范围，虽然化蛇就潜伏在附近一带，芮冷玉一般不下山，但有叶少阳陪着，是不用怕的。
<br/>　　山谷之中，跟人间一样，开满了各种各样的花，而且是原生态的，没有一点人工的痕迹，两人在中间闲庭信步，叶少阳忍不住一时兴起，把她壁咚在一棵木棉树上，附身吻她。
<br/>　　两人正在缠绵的时候，一股风从山谷入口吹来，带来了一股浓烈的血腥味，叶少阳立刻警惕地转头看过去。
<br/>　　芮冷玉拉了他一把，道：“别激动，那是黄妖在献祭。”
<br/>　　“献祭？”
<br/>　　“走吧，我带你去看看，你先贴张隐气符，不要打扰了他们。”
<br/>　　叶少阳于是画了两张隐气符给她和自己贴在身上，手拉手走过去，快到谷口的时候，芮冷玉带路，爬上了一座山峰，从悬崖边上居高临下地看过去。
<br/>　　叶少阳一眼看到，在山谷中间有一座月牙形的深潭，里面荡漾着血色的水，至少有上百只黄鼠狼围在边上绕圈缓慢行走，口中发出低沉的怪叫，不时有黄鼠狼跳进血水中，立刻就是一股白烟，落水的地方会翻腾起浪花，能看到这个跳进去的黄鼠狼转眼就成了一堆白骨。
<br/>　　这场面，有点触目惊心。
<br/>　　“他们是在选出首领。”芮冷玉解释道，“这是他们中间的神圣的仪式，就像人间一样，很多妖族的首领就是靠这种献祭选出来的。”
<br/>　　“这个我知道。”叶少阳想起了当初小九解开封印时的那场献祭，跟眼前这差不多。“我只是没想到，这里居然也有这种仪式。”
<br/>　　“这里就是万千年前的人间，黄狐两族从那时候起，就已经用这种原始的方式来选首领了，这是模仿人类最早的献祭仪式。”
<br/>　　叶少阳望着眼前这残忍的一幕，吸了一口气。
<br/>　　芮冷玉道：“你觉得残忍吗。”
<br/>　　“有点。”
<br/>　　“是啊，不过，这就是妖族的道，对他们来说，却是势在必行，只有强大的首领，才能带领他们崛起，不至于被外族欺负。”
<br/>(本章完)
