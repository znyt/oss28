　　“你不能去，你身份太特殊，你去了，会让事情更乱，况且你这实力，去了也帮不到什么，只能添乱。”
<br/>　　芮冷玉狠狠瞪了他一眼，道：“你倒是会说大实话！”
<br/>　　“还有，我每天需要血液来固本养息，你不能走。”
<br/>　　“等杨宫梓来了，还要我做什么。”
<br/>　　“她不能久留，风之谷没了我，诸方势力都跃跃欲试，她必须坐镇，应付乱局。再说，她是邪灵，并无人类血液。”
<br/>　　芮冷玉皱眉道：“那就让你徒弟来，那个小狐狸。”
<br/>　　“她要与岳恒合修玄功。”
<br/>　　芮冷玉望着他冷笑：“所以就我一个闲人了是吧。”
<br/>　　道风道：“你还能去哪里，外面抓捕你的人，比抓捕我的还多，你安心在这里陪我吧。”
<br/>　　芮冷玉无语，道：“我跟你孤男寡女地在这，你觉得合适吗？”
<br/>　　道风沉默了一会，道：“你是少阳妻子，有什么好怕？”
<br/>　　“正因为这样，才更要避嫌，你算是我大伯，我是你弟媳妇，哪有大伯跟弟媳妇天天一起生活的。”
<br/>　　这下轮到道风无语了，刚要说什么，芮冷玉说道：“算了，看你这么可怜的，我还是留下来陪你吧，谁让我也真的无路可去。”
<br/>　　随即望着道风笑了笑，“之前跟你开玩笑的，你我都成现在这样了，还有什么避嫌不避嫌的。”
<br/>　　道风不做声，他留下芮冷玉在这，自然还有别的原因，只是没法跟她明说。
<br/>　　芮冷玉走过来，将道风抓在手心里，手指按着他的脑袋，笑道：“道风，看你平时总是那么装酷，不可一世的。今天总算落在我手掌心了吧，以后我照顾你，但你要听我的，不然我就捏碎你。”说完手指在他脑袋上轻轻抚摸着逗他。
<br/>　　道风无语。“我反悔了，你还是走吧。”
<br/>　　芮冷玉捂嘴笑起来。
<br/>　　没过几个小时，林三生把杨宫梓带来了，一起来的还有秋莹……之前两人在死寂迷林里迷了路，等找到菩提神木，什么都没看到，怀着绝望茫然地回到风之谷，正六神无主，林三生就亲自来了，对杨宫梓来说，简直是天降福音，赶紧就跟着来了。
<br/>　　看到道风变成了那些小的一个人，杨宫梓悲从中来，双手捧着他哭了一阵，最后还是笑了，虽然这一切凶险万分，虽然有那么一段时间，她真的以为道风死了，那种悲痛……无法形容，而今再次又听到他的声音，大悲大喜，让她也彻悟了一些什么，却又说不上来。
<br/>　　她本想带道风一起走，或者留下来照顾他，道风跟她说了很多，为了让他放心，也只好接受他的安排了。
<br/>　　道风嘱咐了她一些事情，杨宫梓记下来，但又舍不得走。
<br/>　　秋莹看出苗头，拉着林三生和芮冷玉出去，给他们两口子腾出空间说话。
<br/>　　林三生正好要去检阅他那些弟子的情况，告诉两个姑娘，后山有个风景不错的地方，让她们可以去逛逛。
<br/>　　两人就顺着后山下去，在山谷中散步。
<br/>　　果然，山谷之中，到处长着不知名的野花，漫山遍野的，的确是姑娘们喜欢的地方，两人就慢慢地走到花丛里去，静静散步。
<br/>　　“你还未经人事吧。你怎么知道他们两口子要说情话。”芮冷玉逗她。
<br/>　　“看的出来啊。我虽然没经历过什么，但我以前是长在孽镜台边上的，每天看到那些来照镜子的人，什么样的都有，看得多了，对人类的感情，自然也懂一些了。”
<br/>　　秋莹笑着看芮冷玉，“姐姐，你真的好福气。”
<br/>　　“什么？”
<br/>　　芮冷玉一时没懂。
<br/>　　“我跟随老大时间不久，但暗中也观察了些日子的，他……我说不好那种感觉，他跟世人都不一样，是个……用过去的话说，是世间罕见的奇男子，真的，你跟他，倒真的是天造地设的一对。”
<br/>　　一句话勾起了芮冷玉许多心事，轻轻叹了口气，道：“什么天造地设，他是天选之子，我是转世鬼童，亦正亦邪，应该是被诅咒的才对。”
<br/>　　“姐姐，你可不要悲观，老大他也为你做了很多的，他也是不服命运的，只要你们努力争取，我相信你们肯定能够在一起的。”
<br/>　　芮冷玉冲她笑了笑，道了声谢。
<br/>　　“秋莹，你有什么打算？”
<br/>　　“我任务完成了，自然是要去古墓里找老大的，影魅已经进入古墓了，他们设了埋伏，要害老大，我之前跟踪影魅时，略知道一些，要赶紧进去提醒老大。”
<br/>　　见芮冷玉表情紧张起来，拉了她的手安慰道：“姐姐不要担心，军师他们也是要去帮老大的，肯定能够救他出来。”
<br/>　　约莫散步了半小时左右，隐贤观的一个弟子过来，表示观主（林三生）请她们回去，两人跟着回到道观，林三生已经在院子里等她们了。
<br/>　　林三生笑着对芮冷玉道：“你在这里照顾道风，估计要住一阵子，我倒是想请你帮个忙。”
<br/>　　“你看我收的这些弟子，都是这个世界的生灵，他们一直学习人间的礼仪和文化，但终归没有去过人间，以前……都是琳琳在这里教化他们，而今你在这，能够偶尔跟他们说说人间的事情，尤其是关于文化和修行方面的。”
<br/>　　芮冷玉一口答应，然后问他：“琳琳怎么了，好久没有看到她。”
<br/>　　林三生眼中露出一丝酸楚，把李琳琳失踪、自己搜查了好久都没能找到她的事说出来。芮冷玉听了震惊，也只能劝他慢慢寻找。
<br/>　　这时候杨宫梓从里面出来，十分不舍的模样，走到芮冷玉面前，拉着她的手说：“你也知道，我不方便长留下来，还要靠你照顾他，你我这关系，我也没什么要嘱咐的，跟不会对你道谢。”
<br/>　　芮冷玉笑道：“只要你别吃醋就好了。”
<br/>　　杨宫梓嗔了她一眼，道：“他现在那么小一个人，动都不能动，我吃什么醋！”
<br/>　　几个人依依不舍地聊了一会，林三生带着两个妹子离去，临走之前，告诉芮冷玉，东厢房的那个姑娘，不用去管她。
