　　吞天鼎乃是仙器，不但可以炼化万物，内部还有一方空间。
<br/>　　而且和乾坤镯不同，乾坤镯只能盛装死物，而无法将人和活物收入其中，吞天鼎却可以。
<br/>　　当然，前提是不能受到外界的攻击，否则内部空间也会不稳定。
<br/>　　药王宗的这个禁地之中，虽然有不少毒虫毒兽，不过都灵智不高，只要叶尘不主动招惹，应该不会招来它们的攻击。
<br/>　　然而，
<br/>　　就在叶尘准备放出吞天鼎，进入鼎中驱毒的时候。
<br/>　　“吼！！”
<br/>　　洞穴深处，竟然传来一道恐怖的咆哮之声！
<br/>　　整个洞穴都轰轰乱颤，周围的毒虫毒兽，更是被惊的乱窜，有的慌不择路，竟然朝叶尘扑了过来。
<br/>　　叶尘暗道一声麻烦，只好挥起紫琼剑，同时祭出三昧真火，开始奋力抵挡。
<br/>　　唰唰唰！
<br/>　　虽然这些毒虫毒兽，等级普遍不高，可是全都含有剧毒，而且数量众多，纵然叶尘的三昧真火，可以将他们轻松灭杀，可是却也引来了更多毒虫毒兽的报复性攻击。
<br/>　　更可怕的是，叶尘已经感应到，洞穴深处那道恐怖的气息，也正在朝他悄悄靠近！
<br/>　　“吼！”
<br/>　　下方猛然再次发出一道狂吼，周围的那些毒虫毒兽，顿时纷纷四散而逃。
<br/>　　继而，
<br/>　　轰！
<br/>　　一道庞大的黑影，从下方窜出，直奔叶尘扑来！
<br/>　　叶尘定睛一看，脸上顿时露出一抹浓浓的惊愕之色，
<br/>　　“竟然是号称万毒之王的五彩神蛤！”
<br/>　　要知道，哪怕是修真界，这五彩神蛤也是大为罕见，一般都生长在长满各种毒物的秘境绝地之中。
<br/>　　而且这五彩神蛤，不但剧毒无比，而且生性极为懒惰，很多五彩神蛤，一生都未必会走出自己的巢穴一次。
<br/>　　但是，它们的领地观念却极强，一旦有外敌闯入，就会变得极为暴虐！
<br/>　　“难怪之前药王宗的那些人，都不敢追进来，看来他们早知道五彩神蛤的存在！”
<br/>　　“而且眼前这头五彩神蛤，恐怕至少也有千年以上的功力，便是化神天君，若是被它的毒液所伤，恐怕也得玩完！”
<br/>　　在电光火石之间，叶尘已经瞬间想明白了其中缘由，同时整个人也没有闲着，直接施展出缩地成寸的神通，便避开了五彩神蛤的攻击。
<br/>　　与此同时，
<br/>　　嗖！
<br/>　　一道赤红色的毒液，自五彩神蛤的口中发出，犹如一道利箭，瞬间击中在一旁的岩壁之上。
<br/>　　滋滋滋~
<br/>　　一道白烟乍起，那岩壁周围的毒花毒草，瞬间便焦枯了一大片，连那岩壁之上，都被腐蚀出一个巨大的深坑！
<br/>　　“嘶~”
<br/>　　就连叶尘看到这一幕，也不由得倒吸了一口凉气，暗暗心惊不已。
<br/>　　这五彩毒哈的毒力之强，已经超乎了他想象，别说护体真元没用，恐怕就算他施展三昧真火，也不能完全抵挡！
<br/>　　“看来不能和这畜生近身打斗，只能远攻！”
<br/>　　一念及此，叶尘没有犹豫，反而一头朝洞穴深处扎去！
<br/>　　他现在还不能离开这个洞穴，退无可退，也只有孤注一掷了。
<br/>　　“吼！！”
<br/>　　那五彩神蛤看到叶尘，竟然朝它的老巢冲了过去，顿时愤怒的咆哮了一声，当即便也追了过去。
<br/>　　可惜，五彩神蛤并不以速度见长，和叶尘相比就更加差的太远。
<br/>　　一眨眼的功夫，叶尘便已经深入洞穴数千米！
<br/>　　而让叶尘感到奇怪的是，这洞穴越往深处去，毒草、毒虫反而越来越少！
<br/>　　而且，竟然开始出现了许多灵花灵草，而且全都是稀有之物！
<br/>　　“龙胆草！鼠儿果！四合寒香！络灵瑶草！……”
<br/>　　叶尘越看越奇，同时心头狂喜，
<br/>　　“是了！记得古籍中有记载，天下万物，相生相克，但凡剧毒之物，百米之内必有与之相克之物！”
<br/>　　“这里一半为毒药，另外一半却是灵药，当真让人叹为观止！”
<br/>　　暗暗感慨了片刻之后，叶尘忽然心头一横，
<br/>　　“既然来都来了，不如将这些灵花灵草全都给收了，待回去之后，说不定就能为曦月炼制提升神魂力量的灵丹妙药了！”
<br/>　　说干就干，叶尘当即飞扑到那些灵花灵草跟前，也不管它们是何品种，只要不是毒药，统统收入乾坤镯之中。
<br/>　　唰唰唰！
<br/>　　一转眼的功夫，叶尘便已经收了上百株灵花灵草。
<br/>　　而与此同时，后方的五彩神蛤也终于追了过来，看到叶尘的所作所为之后，顿时气的嘎嘎乱叫，狂吼着朝叶尘扑了过来。
<br/>　　叶尘也不和它硬拼，当即便继续朝下方狂奔，迅速拉开了和五彩神蛤的距离，同时也没有闲着，所过之处，又收入一大批灵花灵草。
<br/>　　那五彩神蛤气得肚子鼓起老高，可惜它的速度实在太慢，只能被叶尘不停的放风筝。
<br/>　　然而，好景不长，
<br/>　　就在叶尘正大肆掠夺的时候，怀里的苏蔓忽然醒了过来，
<br/>　　“嘤咛~~”
<br/>　　苏蔓口中发出一道诱人的呻吟之声，此时整个人的俏脸蛋，也娇艳欲滴，完美柔软的娇躯，也开始在叶尘的怀里，不停的扭动起来。
<br/>　　“小尘，我好热……好热……”
<br/>　　叶尘顿时叫苦不迭，
<br/>　　他之前打晕了苏蔓，可是并没有来得及为她解毒，如今再次醒来，已然毒性大发了！
<br/>　　倘若故技重施，再将她打晕过去，她的身体只怕就要吃不消了，经脉非得活活爆裂不可！
<br/>　　“这只臭蛤蟆一直紧追不舍，我根本无法替蔓姐驱毒，这该如何是好？”
<br/>　　叶尘思来想去，也没能想出一个好的办法，而此时毒性大发的苏蔓，已经开始撕扯自己的衣服，顿时春光乍泄~~
<br/>　　叶尘也更加心急如焚，
<br/>　　“实在不行的话，也只有速战速决，先将这只臭蛤蟆干掉，然后才能静下心为蔓姐解毒！”
<br/>　　一念及此，叶尘正打算转过身，主动迎战五彩神蛤。
<br/>　　然而就在这时，
<br/>　　远在后方的五彩神蛤，忽然大吼了一声，声音之中似乎透着一抹急切。
<br/>　　“嗯？这臭蛤蟆怎么突然急了？”
<br/>　　叶尘隐隐察觉到了什么，当即又深入了数百米，顿时便看到前方，竟然出现一道亮光！
<br/>　　“难道这洞穴的尽头，竟然还有出口？”
<br/>　　想到这里，叶尘顿时狂喜。
