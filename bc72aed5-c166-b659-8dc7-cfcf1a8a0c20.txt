        不知过了多久，裤袋里的手机突然振动，童筱颖本能地掏出手机来看了看。
<br/>
<br/>        是邻居夏凌焕发过来的信息，一是询问她宴席已开，她人去了哪里？二是告诉她，她的前夫傅夜沉就坐在他的对面。
<br/>
<br/>        夏凌焕还偷拍了傅夜沉的照片，特意发给了她看。
<br/>
<br/>        照片上，傅夜沉像是望着主席台那边，一袭剪裁合身的深灰色西服，丝质衬衫下隐约还可以看见起伏的胸肌，搭上他修剪得干净利落的短发、深邃的五官，怎么看都像荧幕上的男明星，而不是个集团总裁。
<br/>
<br/>        以前他的帅气在她眼里，好像从来没有这么惊艳过，现在只是看着他的照片，就足矣让她魂不守舍。
<br/>
<br/>        过了许久，她才回复夏凌焕的信息，告知他，她已经回家了。
<br/>
<br/>        但对于傅夜沉的事情，她只字未提，却默默地保存了傅夜沉的这张被夏凌焕偷拍的照片。
<br/>
<br/>        作为市长家的千金大小姐，李忆莎的婚礼上，来了许多政界大人物，这也是傅夜沉受爷爷之命，来参加李忆莎和夏凌焰的婚礼的原因。
<br/>
<br/>        婚礼不算豪华，但也不低调，还能受到这么多人的祝福，也算是美满。
<br/>
<br/>        礼成之后，夏凌焰带着李忆莎一桌一桌的敬酒，在来到傅夜沉这桌的时候，李忆莎还故意问：“夜沉，怎么不见你带女朋友来啊？”
<br/>
<br/>        “她身体不舒服。”傅夜沉淡然一笑。
<br/>
<br/>        夏凌焰却下意识地看了同桌的夏凌焕一眼。
<br/>
<br/>        夏凌焕推了推鼻梁上的眼镜，立马介绍道：“凌焰，他就是我爸的干儿子，傅夜沉。”
<br/>
<br/>        “也是我的前任未婚夫了！”李忆莎娇声道，一双凤眼，直勾勾地盯着傅夜沉。
<br/>
<br/>        夏凌焰则尴尬地笑了笑：“你们吃好喝好啊！”语毕，他赶紧搂着李忆莎去了下一桌。
<br/>
<br/>        夏凌焕挑眉看着傅夜沉，打趣道：“老兄，看不出，你的感情史还挺丰富啊！”
<br/>
<br/>        傅夜沉默不吭声。
<br/>
<br/>        夏凌焕见傅夜沉这不可一世的高冷范，还真想告诉他，他傅夜沉的前妻童筱颖就住自己家隔壁！
<br/>
<br/>        他倒是挺期待他傅夜沉知道这个消息后会是什么反应？
<br/>
<br/>        但是，他又不太想插手别人的事情，所以，这事，他忍着没说。
<br/>
<br/>        这明明是婚宴，没过一会儿，就成了政界大佬们之间的交流宴。
<br/>
<br/>        夏凌焕见傅夜沉身边时不时有人走过来敬酒攀谈，没机会跟他闲聊，只好自顾自地吃完饭，先行离开了。
<br/>
<br/>        怎么说，他夏凌焕好歹也在检察院里上班，是个检察员！
<br/>
<br/>        居然比不上傅夜沉这个商人！
<br/>
<br/>        算了，不计较了，就连他亲生父亲，都把傅夜沉看得比他更重要。
<br/>
<br/>        因为夏凌焰婚后和李忆莎去度蜜月了，所以，夏凌焰接回来的项目现在全由童筱颖一个人在做。
<br/>
<br/>        这些天，白天她忙着赶设计图，晚上她还得去夜校上唇语培训班，但忙碌并未让她忘记那个男人的存在。
<br/>
<br/>        手机里，夏凌焕上次在夏凌焰和李忆莎婚宴上偷拍的傅夜沉的照片，成了童筱颖唯一的心灵寄托。
<br/>
<br/>        而这一次，或许是机缘巧合，夏凌焰从傅氏集团那里接到了关于市体育馆的外形设计的邀请函。而正处于度蜜月的他，自然是把这次项目的交接工作吩咐给了童筱颖。
<br/>
<br/>        童筱颖带着邀请函来到傅氏集团大厦，虽然没被前台小姐冷落，但也遭受了前台小姐不少白眼。
<br/>
<br/>        殊不知，这都是落以柔的功劳。
<br/>
<br/>        原因很简单，在落以柔的暗自煽风点火下，就连前台小姐也认为，她童筱颖明明被傅大总裁给甩了，居然还不要脸地“变着法子”进傅氏集团大厦试图接近她们的傅大总裁。
<br/>
<br/>        “除了设计部你可以去以外，其他地方你最好不要到处乱跑，别怪我没提醒你，我们这儿的保安可是二十小时巡逻的。”前台小姐一边尖酸刻薄地说道，一边给童筱颖做来访人员登记，并给她在电梯里刷了卡放她去设计部的楼层。
<br/>
<br/>        童筱颖落落大方地微笑着，颔首向这前台小姐道了声：“谢谢。”
<br/>
<br/>        到了设计部的会议室，童筱颖才知道这次受邀的人不止她和夏凌焰这一个团队，此外还有好几家建筑设计公司和建筑设计工作室。
<br/>
<br/>        此次市体育馆由傅氏集团承建，因为原来的旧体育馆已经无法满足需求且外形不具特色，所以这次需要对体育馆进行有特色化的外形设计。
<br/>
<br/>        而傅氏集团旗下的设计部里的设计师灵感有限，于是部长才决定对外招标。
<br/>
<br/>        童筱颖坐在会议室里，一边认真地听设计要求，一边认真地做笔录。
<br/>
<br/>        却不知，会议室门外，有一双眼睛正定定地盯着她。
