　　　　校园群蛇出没事件在一个小时后，十几名身穿制服的警员来到了温城大学。
<br/>　　他们一边帮忙抓蛇，一边调查蛇为什么会集体光临这充满书卷学海的校园。
<br/>　　“白队，整个校园已经展开地毯式的搜索，如今已经捕抓到了987条蛇，无毒蛇有687条，有毒蛇300条，蛇品种一共七种，目前虽然绝大部分的蛇已经捕抓，但不少隐蔽的地方，应该还藏有不少蛇”
<br/>　　一位中年制服的男子冲着为首的一位同样身穿制服的冷艳女人报告。
<br/>　　被唤做白队的女人，一身制服将她丰满而曼妙的身材撑得凹凸有致，有着雪光萦绕的肌肤，她的神色带着几分冷清，美艳而精致，只见她柳眉微微皱起，看了一眼熟悉的校园，便道：“查出来这些蛇为什么会集体出现在校园内吗”
<br/>　　“这个目前没有头绪，都忙着抓蛇呢，而且我们是警察，这种案件应该要找个蛇类专家过来，恐怕才能知晓”
<br/>　　冷艳女子淡淡的说：“不用那么麻烦，我们只要保证今后这个校园不会在出现蛇就可以了”
<br/>　　“让人先将所有捕抓到的蛇抬到操场上去”
<br/>　　“白队准备如何做”
<br/>　　冷艳女人高挺的鼻梁显出凌厉的线条，微抿的薄唇似是透出寡情的信号，淡淡的道：“区区畜生，扰乱学堂，自然要全杀了，都杀光，我就不信还有蛇会敢出现在这校园内。”
<br/>　　她的话说出，冷淡透着杀意，顿时让十几名警察开始将捕来的蛇，或是别的学子捕的蛇全部汇聚在操场之内。
<br/>　　一会儿后，操场聚集了很多的人。
<br/>　　绿草坪处，涂小安这个三人组，捕到了满满一个大桶的蛇，这时，一位制服警员走了过来，开口：“辛苦你们了，这些蛇现在就交给我，拿去一起处理”
<br/>　　“处理？如何处理”
<br/>　　苏菲连忙询问这名警员。
<br/>　　他道：“我们队长说了，这些蛇不知所谓，扰乱学堂，等下一起全部火化”
<br/>　　“什么，你们要烧死这些蛇？”苏菲大吃一惊。
<br/>　　李清照呆若木鸡，而涂小安微微的蹙眉？
<br/>　　这校园中的蛇可不下千条，全部烧死，这杀戮也太重了，正常来说，应该是拿出去放生才对。
<br/>　　“走，我们快去看看”
<br/>　　三人立刻朝着操场的位置跑去。
<br/>　　操场上，用雄黄洒了一个大大的包围圈，那些被捕抓的蛇全部倒了出来，汇聚在包围圈内，一条条层层叠起，嘶嘶作响，画面让人毛骨悚然。
<br/>　　由于有雄黄粉，这些蛇只能惶恐的在固定的范围内游曳。
<br/>　　“我的妈呀，好可怕啊，我还从来没见过那么多条蛇呢”
<br/>　　操场的外围，早就围观了很多学子，一个个脸上带着惊悚，三三二二的议论起来。
<br/>　　“我听说来的警察要将这些蛇给全部烧死”
<br/>　　“不会吧，那么多蛇一起烧，也太残忍了点吧”
<br/>　　“我听说是杀鸡儆猴，那样校园内就在也不会在出现蛇了”
<br/>　　“这些蛇咬伤了不少同学，死光了才好，不然时不时冒出来一条，我们怎么上课，每天上课还要当心会不会被蛇咬”
<br/>　　此时足足有上千条蛇被困在雄黄的包围圈中，那位身穿制服而冷艳的警官淡淡的看着这些丑陋的蛇，满是厌恶感。
<br/>　　短短一会儿工夫，校园但凡被捕抓的蛇基本上都汇聚在操场上，冷艳女子下命令道：“洒上汽油，全烧了”
<br/>　　“不能烧，不能烧”
<br/>　　苏菲立刻冲了出来，拦住了要倒汽油的警察。
<br/>　　冷艳警官看了苏菲一眼，有点诧异，随后笑了起来：“呦，苏菲，我的老同学，好久不见”
<br/>　　“白楚艳，你疯了吗，那么多蛇，你怎么能都给烧死，它们也是一条一条生命呀”
<br/>　　苏菲身为生物学老师，是非常的尊敬动物的。
<br/>　　她跟白楚艳曾经都同是温城大学的学生，也是同班的同学，只是后来毕业了，一个当了警察，一个留在母校，当了老师。
<br/>　　“这些畜生，不懂苟且偷生，群体出来伤人，若不杀死它们，你说怎么做”白楚艳长而密的睫毛以一个完美的弧度向上翘起，看着苏菲质问。
<br/>　　苏菲双眸似水道：“自然是将它们拿出去放生”
<br/>　　“那放生了，它们再次回来呢”白楚艳淡淡的说。
<br/>　　苏菲道：“不会的，将它们送到十五公里外的城西山放生，它们就回不来了”
<br/>　　白楚艳摇了摇头：“你想过这些畜生为什么会集体出现在校园吗”
<br/>　　“我不知道”
<br/>　　“对，你不知道，我也不知道，但是它们上千条同时出现，绝对不是巧合，本警官没工夫去调查它们为什么会出现，只有杀鸡儆猴，以绝后患”
<br/>　　苏菲立刻大声道：“不行，你这是犯法的，我们国家有野生动物保护法，不能肆意的杀戮”
<br/>　　白楚艳高傲的露出轻蔑，：“我的老同学，你是当老师当傻了吗，这里的蛇哪一条在野生动物保护法中”
<br/>　　“那是指珍贵，濒危的陆生，水生野生动物和有重要生态，科学，社会价值的陆生野生动物，这里的蛇，都是低贱的品种，死的再多也何妨”
<br/>　　“白楚艳，既然能放生，为什么要妄造杀戮”苏菲气的浑身颤抖起来。
<br/>　　白楚艳淡淡的道：“我这是以绝后患，难不成你想校园变成蛇的巢穴不成”
<br/>　　“你这个女人怎么那么狠心呀，看你长的那么好看，居然是蛇蝎心肠”
<br/>　　李清照见苏老师说不过对方，出来帮腔起来。
<br/>　　“小丫头片子，你懂什么，你知道外面的野味馆一天要杀多少蛇吗，他们一个月，一年杀的蛇累计是多少吗，这上千条蛇，死了问题也就完满的解决了”
<br/>　　对于白楚艳来说，这就是一件小案子，只要将这些蛇处理了，在避免不发生这类似的事件就可以。
<br/>　　为什么一定要在校园操场上杀这些蛇，而不是拿出去。
<br/>　　因为这些蛇死在校园，那么校园满是死蛇的气息，以后自然不会有蛇敢在来。
<br/>　　蛇类对气息是很敏感的，白楚艳这招干脆利落，快刀斩乱麻。
<br/>　　李清照被说的哑口无言，半天说不出话来，气的脸色都绯红起来了。
