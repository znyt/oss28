　　　　“这一批圣者精血，可以给他们，老祖宗不是在凝炼这一件天级灵器吗？正好缺啥大批的丹药，这两百亿九鼎丹，正好可以送过去。”
<br/>　　虚堇侃侃而谈道。
<br/>　　“更何况，这批圣者精血，到了他们的手中，也不会给他们增添实力。”
<br/>　　“我们虚家最为拿手的功法，不正是炼血之术吗？”
<br/>　　虚堇的脸色，露出几分阴狠说道。
<br/>　　“我们可以将那批圣者精血中，融入我们虚家凝聚的魔血，如此一来，他们就算是将这圣者精血炼化了，但是魔血融入到了他们的血脉之中，我们也可以就此机会，控制了那三人。”
<br/>　　“这三人的实力不错，让他们作为我们虚家的奴隶，为我们虚家效力。”
<br/>　　“哦？这倒是不错，将魔血融入到那批圣者精血中，生死境之下的武者，根本不可能看破。”
<br/>　　虚家家族冷笑连连，他了一眼那华瑞，开口道。
<br/>　　“华瑞，你去将仓库里的圣者精血，取出来价值二百亿九鼎丹的，送给他们去，这批圣者精血中已经是融入了魔血，本来是给我们虚家弟子准备的，你去找出来一批送给他们去。”
<br/>　　“是。”
<br/>　　华瑞点了点头，便是直接出去，离开了。
<br/>　　同时另一边，拓跋鼎已经是在隔壁的房间中休息。
<br/>　　而楚风眠跟寒月漓，则是设下了重重禁制，面对而座，一股股的战龙血脉，被楚风眠引了出去，引入到了寒月漓的身躯之中去。
<br/>　　寒月漓身体里的玄寒之力，还没有彻底被引出，楚风眠便是一点一点的，从中缓缓引导出来。
<br/>　　若是一口气引导出来的数量太多，寒月漓还不一定可以掌控，必须要循序渐进才行。
<br/>　　独自击败的夺花尊者，令寒月漓对着玄寒之力的掌控，再进了一步，如今也可以引出更多的玄寒之力来了。
<br/>　　拥有先天神体，本身也就意味着，现在寒月漓的身上算是有着一个大宝库。
<br/>　　她不需要修炼，单单是将身体里的力量引出来彻底掌控，都足以将她的境界，支撑踏入到生死境。
<br/>　　这北邙学院，可不比武胜国度，秦皇国度那等小地方，天才齐聚，不增强实力，只怕是有着无数的人，可以将他们踩死。
<br/>　　不增强自己的实力，只怕就会成为他人鱼肉，任人宰割。
<br/>　　“战龙血脉！交融！引出！”
<br/>　　楚风眠的身上，远古战龙血脉的力量全开，全部涌入到了寒月漓的身躯之中去。
<br/>　　这一股股的远古战龙血脉，便是要将寒月漓身躯之中的玄寒之力，彻底的引出。
<br/>　　寒月漓身躯中的玄寒之力，还远远没有被楚风眠引出到极限。
<br/>　　在楚风眠远古战龙血脉的激发之下，便是有着更多的玄寒之力，被唤醒了出来。
<br/>　　阵阵寒气，就在二人的身上浮现出来，现在若是一尊寻常圣者，敢贸然进入到这房间之中的话，都会被这寒气直接冻死。
<br/>　　“玄寒之力！引！”
<br/>　　轰！
<br/>　　在寒月漓的身躯中，又是大片玄寒之力被唤醒出来，令她的力量，更是节节攀升起来，如今一步之间，便是一下子达到了神力境九重的地步。
<br/>　　这还远远不是寒月漓如今力量可以达到的极限，但是楚风眠却是硬生生的将其他的玄寒之力，给压制住了，不让寒月漓再次突破。
<br/>　　若是一旦踏入到神力境巅峰的话，便是就会随时引来生死大劫。
<br/>　　如今寒月漓的积累，还远远没有达到圣者的极限，生死大劫是提升力量最为重要的手段之一。
<br/>　　必须要将圣者所能够积攒的力量，达到极限，在一举引出生死大劫，踏入生死境，这才是最为完美的选择。
<br/>　　所以许许多多的天才，都是将实力选择压制到神力境九重的地步来，积攒。
<br/>　　只有压制不住的，才会选择踏入生死境巅峰的地步。
<br/>　　不过整个九域中，也是有着无数秘术可以遮掩生死大劫，但是也都太过麻烦，做起事来碍手碍脚的。
<br/>　　楚风眠如今将寒月漓的境界压制到神力境九重，便是最为完美的状态。
<br/>　　寒月漓自然也清楚这一点，她双眼睁开，眼睛苏醒过来，开口道。
<br/>　　“我的力量，又进步了一部分，比起对战夺花尊者的实力，要强大了数倍，在让我遇到她，我可以一招轰杀了他。”
<br/>　　寒月漓自信的说道。
<br/>　　这话中的意思，可不是在盲目自信，而全是实话。
<br/>　　现在的寒月漓，一身的力量，换算成力量指数，只怕是已经突破二十点。
<br/>　　单单境界上，便是比起曾经的夺花尊者要强大一倍，更何况寒月漓有着玄寒之体，真正的战斗力更是远超她的力量。
<br/>　　这等实力，加入到北邙学院，只怕一下子就会成为种子弟子，甚至是一尊冉冉升起的新星，种子弟子中的天才高手。
<br/>　　“我的实力，已经达到瓶颈，必须要在寻找其他奇遇，才可以突破，不过我的杀戮剑术，已经是快要踏入大成的，只要是杀戮剑术修炼到大成，我的实力也可以再进一步。”
<br/>　　楚风眠看着寒月漓，开口道。
<br/>　　“以你现在的实力，就算是加入到北邙学院之中，也不会是无名之辈了，至少也是一尊天才，有着名气地位，做起事来也会方便许多。”
<br/>　　楚风眠平静开口道。
<br/>　　北邙学院，便是竞争最为残酷的地方，在其中，一样是以实力为尊，没有实力，便是原罪，便是罪恶。
<br/>　　只有拥有实力，才可以让众人臣服，得到无数资源。
<br/>　　这一点楚风眠自然清楚，像是七大宗门中，也是这样的，一切都是按照实力办事。
<br/>　　实力越强，地位越高。
<br/>　　“楚公子在吗？”
<br/>　　门外突然传来了敲门声，正是那管家华瑞的声音。
<br/>　　如今两人修炼，倒是过了一下午的时间，如今已经渐入夜色。
<br/>　　“什么事？”
<br/>　　楚风眠开门，走出去，便是看到那华瑞。
<br/>　　在华瑞的手中，拿着一张红色请帖，看到楚风眠出来，便是主动迎了上去，将请帖递了过去。
<br/>　　“这是今日晚上的请帖，晚宴两个时辰后开启，还请各位尽快前来。”
<br/>　　“好。”
<br/>　　楚风眠点了点头，便是将请帖接了过来。
