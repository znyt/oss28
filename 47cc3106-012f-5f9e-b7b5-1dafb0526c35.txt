　　“呵呵，你是情报科的墨子羽科长吧？久违了！我叫黄浦，江湖朋友都喜欢叫我鬼面修罗，我们是奉了尊夫人之命前来带你出去的。”迎着黑子审视的目光，黄浦的露出一个十分诡异的笑容。
<br/>　　黑子在听到黄浦的话之后，愣了一下：“我夫人？我夫人在什么地方？你叫她出来。”
<br/>　　“嘿嘿，墨科长，我想你还没有搞清楚现在的状况，如果我是你的话，我会先担心自己，然后再想自己的老婆！”黄浦说完之后，对着已经拉开架式的几百手下一摆头，‘哗啦’一下，几百支钢弩全部对准了黑子。
<br/>　　如此情况黑子就算是再淡定，脸色也是微微一变，拧着眉毛说道：“你什么意思？不会是觉得抓住我就可以威胁玉凤吧？”
<br/>　　“是又怎么样，墨科长，如果你不想让你身边的人死光光，最好马上过来，否则老子不介意再让这大牢门前再添几缕冤魂。”
<br/>　　“你的目的是什么，玉凤似乎并不值得你如此大动干戈吧？”黑子丝毫不为其所动，看了看身边几人，淡淡说道。
<br/>　　“目的？老子的目的就是要报复。当年李世民杀兄弑弟，我家主上被其乱箭射死，现在我要让他的国家再次陷入战火，老子要让大唐所有对不起我家主上的人全都死！”黄浦得意的说道。
<br/>　　黑子的眼中闪过一丝怜悯，叹了口气说道：“可是你凭什么认为自己能够做到呢？这里是皇城，你搞出这么大的动静，想必现在外面已经被禁军包围了，你觉得你能走出这里？”
<br/>　　“为什么走不出去？老子手中有你，还有这个伪装成‘善德’的女人，李承乾如果不想让你们死，那就只能妥协。”
<br/>　　“如果陛下不妥协呢？”黑子反问道。
<br/>　　“那大不了就一起死好了，不过在你们死了之后，相信李承乾的其他属下会有兔死狐悲之感吧，这样的结局其实也不错，老子认为值了！”事情发展到这一步，黄浦已经没有了任何犹豫与担心，虽然明知道黑子在拖时间，但却没有任何着急的表现。
<br/>　　只是黄浦实在是对于自己现在的处境过于乐观了，千算万算之下他不应该在这个时候把自己的打算全都说出来。
<br/>　　因为他可以保证自己对李建成的忠诚，但却不能保证自己带来的手下全都是死士，那些跟着他一起来的亡命徒虽然不怕死，但却绝对不想成为某个人的牺牲品。所以在听完黄浦的话之后，很多人都将目光投向他的身上，眼中掩饰不出的露出一抹怨毒。
<br/>　　而就在这个时候，异变也突然发生，那被黄浦控制住的‘善德’在不声不响之下，突然抬起右手，肘部猛一用力，狠狠撞向他的胸口。
<br/>　　而与此同时，黄浦的反应也不慢，面对突如其来的攻击，他先是右手用力向前一推，将‘善德’推了出去，同时左手的刀子用力一划，打算将敢于反抗的‘善德’喉咙切断。
<br/>　　这一切说来话长，但整个过程却极为短暂，就在众人都以为‘善德’必死无疑的时候，却见她已经脱离了黄浦的控制，手中不知什么时候已经多了一支左轮手枪，人已经来到了那个李毅恒的身后，将枪顶在了他的头上。
<br/>　　“你，你怎么可能……”黄浦不可置信的看了看左手中微微有些卷刃的刀，又看了看冷笑望着自己的‘善德’。
<br/>　　‘善德’在这个时候声音已经变成了另外一个人，用以种嘲讽的语气说道：“其实你刚刚应该用弩对准我的头。”说罢，在众人错愕的目光中，解开了领口的那条丝带的搭扣，将其丢到一边的地上，发出‘铛’的一声。
<br/>　　那丝带里面竟然藏着一块薄薄的软钢？看着被丢到地上，露出一角金属色泽的‘坚硬’丝带，黄浦的脸色变的十分难看。
<br/>　　不过当他再看向‘善德’的时候，却也稍稍有些欣慰，刚刚他那一刀虽然被软钢所阻，但也不是没有取得任何成果，至少那个‘善德’的脖子上已经被留下了一条细小的划痕，丝丝血迹已经开始渗透出来。
<br/>　　而就在黄浦准备说些什么的时候，‘善德’已经拖着李毅恒开始向着黑子的方向退去，与此同时那位皇子殿下亦面色惨白的叫道：“黄浦，救，救我！”
<br/>　　救还是不救？黄浦皱了皱眉头，很快便已经有了决定，用一种信心十足的语气说道：“毅恒，你放心吧，他们不敢拿你怎么样，如果你真的死了，老子一定给你报仇。”
<br/>　　报仇？老子要的不是报仇，老子要的是皇位啊！从黄浦的回答中，李毅恒已经可以确定这家伙在穷途末路之下已经决定牺牲掉自己。
<br/>　　可是就算想要复仇也不是那么容易的，就在黄浦决定命令手下人放箭，杀死所有人离开的时候，宗正寺四周的屋脊上已经传来一阵稀里哗啦的声音。
<br/>　　而等到那几百人反应过来发生了什么时，无数枝黑洞洞的枪口已经指向了他们。
<br/>　　黑子的声音也在同一时间响起：“投降吧，没机会了！”
<br/>　　“投降？”黄浦抬头向四面扫了一眼，呵呵笑着说道：“老子这里有五百人，而你有多少？就这么一点人就想让黄某束手就擒，墨科长有些自以为是了吧？”
<br/>　　“垂死挣扎而已，有什么意义？你想死的话完全可以用刀抹脖子，何必拉着你的一群部下下水？”
<br/>　　“为主上复仇而已，他们死得其所，这是他们的荣幸！”
<br/>　　“疯子！黄浦，你就是个疯子，难道你还看不出来，现在已经没有退路了，就算是你能把他们都杀了又如何？最后还不是死路一条！”巨大的压力下李毅恒有些崩溃了，他清楚的知道，如果黄浦一声令下，第一个死的一定是自己。
<br/>　　“李毅恒，你是主上的儿子，主上已经去了，你一个人留在这个世界上也没有什么意义，所以不如为主上尽忠吧，咱们一起赴黄泉！”黄浦歇斯底里的吼着，然后抬起手，对着身后的那些犹豫不定的手下喊道：“所有人准备……。”
