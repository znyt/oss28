　　这几个月的时间，海明江几乎是一刻都没闲着，奔赴于苍穹大陆五大洲和四大海域。
<br/>　　他调动了天涯海阁十几万座分阁的力量，在整个苍穹世界，抓捕黄泉魔宗的余孽。
<br/>　　总计共抓住了六千余人，不过，六千余人之中，武王境以下的人，就占去了九成以上。
<br/>　　这些武王境以下的黄泉魔宗余孽，直接全部处死了。
<br/>　　而后，他便对剩下的二百余名武王境的武者，进行刑讯逼供，终于找到了会搜魂术之人，便是眼前之人。
<br/>　　“苏……苏莫，我告诉你搜魂术的修炼之法，只求你放我一命！”戚老望着苏莫，声音颤.抖的说道。
<br/>　　他黄泉圣宗已灭，宗门高层也都彻底隐匿了起来，他东躲西藏，却还是没有躲过苍穹神宫的搜捕。
<br/>　　现在他只求活命，不过想到当初自己追杀苏莫的一事，他的心沉到了谷底。
<br/>　　“我不杀你！”苏莫闻言点了点头。
<br/>　　“你要发誓，并且也不能让其他人杀我！不然就算是死，我也不会说出搜魂术的修炼之法！”戚老眼眸一转，忐忑的说道。
<br/>　　仅凭苏莫一句话，他可不会相信，现在，搜魂术是他唯一的保命条件了。
<br/>　　在没有得到确切的保证之前，他绝不能告诉苏莫搜魂术的修炼之法。
<br/>　　“你觉得你有资格和我谈条件？”苏莫眸光冰冷，直刺戚老的双眸。
<br/>　　“我……！”面对苏莫的目光，戚老全身如糠筛般颤.抖，一时之间居然吓的说不错话来。
<br/>　　“你没有资格谈条件！”
<br/>　　苏莫冷笑一声，继续道：“说出来方可活命，否则抽魂炼魄，魂飞魄散！”
<br/>　　戚老闻言，心念电转，随即他索性闭上了双眼，一副认杀认剐的模样。
<br/>　　戚老作为武王境武者，又是阴罗殿的护法自然不是简单的人物。
<br/>　　他知道，若是不能得到苏莫的保证，他只要说出搜魂术的修炼之法，必死无疑。
<br/>　　苏莫见此眼眸一冷，看来恐吓并不能起作用了。
<br/>　　“好吧！我发誓，绝不杀你，也不会让任何苍穹神宫的弟子杀你，如有违反，天诛地灭！”
<br/>　　少倾之后，苏莫发下了一个重誓，退让了一步。
<br/>　　言罢，他手臂一甩，一枚玉简飘飞到了戚老的额头之前。
<br/>　　戚老闻言，缓缓的睁开了双眼，他凝视了苏莫片刻，沉声道：“希望你记住你的誓言！”
<br/>　　戚老心中松了口气，此种誓言，基本上没有人敢违反，不然会受到天道的惩罚。
<br/>　　就算不会被天诛地灭，也会气运尽失，最终自食其果。
<br/>　　他相信，苏莫作为苍穹世界的霸主，应该不会为了他一个小人物，而违法誓言。
<br/>　　言罢，戚老意念涌动，开始在玉简中括印搜魂术的修炼之法。
<br/>　　几个呼吸之后，搜魂术括印完毕，苏莫立刻便收回了玉简。
<br/>　　随即，他便仔细查他玉简中的内容，的确是一种秘术，而且是精神类的秘术。
<br/>　　“放心吧！绝对不会有错！”
<br/>　　戚老面带忐忑的望着苏莫，道：“现在你可以放过我了吧！”
<br/>　　“不急，我要查探真伪！”苏莫摇了摇头，随即不待对方多言，他手臂一挥，直接将对方收进了虚界神图之中。
<br/>　　随后，苏莫挥退了海明江，便开始静心研究这门搜魂术。
<br/>　　搜魂术颇为玄妙，但修炼难度并不高，只是将精神力量，用一种特殊的方法，形成一种如阵法般的玄妙之阵，而后直接冲进他人的识海之中，窃取他人的记忆。
<br/>　　这种搜魂速，理论上来说，只能探查精神力低于自己之人。
<br/>　　若是探查精神力强于自己之人，不仅会无效，还会让自身精神受创。
<br/>　　而且，这种搜魂术不仅能探查别人的记忆，还能抹去别人的记忆。
<br/>　　不过，搜魂术也是相当的残忍一种秘术，会对被搜魂之人的精神，造成很大的摧残，甚至让被搜魂之人变成白痴。
<br/>　　这取决于施术者的精神力强度，以及搜魂术的造诣。
<br/>　　当然，若是施术者故意为之，那就是另外一说了！
<br/>　　研究了半晌之后，苏莫心中已经了然，便开始正式修炼了起来。
<br/>　　他的精神力强大，而且悟性极高，对于这种秘术，只有了三天时间，便修炼到了小成的地步。
<br/>　　习会了搜魂术之后，苏莫立刻便将屈覃国师和戚老两人，从虚界神图中放了出来。
<br/>　　“苏莫，可以放了我离开吧？”乍一出来，戚老立刻向苏莫喊道。
<br/>　　“待我试验一番再说！”苏莫摇了摇头，随即左手一张，立刻将屈覃国师抓到了身前。
<br/>　　而后，他的右手同样张开，一把按在了屈覃的脑袋上。
<br/>　　霎时之间，强大的精神意念，如同潮水一般，顺着他的手掌，直接冲进了屈覃的识海之中。
<br/>　　随即，屈覃的一生的记忆，如同走马观花一般，呈现在了苏莫的脑海之中。
<br/>　　大殿中寂静无声，苏莫静静的查探着屈覃的记忆。
<br/>　　一旁，戚老脸色难看，他很担心苏莫会反悔，但现在他深处苍穹神宫，根本不敢逃走，也没有实力逃走，只能静心等待。
<br/>　　时间缓缓流逝，约莫小半个时辰之后，苏莫将屈覃的记忆粗略的浏览了一遍，随即松开了手掌。
<br/>　　扑通！
<br/>　　苏莫手掌一松，屈覃整个人如同一滩烂泥一般，瘫坐在了地上，脸色惨白，双眼无神。
<br/>　　苏莫的面上留露出无奈之色，屈覃的确是尸罗殿之人，他隐藏在青俞岛，只是早年尸罗殿的一个任务。
<br/>　　对方也并不知道黄泉魔宗的一众高层，隐藏在何处。
<br/>　　苏莫心中暗叹一声，没有线索，就无法找到黄泉魔宗的一众高层。
<br/>　　即便他实力无敌，但偌大的苍穹世界，也是没有办法找到这些人。
<br/>　　“苏莫，你可以放……！”戚老见苏莫搜魂完毕，立刻开口了。
<br/>　　然后他话未说完，苏莫大手一张，直接便将他抓了过去。
<br/>　　“苏莫你……！”戚老心中大骇，以为对方要杀他，顿时吓的魂飞魄散。
<br/>　　不过，苏莫并不是要杀他，而是手掌直接按在了他的脑袋上，再次开始了搜魂。
<br/>　　而后，戚老一生的记忆画面，开始呈现在苏莫的脑海之中。
