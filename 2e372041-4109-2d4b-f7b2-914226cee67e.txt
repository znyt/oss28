<br/>    星空之中，黑色的潮水开始淹没一切，四大仙国已经收到了消息了。

<br/>        无冕者议会，正在进攻四大仙国！

<br/>        至高无冕并没有露面，主事的都是无眠者。

<br/>        在无字劫发生的时候，无冕者议会，毫不犹豫地发动了他们的攻势。

<br/>        这位大罗金仙怒骂道：“该死的！无冕者议会居然在这个时候进攻，防守的地仙可是都已经前往了神山，如此一来，怎么可能守得住！”
<br/>    
<br/>        “无冕者议会，就没有想过这样一来，谁去灭杀渡劫者吗？”
<br/>    
<br/>        在他的背后，一颗流星落下，从中滚出一个修行者，摔落在地面上，双眼无神，只是道：“我的小九死了，我的小九死了......”
<br/>    
<br/>        他不断地重复着这一句话，忽然听到这个大罗金仙的话，不屑地笑道：“既然他们敢动手，那就是确定渡劫者必死无疑啊，无冕者议会，向来不做选择，他们，全都要！”
<br/>    
<br/>        大罗金仙并没有注意到这个从流星之中落下的修行者，只是安排道：“所有人，天仙，金仙，全部出战，无论如何，我们都不能失去这一颗星辰！这颗星辰若是失守，之后的仙路，将一马平川！”
<br/>    
<br/>        这个大罗金仙终究是没有抵抗太久，每一颗星辰，都绽放出了剧烈的光芒，那不是星辰发威，而是星辰之上在燃起大火。
<br/>    
<br/>        无数的星辰被点燃，战火弥漫在整个星系。
<br/>    
<br/>        一队队黑袍的修行者，开始掌控每一颗星辰，将这些星辰归入无冕者议会的统治。
<br/>    
<br/>        零号的脚步落在仙路之上，周围的无免者都朝着他走来，禀报道：“我们已经拿下了这颗关键的星辰，只要冲过去，便可以一马平川！”
<br/>    
<br/>        “这一次，我们有机会拿下整个星天！”
<br/>    
<br/>        “无眠者阁下，还请下令。”
<br/>    
<br/>        零号的目光落在那个口中喃喃自语
<br/>    
<br/>        的修行者的身上，他伸手，示意所有人闭嘴，“不想死的，先撤到这颗星辰之中。”

<br/>        有无免者问道：“无眠者阁下，到底发生了什么事？”
<br/>    
<br/>        零号走到这个修行者的面前，“眼前的这位，应该就是天庭英灵，北斗星君吧？”
<br/>    
<br/>        躺在地面上的修行者坐了起来，看向零号，“我的太阳死了，那么，总要有人付出代价。”
<br/>    
<br/>        他站起身，双眼之中绽放神光，他的目光所在，一颗星辰不过刚刚被他看到，便已经化作漫天的碎片，无数的执刀人在这颗星辰上惨呼，身陨道消。
<br/>    
<br/>        而后，他看向整个天空，那天空中的星辰，都已经被零号手下的执刀人拿下。
<br/>    
<br/>        他的目光移回，看向零号，“这个代价，便由你来付！”
<br/>    
<br/>        他道完，浑身都是爆涌的杀意，而天空之上的星辰，一颗接着一颗爆裂开来，如同是绽放了无数的烟花。
<br/>    
<br/>        那些星辰之上，都是零号的执刀人。
<br/>    
<br/>        璀璨的烟花之下，众多无免者看向北斗星君，双眼之中都是惧怕之色，眼前这个丝毫不起眼的男人，竟然是天庭的英灵，北斗星君！
