　　此时，慕清霄盘坐于泳池旁，整个人却仿佛融入虚空，宛如消失一般，玄之又玄。
<br/>　　站在落地窗前，王语嫣与小依美眸一眨不眨的望着窗外的景象，眼中充满了惊喜之色。
<br/>　　陡然，以慕清霄为中心，一股磅礴浩荡的灵气，宛如狂涌的潮水一般，源源不断的自他体内弥漫而出。
<br/>　　“是突破了吗？”
<br/>　　望着眼中的恐怖气势，王语嫣与小依相视一眼，脸颊上浮现激动与喜悦。
<br/>　　半年，漫长的等待终于结束了。
<br/>　　在两女惊喜之余，慕清霄眉宇间的黑色神纹闪烁起来，伴随着神纹的闪烁，一缕黑色火焰冒了出来。
<br/>　　灵气源源不断的扩散而出，宛如灵气制造机，充斥着整个金帝庄园，其浓郁程度，堪比低级仙侠位面。
<br/>　　而且，有聚灵阵法的缘故，其中的灵气根本不会四溢，只会弥漫在庄园内部。
<br/>　　“啾——”
<br/>　　伴随着一声嘹亮的凤鸣，浩荡的灵气化作神鸟，仰天凤鸣，凤鸣被结界截下，但神鸟却贯彻云霄。
<br/>　　霎时间，上空的空间一阵扭曲，积云仿佛被某种东西贯穿，最后四散开来。
<br/>　　旋即，拨开云雾见青天，温暖的阳光投射而下，洒落在庄园消瘦的身影上。
<br/>　　片刻后，慕清霄的身形自虚空中浮现，瞬间，一股浩荡磅礴的气息笼罩着整个静海市，眨眼间又消失不见。
<br/>　　“好恐怖的气息，不知是何等境界，青云门掌门与他相比，宛如萤火与皓月一般。”
<br/>　　来到落地窗前，陆雪琪望着院落中的慕清霄，美眸中满满的爱慕与自豪。
<br/>　　不知不觉，他的丈夫已经成长到了这种程度。
<br/>　　感受到他身上那股浩瀚气息的强悍程度，王语嫣与小依也是捂嘴惊呼。
<br/>　　待气息收敛，慕清霄渐渐睁开星眸，其中光泽流转，最终吐出一口浊气，脸上挂上沐浴春风一般的笑容。
<br/>　　站起身来，伸了个懒腰，顿时响起一阵噼里啪啦的声音。
<br/>　　“呼——”
<br/>　　感受到体内那股恐怖的力量，一道气息顺着喉中吐出，顿时游泳池中的池水就翻滚起来。
<br/>　　“好强，这就是合体期的力量，当真有移山填海般的力量。”
<br/>　　慕清霄伸出手，微微发力，在虚空中轻轻一握，面前的空间，直接扭曲成一个怪异的凹陷口。
<br/>　　轻轻一握，仿佛要将虚空都给捏碎。
<br/>　　体内浩荡磅礴的灵气，爆炸性的力量，不由得令他有些迷醉，这股力量，具备令人疯狂的诱惑。
<br/>　　合体期修士，现在的他，总算是颇具实力。
<br/>　　感受着举手投足间，便足以扭曲空间的浩瀚力量，慕清霄俊美的脸庞上，顿时浮现一抹笑容。
<br/>　　合体期在诸天万界的修士中，也绝对是相当高的层次。
<br/>　　要知道，在古剑奇谭位面，天墉城的长老，不过也就练虚期的实力。
<br/>　　当然，掌门和执剑长老例外。
<br/>　　而现在，如若遇到练虚期修士，慕清霄挥手间便可抹杀，根本用不上神火，更不需要化作凤凰！
<br/>　　这完全是境界上的差距，练虚期与合体期完全不是一个层次的存在，后者足以碾压前者。
<br/>　　星眸中光泽流转，旋即眉宇间的神纹消失不见，慕清霄浑身散发出的恐怖气势也收敛起来。
<br/>　　转过身来，两道身影携带着香风就迎面扑了过来。
<br/>　　慕清霄眼中尽是温柔之色，轻轻张开怀抱，便将王语嫣与小依搂入怀中。
<br/>　　陆雪琪则站在一旁，虽不善于表达，但美眸却温柔的望着他，完全是妻子看待丈夫的目光。
<br/>　　温存片刻，三人就回到别墅中。
<br/>　　得知距离自己闭关已经过去半年的时间，慕清霄着实一惊。
<br/>　　在他的感觉中，突破不过是一瞬间，当真是修炼无甲子，全身心侵入其中，完全感觉不到时间的流逝。
<br/>　　坐在沙发上，抱着怀中两位佳人，嗅着幽幽体香，慕清霄的手也渐渐不老实起来，慢慢在两具玉体上游走。
<br/>　　“有没有想我？”
<br/>　　“嗯...”
<br/>　　感受到玉体上游走的炙热手掌，王语嫣与小依微微点头，相视一眼，完美的脸蛋上绯红如霞，皆是羞不可耐，任由慕清霄施为。
<br/>　　片刻后，慕清霄就搂着美眸迷离，衣衫凌乱的王语嫣与小依，快步向卧室中走去。
<br/>　　很快，卧室中就响起一阵高低起伏，令人面红耳赤的婉转呻吟。
<br/>　　...
<br/>　　...
<br/>　　冬去春来，春风带走白茫茫一片，青草钻出土壤，树木长出嫩芽。
<br/>　　金帝庄园内，浓郁的灵气充斥着每一个角落，鲜花绽放，绿意盎然。
<br/>　　道路两旁，竹林中，凉亭四周，花圃中都种满了花草，其中含有低阶到高阶各类灵药。
<br/>　　若是放在修真界，如此多的灵药，绝对能令无数修士眼红，甚至引来一些图谋不轨之人。
<br/>　　仰躺在太阳椅上，沐浴着阳光，慕清霄满脸惬意。
<br/>　　近几个月来，他一直陪着自己的女人，将烦恼全都抛之脑后，尽情享受了几个月的温馨生活。
<br/>　　“休息也休息够了，也是时候前往下个位面了，只是......”
<br/>　　想到地球上可能会突然冒出来其他修士，慕清霄目光一凝。
<br/>　　当下，双手结印，旋即一股无形的能量，将整个庄园都笼罩在内。
<br/>　　这是修为到了合道期，便能掌控的空间结界，释放后能封锁周围的空间。
<br/>　　庄园中虽有幻术，但其中灵气太过浓郁，若有有非常手段，或者路过此处，肯定会被察觉。
<br/>　　其中的灵药，可都是慕清霄幸幸苦苦在各大位面中收集而来，若是给别人做了嫁衣，哭都没地方哭。
<br/>　　当然，在设下空间结界，并在王语嫣她们身上留下一道太虚剑印后，结果就完全不同了。
<br/>　　这种开锁式结界，是仿照天墉城妙法堂的第四层结界而设。
<br/>　　唯有身上拥有太虚剑印，或者本身能凝聚太虚剑印，方能进入庄园中。
<br/>　　如果没有太虚剑印，想要强行闯入其中，后果只有两种。
<br/>　　第一种是被永恒万花筒中的幻术‘月读’反噬，灵魂永远停留在月读空间中。
<br/>　　第二种则比较轻松，直接被九霄琉璃焱焚烧成灰烬。
<br/>　　在王语嫣她们身上设下太虚剑印后，她们便能自由出入庄园，而且不会受到空间结界的反弹。
