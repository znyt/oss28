　　随着紫霄、雷火神王、天尘宇三人开口，所有人的目光，均是紧紧的落在龙腾的身上。
<br/>　　关于屠霄的身份，龙族之人也有猜测，毕竟，关于秦陨的传闻，消息也传到了龙族。
<br/>　　就算是毫不相干之人，心中也难免有些猜测。
<br/>　　吞虚龙帝面无表情，目光平淡，并没有什么表示。
<br/>　　龙腾目光环视紫霄、天尘宇和雷火神王，面色平静，道：“三位，不好意思，屠霄正在闭关，不能出来相见，抱歉了！”
<br/>　　他自然不会主动让苏莫出现，若是能免于让苏莫露面，那就最好不过了。
<br/>　　紫霄三人闻言，并没有什么意外，他们知晓这个龙腾和苏莫关系莫逆，肯定知晓苏莫和他们之间的关系，不会轻易让苏莫露面。
<br/>　　“龙腾小友，这一次祖龙大典，我们来了这么多势力，大家都想要一睹屠霄的风采，你不会这个面子都不给吧？”一位一身赤色长袍的中年前者说道。
<br/>　　此人，便是万化神殿的强者，名外赤云神王。
<br/>　　赤云神王，当初也曾因为麾下天骄古剑山的死，前往终极神王殿寻找苏莫。
<br/>　　“不错，既然屠霄在龙都城，不防出来一见！”
<br/>　　“闭关而已，可以随时出关吧！”
<br/>　　“我等都想一睹屠霄的风采！”
<br/>　　不少强者纷纷开口，他们倒是和苏莫没有仇，一则是真的想见识一下苏莫的不凡，二是，也想要看看，苏莫怎样对抗天王宫和终极神王殿等势力？龙族会不会帮助苏莫？
<br/>　　他们不确定，这个屠霄是不是名震神域的秦陨，但是，看到天尘宇和紫霄等人如此，他们也差不多能确定了。
<br/>　　龙腾闻言，面色阴沉了下来，看来这些人不见到苏莫，是不会罢休了。
<br/>　　地藏神王倒是没有开口，而是目光看向了吞虚龙帝，唯有吞虚龙帝方能做主。
<br/>　　果然，就在此刻，吞虚龙帝开口了，道：“龙腾，屠霄是我龙族的天才，既然大家想要一见，那就让他出关吧！”
<br/>　　龙腾闻言，眉头微不可查的一皱，心中暗怒，龙帝这分明是在打压他。
<br/>　　秦陨的身份消息传出，龙帝肯定能猜到屠霄就是秦陨，现在对方还让屠霄出来，不是故意为之吗？
<br/>　　不过，既然龙帝开口了，他也不好再拒绝，不过，他有足够的底气，能保住苏莫。
<br/>　　“既然各位要见屠霄，那我就让他出关吧！”龙腾环视众人，朗声开口道。
<br/>　　言罢，龙腾手掌一翻，取出一枚传讯符，意念一动，向苏莫传去了消息。
<br/>　　此时此刻，天尘宇、紫霄、地藏神王等人，均是眼眸微眯了下来，屠霄到底是不是秦陨，很快就能见分晓了。
<br/>　　姜枫然、战乱和闻人天都三人，稍稍有些紧张，今日对苏莫极为不利啊。
<br/>　　人王宫的玄天神王，端坐在看台上，面色稍显凝重，他一直未开口，因为他知道，自己就算开口了，也阻拦不住什么。
<br/>　　……
<br/>　　密室之中，苏莫睁开了眼眸，手中拿着一枚传讯符。
<br/>　　他已经收到了龙腾传来的消息，也知道了祖龙大典之上的情况。
<br/>　　“该来的始终会来！”苏莫微微叹了口气，他并不惧怕，此次若是能安全度过，那以后再也不用东躲西藏了。
<br/>　　随即，他长身而起，大步走出了密室。
<br/>　　离开了苍穹祖龙府之后，苏莫身形飞天而起，一步百丈，大步向祖龙大典所在的广场而去。
<br/>　　行走在高空之中，他已经看到了广场，也看到了无数的龙族之人，以及各大势力的强者。
<br/>　　与此同时，他亦是感受到了强烈的杀机，锁定在了自己的身上。
<br/>　　观礼台之上，几乎就在苏莫出现的那一刻，天尘宇、紫霄、地藏神王等人的目光，便齐齐的落在了苏莫的身上。
<br/>　　天尘宇和地藏神王两人，眸中闪过犀利的精光，他们两人虽然没有见过苏莫，但是，现在也能立刻肯定，这屠霄就是轮回之身。
<br/>　　因为，他们对苏莫的各种情况，都非常的了解，凭他们的眼力，一眼就能看出这屠霄九成九是轮回之身。
<br/>　　虽然，他们感知不到这苏莫体内的吞噬元神，但是，综合气息、修为等各方面的情况，他们基本能确定，这个屠霄就是轮回之身。
<br/>　　天尘宇和地藏神王，互相对视一眼，他们都有出手的冲动，但是却不得不生生的按捺住。
<br/>　　因为，屠霄和龙族的关系有多深，他们还不确定，现在就出手的话，搞不好会殒命于此。
<br/>　　而紫霄，目光中的杀机极为浓郁，苏莫就是脱掉一层皮，他也能立刻认出来。
<br/>　　若是说仇恨，他是所有人之中，最痛恨苏莫之人，恨不得立刻灭杀苏莫，方能消心头之恨。
<br/>　　不过，他也是和天尘宇、地藏神王一样，在不清楚情况之前，不敢肆意的出手。
<br/>　　无论是天王宫还是地王宫，亦或者是终极神王殿，在龙族面前，均是不堪一击。
<br/>　　玄天神王则是面露警惕之色，感知死死的锁定着天尘宇和地藏神王，非常担心两人会突然出手。
<br/>　　他已经事先知道了苏莫在龙都城，也给父亲人德宫主传讯了，但现在，他必须尽量保护苏莫。
<br/>　　眼见苏莫大步走来，龙腾环视全场，道：“诸位，这就是屠霄，也是我的兄弟！”
<br/>　　“兄弟？”众人闻言，纷纷诧异。
<br/>　　而天尘宇和紫霄等人，闻听此言，顿时眉头一皱，心中涌现了不好的预感。
<br/>　　“不错，我和屠霄生死与共，早已结拜为兄弟，有福同享有难同当！”龙腾朗声说道，他要尽量拉近和苏莫的关系，才能保住苏莫。
<br/>　　一瞬间，天尘宇和紫霄、地藏神王、雷火神王等人，均是脸色一沉。
<br/>　　有福同享有难同当？
<br/>　　对方这是在告诉他们，想要对付屠霄，就是和对方为敌吗？
<br/>　　他们完全没有想到，这龙腾居然会有这么一出。
<br/>　　一众龙族之人，均是微微有些诧异，显然对于龙腾的话很是意外，不过，转念一想，也都释然了。
<br/>　　龙腾和屠霄的关系，定然极为深厚，不然，屠霄也不会将祖龙精血给予龙腾了。
<br/>　　当然，有些龙族之人，对于屠霄的身份也有猜测，猜测可能就是外界流传的秦陨，只不过他们无法确认，因为屠霄一直没有现身。
<br/>　　吞虚龙帝面无表情，眸中闪过淡淡的精光，不知心中所想。
