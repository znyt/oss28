<br/>　　陈博士等人都是面色有些不好看，一来是饿的，二来也是吓的。
<br/>　　“陈博士，你放心吧，上面交代下来要我保护好你们这些科学家，我王大牛就是死，就是粉身碎骨，也会保护你们的。”
<br/>　　王队长斩钉截铁道。
<br/>　　“咯咯。”
<br/>　　而就在这个时候，一道就好像是夜猫子叫的声音再次响起。
<br/>　　听到这个声音，他们每个人都是忍不住身体震了一下，他们知道，那个变异的六阶丧尸又出现了。
<br/>　　……

<br/>　　周华拿出一副汉市地图道。
<br/>　　“还真是前进一步就要留下许多尸体啊。”
<br/>　　林峰感慨道。

<br/>　　而且医院永远都不缺病人，自然丧尸不在少数，一个个穿着白大褂的丧尸，一个个穿着病人服的丧尸，全部在医院楼道内走着，空气之中有一股古怪的味道，这是属于消毒水外加腐烂尸体混合的味道。
<br/>　　“你们分开行动！”
<br/>　　“是！”
<br/>　　三百人全部分散开来，开始在医院内四处摸索。
<br/>　　林峰和周华也分开行动，林峰直接前往住院部。

<br/>　　吼吼。
<br/>　　几个护士丧尸转过头露出了满是血肉的烂脸。
<br/>　　刷刷刷。
<br/>　　下一刻。
<br/>　　她们的头颅全部都被斩下。
<br/>　　林峰一刀三杀，连晶石都懒得挖，此时此刻救陈博士要紧，毕竟他们已经付出了这么多代价，可不能到头来任务失败了。
<br/>　　林峰一路狂冲，所过之处，所有丧尸全部被无情斩杀，开阔的医院走廊之中，全都是尸体。
