　　吴刚等人都看的怔住了，面面相觑一眼，他们遇到三阶的丧尸，往往都是要利用密集的子弹把对方给射烂，或者运气好忽然爆头了。
<br/>　　可从来没有过如此轻松。
<br/>　　村子里的几百个丧尸，在林峰的屠杀，和吴刚等人的强大火力下，不过一刻钟就被清理殆尽。
<br/>　　留下了一地的尸体。
<br/>　　林峰挖走那两个三阶的，开始打量起这个小村子，小村子没有什么特色，典型的乡下农村。
<br/>　　吴刚等人一边收拾战利品一边也是打量。
<br/>　　“地毯式搜索！”
<br/>　　吴刚大手一挥立即他的小队队员都散开，一间间屋子开始搜索。
<br/>　　“这个小山村看起来也没有什么不一样的地方。”吴刚端着枪四处扫了眼。
<br/>　　“就是不知道为什么一共二三十人的村子会有这么多丧尸在这里。”林峰道。
<br/>　　“你的意思是，他们被什么吸引过来的？”吴刚反应很快。
<br/>　　砰。
<br/>　　也就在这个时候。
<br/>　　一个进屋内去搜索情况的战士直接破墙而出，落地时，肠子都留在了外面，死的不能再死。
<br/>　　吼吼吼
<br/>　　旋即一个似人非人的玩意从屋内跑了出来。
<br/>　　这玩意四只脚的地上爬，浑身都是凸出来的尖刺，一双眼睛之中充满了嗜血的光芒，散发着一股股彪悍的气息。
<br/>　　“什么鬼玩意。”
<br/>　　“大强！”
<br/>　　众人立即面色大变。
<br/>　　“六阶的！”
<br/>　　“队长，这是六阶的怪物！”
<br/>　　猎豹小队队员个个都面色大变。
<br/>　　吴刚也是面色难看，“妈的，居然在这个小山村里遇到了一个六阶的丧尸。还真是走了狗屎运。”
<br/>　　一行人立即子弹上膛紧张无比的瞄准了这个怪物。
<br/>　　而林峰也是目光微微眯起，他还是第一次遇到这么强的丧尸。
<br/>　　这个六阶丧尸冲出屋子后，没有急着对林峰等人进攻，而是在原地来回踱步，就好像在审视林峰等人。
<br/>　　最后他刷的一下就是一跃而起。
<br/>　　“开火！”
<br/>　　几乎在一瞬间，吴刚等人是扣动了扳机。
<br/>　　哒哒哒哒哒。
<br/>　　强大的步枪子弹射了出去形成了密集的弹幕直接笼罩住了这个六阶的丧尸。
<br/>　　然而这子弹落在这丧尸身上不过是打退了他前冲的姿势，并没有对他造成多大的伤害。
<br/>　　“这丧尸皮这么厚？”
<br/>　　吴刚等人都震惊了。
<br/>　　“丢手雷！”
<br/>　　吴刚等人立即丢出了高爆手雷。
<br/>　　轰轰轰。
<br/>　　顿时烟尘四起，火光四射。
<br/>　　“炸死了没？”
<br/>　　一个猎豹小队队员紧张道。
<br/>　　刷。
<br/>　　下一刻一个黑影冲了出来直接落在了他头上。
<br/>　　只是一瞬间，就咬断了他的脑袋，无头尸体直挺挺的倒在地上喷着鲜血。
<br/>　　吴刚等人都眼中带着惊愕。这丧尸再厉害怎么可能无视子弹和手榴弹，这太匪夷所思了，他们遇到的丧尸也不少了，还是第一次遇到这种丧尸。
<br/>　　“你们都退下，这个丧尸不简单，估计是变异的，他有一定的智慧，你们负责掩护，我来主攻。”
<br/>　　林峰大喊道，说完林峰目光就落在了这个六阶丧尸身上，而六阶丧尸也看着林峰，龇牙咧嘴，就好像在嘲笑林峰一样。
<br/>　　“一个丧尸还敢装逼，看我劈死你。”
<br/>　　林峰二话不说掏出天罡符。
<br/>　　噼里啪啦。
<br/>　　顿时一道雷光乍现水桶粗细的雷电稳稳的落在了这个丧尸身上。
<br/>　　砰。
<br/>　　瞬间丧尸被击中，他脚下的地面直接被炸出个大坑。
<br/>　　吼吼吼。
<br/>　　这一击，立即就把这个丧尸劈的是皮开肉绽，鲜血直流。看的吴刚等人瞠目结舌。
<br/>　　“这什么玩意。”
<br/>　　“我去，这，好像是符箓啊。”
<br/>　　“难道他是异能者？？”
<br/>　　一群人都大感不可思议，莫名其妙的就来了一道雷，然后就把那个子弹都射不穿的六阶丧尸给劈伤了。这很显然是异能者才可以做到的。
<br/>　　“看来这个林峰的确是异能者了，藏得可真够深的，哪怕是军部之中，也没有多少异能者。”
<br/>　　吴刚眼中异光闪烁，甚至还有些羡慕。
<br/>　　六阶丧尸怒吼连连，居然主动的朝后退去，一副十分畏惧的模样。
<br/>　　“这个丧尸看样子好像十分惧怕天罡符，这天罡符应该只会一些邪魔外道比如说千年僵尸这样的存在会有暴击的效果，这丧尸怎么会如此惧怕这个玩意？？就好像老鼠遇到猫一样？”
<br/>　　林峰目光微微眯起，他拿这个符箓出来也只是想攻击一下罢了，没想到这攻击效果太显著了。
<br/>　　再看这个丧尸，越看越不对劲，他还是第一次看到丧尸身上长出这种尖刺搞得像是刺猬一样，寻常的丧尸等级就算高，那也是个人形，顶多块头大点，无非就是力量速度上更加出色罢了。
<br/>　　“莫非这玩意不是丧尸？”
<br/>　　林峰目光微微眯起。
<br/>　　砰砰砰砰。
<br/>　　也就在此时。
<br/>　　数道黑影全部从屋子里冲了出来，这些身影全都是身上长满了尖刺的尸体。
<br/>　　实力从三阶到五阶不等，足足有十几只。
<br/>　　这些丧尸全部把林峰包围起来。
<br/>　　一副虎视眈眈的样子。
<br/>　　“小子，能够击伤我的尸傀，你莫非是异能者？”
<br/>　　也就在这时，一个老头忽然出现在一个屋顶上，眯着眼看着林峰。
<br/>　　这个老头穿着一身的黑衣，看上去邪气森森。
<br/>　　“异能者？！”
<br/>　　林峰瞬间心中浮现出这几个字，难道说，这些被叫做尸魁的东西，实际上都是这家伙弄出来的嘛？
<br/>　　“说，你到底是何人！！居然敢跑闯入我的炼尸之地，还杀了我这么多试验品，若是不说出个一二来，杀无赦！！！”黑袍老者忽然一道低喝，眼中杀机毕露。
<br/>　　“我是战门成员！”
<br/>　　林峰心思一转，先借战门的名号用一用。
<br/>　　“战门的？”
<br/>　　黑袍老者眉头微微皱起，“原来是战门成员，怎么，你们这群人还和军部的人合作了？？”
<br/>　　“不过，就算你是战门的，毁了我的试验品，那也要给我跪下磕头，也许我可以既往不咎！！”
<br/>　　老头身上一股无形的气势爆发出来。
<br/>　　赫然是七阶的高手。而且还是个异能者，算是林峰现在遇到的为数不多的异能者之中，已经颇有气候的存在，异能者，非常的稀少，当然也不代表他绝对比黑铁战士要厉害，凡事都相对而言，但是眼下这个黑袍老头，他虽然等级七阶，但是他的战斗力肯定不属于八阶的黑铁战士。
<br/>　　“跪下磕头？你就不怕我战门找你麻烦？”
<br/>　　林峰冷笑道。
<br/>　　“哈哈，战门又如何？老夫七阶异能者，你战门我何须惧哉？更何况，这里就你一个区区战门的毛头小子，还敢用战门来压我？既然如此，就给你点颜色看看！！当真以为你战门可以横行无忌了不成？？”
<br/>　　老头阴冷一笑，瞬间那几头尸魁全部是冲了过来。
