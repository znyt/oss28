　　“呵呵，龙腾，屠霄要败了！”
<br/>　　见到迷宫中的情景，龙洌轻笑出声，心中大定了起来。
<br/>　　本来他还有点担心，怕冰儿和凝雪不是苏莫的对方，但是见到苏莫被冰封，他便彻底的放心了。
<br/>　　不仅是龙洌，其他很多人见此，都认为冰儿和凝雪，应该要胜了。
<br/>　　“哦，这冰封之力这么强？”龙腾不以为然的问道。
<br/>　　“当然，冰儿是玄冰族，冰封之力本来就强，再加上凝雪的辅佐，冰封之力非常的可怕！”
<br/>　　龙洌笑着点了点头，继续道：“不瞒你说，同阶之中，就算是我被冰封住，都有落败的可能！”
<br/>　　“是吗？”龙腾闻言讶然，若是如此的话，那这冰封之力倒的确是不一般。
<br/>　　毕竟，龙洌也是万古天骄，虽然其只是紫血天龙，但是有洪荒祖龙的血统。
<br/>　　不过，龙腾丝毫不担心，就算是再强，又能对苏莫有何用。
<br/>　　“你不信……？”
<br/>　　龙洌见龙腾的面色，知道对方不相信，他正欲再言，声音确实戛然而止。
<br/>　　“这……怎么可能？”龙洌望着迷宫城，面带不可置信之色。
<br/>　　只见迷宫城之中，出口之处，那冰封住苏莫的厚厚坚冰，突然迸裂开来。
<br/>　　轰！
<br/>　　一声爆响，冰屑四溅，坚固无比的坚冰，如同纸片一般，直接炸裂开来。
<br/>　　“这是……！”
<br/>　　冰儿和凝雪两女见此，顿时心中一惊，随即纷纷后退数十丈。
<br/>　　她们完全没有想到，苏莫能如此快速的破开坚冰。
<br/>　　“过来吧！”苏莫手掌再次探出，向两女直接抓了过去。
<br/>　　破开封住自己的坚冰，对于他来说，轻而易举，只是体内五行源力一震便做到了。
<br/>　　这一次，大手的威力更加强大了，苏莫五行源力都动用了。
<br/>　　不过，五行源力他虽然都动用了，但是并没有融合成混沌之力，只是处在半融合的状态。
<br/>　　但即便如此，威力也是奇大无比，这还是他没有动用血脉之力和元神之力的情况之下。
<br/>　　轰隆隆！
<br/>　　大手碾碎长空，速度如电，如同死亡之手，向两女一抓而去。
<br/>　　“快走！”
<br/>　　冰儿向凝雪喊了一句，便急速飞逃，她看出来了，这大手的威力无与伦比，根本无法对抗。
<br/>　　嗖！
<br/>　　凝雪也没有任何的犹豫，转身就逃，不过，两人的速度有限，根本不可能快过源力大手。
<br/>　　几乎是眨眼之间，源力大手便裹挟滔天威势，临近了两人。
<br/>　　“不好！”
<br/>　　凝雪心中大惊，手中出现了一柄雪亮的利剑，转身挥剑，雪花闪动。
<br/>　　这一剑如梦似幻，仿佛让人置身在白雪的世界之中，无尽的寒气肆虐。
<br/>　　冰儿也同样出手了，双掌抬起，掌中寒冰源力吞吐，形成了坚硬的冰柱，企图挡住源力大手。
<br/>　　轰隆隆！
<br/>　　无论是凝雪的攻击，还是冰儿的攻击，都无济于事，根本挡不住源力大手。
<br/>　　凝雪被大手一把擒住了，不过，冰儿倒是运气好，由于冰柱被大手碾碎，强大的反震之力，顿时将其震飞了出去，飞到了通道的尽头。
<br/>　　通道的尽头，有通向两侧的岔道，冰儿身形一闪，便急速逃离。
<br/>　　“居然逃了！”苏莫见此，微微愕然，没想到这个冰儿居然能在他手下逃走。
<br/>　　不过，他也没有在意，逃走也是无用，对方终归是要出来吧。
<br/>　　“放开我！”凝雪被源力大手擒住，也没有挣扎，而是脸色冰冷的喝道。
<br/>　　她知晓，自己挣扎也不用，眼前之人太过强大了。
<br/>　　“你被淘汰了！”
<br/>　　苏莫收取了对方的储物戒，便将对方从出口处丢了出去，不过，他并未抢走对方手中的长剑。
<br/>　　他知道此女是龙洌的奴仆，不想太过得罪，免得日后再生事端。
<br/>　　迷宫城之外，龙洌见到此幕，脸色阴沉如水，虽然心中愤怒，但是他也无法发火。
<br/>　　毕竟，抢夺宝物，这本就是迷宫城的规则。
<br/>　　他怎么也没有想到，这屠霄会这般的强大，简直不可思议。
<br/>　　龙城和龙烨等人，亦是个个面色，这屠霄的实力，未免强大的有些过分了。
<br/>　　看这种情况，若是单打独斗的话，貌似只有龙双双的奴仆，与此人有一战之力了。
<br/>　　“厉害啊！”
<br/>　　“这个奴仆太强了！”
<br/>　　“我感觉他根本没有动用全力，深不可测啊！”
<br/>　　全场议论纷纷，无数的人为之震惊，更有不少人来到龙腾身边，询问苏莫的具体情况。
<br/>　　有些人，甚至想要购买苏莫，想让苏莫成为他们的奴仆。
<br/>　　对此，龙腾自然是一一拒绝，至于苏莫的来历，他也是轻描淡写的糊弄了过去。
<br/>　　所谓奴仆强大，主人也沾光，苏莫的强大，让得很多人对于龙腾羡慕不已。
<br/>　　“龙腾，此奴不简单啊！”
<br/>　　高空之上，那两名中年执事中的白袍中年人，目光望向龙腾，声音洪亮的说道。
<br/>　　“确实不简单！”龙腾笑着说道。
<br/>　　“此奴并未动用元神之力，就有如此战力，修炼的道统非常不简单，应该是混沌神朝的武学道统！”白袍中年人说道。
<br/>　　龙腾闻言，顿时心中一凛，苏莫都没有动用混沌源力，这位执事居然能看出苏莫修炼的是混沌神朝的道统。
<br/>　　不过，其他人听到执事之言，却是纷纷心中一震。
<br/>　　“怎么可能？他没有动用元神之力就这么强大？”
<br/>　　“这不可能吧！若是这样的话，那他岂不是有万古天骄的战斗力！”
<br/>　　“这种天赋甘心为奴吗？”
<br/>　　无数的龙族之人，纷纷震惊，他们在迷宫城之外，虽然能看到其内的情景，但是却是感知不到里面之人有没有动用元神之力。
<br/>　　“凝雪，刚才屠霄与你和冰儿交手，有没有动用元神之力？”
<br/>　　这时，被淘汰出来的凝雪，来到了众人的身前，龙洌极忙问道。
<br/>　　“没有！”凝雪要了摇头，刚才那人用的五行源力，只是单纯的修为之力。
<br/>　　“这……！”
<br/>　　众人闻言，顿时沉默不语，没有动用元神之力，就有如此的战力，那动用元神之力还得了？
<br/>　　万古天骄！
<br/>　　这个屠霄的天赋实力，居然达到了万古天骄的级别，一个小小的奴仆，居然有万古天骄的天赋实力。
<br/>　　“不错不错，此子若是能忠于我族，值得培养！”白袍中年人微微颌首，面带淡淡的笑意。
<br/>　　万古天骄可是极为难得，龙族如此强盛，也才二十余位万古天骄而已。
<br/>　　如此天才人物，若是能为龙族所用，自然是再好不过了。
<br/>　　龙腾叹了口气，眉头微不可查的一皱，心中出现了一丝忧虑，目前来说，这事情若是闹大了，绝对不是好事。
