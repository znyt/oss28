　　　　商城物品都看了一遍后，涂小安退出了系统。
<br/>　　什么都没买，他的信仰点数有限，就别浪费了，先累积到一百万在说。
<br/>　　刚退出系统，涂小安似有所感，蛇眸浑然一眯，一款手机凭空出现，还在响着铃声。
<br/>　　这是有电话打进来了。
<br/>　　显示的是美琳姐。
<br/>　　蛇尾朝着接听的按键一点，一条蛇开始接电话，都是那么的牛掰。
<br/>　　“喂，美琳姐吗”
<br/>　　电话那头传来妩媚的声音：“小安，我刚才看到一段新闻，非常的震惊，想必你会有兴趣知道”
<br/>　　“我现在将视频传给你，你看看”
<br/>　　美琳姐火急火燎，简单的说了几句，电话就挂了。
<br/>　　叮...
<br/>　　手机上出现了一段新闻视频。
<br/>　　一个职业优雅的女性出现。
<br/>　　温城新闻报道：
<br/>　　“昨日在市中心的山野集团出现了一件骇人听闻的事情，一条足足有二十米的惊天巨蟒从野集团的大厦冲了出来，仰天咆哮，凶残无比，顿时间现场陷入了一阵恐惶，一下子照成了数百人的伤亡”
<br/>　　“更为惊奇的是，巨蟒的身上还骑着一位浑身黑袍的神秘男性，身份不详，特征不详”
<br/>　　请看现场传来的视频报道。
<br/>　　视频中，一条二十米长的巨蟒浑身红火鳞片妖艳似火，横冲直撞，就好像是科幻片里面大蟒蛇，闯入了都市，陷入一阵腥风血雨。
<br/>　　好在这条巨蟒虽然凶煞无比，但无意停留，从山野集团大厦冲出来之后，就直接朝着五公里外的深山老林游曳而去。
<br/>　　可就算如此，一路上，也死伤无数，不知道吓傻了多少人，市中心的人集体陷入了一阵惶恐之中。
<br/>　　人类都是有惧蛇症的，更别说那么庞大的一条巨蟒。
<br/>　　看了都让人开始怀疑人生。
<br/>　　“据有关部门的调查，这条巨蟒是山野集团培育出来的实验品，应控制不住，照成巨蟒的逃窜”
<br/>　　“现如今，山野集团已经被有关部门全面进入调查，而这条巨蟒也将会成为抓捕的对象。”
<br/>　　这段视频很快结束，悬浮的手机骤然消失，涂小安看完之后陷入了惊讶的沉思。
<br/>　　随后，他便呢喃自语：“没想到变异吸血蛇居然也从山野集团逃了出来，二十米，这畜生成长的还真够快的”
<br/>　　视频中那条凶煞无比的巨蟒，涂小安一眼就认了出来，就是变异吸血蛇。
<br/>　　被注射了巨蟒基因，让这条蛇的血脉再次提升，潜力无限。
<br/>　　震惊之后，涂小安抓到了一个关键点，吸血蛇背上的人类是谁，莫非是蛇博士？
<br/>　　蛇博士怎么可能带着吸血蛇冲出山野集团，而且这动静也太大了吧。
<br/>　　可谓震惊全世界呀。
<br/>　　难道是山野集团内部发生了什么变故，才让吸血蛇那么肆无忌惮的逃走。
<br/>　　涂小安心中出现了很多的谜团，不过他忽然冷笑起来。
<br/>　　这吸血蛇逃出来也好，自己正好可以找它报仇。
<br/>　　拥有了雷电之力的涂小安，有强大的底气，这次将彻底的收拾了这条蛇。
<br/>　　况且已经见过蛇精的三十米体型，现在吸血蛇成为二十米大蟒，对于涂小安来说，并不算什么冲击力了。
<br/>　　只是涂小安想找吸血蛇，也不是那么容易，它窜进不知名的深山老林，鬼知道现在的位置在哪。
<br/>　　另外还是那个问题，它背上的人是谁，是他控制了吸血蛇吗。
<br/>　　隐隐间，涂小安从黑袍的背影下，确定应该不是蛇博士，而且他完全没必要那么做。
<br/>　　不是蛇博士，当今世上，还有谁能控制的了吸血蛇。
<br/>　　看来近日自己要多关注关注新闻才是，一有这条蛇的消息，就要动手。
<br/>　　也好在吸血蛇没有凶性大发，要是在都市展开一段杀戮，那么不知道有多少人要遭殃。
<br/>　　涂小安想着想着，灵光一现。
<br/>　　吸血蛇逃了出来，那么是不是很有可能会回到佘山，毕竟它出生在佘山的地界上。
<br/>　　这里算是它的家。
<br/>　　可要是回到佘山，那么对于佘山的捕蛇人来说，就是一场浩劫了。
<br/>　　小的吸血蛇都对付不了，更别说现在成为二十米的巨蟒。
<br/>　　思来想去，涂小安觉得自己应该先做点准备工作才行。
<br/>　　只要吸血蛇在佘山出现，那么他就要第一时间出手，解决掉它。
<br/>　　“电蜂...”
<br/>　　涂小安的蛇眸浑然一眯。
<br/>　　霎时间，一营人马，五百只电蜂密密麻麻的笼罩在了金银蛇的上空。
<br/>　　“去吧”
<br/>　　看了一眼，淡淡一语，咻...五百只电蜂朝着佘山分布而去。
<br/>　　只要吸血蛇回到佘山，二十米如此之大的体型，是绝对逃不过电蜂的眼睛。
<br/>　　当初要不是蛇博士插手，这吸血蛇早就被涂小安拿下给吞了，现在这蛇今非昔比，再次对付起来，恐怕会棘手很多。
<br/>　　这也算是吸血蛇的机遇吧。
<br/>　　将电蜂派出去之后，涂小安要做的只能是等待，等待吸血蛇再次出现的消息。
<br/>　　接下来，无事可做的他，最后选择从商城之中购买了两颗强化果实，毕竟才只需要一千信仰点数，无关轻重。
<br/>　　家人的健康对于他来说，是最重要的。
<br/>　　其实老妈跟老姐现在的身体状况都非常的好，想生病都难，不过既然这强化果也不贵，买两颗就全当给她们在补补身体。
<br/>　　系统扣除了二千信仰点数，涂小安的面前悬浮了两颗白色的小果实，跟苹果差不多大小，无色无味，一点果实的果香都没有。
<br/>　　“让老姐先试试在说”
<br/>　　涂小安游曳到老姐的房间，涂小月正在看古装剧，老姐的身体好了，她也无事可做了，家里也不缺钱，到是成了一个闲人。
<br/>　　“姐，吃颗水果”
<br/>　　一颗强化果飘到了涂小月的面前，她用手接住之后，看了看，闻了闻，好奇道：“这是什么果实啊，怎么没见过”
<br/>　　“好东西，你吃了就知道”
<br/>　　小月白了一眼：“神神秘秘，不要以为这样，我就不计较你昨天又消失了一天”
<br/>　　涂小安也是无语，老姐从小到大都喜欢管着他，现在他都变成了一条蛇，就更不放心了，生怕自己会那一天被捕蛇人抓了去。
<br/>　　一句话说完，这丫头也是没心没肺，拿着强化果就直接放进了嘴里，跟吃苹果一样的一口咬下去。
<br/>　　一口吃着，二口吃着，她嫌弃道：“这什么果实啊，一点味道都没有”
<br/>　　“姐，全吃完”
<br/>　　涂小月翻了翻白眼，只能无趣的将强化果都吃进去，表情非常的郁闷。
<br/>　　第一次吃到一点味道都没有的果实，不酸也不甜。
<br/>　　涂小安这时留意老姐的样子，道：“姐，有没有感觉身体有什么不一样”
<br/>　　嗯？
<br/>　　涂小月蹙眉，像是在感受身体的变化，旋即，她脸色大变：“臭小子，你给我吃的是泻药吗，我要上厕所”
<br/>　　咻...
<br/>　　只见涂小月飞速的冲进了洗手间里，半天不见出来。
<br/>　　涂小安乐了，这是效果来了呀，想必强化果的能力会将老姐的身体好好的全方面改善，达到巅峰，气力大涨。
<br/>　　上厕所也正常，排除毒素而已。
<br/>　　系统商城的东西，效果自然不用太担心，而涂小安思量的是要不要接下来给老姐弄颗觉醒果。
<br/>　　犹豫了半天，还是算了，吃了觉醒果，就变成人类觉醒者，还是让她当个普通人吧。
<br/>　　要知道平淡才是福，能力越大，责任也就越大。
<br/>　　他也不想老姐接触那些未知的领域。
