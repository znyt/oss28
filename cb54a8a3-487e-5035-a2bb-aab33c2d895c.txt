　　“放了她们！”八字胡中年人，眸光扫视了一眼苏莫和秦不死手中的巫娴等人，冷漠的开口了。
<br/>　　其声音不大，但听在苏莫和秦不死的耳中，却无异于九天惊雷轰然炸响。
<br/>　　噗！噗！
<br/>　　苏莫和秦不死身形剧震，如遭雷击，纷纷张口喷出一口鲜血。
<br/>　　他们只感觉到一股无形的力量，直接作用在了自己的身上，这股力量之浩瀚，如亿万里江河、如千万里山川，根本无法阻挡。
<br/>　　“什么？”苏莫心中大惊，骇然不已，尊者居然如此恐怖！
<br/>　　“放我们离开，不然他们都要死！”秦不死脸色有些惨白，扬了扬手中的两名人质，大声的吼道。
<br/>　　苏莫亦是紧了紧手中的人质，将巫娴和另外一名瘦弱青年，捏的双眼翻白。
<br/>　　牢牢地控制着手中的人质，他才能有底气对抗尊者。
<br/>　　“梵尊者、尹尊者，救我！”
<br/>　　“尊者大人救我们！”
<br/>　　“我不想死！不想死啊！”
<br/>　　十几名被作为人质的巫族天才，纷纷大呼了起来，个个面带惊恐之色。
<br/>　　“给你们三息时间，放了他们！否则身死魂灭！”
<br/>　　八字胡中年人眼眸微眯，眸中寒光闪动，冷冷的喝道，其身上气息陡然大增，如同狂涛骇浪，形成无形的威压，排山倒海一般，向苏莫两人身上碾去。
<br/>　　若不是顾忌巫娴的安危，他早已出手，将眼前的两人直接镇杀，哪里还会啰嗦！
<br/>　　强大的威压，如同擎天大岳，又如同无形的枷锁，压迫在苏莫和秦不死的身上，让两人全身直冒冷汗，体内骨骼都吱吱作响。
<br/>　　在苏莫两人的眼中，八字胡中年人身形伟岸无比，气势霸绝天地，仿若天地间的主宰，其眼眸开阖间爆射的精光，让人不敢直视。
<br/>　　但是，即便对方再强，苏莫也要面对。
<br/>　　此刻，苏莫咬了咬牙，心中一发狠，左手中劲力一吐，手中的那名瘦弱青年顿时惨叫一声，身躯立刻炸开，四分五裂，变成了一团血雾。
<br/>　　其他人质见此，顿时吓得个个面色惨白，就算是巫娴，眸中都露出了一丝惊恐之色。
<br/>　　“我们死，他们都要陪葬！”苏莫双眸赤红，怒声大喝道，他杀死一名人质，就是要让对方看到他的决心。
<br/>　　八字胡中年人见此，先是微微一怔，随即顿时大怒，一个蝼蚁般的土著，居然敢在他面前残杀他的族人，简直是找死。
<br/>　　“自寻死路！”八字胡中年人厉喝一声，立刻向前踏出一步。
<br/>　　轰！
<br/>　　一步踏出，霎时之间，犹如天降神雷，虚空一阵剧震，苏莫和秦不死如遭重击，再次口吐鲜血，身形忍不住向后暴退。
<br/>　　而秦不死周身气劲被瞬间震散，气劲裹挟的十名巫族天才，直接被震飞了出去，逃脱了秦不死的掌控。
<br/>　　“还有两息！”八字胡中年人再次踏出一步，瞬间便临近了苏莫两人万丈之内。
<br/>　　“停下，不然我杀了他！”苏莫见此立刻怒吼一声，将手中的巫娴向前一扬。
<br/>　　他就算受伤，手掌依旧牢牢的扣着巫娴的脖颈，不敢有半分松动。
<br/>　　他更不能让对方靠近，不然自己必死无疑！
<br/>　　八字胡中年人见此，顿时停下了身形，其眉头微微皱起，眸中寒光闪动。
<br/>　　这是他最忌惮的事情，巫娴决不能有事。
<br/>　　其他人或许可以死，但巫娴决不能死！
<br/>　　巫娴的祖爷爷，可是支族的大祭司，乃是支族二号人物，一旦巫娴有事，他们谁都活不了！
<br/>　　八字胡中年人本以为，刚才可以瞬间将巫娴从苏莫手中震开，哪怕是让苏莫的手掌，有十分之一个眨眼时间的松动，他都会瞬间出手。
<br/>　　但是，他发现自己小觑了面前的这个土著，对方的手掌犹如扎根了一般，牢牢的固定在巫娴的脖颈上。
<br/>　　他不敢轻举妄动，万一伤及巫娴，那就完了！
<br/>　　八字胡中年人心中杀机汹涌的同时，也是有些费解，小小的苍穹世界，怎么会出现此种人物？
<br/>　　这简直不可思议！
<br/>　　见八字胡中年人忌惮，苏莫和秦不死对视一眼，两人身形开始缓缓后退。
<br/>　　他们必须尽快逃离，时间拖的越久，逃走的希望越是渺茫。
<br/>　　唰！
<br/>　　但是，苏莫和秦不死刚刚后退千丈，后方空间一阵波动，那名老妪突兀的出现在了两人的后方。
<br/>　　苏莫和秦不死立刻停下了身形，全身紧绷，他们可不敢小看这名老妪，此人的实力同样不是他们可以抗衡的存在。
<br/>　　“放了她们三人，老夫让你离开！”八字胡中年人冷漠的说道，仿佛是向苏莫妥协了一般。
<br/>　　现在还有三名人质，苏莫手中控制着巫娴，秦不死的手中控制了两名巫族天才。
<br/>　　“是吗？”苏莫眼眸微眯，他可不相信对方的话，手中没有了人质他们必死无疑。
<br/>　　他已经看出来了，对方很重视巫娴，绝不敢轻举妄动。
<br/>　　同时，苏莫的精神高度集中，不敢有丝毫的松懈，预防中年人和老妪突然出手。
<br/>　　“老夫巫梵，可以用尊者的名义，向你们保证！”八字胡中年人淡淡的说道。
<br/>　　“老身巫尹，也向你们保证，只要你们放了她们三人，就绝不出手为难你二人！”那名老妪也开口了，声音幽幽，充满了沧桑与冷漠。
<br/>　　“土……土著，尊者的话……绝对不会反悔！”被苏莫抓在手中的巫娴，脸色涨的发紫，断断续续的说道。
<br/>　　苏莫闻言心中冷笑，他可不是三岁的孩童，这么好骗！
<br/>　　尊者的话就不会反悔？他宁可信世上有鬼，也不信巫族之人的嘴！
<br/>　　秦不死望向苏莫，微微摇头，表示不能相信对方的话语。
<br/>　　他还真担心苏莫受骗，毕竟，苏莫实力虽然，但还很年轻，较容易上当。
<br/>　　苏莫微微点头，随即向秦不死靠拢，两人背靠着背，严防巫梵和巫尹。
<br/>　　一时间，现场有些僵持了下来，巫梵和巫尹两名尊者，既不敢轻举妄动，也不愿放苏莫两人离开。
<br/>　　苏莫心念电转，额头大汗连连，心中急思脱身之策。
<br/>　　不过，面对巫族尊者，好像一切的手段都没有什么用！
