　　逍遥剑圣见大祭司巫仑一脸淡漠的模样，也不想再与对方交谈，招呼一众三神宗的弟子，从天空中落了下来，落在了一处矮丘上。
<br/>　　各大势力的人，扫视三神宗的人群，最终目光都落在了一名锦衣青年的身上。
<br/>　　这名锦衣青年，站在逍遥剑圣的身侧，身形挺拔，面容俊逸，气质卓越，即便与逍遥剑圣一比，都不逊色多少。
<br/>　　“姜神夜，听说你修炼出了问题，还以为你不会来了！”一名黑鳞族的青年朗声喝道。
<br/>　　这名锦衣青年，正是三神宗超级妖孽，姜神夜。
<br/>　　姜神夜闻言，瞥了一眼黑鳞族青年，淡笑道：“小问题，无伤大雅！”
<br/>　　众人闻言有些疑惑，虽然姜神夜说的简单，但众人看其脸色略有苍白，显然是刚刚度过大难的模样，根本不想对方所说的那般轻松。
<br/>　　“那就好，五年不见，希望你别让我失望！”那名黑鳞族青年朗声说道。
<br/>　　“呵呵！黑锋，想要追上我，你还需继续努力！”姜神夜笑道。
<br/>　　这名和姜神夜对话的黑鳞族青年，名为黑锋，真圣榜排名第四，算是整个古灵星河武帝境天才之中，最强大的几人之一。
<br/>　　“这一次在真圣界，我就会打败你！”黑锋冷然的说道。
<br/>　　“我等着！”
<br/>　　姜神夜微微一笑，毫不在乎，随即他目光转动，扫视四方，朗声问道：“不知神武学府苏莫是哪一位？是否已经来此？”
<br/>　　听闻姜神夜问起苏莫，很多人的目光都落在了苏莫的身上。
<br/>　　众人对于姜神夜要找苏莫，没有什么意外，因为正常来说，这个真圣榜第一，应该是属于姜神夜的。
<br/>　　顺着众人的目光，姜神夜看到了苏莫，仔细打量了起来。
<br/>　　苏莫同样也在望着姜神夜，他发现无论是姜神夜、还是黑锋、巫殇、还是飞长云，全部都是武帝境九重的修为。
<br/>　　相比于这些人，他的修为的确是太低了。
<br/>　　“你就是苏莫？”姜神夜凝声问道。
<br/>　　“正是！”苏莫点了点头，大方的承认了。
<br/>　　“或许你的天赋真的很恐怖，希望你能证明自己吧！”姜神夜沉声说道。
<br/>　　苏莫闻言双眉微挑，他抢了对方的真圣榜第一，对方居然对他没有敌意，不像其他人那样，想要打败他，从而证明自己。
<br/>　　此人的心性倒是不错，心胸豁达。
<br/>　　“会的！”苏莫淡淡的点了点头，此时，什么真圣碑留名？什么真圣之气，他都已经不在乎了。
<br/>　　他期盼的，只是能顺利的离开此地。
<br/>　　“嗯！”姜神夜点了点头，便不再理会苏莫。
<br/>　　时间缓缓流逝，在无数人的等待之中，真圣界的开启已经进入了倒计时。
<br/>　　嘭！
<br/>　　就在此时，天空一声闷响，虚空突然炸开，炸出一个方圆千丈的黑洞。
<br/>　　而后，众人便看到一位庞大的巨人，从黑洞中走了出来。
<br/>　　此人是一个光头大汉，身高约有五丈，身躯健壮无比，仿佛是一座小山一般，非常惊人。
<br/>　　其全身肌肉虬扎，仿佛是一块块坚硬的岩石，充满爆炸性的力量，青筋如一条条蛟龙，盘扎与全身肌肉之中。
<br/>　　“好强的巨灵族！”苏莫心惊不已，他虽然看不出这位巨人的修为，但能感受到其身体中蕴含的力量，绝对惊天动地。
<br/>　　都说巨灵族之人，是古灵星河单体实力最强大的种族，看来果然名不虚传。
<br/>　　这个种族应该是走的炼体一道，肉身极度强悍。
<br/>　　“是巨灵族巨昙尊者！”有人惊呼出声。
<br/>　　轰隆隆！
<br/>　　巨昙尊者出现以后，身躯笔直的降落而下，直接落在了大地之上，如同陨石砸落大地，立刻让得大地一声巨响，崩裂出无数的裂缝。
<br/>　　“没想到逍遥剑圣和巫仑大祭司居然来了，幸会！”
<br/>　　巨昙尊者目光扫视四方，见到逍遥剑圣和巫仑大祭司，顿时抱拳说道，其声音浑厚无比，如同闷雷炸响。
<br/>　　“嗯！”逍遥剑圣和巫仑大祭司均是点了点头，并没有与对方多言。
<br/>　　武圣境的武者和武尊境的武者，根本就不是一个层次的人，他们自然不会过多寒嘘。
<br/>　　随即，巨昙尊者一挥庞大的手臂，人影闪烁，他的身旁顿时出现了数百人。
<br/>　　这数百人都是巨灵族人，虽然不像巨昙尊者那般身高五丈，也均是有四丈左右。
<br/>　　“哈哈哈！谁是苏莫？来了吗？”
<br/>　　数百名巨灵族的天才咋一出现，其中一人便狂笑一声，朗声喝道，神态张狂不已。
<br/>　　此人身高四丈出头，体型和巨昙尊者相差无几，皮肤黝黑，只不过并不是光头，而是留着一头毛碎短发，每一根头发都如同钢针一般。
<br/>　　“是巨摩！”
<br/>　　“好强大！”
<br/>　　“一来就要找苏莫，这是要直接发难了吗？”
<br/>　　众人见到此人，暗暗心惊，这位可是真圣榜排名第二的天才巨摩，战力滔天的存在。
<br/>　　巨摩的名声极为响亮，远超飞长云、黑锋等人，因其就是一个战斗狂人，非常的喜欢挑战各族的天才。
<br/>　　没有人是其对手，巨摩的战斗力已然逆天，一拳便可轰杀一切同阶。
<br/>　　苏莫闻听巨摩之言，脸皮一抽，心中几乎吐血。
<br/>　　尼玛！
<br/>　　每一个来到这里的顶级天才，几乎都要找他，每一个顶级天才，都将他当成了对手，这真的是让人吐血的事情。
<br/>　　而这，完全就是因为他是真圣榜第一！
<br/>　　“我就是！”苏莫淡淡的说道，声音虽然不大，但却直入巨摩的耳中。
<br/>　　“你就是？”巨摩铜铃般的眼眸转动，立刻便落在了苏莫的身上。
<br/>　　稍稍打量了一番，巨摩大嘴一咧，轻蔑的说道：“来吧！看看你这个真圣榜第一，在我一拳之下能不能侥幸不死！”
<br/>　　巨摩的言语极为狂妄，言下之意，苏莫根本就挡不住他一拳。
<br/>　　众人闻言无语，这个巨摩居然如此的急躁，现在就要和苏莫一战。
<br/>　　“没兴趣！”苏莫淡淡的摇了摇头，他可没有兴趣与对方做无谓的战斗。
<br/>　　“哼！那就真圣界内，再看看你有多少斤两！”巨摩冷哼一声，他也没有强势出手，毕竟神武学府也有很多尊者境的导师带队。
<br/>　　嗡嗡~~~
<br/>　　就在巨摩话音一落之时，众人所在的这片空间，虚空突然距离的震颤了起来。
<br/>　　前方那座耸立的万丈巨峰上空，空间开始了急速的塌陷。
<br/>　　真圣界，终于要开启了！
