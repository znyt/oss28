　　第535章
<br/>　　黄家终究成了厮杀的战场，兵与匪狠狠撞在一起，在黑夜里乱砍乱杀。
<br/>　　张猛是鹰嘴岭第一战力，但他已死在程大雷剑下。失去张猛的鹰嘴岭，不过就是一群乌合之众。
<br/>　　终究是兖州兵占了优势，人数又多，士兵纪律又强。那方石隆当真也是条汉子，大刀甩开，领着手下冲杀。主将已经如此勇猛，手下人怎敢落在后面。
<br/>　　黄家人杂在兵匪之间，被撞见了，也就随手杀了。这样的夜里，大家可看不清楚谁是谁，而无论官兵还是土匪，似乎都不会在意黄家人的性命。
<br/>　　厮杀声，惨叫声，兵器碰撞声，奔跑摔倒声响成一片。
<br/>　　黄为善万没料到事情会变成如此局面，他挥舞着双手，口中高喊着什么：“大人，不要杀了，他们是无辜的，他们都是我的家人。”
<br/>　　“聒噪。”
<br/>　　方石隆一刀砍在黄为善脖子上，将脸上血一抹，口中道：“弟兄们，杀光这群土匪，一个都不要留。”
<br/>　　黄老汉立在大殿下，三魂七魄早已不在他的躯壳内。他呆呆的看着眼前发生的一幕幕，嘴里只是不停重复着一句话：
<br/>　　“为何会这样，为何会这样……”
<br/>　　他突然哇呀大叫一声，喷出一口鲜血，身体咚地一声倒地，已然气绝身亡。
<br/>　　太多人想要在这场乱世牌局中捞些利益，但不是任何一人都有登上牌桌的资格。就连通向赌桌的这条路，已可令人失去全部。
<br/>　　有靴子从他身上跨过，分不清是官兵的，或者土匪的。反正都差不太多，谁也不会太将一个土财主看在眼里。
<br/>　　“兄弟们，与我杀，莫让走了一个山贼！”
<br/>　　方石隆大喊大叫，领着手下人往前冲杀。在如此状态下，鹰嘴岭的山贼一直往后退，就这短短时间，已经损兵折将严重。
<br/>　　黑河九鬼，还活着的没几个了。
<br/>　　老二长须鬼看着眼前这一幕，脸色越来越难看，眼前不断有弟兄倒下，鹰嘴岭的弟兄怕已扛不过眼前这一关。
<br/>　　“老三！”长须鬼奋力将饿死鬼朱刚推出围墙外，口中大吼：“走啦！”
<br/>　　“与他们拼了！”朱刚双手拖一柄大刀，脸上挂满血水。
<br/>　　“走啦！”长须鬼声嘶力竭的大汗：“再不走都要死在这里，老三，你先走，我挡住他们。”
<br/>　　“二哥……”
<br/>　　“别废话了，老三你是个有本事的人，活下来能为弟兄们报仇。二哥没本事，死在这里也不可惜。”长须鬼大吼：“来也，弟兄替三当家挡住。”
<br/>　　“来呀！”
<br/>　　“三当家先走，这里交给兄弟们！”
<br/>　　鹰嘴岭的山贼一个个涌上去，堵住围墙的缺口，前赴后继。
<br/>　　“二哥！”
<br/>　　“滚！”
<br/>　　朱刚将脸上的血汗抹去，也发了狠心，拎着刀往外闯去。
<br/>　　身后的惨叫一声连着一声，朱刚强忍着没有回头，但却将牙咬得咯吱咯吱响。
<br/>　　想要活下去，有所牺牲是难免的。可活下去的人，总不能忘了死去的人，总要好好活下去，留着一条命，替死去的人报仇。
<br/>　　一路往外逃，黑夜中也看不得方向，就这样迷迷糊糊逃出黄土镇。
<br/>　　天还未亮，四野空旷，夜黑得伸手不见五指。
<br/>　　“三当家，三当家！”手下喽啰气喘吁吁道：“人……人没有追过来。”
<br/>　　朱刚这才回过头，也是大口大口的喘息，看到身后只跟着七八个拼死逃出来的弟兄。
<br/>　　这些人，当就是鹰嘴岭最后活下来的人，原先的黑河九鬼，现在也就剩朱刚这一个孤魂野鬼。
<br/>　　“三当家，咱们去哪儿？”
<br/>　　“还能去哪儿，当然是给大家报仇。方石隆，程大雷……我一个都不会放过。”朱刚将牙磨得咯吱咯吱响。
<br/>　　“报仇……”喽啰看了看身周，心里的话并没有说出口：凭着几个人，谈何报仇。
<br/>　　“哼，我已经有了主意。”朱刚冷哼一声，像是知道他们心里想的是什么，道：“三山五岳，帝国十三州，谁不想杀程大雷。我们将程大雷出现在兖州的消息放出去，还愁找不到帮手。”
<br/>　　……
<br/>　　程大雷和刘发财一路奔出黄土镇，来到一片野柳林，见到身后没人追出来，二人才暂时停住。
<br/>　　刘发财也是累得气喘吁吁，他在牛背上怪笑一阵，冲程大雷道：“好兄弟，我当真是服了你。为了保护我，竟冒充我的身份，把自己置于危险境地。好兄弟，一辈子！我还有些事，咱们暂且别过，山水有相逢，咱们日后再会。”
<br/>　　说罢，双手抱起，拨转牛头就走。
<br/>　　“等一等。”
<br/>　　程大雷的声音不紧不慢的响起，刘发财整个人一下僵在那里，慢吞吞回过头，见程大雷笑盈盈的望着自己。
<br/>　　他翻身从牛背上跳下来，噗通一声跪道：
<br/>　　“程当家，我有眼无珠，不该冒充您的名头。您大人大量，就把我当一个屁放了吧，我再也不敢了。”
<br/>　　刘发财也不是傻瓜，哪里还猜不出程大雷的身份。在意识到这一点后，待在这魔头十步之内，他就觉得浑身冒寒气。
<br/>　　程大雷也从牛背上下来，坐在一块大石头上，看着刘发财表演。
<br/>　　半晌，刘发财试探着抬起头，心里砰砰打鼓。
<br/>　　“好啦，好啦。”程大雷捶着膝盖：“我还忙得很，没时间在这里耽搁。我只问你一件事，以后愿不愿意跟着我做事？”
<br/>　　刘发财愣了愣：“是不是有不愿意这个选项？”
<br/>　　“有啊。”
<br/>　　“假如我不愿意会怎样？”
<br/>　　“杀了你啊。”程大雷慢条斯理道：“你害我走漏了行踪，难道还能容你活在世上？”
<br/>　　刘发财激灵一下，他当然明白跟着程大雷是个怎样结果，程大雷的行踪一旦暴露，多的是人想杀他。而且来的，都是从前难得一见的高手。
<br/>　　可是，如果自己不答应他，似乎眼前的情形更恐怖些。
<br/>　　他挺直了腰杆，口中慷慨激昂道：“不瞒程当家说，刘某平生最佩服的就是您。不然，也不会冒充您的名头。我做梦也想跟在您身边做事。”
<br/>　　程大雷撇撇嘴，一拍他的肩膀：“好兄弟，一起走。”
<br/>(本章完)
