　　刀疤男微微笑了一笑，没有回应，抬脚来到刘哔身边，仔细查探了一圈。
<br/>　　“你可以走了。”没有发现什么问题，刀疤的表情犹豫了一下，冲刘哔挥了挥手。
<br/>　　闻言，刘哔冷哼一声，穿好自己的衣服，从工厂内走了出去。
<br/>　　目送着刘哔离开，刀疤站在原地微动，脸上的表情，若有所思。
<br/>　　从工厂出来之后，外面的行人并不算多，辨别了一下自己所在的方位，刘哔径直向家中走去。
<br/>　　这片地方他有些陌生，但明显是处于基地的边缘，好像是在工业区这一个区域。
<br/>　　作为S省最大的幸存者基地，一号基地的占地面积很广，其中也划分了诸多区域，工业区的建筑大多数都是一些工厂。
<br/>　　而这个地方也给很多没有异能的幸存者提供了一些工作，充分利用各类幸存者的价值，这是一号基地最注重的地方。
<br/>　　只是，基地高层的政策是出于好心，但僧多肉少，工作职位有限，没有一定的后台关系，普通的幸存者想要进来这里工作，几乎是不可能的事情。
<br/>　　身为一名底层的幸存者，他在一号基地生存了许多年了，但很多地方也从来都没有进入过。
<br/>　　顺着高耸的城墙，刘哔一路向西行走着，最西部的地方就是贫民区，他只需要一直走就可以到达。
<br/>　　虽然没用翅膀，但他的脚程也比一般人快上许多，花费了十几分钟的时间，刘哔回到了家里，看到刘莽正搬了个板凳，坐在秦老太太身边说这什么。
<br/>　　和老太太打了一个招呼，刘哔挥了挥手，示意刘莽过来，随后迈进了自己家门里。
<br/>　　刘莽随手也跟了进来，脸上带着灿烂的笑容。
<br/>　　“我要外出一段时间，给你留下这些晶核，你回头去平民区去买一栋房子，搬过去住吧，那边安全一些。”将手上的麻袋提给刘莽，从松散的袋口看去，可以看见一颗颗各色光芒的晶核。
<br/>　　但刘莽的注意力却完全没有在麻袋上，而是直愣愣的盯着刘哔，脸上的笑容逐渐开始凝固。
<br/>　　“哥，去哪啊，我跟着你去呗。”强撑着笑容，刘莽心中突然有些担忧。
<br/>　　心中叹了口气，刘哔换上了一副轻松的语气，笑道：“接了一个团队任务，需要跟团去一个地方，没事的，不用担心。”
<br/>　　脸上的笑容已经完全消失，刘莽猛然拉开麻袋，露出了里面慢慢一麻袋的晶核。
<br/>　　“哥，还接什么任务啊，我们这不是还有晶核吗，其实我觉得，我们完全不需要买房子，留下这批晶核，我们就能用很长一段时间，反正住在哪里都是一样，再说……”
<br/>　　“现在够用，以后呢？”看着有些激动的刘莽，刘哔摇了摇头，打断了他的话语：“莫说了，我已经约好时间了，还得赶着去汇合，你听我的，好好在家待着，等我回来。”
<br/>　　说吧，刘哔不等刘莽回话，直接转头走出了屋门，向城门外面走去。
<br/>　　还有不到半个小时，他没有时间继续从这里消磨下去了。
<br/>　　只是，这次传送还能不能回来，刘哔心中也不清楚。
<br/>　　经过上一个未免，他也明白过来，系统要将他传送去的地方，可都不会是什么善地，一个不小心，或许就会将命给丢掉。
<br/>　　出城的过程很平淡，这个时间进进出出的人很多，他的外出没有引起任何人的关注。
<br/>　　离开了基地，刘哔随意在附近找到了一个残缺建筑，找了个角落躲起，静静的等待传送着。
<br/>　　因为不清楚下一个世界究竟是什么位面，故此他也不知道应该提前做些什么准备，反正账户里还有一些积分，到时候再进行兑换好了。
<br/>　　这些积分都是他最近猎杀丧尸得来的，至于《僵尸先生》位面里获得的那些，都被他拿去兑换紫府的下半部分功法了。
<br/>　　而这套功法经过完善之后，等级也上升到了B级，足够他前期修炼使用得了。
<br/>　　时间缓缓流逝，在这里等的无聊，不到半小时的时间里，他已经抽了整整半盒香烟了，或许，这也和他忐忑的心情有关。
<br/>　　终于，脑海中的倒计时结束，刘哔意识稍稍眩晕了一瞬，四周的画面开始变的有些模糊。
<br/>　　已经有过经验的刘哔丝毫没有慌张，做好了警戒的姿态，周围的景象开始清晰了起来。
<br/>　　这是一个很是繁华的大厅，他好像是在一座办公楼里，身边人来人往，大部分都穿着西装，偶尔可以看见身穿军服的警卫，似乎这里是什么很重要的机关单位。
<br/>　　刚来这个世界，对所有东西都不熟悉，刘哔稍稍有些茫然，下意识的低头，看见了自己胸口处深紫色的领带。
<br/>　　不知道系统这次给他安排的身份是什么，心中嘀咕了一句，刘哔整了整身上有些别扭的衬衫，在这所建筑里面闲逛了起来。
<br/>　　四周的人群都在忙碌，只有他一副无所事事的样子，有些扎眼，但周围的人对他都没有太过理会，他们没有闲工夫去关心这种事情。
<br/>　　“诶，纳尼奥西忒一鲁哇，哔。”就在刘哔百般聊赖的在大楼里闲逛的时候，一道软糯的声音从背后传来，让他下意识转过了头去。
<br/>　　说话的是一个正值青春的少女，一头半长的头发，脸上化着有些怪异的妆容，长相精致，裸露在外的肌肤上，缝合了一排显眼的红线，给见过她的人留下难以磨灭的印象。
<br/>　　可惜的是，这么可爱的女孩子，偏偏胸前一马平川，那个尺寸，目测和旺仔小馒头也没什么区别了。
<br/>　　有人主动搭话，刘哔心中还是有些兴奋的，因为这代表，他可以从这个女孩口中得知自己现在的身份，以及处于一个什么样的世界。
<br/>　　令他蛋疼的是，这里并不是华夏，听这些人的口音，应该是某个弹丸小岛，他可没学过这里的语言。
<br/>　　看着面前正一脸疑惑看着自己的可爱女生，刘哔不免感觉有些头大，不过幸好的是，系统提示音很及时的从脑海中传来。
