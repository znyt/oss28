　　“什么？玄天小儿，我要你死！”
<br/>　　眼见天气祖巫奢比尸被玄天亲手镇压，他目眦尽裂，看向玄天满脸杀气，猛然大喝，乍见天地之间，一片漆黑水光，眨眼之间，天地震荡，无穷无尽的黑色洪流，汇聚在他手中，化作一柄巨大黑色战刀，庞然霸气，震荡乾坤。
<br/>　　“万流斩，灭神击！”
<br/>　　共工一声怒吼，抬手之间转动手中黑色战刀，顿时，无边水光迸现，天地虚空，风云一滞，一道道匹练似的黑芒，化作万道洪流刀芒，呼啸着划破天地长空，携带着崩天裂地之威势，悍然冲向玄天。
<br/>　　“哼！”
<br/>　　“空间，冻结！”
<br/>　　玄天依然不惧，冷哼一声，他那冰冷的声音在虚空中响起，看着向他冲来的万道洪流刀芒，不慌不忙，伸手向着虚空轻轻一指，一点寒光万丈芒。
<br/>　　顿时，天地之间，风云失色，日月无光，乾坤震荡，无穷无尽的寒芒冲击而来，寒气弥漫，瞬间便将玄天周遭的整片空间全部冻结起来，化为一片冰场，整片虚空变成了固体，就仿佛成了一面镜子，触手可及。
<br/>　　共工的万道黑色洪流瞬间便被冻结成了固体，停留在空间中，不得动弹。
<br/>　　“空间，破碎！”
<br/>　　玄天那悠悠的声音再次响起，顿时，那面如镜子一般的空间，瞬间就响起“咔擦”之声，随即镜面上出现道道裂纹，瞬息之间，满面镜子变得破碎不堪，裂迹斑斑。
<br/>　　“轰！”
<br/>　　一声爆响，镜面完全爆炸开来，化为无数碎片，弥散到虚空之中。紧接着，这些碎片并没有摔落在地，反而化为一柄柄虚幻的小刀、小剑，朝着共工祖巫迅速飞去。
<br/>　　无数锋利的空间之刃如同一道锋利的刀刃洪流，划破无尽虚空，宛若漫天暴雨一般冲击到共工身边。
<br/>　　“可恶！”
<br/>　　共工一声怒喝，黑光乍现，手中黑色战刀再次化作一道黑水界，将自己包裹在界中，黑水界外黑光闪闪，形成绝对防御之状，来抵御那向他冲击而来的无尽虚空刀芒。
<br/>　　“兵兵砰砰！”……
<br/>　　一阵剧烈声响，无数刀雨撞击到黑水界上，却被黑水界外的黑光挡住，但公共也不得不再次爆退数千里。
<br/>　　玄天步步紧逼，公共不断后退，就这样，转瞬之间，两人不知不觉中，就已经来到洪荒中心，洪荒第一神山，不周山附近。
<br/>　　玄天抬起一只手，顿时，无数虚空之刃汇聚而来，狂风倒卷，风云汇聚，狂澜激荡之间，无数刀刃化为一柄巨大的通天剑柱，接天连地，震动乾坤。
<br/>　　“剑起风云变！”
<br/>　　极招乍现，引动万古虚空，为之生出莫大变故，赫见玄天手中的虚空剑柱飞速流转，剑锋所指，暴起一道道剑瀑，威势滔天，撕裂苍穹大地，携着无与伦比的庞然伟力，呼啸着，宛若天地洪流，不可阻挡。
<br/>　　“万流斩，戮神击！”
<br/>　　共工满脸怒色，身边黑水界黑光闪烁，氤氲生雾，再次化为一柄巨大黑色战刀，翻手之间，手中战刀斜劈而出，顿时，无数水光漫天而出，天地为之一变，水光化为黑色瀑流，如同九天之上冲击而下的瀑布。形成一道天然的屏障，阻挡那锐利锋芒的通天剑气洪流。
<br/>　　“砰！”
<br/>　　剑柱与瀑布相撞，虚无之剑与战刀瞬息交锋，惊闻一声巨响，震动天地，顿时，无边风云迸爆，可怕的力量，如同惊涛骇浪，波散四方。
<br/>　　共工乃祖巫之躯，玄天修的是武道金身，两者都是肉体强悍之辈，但，公共的祖巫之躯终究是差了一筹，当初玄天传道天下，于那无边雷霆之中可是整整待了百年时光，身躯是破了重组，组了有碎，碎了再重组，如此循环了无数次，肉身早已堪比准圣后期。更何况，这些年他也没有停止修炼过肉体，如今的肉身，早已堪比准圣圆满。
<br/>　　共工那堪比准圣后期的祖巫之体，在玄天的强势交锋之下，整个人顿时承受不住反震之力，向后跌倒抛飞，足足数千里之遥。
<br/>　　“遮天掌！”
<br/>　　玄天不停，再次于那无边高空之上，凝聚出一道巨大的遮天金掌，威压苍穹，堪比天高，他那一掌毫不犹豫地便向着不断后退的共工拍下，巨掌如同一座巨峰一般镇压下来，乾坤震荡，虚空不稳。
<br/>　　仓促之间，共工不得不再次提起黑色战刀抵挡，整个人也随之不断后退。但听得一声巨响，共工整个人再次倒飞数千里之遥，撞断一道巨峰，跌落在地。
<br/>　　“噗——”
<br/>　　“咳咳咳……”
<br/>　　一口鲜血顿时从他口中喷射而出，随即他便咳嗽不止，脸色变得苍白异常。
<br/>　　看着不断向自己逼近的武祖玄天，他不由自主地闭上双眼，原来，死亡，距离自己是那么的近。曾经纵横天下的过往，顺势之间，全部浮现在自己眼前，身为十二祖巫之一，他至高无上，瞧不起任何人。他还真的从未想过，自己有一天竟会败亡于一个诞生不过万年的人族手中。
<br/>　　他想起不久前被玄天斩杀的祝融，想起被镇压的奢比尸，想起昔日的点点滴滴，想到今后巫族有可能要败亡于妖族手中，一时之间，他陷入了迷障之中。
<br/>　　“若是没有我当初向大哥汇报人族魂魄能够炼制戮妖幡一事，令得巫族杀伐人族，也就没有今日的结局。本想着好以此击败妖族，却没想到，如今却牵连到人、巫族两族大战，死伤无数，甚至就连祝融也死了，我共工如今还有何面目去见其他巫族兄弟？哈哈哈！罢了罢了！今日就算是死，也要拖着玄天小儿一起去死，要不然，巫族将会后患无穷！”
<br/>　　“哈哈哈！洪荒生于盘古，祖巫亦生于盘古，我即便是死，也要归于父神。”
<br/>　　共工惨然大笑，只见他全身暴发出巨大潜能，燃烧生命，燃烧血脉、燃烧体内法则，换来短暂的强大力量，但他并没有直接与玄天拼命，反倒是蓄足全身力道，向着一旁的天柱不周山一头猛然撞击而去。
<br/>　　这一撞，无怨无悔！
<br/>　　这一撞，生死由天！
<br/>　　这一撞，天翻地覆！
<br/>　　“咔擦！”
<br/>　　只听“喀嚓”一声，那屹立无数个元会，经历荒古、太古、远古、上古时代，经过无数风雨洗刷，见证无数了奇迹发生的天柱不周山终于于今朝破裂，半截不周山被共工祖巫生生撞得断落下来，直奔武祖玄天当头砸落下来。
