　　看到叶永阳自行脑补想太多的误会，顾凉晨索性就着这一话题道：“我并没有见过叶默本人，只是奉命行·事而已，不过叶默收了三个徒弟，都在狼牙。”
<br/>　　“他还有徒弟？！”叶永阳惊诧无比：“不应该啊，他这个人一生致力研究的目的，都是为了与神秘家族对抗，都是为了叶家的未来，怎么会收徒，将毕生所学传递给非叶家之人？”
<br/>　　“这个我倒是知道一点。”顾凉晨淡淡道：“楚涵长官与叶默分开时，他已经到了人生最后阶段，当时又是末世爆发最混乱的初期，想找个人难如登天，楚涵长官索性就留下了三个人，最终也成为了叶默的接班人，只不过叶默毕生的知识都太过奥妙，那三人都只能选择一个方向钻研，谁也无法将叶默的全部领域吸取。”
<br/>　　叶永阳点头，苦笑道：“我与他也有二十多年未见，离开叶家的时候他就已经够厉害，而这二十年想必也更上一层楼吧？”
<br/>　　“是的。”顾凉晨笑道：“我们长官的那柄修罗战斧，就是叶默亲自锻造，据说材料非常稀有世上仅此一柄，而他那三个徒弟更是狼牙分别三个部门的部·长，分别管理着科技部、工程部和锻造部。”
<br/>　　“狼牙的武器，建筑甚至科技，全部都领先其他基地也是拜叶默所赐，可以说整个狼牙到处都存在着叶默的意识，那些都是叶默的遗产，最珍贵的遗产。”
<br/>　　顾凉晨的这些话发自肺腑，也不是作假，狼牙之所以能到达这个高度，战团之所以能无往不胜，除了楚涵先人一步的谋划，还有部分因素都是当年叶默的那些学识影响。
<br/>　　和上、苏行还有卢宏盛，在幕后的默默付出，才能支持狼牙战团在前方勇往直前。
<br/>　　之前并不知道叶家千年来与神秘家族之间的纠葛，更不知道叶默与叶永阳是亲生兄弟，当此刻秘密揭开后，叶永阳也似乎不再是顾凉晨用尽手段需要去接近的人，反而更像是终于找回了原本就属于狼牙的一份子。
<br/>　　顾凉晨有这样的想法，听到上面这段话后的叶永阳，更如此想。
<br/>　　从一开始的警惕，到此刻忽然涌·出的亲切感，都让叶永阳忍不住的老泪纵横：“想不到，想不到啊！到了我这把年纪，还能再遇到真正可以亲近的人，这么多年来叶家步履薄冰，在刀尖上活着，实在是……”
<br/>　　顾凉晨于心不忍，安慰道：“先把你送到狼牙，我再回头去把您子孙都接来。”
<br/>　　话都说到了这个份上，叶永阳也不再隐瞒，直接道：“阴阳谷的八门有着森严的等级，回炉之下是制裁，制裁之下为水火，这四门之后，才是白罗肖高四偏门并存，而我们叶家在千年前世代为奴。”
<br/>　　“我们不仅是奴，还是阴阳谷的改造者。”话终于说到真正意义上的重点，叶永阳的声音也严肃了起来：“阴阳谷并非一开始就是这个样子，而是经过了我们叶家人的改造之后才是。”
<br/>　　“就在我们祖辈某一天忽然接到命令，是四偏门发布的命令，改造那个阴阳谷，并且工程量空前浩大，整个叶家男女老少都参与了劳作，而所用材料恐怕你们也猜到了，来自上面。”叶永阳说着，指了指头顶上方的天空。
<br/>　　“星际……”顾凉晨的喉咙一紧，声音干涩。
<br/>　　“不错。”叶永阳继续道：“原本的阴阳谷之阴谷虽说也是居住地，但没有禁制，而阳谷也只有制裁之地，没有八门墓地。”
<br/>　　“阴阳谷内的禁止，无数重重阻碍，统统都是利用了特殊材料和地球磁场的结合而成，其中还加入了四正门特殊成员的体制，这些并非那四偏门下达的命令，而是我们叶家祖先在改造阴阳谷的时候，发现了某一重大秘密，而后私下进行的改变。”
<br/>　　“什么秘密？”顾凉晨瞳孔收缩。
<br/>　　叶永阳摇摇头：“我知道的不是全部，但经过之后逃脱出来的叶家人分析，是四偏门集体谋和背叛了回炉一门，想要将四正门的所有成员永远埋葬在阳谷内。”
<br/>　　“一开始的图纸，只有四门墓地，没有他们四偏门的，那四偏门是我们叶家另外秘密建成，但里面还有没有四偏门之人埋葬，信息到我这里已经遗失了。”
<br/>　　“”听到这些，顾凉晨已经不禁冒出了些许冷汗：“所以，那四大神秘家族，当年是想要谋反？”
<br/>　　叶永阳双目冒着精光：“谋反这个词用的恰当，所以我们叶家在察觉到异样之后，就将阴阳谷改造成了让那四偏门也陌生的样子。”
<br/>　　“之后就是一场浩劫，四偏门终于动了手，据说阴谷经历了一场血洗，守护者消失，水火两族隐匿，制裁者不知所踪，但他们四偏门也没讨到好处，做完那些事情之后，阴阳谷就彻底关闭了大门，他们也回不去。”
<br/>　　听到这里，直升机已经距离狼牙地界不远，顾凉晨在松了一口气之后，继续说道：“制裁者隐于世，这些年动作可不少。”
<br/>　　“叶家的一部分人逃出阴阳谷之后，为了隐藏家族就再也没能接触到神秘家族的事，所以这其中有过什么我都不清楚。”叶永阳说着，苦笑：“哪怕想知道，也没有途径打探。”
<br/>　　叶永阳想了想，道：“那个制裁之地，历代制裁者一直都进出自如。”
<br/>　　叶永阳一愣，紧接着说道：“那说明我们祖先当年防的那一手起作用了，在阴阳谷大门开启之前，恐怕也就只有制裁一门的人能回去，但也只能回阳谷吧？”
<br/>　　“对。”顾凉晨想着楚涵用极简话语告知的消息，道：“制裁者并没有因为阴阳谷的关闭而停止行动，每当有四偏门者犯事时，他们都会出现，将其抓到制裁之地关押，这也是哪怕四偏门在阴阳谷外活的至高无上，却也绝对不敢有大动作的原因，一旦有触犯到法则的事发生，都会有制裁者前去。
<br/>　　“只不过……”说到这里顾凉晨顿了顿，道：“制裁者至始至终都只有一个，这是为何？”
