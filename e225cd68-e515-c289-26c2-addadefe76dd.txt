>　　    不过此刻穆广的心情已经放松了下来，他不像来北京的时候那么紧张了，雍和宫给了他很大的心里安定作用，因此看到丁益他还能笑得出来，他笑着说：“是啊，真巧啊，丁总这时跑来北京干什么啊？”<br/>　　<br/>　　    丁益看了穆广一眼，他怀疑穆广是知道他来北京的意图的，不过他不知道关莲现在是一个什么状态，因此倒也不好直接挑明自己的来意，便笑笑说：“”我来是见一个朋友，怎么穆副市长这是要回去啊？<br/>　　<br/>　　    穆广心里暗自好笑，你见朋友，你见个屁朋友啊，你的朋友早就做了鬼了，你去见鬼倒是可以。他笑着点了点头，说：“这一次我来参加一个会议，本来是想学点东西的，没想到会议的组织者根本就是骗钱的，开了一天就觉得没意思了，市里面还有一大堆事务等我处理呢，就赶紧回去了。丁总，你在北京慢慢玩，我先回去了。”<br/>　　<br/>　　    两人就握了握手，穆广就往机场里面走，傅华让丁益现在外面等他，他送走了穆广，就带丁益去海川大厦。<br/>　　<br/>　　    傅华就将穆广送到了安检的地方，穆广再次热情的跟他握了手，嘱咐他在北京好好工作，然后就跟傅华分手了。<br/>　　<br/>　　    送走了穆广，傅华赶忙出来找丁益，丁益正在百无聊赖的等着他，他上前笑着说：“老弟啊，你今天运气真是不错啊，碰到我就省的你打的去我那里了。”<br/>　　<br/>　　    丁益却没有丝毫高兴的样子，他说：“什么运气不错啊，我遇到了穆广算是够倒霉的了。”<br/>　　<br/>　　    傅华愣了一下，说：“怎么了？穆广又惹到你了？”<br/>　　<br/>　　    丁益没好气的说：“这家伙简直就是我的克星，我一个大好的青年竟然争不过他一个快五十的中年人，真是邪门了。”<br/>　　<br/>　　    傅华看了丁益一眼，说：“你不要告诉我你又跟关莲扯上瓜葛了吧？”<br/>　　<br/>　　    丁益有些不好意思了，他说：“傅哥，我真是舍不得关莲，你不知道还没有其他任何一个女人能给我那么美好的感觉。”<br/>　　<br/>　　    傅华有些不满了，说：“老弟啊，关键不是这个女人能不能给你美好的感觉，而是她是一朵有毒的玫瑰，沾惹上她对你有百害而无一利的。”<br/>　　<br/>　　    丁益说：“傅华，那种感觉你不懂得，再说这一次不是我去沾惹他，而是她主动找到我，还跟我说要离开穆广跟我一起生活。”<br/>　　<br/>　　    傅华说：“那穆广是什么态度，他肯放手吗？”<br/>　　<br/>　　    丁益叹了口气，说：“我估计穆广是不肯放手，关莲本来跟我说的好好的，他跟穆广说清楚之后，就来跟我一起生活，可是她回去了之后，第二天就变卦了，说不能信任我会一辈子对她好，还从海川跑回了北京。”<br/>　　<br/>　　    傅华惊讶看了丁益一眼，说：“你是说关莲回了北京？”<br/>　　<br/>　　    丁益说：“对呀，我前天还接到关莲在北京发给我的短信呢，说什么希望我不要骚扰她，她想平静一段时间。可本来是她让我的心不平静的，我不服气她宁愿选择穆广也不选择我，就追来北京想问她究竟是怎么了，是不是受到了穆广的胁迫了？”<br/>　　<br/>　　    傅华笑了起来，说：“老弟啊，你是不是太幼稚了，她短信里说在北京，她就在北京啊？也许她躲在海川不想让你找他，才故意发短信说在北京呢。”<br/>　　<br/>　　    丁益说：“哎呀，你因为我那么傻啊，我是接了短信之后，找了一个朋友帮我查了关莲的手机号码，确信机主正漫游在北京，我才过来的。关莲肯定是在北京了，要不然穆广跑来干啥啊？”<br/>　　<br/>　　    傅华说：“你是说穆广来北京也是追关莲而来的？不像啊，他来北京这两天，没有跑出去见什么人啊？”<br/>　　<br/>　　    丁益冷笑了一声，说：“公开他当然不敢去见关莲了，可私下里呢？”<br/>　　<br/>　　    私下里傅华就不敢保证了，他记得穆广第一天到北京的时候，是他很累，从下午就躲在房间里，晚饭都没出来吃，难保这段时间穆广没溜出去跟关联见面。而且穆广一开始来的时候魂不守舍的，也很可能是为情所苦才那个样子的，不然的话也没别的理由能让穆广这个样子的。<br/>　　<br/>　　    傅华笑了笑，说：“我真服了关莲这个女人了，她竟然能让你们两个大男人为她这么。”<br/>　　最快更新，无弹窗阅读请。<br/><br/><br/>
