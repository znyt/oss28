　　“叶尘哥哥，要不、你还是赶紧走吧！血月宗，我们惹不起的……”
<br/>　　灵溪思量了片刻之后，忽然忍不住小声冲着叶尘催促道。
<br/>　　叶尘哈哈一笑，
<br/>　　“丫头，叶尘哥哥既然答应了要教你炼丹，便一定会教你，又怎能离开？更何况，一个小小的血月宗，你叶尘哥哥还没有放在眼里！”
<br/>　　灵山等人听到叶尘这话，虽然觉得这少年，实在狂妄的厉害，可是却纷纷松了一口长气。
<br/>　　只要叶尘不逃，那此事便还有回旋的余地，等到血月宗大军前来，他们完全可以将此事，全都推到叶尘一人身上，来平息血月宗对他们的黑岩城的怒火。
<br/>　　虽然这么做有些不厚道，可是修真界向来如此，像他们这些低级势力，能够自保就不错了，哪里还管的了这么多？
<br/>　　一旁的柳云飞听到这话，同样喜形于色，连忙道：
<br/>　　“好！叶先生敢做敢当，真是好魄力！柳某佩服之至！哈哈！”
<br/>　　柳云飞嘴上这么说着，心里却在暗暗冷笑，
<br/>　　‘真是一个不知道天高地厚的白痴，既然赶都赶不走你，那你就等着承受血月宗的怒火吧！’
<br/>　　虽然，方才叶尘秒杀血无影的场景，让众人感到无比震撼，可是却没有一个人相信，叶尘具备挑战巨门星域第一大宗门的资格。
<br/>　　众人各怀鬼胎，叶尘自然心知肚明，尤其是这个柳云飞，明面上是在夸他，其实是想让他骑虎难下，然后好借助血月宗之手，来除掉他。
<br/>　　对于这种人，叶尘也懒得理会。
<br/>　　柳云飞见叶尘完全无视自己，心头恨意更浓，却又不敢发作，只好转而又望向灵韵，道：
<br/>　　“灵韵师妹，你可要考虑清楚了？当真不随我们回宗门吗？”
<br/>　　灵韵心中一阵踌躇，不由得咬了咬嘴唇，又悄悄望了一眼，一脸自信神色的叶尘，也说不上来为何，她总感觉这少年，给她一种莫名的安全感。
<br/>　　“柳师兄路上小心，我就不送了！”
<br/>　　灵韵最终拒绝了柳云飞的提议，并且说完之后，直接背过身去，一脸的淡漠。
<br/>　　柳云飞见状，顿时气得牙根直痒痒，忍不住冷笑道：
<br/>　　“灵韵师妹，你该不会真的以为，他能保护得了你吧？我劝你最好还是考虑清楚一点，省的到时候后悔！”
<br/>　　“我说了，我的事情，不用你操心！”
<br/>　　灵韵再次果断拒绝。
<br/>　　柳云飞还想多说，这时一旁的叶尘忽然开口了，
<br/>　　“怎么？听你这话，似乎对我很没有信心啊？”
<br/>　　柳云飞心想说，你这不是废话吗？就凭你，也想对抗巨门星域的第一大势力？简直就是天大的笑话！
<br/>　　不过话到嘴边，柳云飞最终还是忍住了，
<br/>　　眼前的这家伙，可是连血无影都敢说杀就杀，更何况是他？
<br/>　　万一激怒了这个疯子，那后果不堪设想！
<br/>　　“不不！我不是这个意思……”
<br/>　　感受到叶尘冰冷的目光，柳云飞连忙改口，脸上硬生生挤出一丝笑容。
<br/>　　叶尘收回目光，淡淡吐出一字，
<br/>　　“滚！”
<br/>　　柳云飞先是一愣，继而脸色瞬间涨红如同猪肝，双拳紧握，忍不住一阵颤抖。
<br/>　　虽然对方只说了一个字，可是对于柳云飞来说，却比杀了他还要令他难受！
<br/>　　身为飞雪宗的最强天才，柳云飞这一生，何曾受过如此大的屈辱？
<br/>　　即便是血无影，虽然可将他逼退，却也不敢过分羞辱。
<br/>　　“怎么？还不走？你是想和血无影去地下作伴吗？”
<br/>　　叶尘淡淡瞥了一眼，眼眸中闪过一抹杀意。
<br/>　　蹬蹬！
<br/>　　柳云飞顿时浑身一颤，下意识的倒退了两步，继而一股凉气，直接从脚底窜到了头顶，之前胸中那股强大的怒气，瞬间消失的无影无踪。
<br/>　　下一刻，
<br/>　　柳云飞不敢再有任何犹豫，直接转身拔腿便走，转眼便消失不见，简直比兔子还要快。
<br/>　　至于另外一边，邪光宗的众高手，早已经在血无影被叶尘虐杀之时，就已经吓得逃之夭夭，叶尘也懒得去管。
<br/>　　等到柳云飞一走，灵山等人对视了一眼，然后快步走上前来，纷纷跪倒在地，
<br/>　　“多谢叶先生救命之恩！”
<br/>　　叶尘冷冷一笑，连看都没有看众人一眼，淡淡道：
<br/>　　“不用在我面前演戏！我也从来没有想过要救你们！”
<br/>　　众人顿时大为尴尬，一时间也不知道该说什么好。
<br/>　　过了片刻，其中一名黑岩城的长老，腆着脸又道：
<br/>　　“那个，敢问叶先生师承何处？若是血月宗真的找了，先生可有对策啊？”
<br/>　　叶尘也懒得和这帮人多说，摆了摆手，道：
<br/>　　“人是我杀的，你们只管把事情推到我的头上，其他的无需多问！”
<br/>　　说完，拉着一旁的灵溪，直接扬长而去。
<br/>　　众人经过片刻的愕然之后，顿时纷纷议论起来，
<br/>　　“我观这位叶先生如此淡定，多半应该有着什么底牌吧！”
<br/>　　“不错！他年纪轻轻，就有如此实力，说不定是某个超级大宗门的天才人物！”
<br/>　　“既然他敢杀血无影，而且还不逃走，必然有所依仗！”
<br/>　　“我们若是通过此人，结交到他背后的势力，说不定对于我们黑岩城来说，反而是一个天大的机缘呐！”
<br/>　　……
<br/>　　众人议论了一阵之后，之前的忧虑顿时大减，反而开始商量着，如何去巴结叶尘，好去结交他背后的势力。
<br/>　　“韵儿！”
<br/>　　灵山忽然望向一旁，一直呆立在那里，一言不发的灵韵，一脸热切道：
<br/>　　“这位叶先生，年龄应该和你相仿，以你的容貌，若是能再主动些……”
<br/>　　灵韵冰雪聪慧，瞬间便明白了灵山的意识，顿时俏脸一红，羞道：
<br/>　　“父亲，你说什么呢！”
<br/>　　灵山两眼一瞪，
<br/>　　“怎么？为父有说错吗？你不是一直都倾慕少年英雄吗？这位叶先生，论天赋，胜那柳云飞十倍都不止！背后更有可能是某个超级大宗门，你若是能攀上这个高枝，不但可以一步登天，连为父和整个黑岩城，都要跟着沾你的光了！”
<br/>　　“父亲！”
<br/>　　灵韵顿时越发娇羞，也不知道为何，芳心竟然一阵乱颤，犹如小鹿乱撞。
<br/>　　然而，紧接着，
<br/>　　她又忽然想起，之前叶尘完全无视她的情景，俏脸又顿时一黯，不由得幽幽一叹，低声喃喃了一句，
<br/>　　“就怕人家可未必看得上我呢！”
