　　第620章
<br/>　　韩醒的尸体倒在地上，这个一夜屠了塞外巨贼，杀人从不用第二刀的刀客，终究没挡得住程大雷一剑。
<br/>　　包围程大雷的诸人齐刷刷往后退了一步，韩醒已经是莫将难手上有数的高手。至于其他人，自量可没有韩醒的本事。
<br/>　　“与我杀，与我杀！”
<br/>　　莫将难一声声吼着，可谁敢上前，大家都握着兵器，却是情不自禁的颤抖。莫将难苦着一张脸，他本身的武艺其实并不如何，更不用说和程大雷比了，现在他也没胆子上前和程大雷动手。
<br/>　　“来，你不是要争天下么，你我先过过手？”程大雷冲莫将难招招手。
<br/>　　莫将难往后退了数步，与程大雷距离更远了，此时的他，颇有种骑虎难下的感觉。
<br/>　　正这时候，原先悄悄溜走的人又退了回来，梨园外响起乱糟糟的声音。
<br/>　　“莫老大，莫老大……”
<br/>　　一人着急忙慌道：“大事，大事不好了。”
<br/>　　莫将难心里有种骂娘的冲动，我难道还不知道大事不好，这魔头放出来，事情怎么可能好得了。
<br/>　　“官兵，官兵杀过来了。”
<br/>　　“什么！”
<br/>　　莫将难一惊，的确，相比程大雷，还是官兵杀过来这件事更加可怕些。或者说，两件事情是一样的糟糕可怕。
<br/>　　莫将难原先命周桃去长风关盗印，就是想做件大事服众。但周桃阴差阳错捉来程大雷，那方军印自然就用不到了。
<br/>　　可这大印却为他引来祸事，那长风关守将陆长迢岂是善于之辈，东方童回到长风关后，怒发冲冠的陆长迢立刻引军来围剿农神庙前聚义的群贼。
<br/>　　莫将难做事不密，这件事在江湖上传得沸沸扬扬，难免官家就会知道。
<br/>　　现在陆长迢带着三千人马亲自杀来，若论单打独斗，这帮江湖豪杰谁也不输。但论群体作战的能力，他们绝非正规军的对手。
<br/>　　远程用弓箭射杀，近了几个人配合杀敌，陆长迢的人马很快就杀进梨园。
<br/>　　身周箭雨纷纷，程大雷也不敢托大，目前还是先逃命要紧。白元飞跃到程大雷面前，将找回来的匹夫剑交给他。
<br/>　　“大当家，风紧扯呼？”
<br/>　　“扯呼，扯呼。”
<br/>　　二人合力向外冲杀，有程大雷一剑开路，倒是谁也没办法阻拦他。可他这般表现，又委实过于抢眼些。
<br/>　　东方童在人群中发现了程大雷，口中大喝道：“大人，那就是程贼大雷。”
<br/>　　东方童其实并不是很迫切想要杀掉程大雷，如程大雷这种人，无论黑道白道都是要杀的。他主要的仇恨还是针对柳追风和周桃这对狗男女。
<br/>　　他带着一众手下向柳追风迫去，柳追风刚才一轮激战，已经是身受重伤。他本来身手不弱，此刻却是没有丝毫战意，只能像没头苍蝇一般护着周桃向外逃。
<br/>　　东方童跨马拦在他面前，口中喝道：“老贼，还我兄弟命来。”
<br/>　　“你先走。”柳追风推开周桃，独自面对东方童：“你来。”
<br/>　　程大雷和白元飞向外冲，然后他发现一件诡异的事情，自己面前的人越来越多。倒是这件事的始作俑者莫将难，面前没有几个人拦阻。
<br/>　　这是怎么搞的，本当家只是过来摸鱼的，你们找错人了吧。
<br/>　　陆长迢并不知道程大雷怎么到的这里，也不是很想知道。在他想来，程大雷出现在此地，肯定和莫将难一丘之貉，想要谋划造反。这种事，程大雷做得出来，说不得这件事，他还是主谋。
<br/>　　无论怎么想，程大雷都要比莫将难有价值。
<br/>　　他跨马拦住程大雷，口中喝道：“程贼，还不纳命？”
<br/>　　程大雷表情有些无语，道：“陆将，咱们井水不犯河水，你这样不好吧？”
<br/>　　“呔，如你这样的恶贼，人人得而诛之，今天我就为民除害。”
<br/>　　程大雷身边聚的人越来越多，再不走怕就没机会走了。况且，程大雷现在也没有座骑，想要突围也是不易。
<br/>　　“那就对不住了。”
<br/>　　程大雷突然跃起，身子跃在半空中，飞一脚将陆长迢从马上踹下来。陆长迢还没站稳，白元飞的刀就已架在他的脖子上。
<br/>　　“呔，想要他活着么？”白元飞喝道。
<br/>　　身为一军之帅，陆长迢不该铤而走险。更何况，他的武艺本就不如何，今天面对的又是程大雷这等级别的恶贼。
<br/>　　“都不要动，都不要动。”陆长迢摆着手，道：“程当家，咱们井水不犯河水，不至于如此吧？”
<br/>　　“不是如我这等恶贼，人人都该杀么？”
<br/>　　“玩笑了，委实玩笑了。”
<br/>　　程大雷将剑搭在他肩膀上，道：“说真的，我不想要你的性命。今天兄弟也不想趟这次浑水，让你手下人安生些，不然我认得你我的剑可未必认得你。”
<br/>　　“明白，明白。”
<br/>　　程大雷押着陆长迢往外走，长风关的士兵让开一条通道。
<br/>　　正这时候，前面有一匹快马冲过来，正是小白狼寻到此地，过来搭救程大雷的性命。
<br/>　　那头黑牛还跟在她后面，程大雷在陆长迢踹翻在地，翻身跃上牛背，白元飞夺了陆长迢的马，三人一起往外冲。
<br/>　　陆长迢晕头晕脑从地上站起来，扶正头盔后，口中大吼道：“追，追上他，一定要杀掉他。”
<br/>　　柳追风夺了一柄剑，在生命的最后关头，激发了他骨子里的潜力，长剑舞动，只见剑光不见人，眨眼间就有数人遭了他的毒手。
<br/>　　东方童虽然报仇心切，但也不敢硬攻。周桃一拉柳追风肩膀，口中喝道：“走啦。”
<br/>　　一群江湖豪杰被赶急了，也只得拔出兵器厮杀，向四面八方突围。
<br/>　　东方童只是稍微不注意，眼前就没了柳追风的身影。周围乱糟糟一片，也不知道他们跑到那里去了。
<br/>　　“找到他，一定要找到他。”
<br/>　　莫将难几个人也随着人群往外冲，他一腔雄心壮志，还未来得及施展就因为这场乱战被扼杀在摇篮中。
<br/>　　今日农神庙前聚义，绿林道死伤不计其数。但这当然无法满足陆长迢，他的目的是要赶紧杀绝，命令手下将士追杀，主要目标就是莫将难、柳追风、程大雷三人。
<br/>(本章完)
