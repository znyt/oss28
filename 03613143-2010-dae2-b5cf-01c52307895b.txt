        秦烈微微耸肩，黯然神伤地回答道：“我从她的弟弟口中得知，她是被人害死的。至于具体是被谁害死，怎么死的。我打听过，但是没有结果。”
<br/>
<br/>        “难道连警察都查不出来吗？”童筱颖不禁纳闷起来。
<br/>
<br/>        秦烈垂下眼帘，回答道：“连尸骨都未找到，怎么破案？”
<br/>
<br/>        “没找到尸骨的话，那应该叫失踪啊！为什么你就确定她死了呢？”童筱颖疑惑不解地反问。
<br/>
<br/>        秦烈既无奈又伤感地接着说道：“我本来是想去烟儿的坟头看看她的，但是她弟弟说，她的尸骨未找到，所以没有坟。而他们一家人又肯定，烟儿已经死了。”
<br/>
<br/>        “没找到尸骨的话，说不定她没死呢？”童筱颖安慰道。
<br/>
<br/>        秦烈苦笑：“但愿吧！但愿她还活在这个世上我不知道的角落里。”
<br/>
<br/>        “那你到现在，都还没交女朋友吗？没结婚？”童筱颖好奇地问。
<br/>
<br/>        秦烈微微点了点头。
<br/>
<br/>        顿时，童筱颖对这个男人刮目相看了。
<br/>
<br/>        痴情又专一……
<br/>
<br/>        “好吧！看在你对你的初恋女友痴情又专一的份上，我决定跟你交朋友了。不过，你也得相信我，我现在真的是已婚妇女。”童筱颖再三强调。
<br/>
<br/>        秦烈心里虽然仍旧对她说自己是已婚妇女这事表示怀疑，但嘴巴上还是应许了她的话：“嗯，好，我相信你。”
<br/>
<br/>        “我现在想去玩那个‘狂野飞天’的项目，你要不要跟我一起去？”童筱颖咧着嘴，笑着问。
<br/>
<br/>        秦烈毫不犹豫地点了点头。
<br/>
<br/>        而接下来的这几天，因为有了秦烈的陪伴，童筱颖的旅途过得很开心，他俩实行AA制，把澳洲好吃好玩的地方都去了一遍。
<br/>
<br/>        另一边，中国，滨江城。
<br/>
<br/>        连华生原本不想把这事告知傅夜沉，但是，事情似乎比他料想中的还要严重了。
<br/>
<br/>        那个该死的童筱颖，居然跟一个刚认识的男人结伴旅行，这不是在外给他家阿沉戴绿帽子是什么？！
<br/>
<br/>        真是“是可忍，孰不可忍”！
<br/>
<br/>        连华生找上傅夜沉的时候，傅夜沉还正在陪几位国外商界大佬们打高尔夫球。
<br/>
<br/>        他果真是对童筱颖很放心啊！
<br/>
<br/>        放心到他连华生都看不下去了。
<br/>
<br/>        趁着傅夜沉休息的期间，连华生走过来时，他傅夜沉还有心情问他连华生，要不要一起打高尔夫球？
<br/>
<br/>        连华生自然是没这个心情，反倒是很好心地提醒道：“我觉得吧！你有时间在这陪大佬们打高尔夫，倒不如去问问你老婆，最近是不是有了什么桃花运。”
<br/>
<br/>        “我不是让你关掉筱颖的监听定位器吗？怎么，你没关？”傅夜沉一边拧着矿泉水瓶盖，一边漫不经心地问。
<br/>
<br/>        连华生倒是振振有词起来：“你应该庆幸，还好我没关！要是关了，你老婆跟别的男人跑了，你都还被蒙在鼓里。”
<br/>
<br/>        他这算不算是好心的了？
<br/>
<br/>        虽然恨不得童筱颖离开傅夜沉，但是，他见不得童筱颖是以这种方式离开，这样太伤傅夜沉的自尊心了。

<br/>        傅夜沉听着连华生这话，不痛不痒地应道：“她不会。”
<br/>
<br/>        “有什么不会？我告诉你，没有哪个女人受得了男人对自己的‘冷暴力’，你现在对童筱颖不闻不问，正好，她的身边现在多了一个男人对她嘘寒问暖，还陪她游山玩水。”连华生接着说道。
<br/>
<br/>        傅夜沉仍旧没有一丝危机感，反而瞥了连华生一眼，不痛不痒地说道：“你是不是闲的没事做？”
<br/>
<br/>        “得了！我就直白地告诉你！我替你查了那个男人的底细，他叫秦烈，长得帅也很有钱，目前单身，更重要的一点是，他是夏玄烟的初恋男友！”连华生气急败坏地说。

<br/>        傅夜沉顿时凝了凝眉头。
<br/>
<br/>        连华生又说道：“你不会连夏玄烟是谁都没印象了吧？她是夏凌焰的姐姐，夏七叔的女儿，同时就是十爷的妻子婴宁！这么说来，算得上是你堂婶了哈！那个叫秦烈的男人，靠近童筱颖，目的不是已经显而易见了吗？”
