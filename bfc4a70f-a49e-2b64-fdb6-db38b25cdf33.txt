　　调整了一下麦克风的位置，台上那位搜查官缓缓开口，声音中带着一股久居上位的威严。
<br/>　　“大家好，你们大部分人，大概都认识我，但新来的小家伙们，可能对我还不太熟悉，那样，我不妨先做一个自我介绍。”
<br/>　　停顿了一下，干瘦搜查官用手指轻轻敲击着桌上的银色密码箱，继续说道：“我是丸手斋，特等搜查官，负责一区的指挥和调动，也是这场会议的发起人。”
<br/>　　听到丸手的话语，台下的搜查官们开始小声议论了起来，很多人早就听说过他的名号，但也是第一次见到本人。
<br/>　　“你们肯定很好奇这次会议的内容，相必也听到过一些传言，没错，之所以组织这场会议，是因为有一个大型的行动要大家共同参与。”
<br/>　　“在场的诸位都是经过严格选拔的优秀搜查官，有一些新人，但大部分人也都跟随我参加过一些战役了，所以，多的话我也不讲，你们就按照之前的流程，好好进行准备。”
<br/>　　“这次行动会在今天晚上正式开始，在场所有人，都要在十点之前赶到这里集合，届时会给你们安排具体的部署，希望大家都不要迟到。”
<br/>　　敲击密码箱的动作停顿了一下，丸手脸上露出有些冷酷的笑容，道：“当然，也不要忘记准备好自己的遗书。”
<br/>　　说完之后，丸手就把麦克风递交给了另一名搜查官，双手抱胸，靠在背后的椅子上，看着台下的众人。
<br/>　　另外一名搜查官接过话筒，简单介绍了一下自己，名字很平庸，刘哔对他没什么印象。
<br/>　　接下来的会议内容也比较乏味，大抵是有关这次行动的一些事情，已经需要提前做好的准备。
<br/>　　至于任务的目的性，却是没明确的指出来，不过刘哔计算了一下时间，也大致猜到了CCG的目的。
<br/>　　讨伐青铜树，这次战役在CCG的历史上，也是无比重要的一役，事情来得如此突然，略微打乱了一些他的计划。
<br/>　　先前在古董店里见到金木研的时候，他还没多大的紧张感，本来以为还会有一段缓冲的时间，没想到却即将开始了。
<br/>　　看了一眼坐在自己身边的木嶋式，刘哔犹豫了一下，伸手拉了拉木嶋式的衣服。
<br/>　　“班长，这几天我调查出了一件很重要的事情，应该和‘枭’有关，你看我要不要汇报上去？”刘哔语气有些严肃，征询着木嶋式的意见。
<br/>　　他要等的，就是现在的契机，CCG一旦得到了‘枭’的具体消息，必然会进行讨伐，而青铜树一役，失去店长的帮助，独眼‘枭’肯定会从暗处显露出来。
<br/>　　那样一来的话，两只‘枭’就被强制分离开了，而看过原著的刘哔可以肯定，CCG一定会将店长击杀。
<br/>　　那只不过是一枚不听话的棋子，CCG的高层中，没有人愿意店长继续存活于这个世界上。
<br/>　　而独眼‘枭’届时也被纠缠住，必然不会有机会捡走店长的尸体，那样的话，他有很大的希望将店长给吞噬掉。
<br/>　　一个SSS级的赫者，如果能按照计划将他吞噬掉，自己的实力一定会得到一个极大的提升。
<br/>　　即使失败，也无伤大雅，反正他也不会损失什么，况且因为店长的态度，不管是CCG这边还是喰种，对其都没有什么太好的印象。
<br/>　　他也不会因此惹上什么麻烦，没有人会因此怪罪他。
<br/>　　除了古董咖啡店里那几个可怜虫，反正那些人，他迟早都要把他们吞噬掉的。
<br/>　　“消息能确认吗？”木嶋式闻言转过头来，目光有些诧异，语气却是异常的凝重。
<br/>　　这可是一件不得了的事情，但他并没有对消息的真伪有多大怀疑，毕竟，‘枭’的资料在CCG属于绝密，绝对不是一个刚加入不久的新人可以查看的。
<br/>　　既然提到了‘枭’，想必也确实掌握了一些东西。
<br/>　　“虽然还有一些小的问题，但基本上已经可以确认了。”点了点头，刘哔一脸认真的回答道。
<br/>　　眼中闪过一抹惊喜，木嶋式拍了拍刘哔的肩膀，却也没多说什么，而是当即站起了身来。
<br/>　　“咦，木嶋班长，您有什么事情吗？”台上主持会议的搜查官一愣，有些不满的开口问道。
<br/>　　会议的事情被强行打断，他心中未免有些不爽，但碍于木嶋式的身份，他也没办法说些什么。
<br/>　　原则上来说，每一个小组班长都有在会议上主动发言的权利，无论多重要的会议，这项权利都不会被剥夺。
<br/>　　“我班上的成员刚才和我说了一件事情，我觉得有必要提一下。”拉了一把刘哔，木嶋式裂嘴笑了一下：“你们一定会对这件事情感到吃惊的。”
<br/>　　顺着木嶋式的力道，刘哔站起了身来，会议厅众人的目光，都被吸引到了他的身上。
<br/>　　不远处，铃屋什造坐在一群黑西服中间，正一脸兴奋的冲他挥着手臂。
<br/>　　虽然不是第一次被这么多人注视，但刘哔还是略微感到紧张，毕竟在这种地方，他万一露出什么马脚，就死无葬身之地了。
<br/>　　“我是三等搜查官，刘哔，前几天我在二十区追查一名喰种的时候，发现了一处隐秘的地方，似乎是二十区的喰种聚集地。”清了清嗓子，刘哔表情有些忐忑的说道。
<br/>　　他的话语，让会议厅掀起了一阵议论，丸手斋坐在台上，微微皱起了眉头。
<br/>　　只不过是发现了一处喰种聚集地而已，这个消息在平时自然是无比重要，但是在这种时刻，就有些不值一提了。
<br/>　　和底下那些搜查官不同，他可知道这次会议的目的性。
<br/>　　这种事情，私下里做一份报告就可以了，有必要在这样重要的会议上提出来吗？
<br/>　　丸手心中有些不耐，但是看到木嶋式脸上自信的笑容，想了想，还是示意主持人不要打断，耐心的倾听了下去。
<br/>　　这个家伙，虽然总是做一些奇怪的事情，但大部分时候，都不会让他失望。
<br/>　　“为了进一步确认事件的真实性，我跟进调查了一段时间，发现这件事情，和‘枭’有关。”脸上的表情归于平静，刘哔的话语如同一颗重磅炸弹一般，顿时将会议厅的气氛点燃。
