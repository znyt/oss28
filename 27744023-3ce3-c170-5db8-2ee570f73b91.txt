　　因为突破引来的浩大声势，吸引大批修士闯入酆都，并且触发杀阵的事情，慕清霄并不知情。
<br/>　　哪怕知情，他也不会皱一下眉头。
<br/>　　正所谓，人为财死鸟为食亡，他们为了子虚乌有的至宝，触动了鬼皇宫中的杀阵，哪怕死了也是活该。
<br/>　　此时，他已经跨越数百里，来到川西地界一座偏僻的山谷中。
<br/>　　传说，蜀山派似乎就坐落在这附近，当然，这都是民间传闻，但可信度很高。
<br/>　　蜀山派位于川西崇山峻岭之中，山峰淩空，宛若浮云，相传为仙家赐予人间的胜地。
<br/>　　自开派至今，历代弟子勤修仙术，入世降妖，并囚妖于锁妖塔，庇佑人间不受妖孽滋扰，深得百姓爱戴，在蜀地更是被尊为神仙。
<br/>　　来到川西后，慕清霄一路上，的确没有遇到一只妖魔鬼怪。
<br/>　　哪怕穿过崇山峻岭，就连小妖都没有见到一只，可见蜀山在周围的确起到了一定震慑作用。
<br/>　　蜀山与武林众门派交往不深，其修炼的蜀山仙术也是界于内功和道家炼养之间，但蜀山门徒修炼不为成仙，而为济世救人，和寻常道家又有所不同。
<br/>　　在《仙剑奇侠传》位面中，蜀山仙剑派建立在神树之根依附于盘古之心而形成的悬空山上。
<br/>　　蜀山为仙剑派是人界最大修仙门派，在一定时期同时也被视为武林中的第一门派，乃是求仙问道，拜师学艺的不二选择。
<br/>　　当初，慕清霄在古藤林中，获得了古藤老怪手中的一卷手札，其中记载了诸多蜀山的修仙法术。
<br/>　　当然，他来蜀山仙剑派的目的，自然不是为了学习蜀山的修仙法术，而是蜀山的修仙法术能给他带来多少掠夺积分。
<br/>　　掠夺积分自然是多多益善。
<br/>　　要知道，慕清霄当初获得的仅仅是一名蜀山修士的手札，其中记载的修仙法术只是一部分。
<br/>　　偌大的蜀山仙剑派传承数千年，根本不可能只有可怜的一卷手札，对于他来说，蜀山简直就是金窟。
<br/>　　不过，掠夺积分只是其一，更重要的是，紫萱住的灵溪谷似乎就在蜀山附近。
<br/>　　慕清霄寻找的同时，正好也要找个落脚的地方，蜀山仙剑派正好不错，虽然都是些道士，却也勉强凑合一下。
<br/>　　如果让蜀山各长老知道，在慕清霄眼中，蜀山只是个收集掠夺积分的落脚点，不知道会不会气的吐血三升。
<br/>　　慕清霄并没有御剑飞行，而是行走在崇山峻岭间，游山玩水的同时，也向着灵气浓郁的地方前进。
<br/>　　人间有十大洞天，三十六小洞天，七十二福地等‘仙界’，其中最著名的仙界，就是蜀山，因为那里是‘盘古之心’所在之处。
<br/>　　蜀山所在，无疑是灵气最浓郁的地方。
<br/>　　慕清霄对自然界的气息非常敏锐，跟着空气中灵气的浓郁度，估计就能找到蜀山。
<br/>　　“咦？”
<br/>　　就在这时，不远处的一条山间道路上，出现一批人群，其中两名元婴期修士，身后则跟着数百名普通人。
<br/>　　虽说是些普通人，但他们衣着华贵，有的甚至已经步入练体期。
<br/>　　至于最前面的两名元婴修士，身穿白袍，腰间挂着佩剑，头顶玉冠，丰神俊朗，似乎是蜀山的弟子。
<br/>　　踏破铁鞋无觅处，得来全不费工夫。
<br/>　　慕清霄本就打算去蜀山，现在正好遇到蜀山的弟子，倒不如一同前往，一路上也有个伴。
<br/>　　“律律……”
<br/>　　嘶鸣声响起，只见两只马蹄腾空，队伍前方的两名元婴修士已经拔出佩剑，脸上露出警戒的模样。
<br/>　　当两人看到道路中央的慕清霄后，脸色微微一愣，收起佩剑，抱了抱拳道：“阁下为何挡住我等去路？”
<br/>　　“两位可是蜀山弟子？”
<br/>　　其中一名弟子道：“正是，在下蜀山弟子李恒与师弟李毅。”
<br/>　　“正好，我与蜀山有些渊源，正好要去蜀山一趟，可否请两位带个路？”
<br/>　　李恒与李毅面面相觑，有些犯难，但后者似乎不是普通人，不好得罪，但他们奉命接引世俗弟子进入蜀山，时间有些紧迫。
<br/>　　“阁下，我师兄弟两人奉命接引世俗弟子进入蜀山，时间有些紧迫，蜀山山门就在前方千米之地，恐怕只能阁下自行前往了。”
<br/>　　话语刚落，李恒与李毅就快马加鞭，带着身后一群世家子弟向蜀山奔腾而去，速度比刚才还要快上一些。
<br/>　　微微摇头，看来蜀山也有规矩，并非蜀山弟子，不得进入山门，许多宗门都有这样的规矩。
<br/>　　慕清霄也没有为难两人，毕竟，不以规矩，不成方圆。
<br/>　　不过，知道了蜀山派的位置，事情就好办多了，这一路上的风景，慕清霄也已经看腻，快些办完事情离开才是。
<br/>　　想到此处，慕清霄就化作一道剑光，划过天际，转眼就跃过蜀山山门，出现在蜀山大殿上空。
<br/>　　突然，慕清霄从己身化剑术恢复过来，只见他的前面已经出现一名白发老者。
<br/>　　老者身穿黑白道服，手掌拂尘，发丝眉毛均以花白，仙风道骨，声音如微风和煦。
<br/>　　“阁下，蜀山重地，不得御空飞行。”
<br/>　　慕清霄微微一愣，道：“在下慕清霄，见过清微掌门。”
<br/>　　见清微目光一直在自己身上打量，殊不知后者内心当中早已掀起惊涛骇浪，对于慕清霄的容貌与修为发自内心的震惊。
<br/>　　回过神来，清微微微一笑，一张老脸顿时皱成菊花。
<br/>　　“阁下来蜀山可有何事？”
<br/>　　“纵观妙法，整个仙界中也唯有蜀山乃是妙法齐聚之地，在下几年前偶获一卷手札，上面记载的正是蜀山仙法，关法过后，在下对蜀山仙术颇感兴趣，与蜀山也算有些因果，不知可否应个长老职位？”
<br/>　　慕清霄也不想找弯弯绕绕的理由，至于能不能进蜀山，他有绝对的自信。
<br/>　　清微眯了眯浑浊的双眼，道：“阁下口气到不不小，开口就谋取长老一职，你可知蜀山乃在仙界的地位，长老一职何其重要？”
<br/>　　“在下对丹道有些成就，正好是一位九品炼丹师，十品以下皆可炼制，不知可否胜任？”
