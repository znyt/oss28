　　叶尘微微一愣，
<br/>　　显然没有想到对方竟然如此直接。
<br/>　　若是换成其他人，肯定可以做梦都想得到灭世仙君的传承吧。
<br/>　　可惜，他面对的叶尘！
<br/>　　且不说，叶尘前世乃是渡劫帝君级的存在，而灭世仙君却止步于合道仙君，两人根本就不是一个等级的存在，叶尘自然也不需要继承他的传承。
<br/>　　更何况，叶尘清楚的知道自己要走什么样的道，和灭世仙君所走的毁灭之道，可以说大不相同，也不需要拜师任何人。
<br/>　　“不好意思，前辈，我只想要灭世魔刀，不想做你的传承者！”
<br/>　　叶尘想也没有想，直接果断拒绝。
<br/>　　“你——说——什——么！”
<br/>　　灭世仙君先是一愣，继而两眼一瞪，仿佛要喷出火来，身上那原本就令人心悸的毁灭气息，顿时也越发的浓郁，似乎随时都能把周围的一切摧毁一般。
<br/>　　轰隆！
<br/>　　一股强大到令人胆寒的气息，朝叶尘铺天盖地压倒而来。
<br/>　　然而，
<br/>　　叶尘却依旧一脸云淡风轻，仿佛并没有感受到对方身上散发出来的毁灭气息，淡淡开口道：
<br/>　　“我有我自己的道，前辈的道，并不适合我，所以，我拒绝！”
<br/>　　“嗯？”
<br/>　　灭世仙君显然没有想到，面对自己如此强势的毁灭气息压迫，对方居然还能丝毫不为所动，
<br/>　　“小子！你可要想要好了，你当真不愿意成为本座的徒弟？”
<br/>　　灭世仙君继续恶狠狠道。
<br/>　　叶尘也依旧淡然，果断摇了摇头，
<br/>　　“不愿意！”
<br/>　　“你！”
<br/>　　灭世仙君顿时气得胡子都翘起来了，
<br/>　　“好好好！既然你不愿意，本座也不勉强，念在你为人还算诚实，本座不取你性命，你这就请吧！”
<br/>　　说完这话，灭世仙君长袖一挥，便要转身返回宫殿之中。
<br/>　　不料，
<br/>　　叶尘嘴角微微一扬，
<br/>　　“那不行！我好不容易才来到这里，灭世魔刀必须带走！”
<br/>　　霍！
<br/>　　灭世仙君听到这话，猛然转过头来，死死盯着叶尘，寒声道：
<br/>　　“怎么？小子，难道你还想从本座手中硬抢不成？”
<br/>　　说话间，灭世仙君身上的气息，一下子攀上到了极点！
<br/>　　就连悬浮在大殿中央的灭世魔刀，也开始发出嗡嗡作响的声音，似乎战意盎然。
<br/>　　“嗯？”
<br/>　　面对对方强大的气势压迫，叶尘眉头微微一皱，当即暗暗运转逆生百转神功，气息也开始节节攀升，最终在运转到第六重的时候，便一举盖过了灭世仙君，大有喧宾夺主的味道。
<br/>　　灭世仙君虽然生前修为超绝，如今毕竟只有一道残魂，而且又过了两千余年，实力早已经大不如前。
<br/>　　轰！
<br/>　　灭世仙君的残魂虚影，在叶尘的强大气势压迫之下，顿时一阵晃动，瞬间黯淡了许多，脸色不由得露出浓浓的惊骇之色。
<br/>　　很显然，灭世仙君也没有想到，对方这个只有元婴境的区区小子，体内竟然蕴藏着如此强悍的能量！
<br/>　　好在，叶尘的那股强大气势，一放即收，很快便消散于无形，这才让灭世仙君微微感到好受一些，不过心中却依旧掀起了惊涛骇浪，不敢再小瞧眼前这个年轻人。
<br/>　　叶尘淡淡一笑，冲着灭世仙君微微拱了拱手，道：
<br/>　　“晚辈自然不想冒犯前辈！不过前辈已经离世千年，这灭世魔刀留着也无用，只会让珍珠蒙尘，不如前辈再换一个条件，只要是晚辈能够办到，一定在所不辞！”
<br/>　　叶尘也不想白拿对方的好处，当即提出了一个折中的办法。
<br/>　　灭世仙君眉头微微一皱，并没有立即开口答应。
<br/>　　以刚刚叶尘所表现出来的实力，完全可以出手硬抢，应该也有很大的胜算，即便他拼尽底牌，和对方同归于尽，也是得不偿失。
<br/>　　“罢了！”
<br/>　　灭世仙君沉吟了片刻之后，忽然幽幽叹了一口气，道：
<br/>　　“你若是能立下天命誓言，为本座杀一个仇人，本座不但将灭世魔刀送给你，还将灭世魔刀中最大的秘密也告知与你！”
<br/>　　“哦？”
<br/>　　叶尘听到这话，顿时眉头微微一挑，
<br/>　　“不知道前辈，想要晚辈杀什么人？”
<br/>　　提及这个，灭世仙君深吸了一口气，道：
<br/>　　“本座要你杀的这个人，可不是寻常人物，他就是八大仙门之一无极宗的宗主，云苍生！”
<br/>　　叶尘顿时一愣，旋即想起了一些关于灭世仙君的生平，顿时便明白了。
<br/>　　据说，这灭世仙君和无极宗宗主云苍生，原本乃是挚友，后来不知道为何反目成仇，灭世仙君陨落，应该也和云苍生乃至无极宗，都脱不了干系。
<br/>　　灭世仙君见叶尘默然不语，还以为他是怕了，不由得冷冷一笑，
<br/>　　“如何？小子，本座提出对这个条件，一定把你都吓坏了吧？无极宗的宗主云苍生，当年的修为便不在我之下，如今应该早已经踏入渡劫帝君的层次，而且手下还有数以亿万计的门徒弟子，是真正的宇宙巨擘……”
<br/>　　灭世仙君的话还没有说完，叶尘直接一口打断，
<br/>　　“我答应你的条件！而且我不但要帮你杀了云苍生，我还要灭了无极宗，前辈意下如何？”
<br/>　　“你——说什么？”
<br/>　　这下顿时轮到灭世仙君傻眼了，不过微微错愕之后，他又忍不住哈哈大笑起来，
<br/>　　“小子，吹牛皮谁不会？你可敢立下本命誓言吗？”
<br/>　　“有何不敢？”
<br/>　　叶尘当即抬手在空中划出一道咒印，再次缓缓开口，道：
<br/>　　“我叶尘，以吾之本命起誓，百年之内，必杀云苍生，屠灭无极宗，若违此誓，天地不容！”
<br/>　　轰隆！
<br/>　　随着叶尘这道誓言落下，虚空之中顿时轰轰作响，连同整个宫殿，都一阵剧烈晃动，似乎随时都要崩溃了一般。
<br/>　　这一次，
<br/>　　灭世仙君顿时彻底傻眼了，
<br/>　　他怎么也没有想到，叶尘竟然真的会立下本命誓言，而且居然还限定了一百年的期限！
<br/>　　“疯了！疯了！你这小娃娃，简直就是疯了！”
<br/>　　“你知道云苍生是什么人吗？你知道无极宗意味着什么吗？”
<br/>　　“即便是你修炼天赋是不错，但没有四、五百年的修炼，也绝对不可能达到渡劫帝君的层次！”
<br/>　　“你知不知道，一旦立下本命誓言，倘若到时候无法做到，就会被天雷轰杀！”
<br/>　　“你真是本座见过的最不知所谓的疯子！”
<br/>(本章完)
