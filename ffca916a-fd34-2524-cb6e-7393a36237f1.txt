　　地狱三头狼和苏莫离开了，整个帝玄山脉区域，一片安静，久久无声。
<br/>　　时间缓缓流逝，直到过去了数十个呼吸的时间之后，无数的武者才逐渐的回过神来。
<br/>　　霎时间，声浪震天，惊呼不断！
<br/>　　“我的天！我不会看错了吧！”
<br/>　　“苏莫胜了，真的胜了！”
<br/>　　“不可思议，太不可思议了！”
<br/>　　“逆斩武帝，以武皇境四重修为，逆斩武帝之尊，太恐怖了！”
<br/>　　“苏莫无敌了，在苍穹世界再无敌手！”
<br/>　　“……”
<br/>　　无数嘈杂的声音，彼此交织，冲天而起，形成排山倒海的声浪，让得整片地域都沸腾了。
<br/>　　夕儿、司空炎、龙腾、彩芸等苍穹神宫的人，均是面上露出了笑容。
<br/>　　苏莫打败了黄泉魔宗宗主，武帝强者，在苍穹世界无敌了，那日后的苍穹神宫，将唯吾独尊。
<br/>　　帝释、海明江等帝玄宫和天涯海阁之人，亦是心中暗喜。
<br/>　　巫森一败，黄泉魔宗不足为惧了，他们的仇也可以报了！
<br/>　　剿灭黄泉魔宗、阴阳神宫、北堂世家，轻而易举。
<br/>　　“哈哈，帝宫主、海阁主，恭喜恭喜，恭喜你们找了苏莫这个盟友！”
<br/>　　这时，一声大笑声响起，战神殿殿主战空，遥望帝释等人朗声说道。
<br/>　　“帝宫主，你真是找了一个好女婿啊！”另一边的太玄宗宗主究楚北，满脸唏嘘之色。
<br/>　　“恭喜帝宫主，大仇即将得报！”
<br/>　　“海阁主，有苍穹神宫这个盟友，是你天涯海阁之幸啊！”
<br/>　　“苍穹神宫将要彻底复兴了！”
<br/>　　“……”
<br/>　　很多强者纷纷开口了，一方面是真心的恭喜，一方面是传达一个善意。
<br/>　　如今苏莫无敌了，苍穹神宫的崛起于世界之巅，众人不得不表达善意。
<br/>　　……
<br/>　　无尽虚空深处，空间之力肆虐，充满毁灭般的气息。
<br/>　　两道身影一前一后，疾驰在黑暗的虚空之中，快到极致。
<br/>　　前方的身影乃是一只庞大的妖狼，长着三只狰狞的脑袋，正是地狱三头狼战魂，其后方的身影自然便是苏莫。
<br/>　　战魂脱变与武魂，早已脱离了武魂的限制，几乎是飞天遁地无所不能，所以，即便巫森身死，其战魂依旧能逃走。
<br/>　　“你逃不掉的！”苏莫冷声大喝，今日无论如何，他也不能让对方逃掉。
<br/>　　虽然他不知道没有了肉身的战魂，还有多少的实力，但不彻底除掉，总归会留下祸患。
<br/>　　而且，宏青璇还在对方的身上，若是今日救不出来，就不知道要等到什么时候了！
<br/>　　“苏莫，你会付出代价的！”地狱三头狼口吐人言，声音浑厚异常，但依稀可辨，正是巫森的声音。
<br/>　　“果然！”苏莫闻言心中讶然，巫森果然没有死，这战魂活着，巫森就不会死。
<br/>　　战魂当真玄妙，肉身毁灭的情况下，居然都能让人存活。
<br/>　　不过，没有了肉身，想必这战魂也不能发挥出多强的实力吧！
<br/>　　地狱三头狼的速度快的不可思议，但苏莫御剑飞行的速度，并不比对方慢，他将本命灵剑催动到至极，稳稳的追在对方的后方。
<br/>　　不过，两人的速度相差无几，短时间内，苏莫很难追上对方。
<br/>　　咻！
<br/>　　咻！
<br/>　　就这样，两人一追一逃，很快便过去了半个时辰，两人飞行了无数万里。
<br/>　　这个时候，苏莫和地狱三头狼之间的距离，稍稍有些拉近，约莫还有三千余里。
<br/>　　只需在拉近千里距离，苏莫就有把握展开攻击，击杀对方了。
<br/>　　但就在此刻，苏莫面色微变，立刻眉头紧皱了起来。
<br/>　　因为，他大三宝的时效到了，一股无比虚弱的感觉，立刻席卷全身。
<br/>　　“糟糕！”
<br/>　　苏莫暗道不好，虚弱之感排山倒海一般席卷他的身心，让他的速度立刻大减了下来。
<br/>　　他消耗了六成的精气神，头发已经半白，现在大三宝术的时效一过，顿时虚弱无比。
<br/>　　苏莫速度大减了下来，但地狱三头狼的速度却是丝毫不减，呼吸之间便如同一道黑色的流星，消失在了他的视线之中。
<br/>　　“来的真不是时候！”苏莫无奈的停下了身形，满脸的郁闷之色，还是被对方逃了！
<br/>　　秘术终究是秘术，还是不如自身的实力重要，关键时候保命可以，但难堪大用。
<br/>　　沉吟了一番，苏莫认清方向，向九幽魔渊的方向飞去。
<br/>　　既然无法追上地狱三头狼，那就先去毁灭黄泉魔宗的老巢。
<br/>　　此事不能耽搁，不然给了黄泉魔宗时间，怕是整个黄泉魔宗都会隐藏起来。
<br/>　　苏莫在灵剑上盘膝坐了下来，分出一缕心神控制本命灵剑，随即闭目调息，争取早日恢复消耗的精神力。
<br/>　　即便苏莫没有极力催动灵剑，他的速度也是奇快无比，大概两日之后，他便来到了九幽魔渊。
<br/>　　伫立在虚空中，望着下方深不见底的魔渊，苏莫眸中杀意沸腾。
<br/>　　随即，他毫不犹如，身形向下方的魔渊中冲去。
<br/>　　一路畅通无阻，很快苏莫便来到了魔渊之下那处庞大的地下世界。
<br/>　　远处，阴罗城、阎罗城等四座城池，尽数出现在了他的视线之中。
<br/>　　“嗯？”
<br/>　　苏莫灵识一扫前方的四座城池，顿时眉头一皱，因为此刻的四座城池之中，虽然有数十万黄泉魔宗之人，但全部都是低级武者。
<br/>　　四座城池之中，甚至连一名武王境的武者都没有，全部是武王境以下的弟子。
<br/>　　这说明什么？
<br/>　　说明黄泉魔宗的强者，已经全部撤离了九幽魔渊，而这些低级的弟子，已经全部被舍弃了。
<br/>　　“动作真够快的！”
<br/>　　苏莫低声自喃，不过，那些逃走的黄泉魔宗之人，包括巫森和二魔老等人，别想逃出他的手心。
<br/>　　日后就算翻遍整个苍穹世界，他也会将这些人找到，然后全部灭杀。
<br/>　　唰！
<br/>　　随即，苏莫身形如电，直接向四座城池上空飞了过去。
<br/>　　“哈哈！听说苏莫已经宗主被杀死了！”
<br/>　　“不错，现在圣宗的所有强者全部出动了，要扫荡整个苍穹世界！”
<br/>　　“哈哈哈！要不了多久，我们黄泉圣宗就要一统天下了！”

<br/>　　四座城池之中，嘈杂的声音此起彼伏，兴奋的呼声不断。
<br/>　　黄泉圣宗的无数弟子，根本不知道外界的情况。

<br/>　　嘭嘭嘭！！！！
<br/>　　下一刻，无数的闷响声、爆裂声，疯狂的响彻而起。
<br/>　　四座城池之中，数十万黄泉魔宗的弟子，受到吞噬之力的影响，身体全部爆炸开来。

<br/>　　只是眨眼之间，数十万黄泉魔宗弟子，便齐齐陨落，变成了碎片残渣。
<br/>　　一时间，无数的玄力、血气、尸气和武魂混杂在一起，形成滚滚浪涛，向天空上狂涌而去。
<br/>　　天空之上，苏莫身形挺拔，脸色冷漠，正冷眼望着这一切。
