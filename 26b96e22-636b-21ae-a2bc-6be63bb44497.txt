　　　　两地遥遥而望，涂小安的意识力却掌控一切。
<br/>　　他那双妖治般的明眸浑然一眯，数里定生死，成为蛇精层次的涂小安，实力难以想象。
<br/>　　昏暗的小巷子中，鹰钩鼻男子耳朵里哄了一声，如同被尖针刺了一下，全身都有些麻木了，一股强大而阴冷的气息瞬间锁定住他，让他甚至都难以动弹。
<br/>　　不论是他，还是空中的蚊子，都陷入了一只无形的巨大掌心中。
<br/>　　“好强...”
<br/>　　区区一个小镇，既然隐藏着如此可怕的一位高手，鹰钩鼻男子内心震撼的难以复加。
<br/>　　未见其人，未闻其声，可对方的威势却如涛涛江水汹涌而来。
<br/>　　他甚至完全不知道对方在哪，这种未知的恐惧感更为折磨人。
<br/>　　猛然间，鹰钩鼻男子狠狠的倒吸了一口凉气，空气中出现了一道道爆响声，他放眼望去，肝胆俱裂。
<br/>　　只见空中一百只白色蚊子纷纷的自爆了起来，簌簌而落。
<br/>　　仅仅片刻的工夫，这一百只他尽心培养出来的白色蚊子全军覆没，留下了一地的蚊子尸体。
<br/>　　怎么会这样。
<br/>　　他的眼珠子都要惊的掉下来，忽然，他想到了最关键的一点，朝着那只银白大蚊子看去。
<br/>　　银白大蚊子距离他最近，但此时这只大蚊子发出尖锐的怪叫声，像是非常痛苦，好像有什么东西挤压着它的身体。
<br/>　　“不好，它不能死”
<br/>　　白色蚊子全死了，鹰钩鼻男子心疼归心疼，但还是能够接受，但这只银白大蚊子要是也自爆了，那么等于要了他半条命。
<br/>　　“走...”
<br/>　　鹰钩鼻男子的体表外迸发一股惊人的能量波动，将自己身上的束缚缓解了不少，一只手朝空中伸出，将银白大蚊子一把抓住，直接抽身遁走。
<br/>　　小巷子中，人影消失，只有一地白色蚊子的尸体。
<br/>　　半晌后，一位清秀少年出现了，看着一地的蚊子尸体，不为所动，目光朝着某个方向看去：“在白镇，在我的手里，你逃的了吗”
<br/>　　又是一位人类觉醒者，看起来也不过寥寥。
<br/>　　他虽然挣脱了涂小安的意识力掌控，但也最多做到这种地步而已。
<br/>　　涂小安目光深远，他已经锁定住了那位人类觉醒者的气息，敢破坏本蛇王的蛇王节，我看你是活腻了。
<br/>　　咻...
<br/>　　清秀少年瞬间原地消失。
<br/>　　此时，早已入夜，白镇这一天到晚都不平静，开始在一片喜庆的节日氛围中，晚上却笼罩在一片恐慌中。
<br/>　　如今，一道黑色影子在挨家挨户的屋檐中极速穿梭，最后来到了最人烟罕至的佘山山脚下。
<br/>　　因为大晚上，恐怕没有人会来到这里。
<br/>　　黑色影子出现在了佘山大青山的上面，鹰钩鼻男子一脸苍白，挂着心有余悸。
<br/>　　“应该甩掉了吧”
<br/>　　他重重的吐了一口浊气，心中苦笑，万万没想到，在这个小镇，还能碰到一位隐藏的高手，杀人于无形。
<br/>　　早知道这里有隐藏的高手，他绝对不会选择在这里犯案。
<br/>　　现在好像惹到了当地的地头蛇。
<br/>　　如今别说继续犯案，自己还要被追杀，他现在在这里落脚，地利位置还算不错。
<br/>　　一是这里够隐蔽，二是后方便是山林，要是一个不对劲，他可以一溜烟的进山躲避。
<br/>　　此时，山风从山上肆无忌惮的席卷起来，带着一抹阴冷跟寒气，犹如无形间有人在你耳边吹风，让人毛骨悚然。
<br/>　　鹰钩鼻男子摊开手中，十公分的银白大蚊子缓缓的飞了起来，发出了几声的怪叫。
<br/>　　看到这只大蚊子没事，他总算松了一口气，刚才真的好悬，要不是眼前这只大蚊子是觉醒生物，生命力异常的顽强，恐怕也会随着那些白色蚊子一样的原地自爆。
<br/>　　“这下麻烦了，不但惹了一个未知的高手，自己手中的白色蚊子也全部死光”
<br/>　　鹰钩鼻男站在大青石上面，一脸的阴郁。
<br/>　　光靠这一只银白大蚊子，他是没办法快速的获得血精，完成任务回去。
<br/>　　几番思忖下，他越想也越气，有种偷鸡不成蚀把米的失败，就在这时，他的耳朵动了动，山风轻拂间，一阵不轻不重，如闲庭信步的脚步声在山脚下响起。
<br/>　　鹰钩鼻男子的目光骤然看去，暗地里惊涛骇浪：是他追上了？
<br/>　　柔和月光下，一位清秀少年噙着一抹淡笑走到了大青石的外围。
<br/>　　他轻吟浅笑，一袭白衣，清秀俊逸的脸庞在月光的映照下，透着一抹邪魅，眼神似是冷淡无波又似是含着高高在上的桀骜。
<br/>　　来人一入眼，鹰钩鼻男子大吃一惊，好年轻啊，能杀人于无形的高手，就是他？
<br/>　　清秀而透着邪魅的少年走来之后，微微抬头，掀开眼，望着他，淡淡的道：“从大青石上滚下来，这块石子可不是你这种外人可以站的”
<br/>　　“你是谁”鹰钩鼻男子沉着声的开口。
<br/>　　涂小安也细细的打量了这位人类觉醒类，三十多岁，其貌不扬，身上隐隐有能量的波动散发。
<br/>　　旋即，他注意到了对方旁边悬浮着一只银白大蚊子，月光映照下，通体既然闪烁光泽，不丑陋，不恶心，到还有点惊艳。
<br/>　　还真是有意思，一只觉醒的蚊子，等级在十级以上。
<br/>　　鹰钩鼻男子见对方无视了自己的话，反而将目光投放到大蚊子身上，还露出饶有兴致的表情。
<br/>　　这是无视他，赤果果的不将他放在眼中。
<br/>　　如果要是平时，鹰钩鼻男子岂能受这种气，可之前对方无形中就杀死了他一百只蚊子，这种手段，让他非常的忌惮。
<br/>　　涂小安深深的看着这只银白大蚊子，像是看的不真切了，妖治的眼眸浑然一眯，空间荡漾。
<br/>　　只见那只悬浮的大蚊子浑身僵硬起来，顿时发出了怪叫声。
<br/>　　“给我过来”
<br/>　　涂小安含此轻吐，话音刚落，只见那只大蚊子诡异的朝着他这边飘了过来。
<br/>　　鹰钩鼻男子大惊失色，想阻止都晚了，内心对这位浑身邪气的少年更为的忌惮。
<br/>　　十公分的银白大蚊子就这样轻飘飘的飞到了涂小安的眼前，除了怪叫，浑身被强大的意识力束缚。
<br/>　　“给我洞察”
<br/>　　“叮...觉醒类变异银蚊，等级十二级，能够吸食人的精血”
