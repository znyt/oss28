　　在众人一脸怪异的眼神下，两位祖巫后土、玄冥与月神望舒和他的徒弟嫦羲留下来。之后，众神心思各异的走了。
<br/>　　他们也没有心思来管别人的私事，此次论道，他们也收获巨大，需要回去好好感悟消化。
<br/>　　“见过三位道友，一别经年，想不到元始道友更是成圣了，小妹在此恭喜了！”
<br/>　　众人走后，后土带着玄冥上前，对着元始三人拱手恭喜道。
<br/>　　“后土妹子，多年不见，你们过得不错吧。有没有找到适合你们巫族的道了？”元始明知故问道。
<br/>　　“多谢元始大哥关心，我们巫族很好。”
<br/>　　后土淡然一笑，并没有多说。
<br/>　　元始也知道贸然问别人族中之事不好，所以也没有在此事上多问，而是对她说道：
<br/>　　“你们巫族虽无元神，但却有灵魂。吾此处有一套锤炼灵魂之法，或许对你们巫族有用，今日便传给你们。”
<br/>　　说着，元始伸出拇指一点，顿时，一篇名为《三魂九炼》的灵魂法门便直接传入后土与玄冥的脑海之中。
<br/>　　这是元始闲暇时创出来的一门简单锤炼灵魂的法门，没有过多要求，方法很简单，就是需要不断地锤炼灵魂，将灵魂当做钢铁一般，像炼铁一样不停的淬炼。让灵魂变得更加凝实，更为强大。
<br/>　　灵魂到最后并不比元神差，甚至更强。因为万物有灵，但其中却很多都不会有元神。灵魂是天地间最为玄妙的东西，也是最为普通的东西。普通到随处可见，但它又最为神奇，因为没有人能够完全参悟透它。
<br/>　　灵魂是上天赐予众生的东西，它不可复制。世界上没有完全相同的两道灵魂。
<br/>　　将灵魂修炼到最后，同样强大无比。
<br/>　　“啊？！”后土错愕，感受到脑海之中的东西后，连忙感谢道：
<br/>　　“多谢元始大哥！”
<br/>　　就算她不修炼元神，也知道灵魂的重要性，要知道灵魂的修炼，在洪荒之中，也不可多见啊。若是知道哪里有灵魂修炼之法，那都可以让众人争破头颅了。
<br/>　　如今元始竟然将如此重要的东西给她。她真不知道说些什么是好。此时说那些感谢的话，那就太虚了。
<br/>　　“不必多礼，只要对你们有用就行了。”
<br/>　　元始摆摆手，满不在乎道。这门灵魂修炼法门只是他手中最为简单的一门，他还有更好的，但巫族与他又没有太大的关系，他又何必多管闲事。给他们一套简单的法门，还是看在后土的份上。
<br/>　　他知道后土日后会身化轮回，从此后土不复巫。后土虽然没有元神，但若是如今将灵魂锻炼强大起来，或许以后可以保存一丝真灵，不至于彻底陨落。
<br/>　　当然，这些还只是他的猜想，至于能不能成功，那就要看她自己的了。
<br/>　　“望舒道友，我们又见面了。呵呵！”
<br/>　　元始看向边上的这位高冷的太阴之主、绝世月神望舒，微笑着说道。要知道，她如今还是自己的俘虏呢。
<br/>　　“哼！若是可能，我希望一辈子不会再见到你。”望舒冷哼一声，冷冷道。
<br/>　　一旁的后土与玄冥见此，则有些好奇的看着元始与望舒两人，想知道他们之间发生了些什么，为何这位望舒道友会如此生气。
<br/>　　不过，这是人家的私事，她们尽管好奇，但也感到此时站在此地，颇有些不适。于是，她们便向着元始告辞离开了。
<br/>　　“哈，道友这是怪我咯，本座当初只是想取你太阴星上的一点本源，又不是要毁去你太阴星，你至于那么拼命吗？”在两人走后，元始看着望舒淡淡道。
<br/>　　“哼！”
<br/>　　望舒冷哼一声，便不再说话，显然是对于当初之事，她一直耿耿于怀。
<br/>　　“哼！你哼什么哼？说起来，道友你如今还是本座的俘虏呢。”
<br/>　　说到此处，元始有些好笑。当初自己只想取点太阴星本源，谁又曾想到会得到两个俘虏？
<br/>　　“你——”
<br/>　　望舒冷眼横翘，表示自己十分生气。
<br/>　　“好了，本座不是叫你来与你废话的。本座是要提醒你一句，今后小心妖族些。”
<br/>　　“你什么意思？”
<br/>　　“妖族如今占据太古星辰修炼，而你太阴星又是众多星辰之中最为强大的两颗至尊星之一。若是你是妖族，又会如何？”元始淡淡道。
<br/>　　望舒若有所思，她本身就是太古强者，自然不笨，听元始这么一说，她也知道了其中的厉害之处。不过，她的嘴硬，显然不会承认，而且还冷声道：
<br/>　　“要你多管！”
<br/>　　“你们如今可是本座的俘虏啊。好歹也算作是本座的人了。若是今后被妖族算计了。那岂不是丢了本座的脸吗。”
<br/>　　“你——”
<br/>　　听到元始再次提起俘虏一事，她显然是有些生气了，更是后悔当初不应该阻拦他。
<br/>　　“好了，这是你的月精轮，今日还你。言尽于此，听不听在你？”
<br/>　　说完，元始扔下一个银月般的银轮给她，便不再理会她。
<br/>　　“哼！羲儿，我们走！”
<br/>　　望舒接过月精轮，冷哼一声，便带着自己的徒弟嫦羲头不回地走了。
<br/>　　“呵呵！”
<br/>　　两人走后，元始轻笑一声，便不再多管。反正自己已经提醒了她们，听不听在她。今后若是出来事，也与自己无关。
<br/>　　今日提醒她二人，还是看在昔日自己到太阴星上取过本源的份上，她们与自己有些交情。若是换做他人，他才懒得管呢。
<br/>　　而且自己当初对她有那么一瞬的心动。但如今，这么多年已经过去了，那些感情啊什么的，他也早已看淡了。他如今追求的是大道，在这途中，所有会拖累他的东西，他都能抛弃。
<br/>　　不能够追上自己的，还想它干什么。渐渐地，随着时间的流逝，我们或许就会遗忘一些。直到最后，什么都没有剩下了，在他眼中，只有大道。
<br/>　　或许正是因为如此，他才能登上大道之巅。
<br/>　　“太上忘情吗？不知道这样是对是错？”元始喃喃道。
<br/>　　“管他的，今朝有酒今朝醉，明日愁来明日忧！管那么多干什么？自己高兴就行了！”紧接着，元始放开心情，便不再管这些。
<br/>　　“无论前路之上有何种困难，吾都无所畏惧！管他诸天神佛魔，还是漫天仙圣邪，只要敢来阻本座。本座便能将他们轰成碎片。”
<br/>　　“谁阻，谁死！”
<br/>　　说完，元始迈着坚定的步法，转身走进玉虚宫悟道台。
