<br/>				场中两人的较量让周围空气中弥漫着一股强大的灵力气息以及威压，几个隐世家族的家主感觉到他们身后家族子弟在那股强大威压之下感到不适，当即挥手为他们摒去强大威压。<br/>　　<br/>　　看着两人越战越激烈，他们不由相视一眼。虽然卓君越的实力身手都很强，但，久战之下已经不渐处于下风，而罗家大长老的攻击却越发猛烈，似乎还真如那罗家家主之言，想要让他打伤出一口气，因此，他出手之凌厉狠辣，让他们见了都不由的心口微提。<br/>　　<br/>　　罗家大公子和二公子见了，也不禁有些担心，看来他们父亲的意思还真的是在这较量中给他们一个狠狠的教训，要不然大长老也不会得令出手，招招狠厉。<br/>　　<br/>　　只是，在他们两人看来，这做法则有些过了，毕竟卓君越他们也没做出什么不可饶恕的事情来，若只是单单为了出气而将人重伤，似乎……<br/>　　<br/>　　想到这，两人相视一眼，来到他父亲的身边劝说着：“父亲，分出胜负就好，让大长老手下留情吧！凡事留一线，对谁都好。”<br/>　　<br/>　　“这事情你们两个不用管，为父自有主张！”罗家主沉声说着，警告般的扫了两人一眼。<br/>　　<br/>　　“父亲……”罗大公子还想劝说，却被一旁的两名长老拉到一旁了。<br/>　　<br/>　　“大公子，家主做事自有主张，这亲事不成还让我们罗家闹出这场笑话，而且他们两人又强行闯入我们罗府，于情于理，我们都应该给他们个教训的。”一位长老说着，示意他们两人不要再说。<br/>　　<br/>　　见此，罗大公子轻叹了一声，摇了摇头，不再多说。<br/>　　<br/>　　那边，凤九将他们的话听在耳中，只是朝那罗大公子看了一眼便淡淡移开了。她的目光落在卓君越的身上，正好看见他在避开罗家大长老拍出的一掌时，正好身体往后退去，旋身一落，稳稳落在圈外。<br/>　　<br/>　　当那罗家大长老还想趁势追击时，却见他已经落在圈外，那一瞬间，他微愣了一下，眉头微拧的看了卓君越一眼，这才收手退回。<br/>　　<br/>　　只是那神情，似乎对没有将他重伤便让他落败有些不甘。<br/>　　<br/>　　看到这一幕的罗家罗主目光微沉，朝那卓君越瞥了一眼，抿着唇将阴沉的目光收回。<br/>　　<br/>　　那罗家大公子见此，则暗暗松了一口气。而那罗家的四小姐躲在不远处看着并没过来，见他没伤着，也是轻松了一口气，提起的心总算是因此而放了下来。<br/>　　<br/>　　她目光痴痴的看着那一手负在身后站在圈外的卓君越，看着他冷漠不带一丝表情的面容，不由的心中有着不舍与迷恋。<br/>　　<br/>　　打从第一眼她就看中了他，就是他们罗家中的年轻一辈的子弟也没有一个如他一般出色的。且不说他们罗家了，就是那另外的几个家族当中也没有可与之相比的年轻男子。<br/>　　<br/>　　尤其是在看到他与大长老交手时的英姿之后，更是痴迷得无法自拔，就这样让他离开，她心里真的很不甘。<br/>
