　　　　蛇博士不断的盯着涂小安的左眼看，慢慢的起身，慢慢的凑进，然后姿势变的极为的暧昧。
<br/>　　涂小安稍稍的向后仰，看博士这样子，大有将涂小安的左眼挖出来做研究的意思。
<br/>　　都说科学家都是疯子，碰到自己感兴趣的事情，就什么不管不顾了。
<br/>　　“博士，喝茶”
<br/>　　涂小安不得不打断他，在凑过来，他都怕自己的初吻要没了。
<br/>　　咳咳...
<br/>　　陈默自己知道有点失态，尴尬的笑了起来，但尴尬的脸上还挂着浓浓的遗憾。
<br/>　　“博士，你都这样了，还在做研究？”
<br/>　　涂小安忍不住的问。
<br/>　　陈默瞪了他一眼：“我是断腿，不是断手，有什么影响”
<br/>　　好吧。
<br/>　　涂小安讪讪一笑。
<br/>　　“那次在蛇王庙，我看你拿出了一条红白相间的蛇，那蛇现在还在吗”
<br/>　　“怎么了”涂小安知道他说的是太阴蛇。
<br/>　　陈默道：“那天我没看仔细，没看出是什么品种的蛇”
<br/>　　敢情他惦记上了自己的太阴蛇。
<br/>　　“现在不在”
<br/>　　陈默忙道：“我就看看，不做研究”
<br/>　　“真不在，借别人当宠物玩两天了”
<br/>　　顿时，陈默猛然吸了一口气，那么极品的蛇，借给别人当宠物？
<br/>　　暴殄天物呀。
<br/>　　那日，深度中毒晕迷者完全是靠着那蛇的蛇血才清了毒素。
<br/>　　一条比太玄蛇更要有药性的蛇，陈默怎能遗忘，按他以前的话来说，这是一条绝世蛇种呀。
<br/>　　“博士，改天我也借你玩两天”
<br/>　　“好，一言为定”他激动了起来。
<br/>　　涂小安翻白眼，听不出这是客套话吗。
<br/>　　两人短暂的沉默了下，蛇博士盯着涂小安看个没完，不知道打着什么坏主意，然后问：“你来温城有什么具体的打算吗”
<br/>　　“练级打怪”
<br/>　　“什么！”
<br/>　　涂小安解释：“我是一条正蛇，自然要为人类做点贡献，我相信这大都市应该藏着不少为非作歹的精怪吧”
<br/>　　二阶为精，精怪之称，就是一些做伤天害理的二阶觉醒生物，如猫精这种知恩图报，至善至美的二阶生物，可称为二阶精灵。
<br/>　　就如三阶生灵一样，是正，三阶妖灵是恶。
<br/>　　涂小安想快点证道，就要赶紧将等级提升上去，那么必须去杀一些高等级的精怪了。
<br/>　　说好听点，就是降妖伏魔。
<br/>　　正义的化身。
<br/>　　“你有这种觉悟，我很高兴”陈默欣慰的看着涂小安，他还真怕这条蛇精来温城胡作非为。
<br/>　　“但凡开了灵智，成为二阶觉醒生物，都很容易迷失，伤天害理，胡作非为，这种精怪，就算一脚迈过去，也会被天雷天火劈的灰飞烟灭，你的觉悟很高，不愧是涂兄培养的正蛇”
<br/>　　“唯有正途，得福报，才能减少天雷天火的数量，从而扛过去，步入三阶生灵”
<br/>　　陈默想说，你有这种觉悟，怪不得能证道成蛇王。
<br/>　　其实他压根不知道，涂小安距离证道还远着。
<br/>　　福报？
<br/>　　涂小安对于福报一直很模糊：“博士，做好事的二阶生物，就能得到福报吗，得福报还会影响到渡劫？”
<br/>　　陈默点了点头：“善有善报，恶有恶报，苍天饶过谁，那样已阴邪手段提升实力的二阶生物，最后渡劫，它们就会知道怕了”
<br/>　　涂小安慢慢的听着，最后明白了，这个渡劫的天雷天火的数量是不固定的。
<br/>　　二阶觉醒巅峰生物渡劫，天雷天火一般各为九道，但是你善事做多了，得了福报，那么这个数量会减少。
<br/>　　比如降下的天雷变成了六道就结束，网开一面。
<br/>　　但你如果大量的屠杀，有伤天和，那么就不是九道了，十八道也不一定。
<br/>　　劈不死你，也用天火烧的你灰飞烟灭不可。
<br/>　　也正因为如此，人类世界才会如此和谐。
<br/>　　当然，也依然有不少觉醒生物为非作歹，想让自己变的强大，只能走捷径。
<br/>　　因为自身实力弱，也很容易成为人类修武者的灵丹妙药。
<br/>　　涂小安并不怕天雷天火，但他想快度的提升等级，又不想大造杀戮，也就只能去找那些“坏蛋”了。
<br/>　　还能一举二得。
<br/>　　“我们温城的修武界鱼龙混杂，隐藏世家自称一派，还有修炼有成的精怪，人与妖的相互争斗，谁输了就成了谁的修炼资源”
<br/>　　“除了世家，很有不少门派，为争夺资源，随时都是一番腥风血雨，你来到温城，也要小心点”陈默提醒。
<br/>　　涂小安听着：“博士，有那么多精怪吗”
<br/>　　“若是凡人，什么都看不到，若是修炼者，你随处可见，甚至一家大公司的老板，都有可能是证道的三阶生灵所化”
<br/>　　陈默目光深远，他显然知道的比涂小安这种小白要多的多。
<br/>　　涂小安对他这句话深以为然，早上他不就进一家面馆发现了老板娘应该就是三阶生灵吗。
<br/>　　就是不确定是生灵还是妖灵了。
<br/>　　这大都市，水就是深。
<br/>　　一只猫，就是二十九级的猫精。
<br/>　　一位面馆老板娘，就是三阶生灵。
<br/>　　安分守已的，跟人类和谐同处到没什么，要是那种胡作非为的，就是一场灾难了。
<br/>　　陈默又给涂小安补充了不知未知层面的知识，随后，犹豫的说：“你可否帮我一忙”
<br/>　　“博士你说”涂小安没一口就答应，毕竟两人之前关系可是仇敌也差不多，还没到两肋插刀的地步。
<br/>　　“你昨天在温城校园，可否发现校园又所不同”陈默忽然问。
<br/>　　涂小安立刻知道他有所指：“我发现还有一件隐藏的古学院”
<br/>　　“果然瞒不过你，那间古学院是人类修武者的启迪，为国家提供新鲜的血液”
<br/>　　“每一年，都会招收一些刚突破身体而觉醒的人类，作为培养”
<br/>　　陈默继续道：“我要你帮忙的是，我朋友的儿子即将要进入古学院，我希望你能护他周全”
<br/>　　“进入古学院还能有生命危险不成？”涂小安好奇的问。
<br/>　　陈默开口：“修炼一道，好勇斗狠，血气方刚，一个受不住力，就会出现伤亡”
<br/>　　“古学院虽然禁止不准生死斗，但每年也有不少的死伤”
<br/>　　涂小安明白了，这是让他当别人的保镖。
<br/>　　这种事情，他可没时间去做，况且：“古学院能让我进去吗，我可是蛇精，应该瞒不住人类高手吧”
<br/>　　“这种事情，我自由分寸，我跟古学院的副院长有交情，可以通融”
<br/>　　涂小安呆滞，人类的修炼学院，让一条蛇精进去，这都能通融呀...
<br/>　　要求挺宽呀。
