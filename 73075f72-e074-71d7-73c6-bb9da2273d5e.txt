　　苏莫回到火元峰之后，火元圣师也没有再叮嘱什么，便让他回去了。
<br/>　　回到洞府之后，苏莫便进入玉镯空间，立刻就开始了修炼。
<br/>　　这一次真圣界之行，不仅要在真圣榜上留名，还要面对巫族的杀机，可谓是危险重重，让他感受到了很强的压力。
<br/>　　而且，他被排在真圣榜第一，到时候针对他的人，估计不可能仅有巫族之人。
<br/>　　所以，他现在要争分夺秒的提升实力。
<br/>　　木属性功法《生生不息诀》运转，苏莫很快便沉浸在了修炼之中，水仙录和三千炎焱诀早已达到第一层的巅峰，现在只差将生生不息诀也修炼到第一层的巅峰了。
<br/>　　生生不息诀他早已入门，时间上倒是足够了。
<br/>　　不过，距离真圣界开启，只剩下两个多月的时间了，到时候肯定要提前出发，他不得不加紧修炼进度。
<br/>　　时间如白驹过隙，灵晶大量的被消耗，苏莫的修炼的速度着实不慢，一个半月的时间，便将生生不息诀顺利的修炼到第一层的巅峰。
<br/>　　这个时候，他即便没有进入属性玄力的融合，实力也提升了很大一截。
<br/>　　水仙录，让他的水属性玄力犹如无尽之海，磅礴无比，威力恐怖。
<br/>　　三千炎焱诀，让他的火属性玄力强大的数倍，狂暴的火属性玄力，足以焚烧万物。
<br/>　　生生不息诀，更是不简单，何为生生不息，意为没有终止，绵绵无尽，无休无止。
<br/>　　苏莫停止了修炼，三种功法都达到了第一层的巅峰，那么火属性玄力和木属性玄力，该尝试融合了。
<br/>　　意念一动，两种玄力流转到了指尖之上，他又按照老办法，开始尝试融合。
<br/>　　连续尝试了十数次，均是立刻失败，原因则是火属性玄力稍弱。
<br/>　　苏莫无奈，只能再次修炼三千炎焱诀，将玄力的威力，再次凝练一番。
<br/>　　两日之后，他便再次开始尝试融合。
<br/>　　这一次，两种玄力的威力，基本上保持在了一个相差无几的地步，互相融合之间，各有胜负。
<br/>　　不断的尝试，不断的崩灭，由于苏莫对于玄力的融合已经颇有经验，在上千次失败之后，他终于掌握了其中的平衡，让得两种属性玄力彻底的融合在了一起。
<br/>　　这两种属性玄力融合在一起，威力几乎是直线暴增，瞬间提升十倍，气息相当的恐怖。
<br/>　　而后，苏莫深深的吸了口气，开始在体内融合。
<br/>　　意念一动，他体内火属性神丹和木属性神丹催动了起来，两种玄力倾泄而出，立刻开始在丹田中融合。
<br/>　　嗡嗡嗡！！！
<br/>　　两种玄力交织在一起，犹如蜂鸣一般，发出嗡嗡的声响，而后快速的融合。
<br/>　　与此同时，一股排山倒海的磅礴气势，从融合的玄力中，立刻狂涌而出，气息惊悚，让苏莫都感觉骇然不已。
<br/>　　少倾之后，两种属性玄力彻底融合，再次化为了一种紫色的特殊玄力。
<br/>　　这种玄力乍一融合而出，苏莫体内的五枚神丹，都被强烈的气势逼迫的后退开来。
<br/>　　而后，苏莫意念再动，三色玄力涌上手掌，璀璨耀眼。
<br/>　　“强大！”苏莫拳头微微一握，面上露出了一丝笑容，这三色玄力自然便是木火属性融合的全新紫色玄力，和水仙录修炼而出的淡蓝色玄力，以及土金属性融合的灰色玄力。
<br/>　　现在，三色玄力之中，明显是紫色玄力最强，占据主导地位，蓝色玄力和灰色玄力较弱，成为辅佐的力量。
<br/>　　这种玄力的融合，约莫维持了半柱香的时间，才开始不稳定起来，苏莫立刻将玄力恢复原状。
<br/>　　半柱香的时间已经非常好了，当年他第一次融合成功的时候，只能坚持十几个呼吸的时间。
<br/>　　当然，随着一次次的融合成功，以及时间的流逝，能融合在一起的时间逐渐的延长。
<br/>　　就比如他现在的土金属性玄力，就可以一直维持着融合的状态，根本没有时间的限制。
<br/>　　苏莫感觉，等到他将五门尊级中品功法全部转换，玄力的融合成为常态之后，或许可以考虑进一步的融合了。
<br/>　　比如，将水属性玄力与其它两种已经融合的属性玄力，再一次的融合。
<br/>　　若能进一步的融合成功，三种属性的玄力融合，绝对的恐怖。
<br/>　　但目前，显然不是尝试的时候。
<br/>　　玄力的融合已经搞定，接下来就是提升修为了，这对于苏莫来说并不难。
<br/>　　“嗯？”就在此刻，苏莫双眉一挑，因为他发现外界有人在喊他，是任云都的声音。
<br/>　　随即，苏莫意念一动，立刻离开了玉镯空间，随后走出了洞府。
<br/>　　此刻，任云都正在火元峰之前，一袭青衣，面带笑容。
<br/>　　“任兄！进来坐吧！”苏莫向对方抱拳说道。
<br/>　　“苏兄，不了，我这次来是要告诉你，真圣界开启还有二十天，半个月后在学府大门前集合，由学府导师带队，一起前往真圣界！”任云都说道。
<br/>　　“嗯！到时候一起！”苏莫点了点头，他早知道学府会有导师带队，因为真圣界的所在，非常的远。
<br/>　　从古周星到真圣界所在的星辰，就算是有星空传送大阵，一年之内都到不了。
<br/>　　“好！我先走了！”任云都来去匆匆，告知苏莫之后，便转身离去。
<br/>　　苏莫返回洞府，进入玉镯空间，开始吞噬神丹修炼，冲击武帝境六重境。
<br/>　　他身上还有不少神丹，就算不用灵晶，神丹也足以支撑他突破境界了。
<br/>　　吞噬战魂催动，无数玄力如同，苏莫的修为开始了急速的暴涨，直到武帝境五重的巅峰境界，增长势头才停止下来。
<br/>　　随后，他便开始冲击境界瓶颈。
<br/>　　不得不说，随着修为的不断提高，突破境界越发的困难。
<br/>　　这一次，虽然苏莫的修为沉淀了大半年之久，但依旧用了整整三天时间，不停地冲击瓶颈，几乎将所有神丹全部耗尽，才成功突破到武帝境六重境界。
<br/>　　玉镯空间中。
<br/>　　苏莫全身气息缓缓收敛，感受到增加的修为，体内暴涨的实力，他心中算是有了底气。
<br/>　　虽然他不清楚，那些顶尖的武帝境九重妖孽天才的实力如何，但想来他绝对不会比那些人弱。
<br/>　　现在的他，比几个月之前，简直是强大了几个档次。
<br/>　　目前，距离真圣界开启，还有十余日的时间，苏莫就静等那一日的到来了。
