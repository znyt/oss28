　　“所以，必须只有妖精，才能进入妖灵山脉，才能去救人”副院长看着涂小安那张清秀的脸庞：“这也正是老夫要找你帮忙的原因”
<br/>　　人类进入妖灵山脉，会被围攻，但同样是妖精就不会被引起重视。
<br/>　　“哦，原来如此...”
<br/>　　涂小安明白副院长的意思了，他要找的帮手，一是妖精，二还要靠得住，心地善良。
<br/>　　貌似好像自己是挺合适。
<br/>　　他不禁恶趣味了一下。
<br/>　　不过这个妖灵山脉到是有点意思，妖精的大本营，集中地，没想到在现代无神论的世界，还有这样的地方。
<br/>　　要都是妖精，觉醒生物，不正是涂小安刷怪练级的好地方？
<br/>　　“老夫也不会白白让你帮忙，我会给你一千块灵石当报酬”
<br/>　　“一千？”
<br/>　　涂小安倒吸了一口凉气，这老头还真是大手笔呀。
<br/>　　不是说灵石都在国家的掌控之中吗，一块都来之不易，那你这一千块是什么路数。
<br/>　　副院长见涂小安一脸古怪的看着他，好像明白他在想什么：“这一千块，是老夫在古学院当院长全部的家当了，多年攒下来的”
<br/>　　呵呵...我信你个鬼。
<br/>　　按照一块灵石100进化能源来算，那么涂小安要是有一千块灵石，全部吸收了，那么岂不是十万的进化能源？
<br/>　　要知道他现在提升一级是5000进化能源？
<br/>　　我的天，这买卖好像可以做。
<br/>　　很显然，有着一千块灵石，让涂小安步入三十级都有富余。
<br/>　　“先说说具体情况吧”涂小安沉思一下，开口。
<br/>　　“哎...”副院长先是哀伤的叹了一口气，做了一番面部表情工作，然后唏嘘的道：“老夫让你救的不是别人，而是老夫的亲孙女”
<br/>　　妖灵山脉，孙女？
<br/>　　答案呼之欲出呀，涂小安道：“你孙女被妖精抓走了？”
<br/>　　“哼...老夫虽然年迈，但还不至于没用到让自己的孙女被妖精抓走”副院长一双眼睛精光闪闪，他说自己年迈都是谦虚，他在活个一二百岁都不是问题。
<br/>　　涂小安静静的看着老头装逼。
<br/>　　“老夫的孙女，乃为天之骄女，她有一张天使般美丽的面容，菩萨般的心肠，天真无邪，宛如世间的一张白纸，纯洁无瑕，她就是...”
<br/>　　“她就是九天仙女下凡尘是吧”涂小安一头黑线，这老头怎么回事，谁让你夸自己的孙女了。
<br/>　　能不能正经严肃点，能不能直入主题一点。
<br/>　　咳咳...
<br/>　　副院长咳嗽了一下，一想到自己的孙女就情不自禁的夸奖起来。
<br/>　　“命运弄人啊，老天将世间一切的美好都给那丫头，万千宠爱集一身，可又偏偏跟她开了一个天大的玩笑”
<br/>　　“那丫头出生，体质就异于常人，多病多灾，到了她九岁的时候体质出现了明显的征兆，老夫那时才知道，原来这丫头的体质既然是传说中的太阴广寒之体，也可以称之为阴寒之体”
<br/>　　太阴光寒之体，体蕴太阴，阴气转寒气，这种人几百年难出一个，出了基本上多半夭折。
<br/>　　也就是说，这种人注定就活不久，甚至活不过成年。
<br/>　　“阴寒之体？”涂小安闻言就微微的蹙眉，他都不需要多问，从老头的表情中，就知道得了这种体质的，要GG。
<br/>　　副院长哀痛道：“这种极为罕见的体质，让我的孙女体内的寒气一年胜似一年，阴气重，寒气盛，一般情况下到了十五六岁，将会必死无疑，若想活命只有两种方法，其中之一就是请一位传说中的四阶圣灵出手，连续十年，每年用强大的力量将她的寒毒驱散开，更不惜用真元助其洗髓经脉，方可保命”
<br/>　　四阶圣灵？呵呵...
<br/>　　按涂小安现在的理解，四阶圣灵，可谓陆地神仙了，劈山断河，飞天遁地，无所不能，更有点像落子烟跟他说的那种远古古妖一样。
<br/>　　不用说第一个办法是没戏的，当今世上，三阶生灵不少，但是四阶圣灵就不可见了，不然也不会用传说形容。
<br/>　　或许有四阶圣灵，但这种存在，是你找的到？请的动？
<br/>　　这老头虽然实力强大，让涂小安都洞察不出数据，但看他一脸的哀痛，也知道他不可能是四阶。
<br/>　　无法帮自己的孙女驱寒毒，散阴气。
<br/>　　心照不宣，不用多做解释，院长继续道：“第二种救命的办法，就是我孙女必须每年服用一种非常珍贵的灵果，叫百年冰圣果。

<br/>　　一听名字，涂小安就知道是天材地宝，天地奇珍，大概跟涂小安在蛇谷得到了蛇莓差不多。
<br/>　　但显然，价值肯定比蛇莓要珍贵的多，光听名字就能听出不凡之处。

<br/>　　动不动还送人几颗。

<br/>　　十年寿命，可以让很多人倾家荡产，也在所不惜。
