　　神魂，是生命的本质，亦是精神意念的源泉。
<br/>　　神魂融入武魂，武魂进化战魂，便能做到战魂不灭，神魂不死。
<br/>　　即便肉身毁灭，也能存活于世。
<br/>　　当然，所谓的神魂和武魂融合，并不是完全的融为一体，而是做到彼此交融，彼此相辅相成。
<br/>　　苏莫的神魂，化为了浩大的精神意念，钻入了吞噬武魂之中。
<br/>　　按照阎魔鬼圣之言，神魂要做到融入武魂，要以奥义为媒介。
<br/>　　因为吞噬奥义已经和武魂融合，而吞噬奥义来自于苏莫的精神感悟，是无穷无尽的力量，可以不听的从精神意念中涌出。
<br/>　　这就是一座桥梁，一座连接神魂和武魂的桥梁。
<br/>　　苏莫完全沉浸在了修炼之中，努力的让神魂和武魂交融，尝试融合。
<br/>　　时间缓缓的流逝，一个时辰又一个时辰。
<br/>　　十几个时辰之后，苏莫缓缓的睁开了双眼。
<br/>　　他已经有些一些眉目，但很难立刻成功，而时间已经不多了。
<br/>　　叹了口气，苏莫停止了修炼，准备先返回神武学府。
<br/>　　……
<br/>　　苍穹神宫，新建的议事大殿之中，苏莫召集了所有武皇境以上的高层。
<br/>　　“诸位，我要再次离开了，苍穹神宫的发展，就要靠你们了！”苏莫环视众人，朗声说道。
<br/>　　“宫主放心，我苍穹神宫发展一日千里，宫主无需挂心！”众人说道。
<br/>　　苏莫微微点头，目前苍穹神宫只是在这小镇上，只要不招惹强大的势力，一般不会有事。
<br/>　　“苏莫，我们几个也想出去闯荡！”
<br/>　　司空炎望着苏莫，面色郑重的说道：“留在这里，修为提升的太慢了！”
<br/>　　苏莫闻言一怔，随即忍不住眉头微皱了起来，司空炎居然想要离开。
<br/>　　“你们几个呢？”苏莫又看向姜枫然、战乱、龙腾等人。
<br/>　　“苏莫，我也有这个意思！”龙腾说道，其他几名九宫之主，尽皆点了点头。
<br/>　　就连夕儿，也不例外。
<br/>　　苏莫见此心中暗叹，他知道九人都不想被束缚在小小的苍穹神宫中。
<br/>　　这个世界这么大，这么精彩，强者无数，他们都想去追求更高的层次。
<br/>　　而且，九人的天赋都不弱。
<br/>　　姜枫然和司空炎是属性灵体，战乱是双生武魂，龙腾本体是金蛟，翼晓晓是神兽八翼飞龙蛇后裔。
<br/>　　他们五人的天赋，比之神武学府中大多数的弟子，天赋都要高的多。
<br/>　　留在苍穹神宫的话，的确是埋没了他们的天赋。
<br/>　　夕儿、闻人天都、彩芸和青雅四人，虽然天赋略低一些，但也算是天才。
<br/>　　苏莫陷入了沉默之中，他也不想束缚九人，但九人都走了，苍穹神宫怎么办？
<br/>　　“苏莫哥哥，你不必担心，大家只是出去闯荡，并不会脱离苍穹神宫！”夕儿知道苏莫的担忧，柔声说道。
<br/>　　夕儿想要出去，并不是要追求武道巅峰，而是不想被苏莫甩的太远。
<br/>　　她知道，以苏莫的天赋，将来晋升武帝、武尊，都没有多大的问题。
<br/>　　而她若是不出去寻找机缘的话，就会落后苏莫太多太多。
<br/>　　“不错！苏莫，我们虽然离开了，但苍穹神宫是我们的根，我们永远都是苍穹神宫之人。”战乱点头说道。
<br/>　　其他几人亦是纷纷点头，表示赞同，他们只是出去闯荡，或许也会加入其他势力，但并不会脱离苍穹神宫。
<br/>　　苏莫闻言沉默了片刻，只能心中暗叹，既然几人要出去闯荡，他要不好阻拦。
<br/>　　“晓晓，你也要走吗？”苏莫目光望向翼晓晓。
<br/>　　其他人出去闯荡，他倒是没有多少担心，唯独担心此女。
<br/>　　现在的翼晓晓，看起来也不过是一个十四、五的小丫头，因为是妖兽化形，心思单纯，如何能出去？
<br/>　　“苏莫大哥，我和龙腾大哥已经商量好了，一起出去闯荡！”翼晓晓笑着说道，小脸上充满了兴奋。
<br/>　　两人都是妖族，平常的交流比较多，关系也是最好。
<br/>　　“好吧！我同意你们出去！“
<br/>　　随即，苏莫点了点头，又继续道：“但苍穹神宫不能无人坐镇，你们过些日子再走吧！”
<br/>　　苏莫的意思，是让苍穹神宫再发展一段时间，足够强大之后，九人再离开。
<br/>　　“宫主，让他们先走吧！我留下来，待海前辈和帝前辈突破到了武帝之境，我再离开！”青雅轻声说道。
<br/>　　青雅的实力，在九人中算是最低的，所以他也不急着离开。
<br/>　　“哦？”
<br/>　　苏莫闻言讶然，转头看向海明江和帝释，仔细打量了两人一番，发现两人身上的气息，隐隐又突破的征兆。
<br/>　　“苏莫，我最多两个月之内，便能晋升武帝之境！”帝释笑着说道。
<br/>　　“宫主，老朽也差不多，一个月之内，必定能突破！”海明江说道。
<br/>　　他们两人的修为，都卡在了准帝境很久了，尤其是海明江，足足卡了数百年。
<br/>　　现在来到了外界以后，浑厚的积累顿时爆发，突破已经近在眼前了。
<br/>　　“呵呵，那就好！”
<br/>　　苏莫闻言顿时放心了，两人只要突破到武帝之境，足以坐镇途山镇了。
<br/>　　随即，苏莫详细的交代了一番，待司空炎九人离开之后，苍穹神宫由帝释和海明江共同管理。
<br/>　　他也给两人安排了一番接下来几年的发展计划，还留下了少量的下品灵晶。
<br/>　　最后，他才和众人依依告别，离开了途山镇。
<br/>　　途山镇十万里之外，高空之中。
<br/>　　苏莫和夕儿相对而立。
<br/>　　“夕儿，你回去吧！不用再送了！”苏莫面带柔情，抓着对方的玉手。
<br/>　　他离开途山镇之后，夕儿送了他十万里。
<br/>　　“苏莫哥哥，这一次离别，不知道我们何时才能相见？”夕儿的俏脸上，挂着落寞之色，她仅仅的抓着苏莫的手，不舍得放开。
<br/>　　她与苏莫成亲之后，在一起的时间并不多。
<br/>　　即便是在苍穹世界，试炼者到来的前几年，也是各自忙着修炼，能相处的时间，非常的短暂。
<br/>　　苏莫见此心中暗叹，人生难得是欢聚，唯有别离多啊！
<br/>　　轻轻的将夕儿揽在怀中，鼻中嗅着对方的发香，苏莫心中也是充满了不舍。
<br/>　　“不会太久的，我还指望你给我生个孩子呢！”苏莫心中虽然不舍，但口中却是调笑着说道。
<br/>　　夕儿闻言，顿时俏脸一红，忍不住的将脑袋埋在了苏莫的怀中。
<br/>　　她，早已想为苏莫生一个孩子了！
<br/>　　只不过，在武道的世界，对于高级武者而言，想传下后代并不是很容易的事情。
<br/>　　修为越高的武者，生子的几率越低。
<br/>　　当然，父母双方的修为越高，后代的天赋也会越好，有利也有弊。
<br/>　　武者的修为越高，身体素质则是越高，每一个大境界都是一次本质的脱变。
<br/>　　但也会因为一次次的脱变，让得武者血脉不断的进化，逐渐脱离凡人的层次，成为更高等的生命。
<br/>　　这也就造就了，高级武者很难留下后代的事实。
<br/>　　这也算是天道使然，使万物生灵保持在了一个平衡。
<br/>　　不然的话，那些武圣境、乃至虚神境的武者，若是生下很多子嗣，个个继承了父母的天赋，岂不是要称霸宇宙了。
<br/>　　“以后会有的！”夕儿俏脸绯红，声如蚊蝇的说道。
<br/>　　两人静静的拥抱在一起，感受着这即将分离的时刻。
<br/>　　“对了，苏莫哥哥，那三个巫族之人，还在虚界神图之中！”良久之后，夕儿突然说道。
<br/>　　她说的是巫娴和另外两人巫族天才，苏莫将虚界神图给了她，三人也一直在神图空间之中。
<br/>　　“都交给我吧！”苏莫说道，这三人以后对他还有用。
<br/>　　随后，巫娴三人被夕儿从虚界神图中放了出来，又被苏莫收进了玉镯空间之中。
<br/>　　苏莫和夕儿静静的相依了半个时辰之后，才告辞离去，向神武学府而去。
