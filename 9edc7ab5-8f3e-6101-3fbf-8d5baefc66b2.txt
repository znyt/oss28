        &lt;h3&gt;第三卷第三百三十二章  疑问&lt;/h3&gt;
<br/>
<br/>        陈家兄妹。聚在一起讨论冯睿的伤势。陈伟冯没有想到他的妈妈真的不在了，一口气没有上来竟然晕过去了。
<br/>
<br/>        陈家乱成一团，又开始抢救陈伟冯，好的是陈伟冯只是伤心过度，一时接受不了，晕过去了，过了一会才醒转过来。
<br/>
<br/>        大家对陈伟冯是又恨又爱又无奈，这都快30多岁的人了，能不能有点出息，陈伟冯醒过来后，一句话也不说，只是默默流泪，这么多年嫌弃他妈妈管得太严了，成家搬出去之后觉得是飞出笼子的小鸟，自由自在了。
<br/>
<br/>        高兴了偶尔打个电话，不高兴了连面都不想见，父亲和母亲的事情也从来都不过问，总之就是管得太多，让他们父子都不舒服，自己成家了有孩子了，下班放假就回了自己的小家，很少来看冯睿，他都忘记有多久没有回来了，谁知道回来就人天两隔。
<br/>
<br/>        想起冯睿的唠唠叨叨，陈伟冯又哭起来，老太婆再也不会唠叨他了，此时，他多么想他妈妈能站在他的面前说：“臭崽子，你给听着，你要不听话，老娘再也不管你了”。
<br/>
<br/>        “大家商量一下，二嫂在家里也不能久呆，毕竟这个院子还住着其他的邻居，谁明天和我一起去看看墓地？”陈子霖将大家叫在一起商量。
<br/>
<br/>        陈子建手头没有太多的钱，陈伟冯拿工资，还要养家糊口，平时都是冯睿接济，西城这几天的发展很快，地价飙升，好一点的墓地都是安万来计算的。
<br/>
<br/>        “七哥，明天我和你一起去，二嫂辛苦了一辈子，墓地就选好一点的地方吧，生前爱清净、爱干净，就给她找一处好住所。钱呢……”
<br/>
<br/>        “钱大家均摊。以后家里有什么事情了，大家都均摊，别一个人大包大揽的，现在大家日子都过得差不多，一起分担才能显得我们齐心协力。”陈子慧开口了。
<br/>
<br/>        “行，就按照大姐说的办，大家均摊，子昂你一个女孩子就不要去那种地方了，我和……”
<br/>
<br/>        “我和七哥一起去吧。”江俞轩在旁边突然发声。
<br/>
<br/>        江俞轩一开口，大家才发现江俞轩也和他们在一起，陈家人原本都知道江俞轩，也都知道江俞轩有老婆和陈子昂是朋友关系，大家也就见怪不怪了。
<br/>
<br/>        “也行，我明天和俞轩还有伟冯一起去看墓地，家里你们就招呼着。”
<br/>
<br/>        “二嫂出车祸，现场有没有看到？或者你们有去警局看看情况？”陈子昂问陈子霖和陈子建。
<br/>
<br/>        “没有，就当是在医院询问了一些情况，直接就回来了。”
<br/>
<br/>        “那我明天去看看情况吧。”
<br/>
<br/>        “子昂妹子，我和你一起去吧。”冯乔看着陈子昂。
<br/>
<br/>        “行。那就辛苦你了，明天你和我一起去。”陈子昂并没有拒接冯乔。因为她觉得，冯乔跟着她一起去警局，能知道事情发生的状况，才不会记恨陈子建。
<br/>
<br/>        “小姑，我也想和你一起去。”陈天天眼含热泪，她也想知道她妈妈是被谁撞死的。
<br/>
<br/>        如果找出来真凶，她要将他千刀万剐。她要问问他的心是黑的还是红的？如果他不逃跑，那么他的妈妈是不是还会有机会活下去？
<br/>
<br/>        她才找回他妈妈多久啊？他就是去了？陈天天好后悔，后悔这么大都没有好好的陪一陪冯睿。
<br/>
<br/>        “行。明天我们一起去吧。”陈子昂答应了陈天天的请求。
<br/>
<br/>        折腾了一宿，天已经发白了，东方已经出现了一次丝曙光。
<br/>
<br/>        “有谁知道二嫂临走时留下什么了吗？”陈子昂跟陈子健。
<br/>
<br/>        “就他的包。还有手机。手机里头还有没编辑完的短信。对了，这个短信是准备发给你的。”陈子建从口袋里摸出冯睿的手机交给陈子昂。
<br/>
<br/>        陈子昂看了一下手机里头的短信，她觉得冯睿一定是碰到什么事情了，要不然不会平白无故的给她发短信。
<br/>
<br/>        “俞轩，你看。”陈子昂将手机递给江俞轩。
<br/>
<br/>        “二嫂有可能正在给你编辑短信，但是车来了，她来不及发出去这个短信。她去干什么？她为什么会突然发这样一个短信？”江俞轩看着陈子昂了。
<br/>
<br/>        “还有啊！她去了哪里？她为什么会在那个路口出现？”陈子昂充满了怀疑。
<br/>
<br/>        “你们这一段时间，没有和二嫂联系吗？”江俞轩问大家。
<br/>
<br/>        “大家都在忙。”陈子霖闷着声音说，平时就不怎么待见冯睿，局不想和她联系。
<br/>
<br/>        “天天，你有和你妈联系吗？”陈子昂是知道陈天天前一段时间是和冯睿在一起的。
<br/>
<br/>        “我这一段时间陪着爷爷和奶奶，我很少，很少去看妈妈。”陈天天哭着说，从昨天到今天她一直在哭，眼睛都肿了。
<br/>
<br/>        “我姐和她的干女儿关系很好，她们联系很紧密，上午她那个干女孩儿去我家找她呢。”冯乔开口说。
<br/>
<br/>        “干女儿。”陈子昂和江俞轩都不知道冯睿的干女儿是谁。
<br/>
<br/>        “就是那个夏颜呀！”陈天天抽泣着的接话。
<br/>
<br/>        “如果西城也智慧城市的项目就好了。那样就有天眼，谁走到哪里都可以检查。”陈子昂叹了口气，不能让冯睿就这么不明不白的走了。
<br/>
<br/>        “以后肯定会有的，但是不知道在什么时间启动。沈城的智慧项目当是交接可是轰动一时，如果西城有项目不知道会花落谁家？”江俞轩叹了口气，他们在西城的发展是越发的难了，不知道易倾秋能不能翻盘，将莲鱼的项目拿过来？
<br/>
<br/>        “智慧项目我们就不插手，这个不是我们强项，我们还是专注我们自己的行业。”陈子昂微微一笑，她知道江俞轩为何叹气。
<br/>
<br/>        其实，他们说的智慧项目西城已经准备开始启动了，这个项目设计资金几个亿，夏颜怎么会不启动呢？况且有了陆家的支持，夏颜是绝对会放手大干一番的。
<br/>
<br/>        “你要不要去休息一会？”江俞轩看着陈子昂有些憔悴的脸悄声问陈子昂。
<br/>
<br/>        “不，我去趟医院看看我爸和我三姐，等上班了去趟警局，我总感觉二嫂的死不一般。”陈子昂眉头紧皱。
<br/>
<br/>        “那我送不过去。”
<br/>
<br/>        “你干嘛老是要跟着我啊。”陈子昂有些无奈，江俞轩这个样子真的会让人多想的。
<br/>
<br/>        “我不跟着你我跟着谁啊？”
<br/>
<br/>        “你......你不用去了，就在这里吧，不是要和我哥一块去看墓地吗？”陈子昂拒绝了。
<br/>
<br/>        陈子昂走后，江俞轩突然就觉得空落落的，他不知道该干什么，在旁边闷闷的坐着。
<br/>
<br/>        陈天天走了过来，坐在他身边：“江俞轩，我妈妈没了。”
<br/>
<br/>        “我知道。”
<br/>
<br/>        “你难道就不知道安慰一下我吗？”
<br/>
<br/>        “节哀！”江俞轩看了一眼陈天天红红的眼睛，心一软：“天天，人死不能复生，你......”
<br/>
<br/>        陈天天的头往江俞轩的肩膀靠了一下：“借你的肩膀给我靠靠！”
<br/>
<br/>        陈子昂出门的时候，发现车钥匙没有拿，返回来拿车钥匙的时候，看到了陈天天靠在江俞轩的肩膀上。
<br/>
<br/>        陈子昂的眼眶一热，泪水夺眶而出。
