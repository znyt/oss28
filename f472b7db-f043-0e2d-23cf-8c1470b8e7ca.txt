　　幻影旅团众人，全都脸色凝重。
<br/>　　窝金石强化系念能力者，身体被淬炼的如同金铁。一身怪力，更是旅团中第一。而且还有超破坏拳的杀招。
<br/>　　虽然那只是简单的念力凝聚，但却被窝金练的恐怖无比。窟卢塔族村中的百米巨坑，就在眼前。
<br/>　　而能够和窝金对抗，对手很厉害。
<br/>　　“不用担心，窝金虽然白痴混账。实力还是不错的，弄出那么大动静，估计又是打发了性子。说不定，遇到了什么古怪的能力者。”信长浑不在意的甩了甩长刀鲜血，还到入鞘。
<br/>　　他和窝金是老搭档，对窝金的实力记忆犹新。那就是一个人形重装坦克，想要打败窝金，可没那么容易。
<br/>　　“估计是遇到了什么缠人的招式吧。”侠客笑嘻嘻道：“窝金那家伙最没耐性，大概是被对方给搅扰的心头起火，又开大大招啦。”
<br/>　　“走吧，去看看。”库洛洛脸色不变，他谨慎惯了。虽然觉得窝金不会出生意外，但还是坚持去看一看。
<br/>　　库洛洛作为旅团团长，有着绝对的权威。他既然发话，几人也不怠慢。加进速度，把整个窟卢塔族所有人的眼睛挖出来放好。几人快速消失在山村，只留下一片血火狼藉。
<br/>　　……
<br/>　　苏重扫了一眼不远处狼狈不堪的窝金，举起自己右手查看。
<br/>　　晶莹如玉，软中带硬，一丝丝鳞甲纹路若隐若现。
<br/>　　“只是一个月的时间，身体强度提升了一倍还多。肌肉力量更强，皮肤防御更加强大。魔龙改造，竟然拿和纯阳七段意外的贴合！”苏重严重闪过一道异色。
<br/>　　“只是这黑色鳞甲纹路，不知道会不会有妨碍。毕竟是在以恐怖魔龙为模板改造，自己难道真的要变成魔龙？”苏重旋即摇头。
<br/>　　虽然有魔龙虚影影响，但本质上苏重修炼的还是纯阳七段，只不过多了个目标模板。
<br/>　　以前纯阳七段，只是单纯的强化身体。但强化的在厉害，依然是人体模板。能够有那么强的功效，说到底还是走的量变引起质变的堆砌道路。
<br/>　　“看似强大，其实是在摸着石头过河。魔龙虚影，却给我提供了一条变强方向！”
<br/>　　苏重不由想到，很多秘术都需要观想。那其实就是以某种形态为目标，进行有目的的模仿改造。就像龙象般若功，它就需要观想龙象。
<br/>　　“纯阳七段，以前只是堆积量变。如今有了魔龙虚影提供方向，立刻就有了质变。无怪修炼迅速无比。”苏重已经对纯阳七段的改变彻底了解。
<br/>　　不管暗黑奏鸣曲作者的本意是好还是坏，但却给出了魔龙最真实的描述，被苏重取得其中真意。那所谓吸食生命的黑烟，也早就融入道种，成了他力量的一部分。
<br/>　　“好大的力量，竟然把握的胳膊打断了。哈哈！这才我期待已久的战斗！”窝金摇摇晃晃站起来，不仅没有因为受伤而恼怒，反而越发疯狂。
<br/>　　超破坏拳！
<br/>　　轰！
<br/>　　赤红色念力瞬间就把它包裹，身影一片模糊，炮弹一般射向苏重。
<br/>　　苏重一脸漠然，不闪不避开。全身筋肉皮膜好似弓弦一般蹦蹦作响，鳞甲一般的纹路瞬间弥补手背。
<br/>　　暗杀拳！
<br/>　　砰！
<br/>　　两只拳头狠狠的撞在一起。
<br/>　　一阵气浪红的扩散开来。窝金身形好似撞上了墙壁的弹力球，猛然飞退。砰的一下砸进地面。
<br/>　　一个三米大坑出现在地上，窝金深陷其中，双手软绵绵呈怪异角度扭曲。嘴里不断吐出血液，身体挣扎几下，却始终无法坐起。
<br/>　　受了这么重的伤，窝金情绪丝毫没有低迷，眼中反而满是疯狂兴奋。
<br/>　　“还是个武疯子？”苏重挑了挑眉。
<br/>　　被自己全力两拳砸趴下，双臂粉碎性骨折，内脏受创严重。即便如此，仍然战意不减，可不就是个武疯子。
<br/>　　“也是，如果不是武疯子，也不会练成这一身强悍肌肉。”和苏重相比，窝金的身体强度自然没法比。可比其他念能力者，窝金简直就是个人性怪兽。记得在原著中，这家伙可是凭空借助了火箭弹的攻击！
<br/>　　虽然可能有念力防护，但能够抗住火箭弹，身体强度也已经十分变态了。
<br/>　　可惜他遇到了苏重，被他两拳就给废了。成了测试纯阳七段的试验品。一个月苦修，身体强度急剧增强，即便是苏重自己，也不清楚自己身体的极限在哪。
<br/>　　“能够只用身体力量，就打残废窝金，我这身力量，也算能够在这个世界上立足了吧。”苏重满意点头。
<br/>　　来到这个世界，苏重几乎疯了一般修炼。就是因为他知道，猎人世界太过危险。暗处隐藏着数不清的高手。而且念能力又特别诡异，这让苏重一直有一种危机感。不然也不会俺么疯狂修炼。
<br/>　　咦？其他人都来啦？苏重突然心中一动。一直扩散开的圆，被人给触动了。
<br/>　　“窝金！窝金！你竟然敢让我帮你挖眼睛？如果让我知道，你躲在这里偷懒，一定一刀砍死你！”人还没到，信长就不满的嚷嚷起来。
<br/>　　这里风平浪静，一点没有动静。肯定是窝金解决了敌人，这会儿一定在偷懒！
<br/>　　信长越想就越气，挖眼睛可比砍人辛苦多啦！见了面，一定要砍他两刀！
<br/>　　一个大踏步，信长一马当心冲出树林，伸手握住剑柄，抬头就要出刀砍过去。
<br/>　　“咦？人呢？”除了一个黑头发小子，竟然没看到窝金。
<br/>　　突然察觉身旁有动静，转头看去，顿时大吃一惊：“窝金！你这是怎么啦！”
<br/>　　几乎下意识的，他立刻半蹲身体，右手紧紧握住剑柄，眼睛死死顶住不愿出手苏重。
<br/>　　那个难看的矮胖子瘫倒在地一脸苍白，危险只可能来自于那个一脸冷漠的小子。
<br/>　　身上的念瞬间释放出来笼罩全身。他和窝金是搭档，最了解窝金的实力。身体强悍的变态。
<br/>　　可就是如此，窝金现在却满脸是血，被人生生砸进了地里？对手该死何种强大？
<br/>　　扫了一眼窝金伤势，信长越发凝重。他可是武士，擅长兵刃，身体强度可不高。窝金那么强大的身体，都被对手砸断双手。以他的身体，一个不小心挨上一下，不死也得重伤！
<br/>　　“是他吗？”信长双眼紧紧盯住苏重，小心翼翼开口问窝金，丝毫不敢分神。
<br/>　　窝金咔的吐出一大口鲜血，努力喘息片刻，眼中带着丝丝不可思议：“信长，要小心了。他的力量比我还大！”
<br/>　　力量比窝金大？信长心中顿时一紧。
