　　第702章
<br/>　　三千兵马不算少，但也绝对不算多，就算程大雷现在的兵力紧张到这个地步，也是绝对拿得出来的。可当朝天子，三十万禁卫军，真的连三千兵马也需要借么？
<br/>　　面对程大雷的这个问题，田七没有回答，只是道：“陛下让我转告程当家，借你三千兵，送你一个凉州王。”
<br/>　　程大雷心里琢磨着，目前自己这凉州王是有实无名，否则也不会是田七单独送信，真正忌惮的，便是崔相而已。
<br/>　　“陛下现在的情况很糟糕，难道已经糟糕到狗急跳墙的地步？”
<br/>　　田七没有回答，关键是不好回答，这程大雷口里对天子一丁点尊敬都没有。
<br/>　　程大雷忽地长叹口气，道：“田七先生放心，陛下的命就是我的命，陛下想做的事我冲在最前头，可借兵……实不相瞒，我现在的情况很紧张呐。”
<br/>　　田七道：“若这件事程当家有些为难，陛下也并不打算勉强。”
<br/>　　程大雷摸了摸鼻子，这么容易就糊弄过去了，自己准备了许多话，还没来得及说呢。
<br/>　　“陛下还准备了一物，让我转交给程当家。”
<br/>　　程大雷好奇的看着田七，见他从怀里掏出一样物事，看仔细后，眼睛瞬间瞪直。
<br/>　　一份圣旨，一份空白圣旨。
<br/>　　程大雷：“陛下有让你带什么话没有？”
<br/>　　田七摇摇头：“陛下说，当程大雷看到后，便会明白一切，不需要我说什么。”
<br/>　　程大雷歪坐在椅子后，半晌，楠楠道：“还真是要狗急跳墙啊。”
<br/>　　喜字队在长安也有探子，负责刺探长安的情报。自从尉迟离死后，李乐天便渐渐被相府架空，再如此下去，李乐天就当真要成为傀儡皇帝了。李乐天这种性情，当然不会让事情演变到这一步，如此，是准备还有力气时，和相府来场生死肉搏。
<br/>　　今次向程大雷借兵，便是为了这个目的。千金之子，坐不垂堂，能让当今天子走出这一步，看来他委实是没有办法了。
<br/>　　不过，他似乎已料定程大雷不会借兵。给程大雷一份圣旨，便是假若事败，程大雷可以借他的名义，给拥兵自重的相府添点恶心。
<br/>　　当然，李乐天活着，这份圣旨没什么作用。假若李乐天死了，程大雷就有了名正言顺的幌子。
<br/>　　能做出这个决定，可见李乐天已经下定决心。
<br/>　　大概理清了事实，程大雷轻轻叹了口气，便是当朝天子，也有这样的无奈。与之相比，自己遇到的这点麻烦似乎就不算什么问题了。
<br/>　　将书信送到，话也说明白，田七告辞离开。从长安到此地，一路千山万水，还要担心被相府发现，田七这一路走得并不容易。
<br/>　　程大雷对这个沉默寡言的汉子，也是暗暗佩服。
<br/>　　田七离开凉州城后，并没有直接回长安，而是绕过三水关，进入京州界后，又往北行了一段路。
<br/>　　山路崎岖骑不得马，他弃马之后，拨开密密麻麻的丛林，面前出现一座山寨。
<br/>　　前方山坡上一棵大树的树冠轻轻晃动，田七知道这是山寨的探子发现自己后，回山中禀报。
<br/>　　他也并不着急，立在大门口等待。
<br/>　　“来者何人，无故闯山所谓何事？”山寨大门上有人瓮声吼道。
<br/>　　田七站直身子，也不行礼，像是不懂江湖规矩。
<br/>　　“长安来的人，要见百里侯爷。”
<br/>　　沉默片刻，一个声音响起：“这里没有百里侯爷，你哪里来的回哪里去吧。”
<br/>　　田七沉默不言，笔直的立在原地，不声不响。
<br/>　　山寨大门口的人也撤了，只是有个喽啰守在栅栏上，奇怪的看着这个陌生人，感觉有些事情，怕自己一辈子都想不明白。
<br/>　　田七在山寨门口站了一天一夜，纹丝不动，到第二天黄昏，大门缓缓打开，那小喽啰扯着嗓子喊了一声：“喂，你进来吧，我们大当家答应见你。”
<br/>　　田七身子踉跄一下，险些跌倒，他终于站直了身子，迈步走向山寨。
<br/>　　这小喽啰看他的目光有些佩服，自己就是站岗站一个时辰，就是两腿酸疼，可这个汉子竟真撑了一天一夜。
<br/>　　山寨聚义厅，百里胜歪坐在太师椅上，他本身也是正经人，向来腰杆笔直如石碑。可也不知怎地会事，落草成贼后，腰背就始终挺不直了。
<br/>　　田七迈步走入，二话不说便屈膝跪倒：“鱼龙卫田七，见过侯爷。”
<br/>　　百里胜轻叹口气：“起来说话吧。”
<br/>　　田七站直身体，从怀中掏出一面令牌：“陛下口谕，百里胜接旨。”
<br/>　　百里胜发现一个尴尬的事实，田七刚刚给自己磕头，自己立刻就要与他磕头。
<br/>　　他心里想着自己特么已经是山贼，根本不在意什么天子王法。可身体竟如不由自主般，慢慢站起，屈膝跪倒在地上。
<br/>　　“臣在。”
<br/>　　“百里胜，朕知道你委屈，若还想做些事，就来长安，如果不愿意，就杀了田七，做你的逍遥山大王。”
<br/>　　百里胜慢慢抬起头，目光看着田七，这田七自然不是李乐天，可百里胜宛若见到李乐天一般。
<br/>　　“陛下，原来还没忘了我。”
<br/>　　说罢，泪如雨下。
<br/>　　田七将百里胜掺起，百里胜擦干眼泪，将牙一咬：“陛下要我做什么事？”
<br/>　　“杀人。”
<br/>　　百里胜眼睛眨了眨：“我窝在这山里，许多消息都收不到，如今的长安是什么情况？”
<br/>　　田七看了看左右，大厅之上还有其他人，百里胜摆摆手：“都是自家兄弟，有话直说无妨。”
<br/>　　田七这才开口：“陛下现在情况极其不易，让我与侯爷传话，侯爷现在能用的兵有多少？”
<br/>　　百里胜目光转了一圈，看着聚义厅上的弟兄，道：“八百。”
<br/>　　田七叹了口气：“抬手，至少需要有三千兵，方才能争一线生机。”
<br/>　　百里胜摇了摇头：“虽然只有八百，却都是与我从生死场中冲杀出来，八百破八千都没有什么问题。况且，我有一将，能以一敌万。”
<br/>　　田七顺着百里胜手指的方向看去，只见一身猿臂蜂腰，身材魁梧。
<br/>　　“侯爷，这位是？”
<br/>　　这大汉双手一抱拳：“草民卢俊义。”
<br/>(本章完)
