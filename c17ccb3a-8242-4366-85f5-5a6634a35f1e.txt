　　“叮，《天剑》收录中，高阶高级剑诀，价值70000掠夺积分。”
<br/>　　《天剑》是蜀山高等剑诀其一，人剑合一，身化利剑，攻击全体，追求一击必杀，在高阶剑诀中，都是最顶级的剑诀，威力远比乱剑诀要高的多。
<br/>　　此剑诀倒是与《已身化剑术》非常相似，只是化剑术不具备攻击力而已。
<br/>　　若是推演，慕清霄说不定也能将《天剑》推演出来。
<br/>　　若是在合道期，他对于《天剑》或许会有些想法，但是见证了地仙的强大后，慕清霄渴望更强的剑诀。
<br/>　　于是，将竹简放回原位，拿起更上一层的竹简。
<br/>　　“叮，《摧岳撼》收录中，仙品低级剑诀（伪），价值200000掠夺积分。”
<br/>　　系统提示声刚落，慕清霄眼中闪过一抹狂喜，居然真的是仙品剑诀，这可是地仙强者才能修行的剑诀。
<br/>　　不过《摧岳撼》似乎只是不是完整的仙品剑诀，带有伪字，恐怕是没有达到正仙品的要求。
<br/>　　此剑诀需凝神将意识全部注入武器，奋力击地，一招地动山摇，土崩山摧，威力绝对恐怖无比。
<br/>　　慕清霄虽有《空明幻虚剑》，但剑诀却是紫胤真人成名剑诀，可以说是仙品以下顶尖，但并非仙品。
<br/>　　《摧岳撼》虽没有达到正仙品的要求，但威力恐怕要比《空明幻虚剑》还要强一些。
<br/>　　“叮，《裂穹斩》收录中，仙品低级剑诀，价值300000掠夺积分。”
<br/>　　当慕清霄拿起更高一层的玉简时，果不其然，真的是仙品剑诀。
<br/>　　《裂穹斩》将浑厚灵气积聚掌心，凝聚出一柄巨剑，威能洞穿天地。
<br/>　　将两种仙品剑诀收录后，慕清霄已经有些迫不及待，修成剑诀后，不知实力会比现在强多少。
<br/>　　这时，慕清霄看向这座书架的最上方，其上放着最后两卷卷轴，卷轴与下面的卷轴有些不同，颜色为青色。
<br/>　　“叮，《刑锁念刃》收录中，仙品高级剑诀，价值400000掠夺积分。”
<br/>　　此剑诀乃需凝聚精气神，凝聚数把巨剑，虐杀敌阵，刚劲之力，震煞宇宙，属于仙品高级剑诀。
<br/>　　居然是仙品高级剑诀，恐怕是蜀山最顶尖的剑诀了吧。
<br/>　　“叮，《御剑伏魔》收录中，仙品高级剑诀，价值500000掠夺积分。”
<br/>　　此剑诀乃是蜀山掌门姜清的最强剑诀，以一剑化万剑，乃是他一生对剑的领悟所聚。
<br/>　　将竹简放回原位，慕清霄有些震惊，没想到清微老头子藏的如此之深，居然是以剑证道地仙，实力绝对恐怖无比。
<br/>　　经过半个月的接触，在慕清霄眼中，清微只是个和蔼狡猾的老头，没想到一身锐气早以收敛的干干净净。
<br/>　　哪怕他对自然非常敏感，都没有察觉出他的身上有一丝剑意的痕迹。
<br/>　　压下心中的震惊，喜悦涌上心头，总共收录了六种剑诀，其中有三种是高阶剑诀，另外三种是仙品剑诀。
<br/>　　如果《摧岳撼》算在内，那就是四种仙品剑诀。
<br/>　　对于高阶剑诀，慕清霄此刻已经没有多少想法，令他感兴趣的是仙品剑诀。
<br/>　　无论是或者是《摧岳撼》，或者是《裂穹斩》，亦或者是《刑锁念刃》《御剑伏魔》，他都非常的感兴趣。
<br/>　　但慕清霄知道，虽说技多不压身，但杂而不精的道理他也懂。
<br/>　　他现在只是大乘修士，能够施展《空明幻虚剑》，但这四种可都是仙品剑诀，想要完全掌握，不知需要多长的时间。
<br/>　　慕清霄追求的是一击必杀，其中唯有《裂穹斩》与《御剑伏魔》比较适合。
<br/>　　四选其二，伪仙品剑诀首先被排除，至于《刑锁念刃》凝聚全身力量爆发需要耗费太多的时间。
<br/>　　想了想，慕清霄不禁苦笑摇头。
<br/>　　两种仙品剑诀威力自然不用多说，但他只是大乘修士，修炼起来肯定需要耗费大量的时间，倒不如先将修为提高上来。
<br/>　　将复杂的情绪抛之脑后，慕清霄目光落在其余的书架上，其上依旧零零散散的放着一些竹简。
<br/>　　只是这些竹简颜色与第一座书架上的完全不同，拿起一卷竹简，脑海中再次响起系统的提示声。
<br/>　　“叮《天音阵》收录中，高阶高级阵法，价值70000掠夺积分。”
<br/>　　微微一怔，难怪竹简颜色不同，原来其上记载的并非剑诀或者心法，而是阵法。
<br/>　　对于阵法，慕清霄平时也有些研究，如果有强大的阵法，不动一丝灵气轰杀强敌也并非不可能。
<br/>　　不过还是要看谁来布置，相同的阵法，布阵的修士不同，威力也会不同。
<br/>　　《天音阵》并非攻击阵法，而是幻化术的一种。
<br/>　　将自身实力分成五份，幻化出五位化身，但化身的实力也会下降，显得有些鸡助，他自然不会耗费大把的时间去学习。
<br/>　　有些失望的将竹简放回书架，随后拿起上一层的一卷竹简。
<br/>　　“叮，《天璇迷局》收录中，高阶高级阵法，价值70000掠夺积分。”
<br/>　　此阵法利用青石所施棋阵，演化出一座庞大的迷宫，若是不懂阵法之人，可能一生都会被困于其中。
<br/>　　《天旋迷局》与《天音阵》虽然都是高阶高级阵法，可对于慕清霄来说，无疑是《天旋迷局》的价值更高。
<br/>　　地球的庄园中，他所设置的阵法不过是中阶迷幻阵，并且有永恒万花筒的加持，迷幻阵中带有一定的精神攻击。
<br/>　　而中阶迷幻阵比起《天旋迷局》，显然要落了下乘。
<br/>　　握着卷轴，慕清霄微微一笑，道：“看来，回去以后，庄园中的阵法或许要换一换了。”
<br/>　　“叮，《七星伏魔阵》收录中，仙品低级阵法，价值300000掠夺积分。”
<br/>　　仙品阵法，此阵法威力绝强，地仙以下修士进入绝无生还。
<br/>　　“叮，《蜀山护体结界阵》收录中，仙品高级阵法，价值400000掠夺积分。”
<br/>　　此阵法乃是结界类阵法，在蜀山整个山体设下的巨大法阵，以八卦各个方位的开关，可以起到封锁建立结界。
<br/>　　并且拥有阻击敌人的作用，乃是蜀山派的护派大阵。
<br/>　　“叮《太极剑阵》收录中，仙品高级阵法，价值400000掠夺积分。”
<br/>　　《太极剑阵》又称为《兵解剑阵》。
<br/>　　太极八卦图阵环绕罡斩周围地面，以玄铁重剑引动数万剑气轰击并加固神魔之井封印。
