
    茅小草一阵懵逼，手心忽然之间就出汗了，两人就走在林间的小路上，准备前往下一站的苗寨，但谁能想到王惊蛰嘴里会忽然蹦出这么一句话来，而且她还看出来了，人家说的老认真了，不是调侃也不是开玩笑。<br/>　　<br/>　　    “我要是过段时间就……”<br/>　　<br/>　　    王惊蛰攥着她的手，说道：“你没了我就再找一个，你放心我不能可着你一棵树上吊死了，毕竟我还年轻着呢”<br/>　　<br/>　　    “妈蛋，你就不能好好安慰我下，说些海誓山盟什么的，海可枯石可烂，你对我的感情一直不会变这种话么”小草扭过脑袋，眼圈忽然就红了。<br/>　　<br/>　　    王惊蛰伸出左手，摸着她脑袋说道：“我怕我那么说了，你会难受，所以我就只好装的没心没肺一点了”<br/>　　<br/>　　    茅小草咬了咬嘴唇，往王惊蛰那边靠的更紧了一些。<br/>　　<br/>　　    一场突如起来的恋爱就这么出现了，来的很突然，其实也不突然，没有什么甜言蜜语也没有单膝跪地，更没有那些感人的网络段子，他就只是问了一句我们处对象吧，她就没有任何矜持的点头了。<br/>　　<br/>　　    就像王冬至和王仙芝说的那样，其实都是宿命！<br/>　　<br/>　　    王惊蛰和小草，无论怎么躲怎么逃，都没能走出两人宿命间的轮回，他们兜兜转转的还是在一起，只是略微有点坎坷了些。<br/>　　<br/>　　    时间缓缓而过，四十九天的时间已经走过了大半，他们就像是和时间在赛跑的两个孩子，努力着没有放弃，至少在最后一天来临之前，仍然还得朝着第一天定下的目标出发。<br/>　　<br/>　　    在最后这十来天的时间里，稍微有些不同的是，王惊蛰和小草的举止更加亲昵了一些，以前牵着彼此的手是因为需要赶路，现在再牵着手，则是心连着心。<br/>　　<br/>　　    晚间睡觉的时候，他们也是相拥而谁，没像之前那样，背靠着背。<br/>　　<br/>　　    都是第一次搞对象，谁也没啥经验！<br/>　　<br/>　　    另外一头，阎王坟场附近的山头上，午桥和菜刀文眺望着远处的那块地皮，这里就是当初王惊蛰发现此地有恙的地方。<br/>　　<br/>　　    “他们到底在捣鼓些什么呢？”菜刀文不解的说道。<br/>　　<br/>　　    在地皮到手之后的第三天，这里就破土动工了，当时是清晨左右，开过来大概两队能有两百多人的施工队伍，还有几辆运输车，没用三五天的时间，队伍就把整块地皮给清理干净了，然后全都运了出去，变成了一块平坦的地面。<br/>　　<br/>　　    紧接着建筑材料就进场了，铁皮，钢架，还有建材等等，随即就进行了施工，先是外面被围起了栅栏，把整块地都给圈上了，接着平地上面建造出了两排厂房，占地面积十分广泛，留下的空地也很少，几乎占了三分之二的地方。<br/>　　<br/>　　    再往后，就不知道厂房里面是什么操作了，因为从外面是没办法看得见的，只能够看到进进出出的工人还有一辆辆拉着碎石泥土的车子，午桥和菜刀文观察了几天之后，就断定厂房下面肯定被挖开了，上面的钢板房只不过起到的是掩人耳目的作用，不让人看见里面在干啥。<br/>　　<br/>　　    而且，施工的时候只在白天进行，晚上所有的人都撤了出去，一个人都没有留下，午桥就说阎王坟场这种地，白天的话阳气重，一堆精壮汉子干活没啥事，要是晚上了人睡在那块地上，时间一长是要出大事的。<br/>　　<br/>　　    “要不，咱乔装一下装成工人进去看看？”菜刀文问道。<br/>　　<br/>　　    午桥寻思了片刻，说道：“不太行，你看见那些工人没有，穿着的都是统一的工地制服，一共不过就两百人左右，估计最多也就两个工程队，人数不算多，咱俩要是谁进去了那就是生面孔，一旦给人发现就该打草惊蛇了，这么闯进去肯定是不行的”<br/>　　<br/>　　    “那谁知道里面能干啥呢，光是在外面看，咱们瞧不出个一二三来啊”<br/>　　<br/>　　    “倒也不是没招……”午桥眯着眼说道：“唉，你发现没有，这个工地干了能有一个月了吧，怎么没有来现场检查的呢？”<br/>　　<br/>　　    “你说监管的啊？”<br/>　　<br/>　　    午桥点头说道：“对啊，不管是干啥工程，就他么是在山里面干，那也得有监管方吧，消防，施工资质什么的，各种批文都得要好几个，但到现在为止，咱们好像一次检查都没有碰上过，这说明啥？说明有人打招呼了，他们干他们的，肯定没有人会过问就是了”<br/>　　<br/>　　    “那你是什么意思呢？”<br/>　　<br/>　　    “找陈重，让他说句话，请有关部门过来走一趟，以检查的名义，然后咱俩谁跟着检查的队伍进去，到时候一看不就知道了”<br/>　　<br/>　　    菜刀文皱眉说道：“这个办法到也行，陈重现在和这边都和解了，他能这么干明显得罪人的事么？”<br/>　　<br/>　　    “呵呵，你把他想的太大度了，我跟你说吧，陈重肯定没把这个仇给忘了，只是要等惊蛰回来才能报复，那咱们找他，给对方来个穿小鞋的机会，陈重能不愿意？”<br/>　　<br/>　　    “确实靠谱！”菜刀文笑了。<br/>　　<br/>　　    当天，两人就回到了川中，然后找到陈重把事一说，他那边果真就没啥犹豫，直接打了几个电话，就捅咕出了一个检查组来。<br/>　　<br/>　　    由消防，城管还有安监局的人员组成的，目的就是进去看看施工现场，查一下资质什么的，这事吧其实也不难操作，因为你只要是在建的工地就避免不了被检查，说你没事那是因为你有人了，说你有事，就能在鸡蛋里挑出骨头来。<br/>　　<br/>　　    而陈重，午桥和菜刀文也不是非得要鸡蛋挑骨头，他们就是想看看那两处厂房下面到底藏着什么猫腻。<br/>　　<br/>　　    过了两天后，联合检查组就成型了，午桥穿上了一件安监制服，夹在了检查队伍里面。<br/>　　<br/>　　    这天上午九点多钟，一排车队从川中开出，直奔施工现场而去，到了现场之后停到工地大门外，得到消息的施工负责人就出来了，看见一排印着各种部门字体的车就挺诧异的，因为老板上面已经打好招呼了，说这个地方不会有人来检查的。<br/>　　<br/>　　    想和更多志同道合的人一起聊《天命赊刀人》，微信关注“热度网文或者rdww444”与更多书友一起聊喜欢的书
