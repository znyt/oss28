　　“苏莫，现在我们修为到了这个境界，纵然功法高深，资源不缺，提升境界也是变得非常慢了！“司空炎说道。
<br/>　　“是啊！我修为到达武皇境七重，再想提升，难如登天！”战乱点头道。
<br/>　　“苏莫，这急不得，好在距离试炼者降临，还有几年的时间！”姜枫然道。
<br/>　　“……”
<br/>　　“……”
<br/>　　其他几人也是纷纷开口，他们修为的提升速度，实际上已经非常的快了，比之一般的武者，快上十几倍都不止。
<br/>　　比如修为最低的青雅，两年的时间，修为从武王境三重，提升到了武王境八重。
<br/>　　两年五个境界，这在外界来说，已经非常的惊人了！
<br/>　　“好吧！我助你们一臂之力！”良久之后，苏莫沉声说道。
<br/>　　……
<br/>　　岁月悠悠，很快又过去了三年。
<br/>　　南海之中。
<br/>　　一望无际的海面上，一道璀璨的剑光直冲苍穹。
<br/>　　剑光锋锐无匹，爆射数十万丈，冲上九天，磅礴的剑威弥漫整片海域。
<br/>　　“斩！”
<br/>　　苏莫手提十亿魔剑，伫立在海面之上，一声轻喝，魔剑猛然一挥而出。
<br/>　　咻！
<br/>　　瞬息之间，一道恐怖的三色剑气，斩在了海面之上，瞬斩数万里，仿佛将整个浩瀚的大海都一分为二。
<br/>　　轰隆隆！
<br/>　　剑气斩过，海水向两边卷起，露出了下方深达数千里的海底，海水冲起数万丈之高，形成滔天大浪，排山倒海一般，向远处狂涌而去。
<br/>　　大量的海中妖兽，在这一剑之下化为了碎片，大量的气血纷纷向十亿魔剑汇聚，让得十亿魔剑血光更甚。
<br/>　　少倾，苏莫收剑伫立，一切都归于了平静。
<br/>　　“终于大成了！”苏莫长长的出了口气，整整五年时间，他终于将大苍穹剑法，修炼到了大成的境界。
<br/>　　不愧为帝级上品武技，修炼难度着实不小，即便是以他的悟性，都修炼了五年，若是换成其他的同阶武者，就算是二十年，都不一定能练到大成。
<br/>　　这五年时间，苏莫的修为只提升了三重，目前是武皇境八重境界。
<br/>　　这个提升速度，相比于他以前来说，简直是慢如龟爬。
<br/>　　但是，即便是提升到武皇境八重的境界，他也是相当不容易了。
<br/>　　因为，苏莫现在每提升一重修为，所需要的资源简直夸张到了极致。
<br/>　　从武皇境五重提升到六重之时，他耗尽了苍穹神宫内所有的资源，差点让苍穹神宫无法运转。
<br/>　　从武皇境六重提升到武皇境七重之时，他将苍穹世界所有大势力狠狠的剥削了一遍。
<br/>　　而从武皇境七重提升到武皇境八重，整整间隔了三年的时间。
<br/>　　三年之中，苏莫斩杀了无数的妖兽，囤积妖丹，再加上苍穹神宫三年为他搜刮的资源，他才在月余之前得以突破。
<br/>　　可以这么说，苏莫一人的修炼，便让苍穹神宫和无数大势力，变成了穷光蛋。
<br/>　　现如今的苍穹世界，资源贫乏不已，极品灵石的数量极为有限，早已被苏莫耗尽，他大多都是只能用上品灵石和中品灵石来修炼。
<br/>　　而现在提升到了武皇境八重的苏莫，即便再给他三千亿极品灵石，都很难突破到武皇境九重了。
<br/>　　不过，虽然修为提升遭到资源的限制，但苏莫改变了方向，将大部分的时间，都用在了修炼武技之上。
<br/>　　他不仅将大苍穹剑法练到了大成，还将一门叫做《无影魔踪》的帝级中品身法练到了大成境界。
<br/>　　不仅如此，对于他有用的魔影分身，更是被他练到了大圆满境界。
<br/>　　除此之外，苏莫还让天下所有势力为他收集血纹籽，终于又找到了五枚。
<br/>　　他便再次炼制出了五柄本命灵剑，他的本命灵剑达到了二十九柄之多。
<br/>　　他也修炼了由二十八柄灵剑组成的大星宿剑阵，威力比之二十四小星宿剑阵，强大了两倍不止。
<br/>　　而且，本命灵剑，经过苏莫常年的温养，无论是品质还是威力，都不是几年前可比了。
<br/>　　现如今，他一身绝学都练的炉火纯青，比之几年之前，根本不可同日而语。
<br/>　　这几年之中，他也尝试着想把属性玄力再次融合，但无论融合多少次，无一不是以失败告终。
<br/>　　最终，他干脆放弃了融合属性玄力的念头。
<br/>　　唰！
<br/>　　少倾之后，苏莫身形一闪，瞬间便消失无踪。
<br/>　　……
<br/>　　苍穹神殿之中。
<br/>　　苏莫高坐于帝座之上，他的左右两侧端坐着司空炎等九宫之主。
<br/>　　现如今，司空炎等九人，除了彩芸和青雅之外，尽皆达到了武皇境九重的地步。
<br/>　　之所以能提升的如此之快，完全是苏莫数次的帮助，就算是修为最低的彩芸和青雅，也已经达到了武皇境八重。
<br/>　　而在下方的大殿中，熙熙攘攘坐着数百人。
<br/>　　这些人全部都是苍穹神宫的长老，而且全部是高阶武皇。
<br/>　　其中更是有四名准帝，分别是帝释、海明江、钱副阁主以及华太山。
<br/>　　半年之前，原帝玄宫大长老华太山，也晋升到了准帝之境。
<br/>　　“诸位，大劫将至，你们有什么看法？”苏莫目光如电，扫视众人。
<br/>　　大殿中寂静无声，片刻之后，海明江率先开口，道：“宫主，大劫千年一次，理论上还有两年多的时间，但试炼者具体的降临时间，会有一个小范围的波动，可能推后一两年，也可能提前一两年，所以我们要早做准备。”
<br/>　　“嗯！”苏莫闻言颌首，的确，距离大劫还有两年多的时间，但试炼者很可能会提前到来，不得不防。
<br/>　　“诸位，你们见多识广，可知试炼者会从何地降临苍穹世界？”苏莫问道。
<br/>　　“这个不确定，按照以往的记载，巫族会打开外界和苍穹世界的通道，通道可能会出现在任何地方！”
<br/>　　帝释面色凝重，沉声道：“他们有特殊的方法，能查探到苍穹世界强者的所在，直接就会大开杀戒！”
<br/>　　海明江也是面色沉重，道：“宫主，黄泉魔宗之人，隐匿数年都查探不到踪迹，我们也不得不防！”
<br/>　　苏莫闻言低头沉思，巫森是巫族的后裔，到时候黄泉魔宗之人，说不定会和试炼者里应外合。
<br/>　　黄泉魔宗的实力，现在已经不足为惧，但对方很可能会充当试炼者的耳目。
<br/>　　如此的话，他想将试炼者杀个措手不及，就会有些困难了。
