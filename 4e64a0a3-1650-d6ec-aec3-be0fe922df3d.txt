　　“祝英，你难道还想强行向我们动手不成？”见到对方眸中的杀机，任云都冷喝道。
<br/>　　任云都话音一落，一众圣武学府的地弟子，纷纷体内玄力涌动。
<br/>　　祝英沉默不语，心中急速思忖，他很想将苏莫等人就地斩杀，但又怕此事会惹下麻烦。
<br/>　　毕竟，面前的这些人，可都是八皇子的人。
<br/>　　他若是能给众人按上罪名，杀了也就杀了，但没有罪名的话，强行抹杀必定遭到八皇子的报复。
<br/>　　“任云都，你们是要违抗军令吗？”祝英冷冷的说道。
<br/>　　“违抗军令又怎样？”
<br/>　　任云都非常的霸气，大喝道：“有种你就将我们杀光，不然你休想冤枉我们！”
<br/>　　言罢，任云都转头向众人喝道：“我们走！”
<br/>　　随即，众人的身形缓缓升空，准备离去。
<br/>　　众人的目光均是紧紧凝视着祝英，一旦对方出手，他们随时准备迎战。
<br/>　　祝英见众人要强行离开，脸色阴沉如水，不过，他并未阻拦。
<br/>　　少倾之后，众人的身形升到了高空，随即纷纷破空而去，转眼间便消失无踪。
<br/>　　直到众人彻底离去，祝英也没出手。
<br/>　　“尉将大人，就这么让他们走了？”王副尉面带不甘的向祝英问道，若是将这些人强行擒拿，也无不可吧。
<br/>　　“还能怎样？没有大的罪名，私自动手将他们击杀，八皇子肯定不会善罢甘休！”祝英望着苏莫等人离去的背影，沉声说道。
<br/>　　言罢，他不由的叹了口气，没想到计划不仅失败了，还弄巧成拙，赔上了一座灵石矿脉。
<br/>　　不过，这些人都是八皇子派系的神武学府弟子，迟早都会加入皇祖圣朝，以后有的是机会除掉。
<br/>　　……
<br/>　　苏莫等人离开了矿脉之后，便一路不停，再次向圣皇城而去。
<br/>　　回到圣皇城之后，任云都也没有回兵府禀报，直接就带着众人离开了赤粱星。
<br/>　　一众人的心情都不好，这一次历练，不仅毛没捞到一根，还受了闷气，更陨落了十几名同伴。
<br/>　　当然，苏莫除外，他得到了整个灵石矿脉，心中还是颇为高兴的。
<br/>　　只不过，因为他暗中收取了矿脉，差点害了众人，让他心中稍稍有些过意不去。
<br/>　　月余之后，众人回到了古周星。
<br/>　　不过，苏莫并未返回神武学府，而是和谢彬一起脱离了众人，向谢彬的家乡而去。
<br/>　　“苏兄，我的家乡大荣城，可是古周星有名的大城池，每年一次的拍卖会，都是一次盛会！”
<br/>　　高空之中，谢彬一边急速飞行，一边向苏莫说道。
<br/>　　他要去参加大融城的拍卖会，之所以邀请苏莫，当然是想和苏莫拉近关系。
<br/>　　毕竟，苏莫的天赋太强大了，与苏莫这种天才成为好友，有百利而无一害。
<br/>　　“都有什么级别的宝物拍卖？”苏莫微笑着问道，他之前听闻对方说拍卖会，便毫不犹豫的和对方一起来了。
<br/>　　主要是他想购买尊级的五行功法，毕竟，他修炼的五行功法，只是帝级功法，若是修炼了尊级的五行功法，战力还能再次爆增。
<br/>　　唯一让他不确定的便是，不知道大混沌五行神诀，能否融合尊级的功法？
<br/>　　无论能否融合，他都要弄到尊级的五行功法试验一番。
<br/>　　若是不能融合的话，那他就只能修炼混天魔功了。
<br/>　　“基本上除了圣级的宝物，什么级别的宝物都有！”
<br/>　　谢彬面带轻笑，颇为自豪的说道：“苏兄有所不知，大荣城的拍卖会，乃是几个非常强大的商会联合组建，规模相当宏大，在古周星乃至周围几十颗星辰上，都是极负盛名！”
<br/>　　“如此最好！”苏莫点了点头，规模越大的拍卖会，宝物越多，到时候不仅能购买到合适的功法，说不定还能淘到一些其它适合自己的宝物。
<br/>　　“哈哈！在大荣城的拍卖会上，只要你有灵石和灵晶，什么珍贵的宝物都能得到！”谢彬哈哈大笑道。
<br/>　　两人一边闲聊一边赶路，古周星非常的巨大，以两人的速度，都足足飞行了九日时间，才到达了所谓的大荣城。
<br/>　　大荣城果然如谢彬所言，是一座巨大的雄城，方圆近万里，城墙高达千丈，屹立在大地之上，气势巍峨雄壮。
<br/>　　“苏兄，距离拍卖会还有半个月，先到我家府上小住几日！”走在大荣城繁华的街道上，谢彬微笑着说道。
<br/>　　“那就叨扰几日了！”苏莫沉声道，他对谢彬的感觉还算不错，此人颇为健谈，心性也不差。
<br/>　　“哈哈！哪里的话？苏兄住在我谢家，是我谢家的荣幸！”谢彬心情甚好，哈哈大笑道。
<br/>　　不多时，两人来到了谢家的府邸。
<br/>　　谢家只是一个小家族，府邸不是很大，坐落在大荣城的城北，占地五百余亩。
<br/>　　“少爷回来了！”
<br/>　　谢彬和苏莫两人刚刚到来，谢府门前的几名护卫便立刻眼前一亮，急忙迎了上来。
<br/>　　随即，苏莫和谢彬两人，被护卫簇拥着进入了谢府。
<br/>　　进入谢府以后，谢彬为苏莫引荐了其父谢丰。
<br/>　　谢彬的父亲谢丰，便是谢家的家主，但修为并没有多高，只是武帝境七重修为的武者。
<br/>　　随后，谢丰为苏莫安排了一处幽静雅致的院落，在此暂住了下来。
<br/>　　院落中。
<br/>　　苏莫盘膝坐在池塘边，望着池塘中盛开的荷花，鼻中呼吸着淡淡的芳香，感觉神清气爽，心境也是无比的平和。
<br/>　　“血脉瞳术！”
<br/>　　苏莫口中呢喃自语，思索着那些血瞳族之人的瞳术本领。
<br/>　　他杀死众多血瞳族之人后，基本上都将那些人吞噬了，自然也吞噬了血瞳族之人体内的血脉。
<br/>　　这种血瞳族之人的血脉，不同于特殊体质的血脉，是一种传承血脉，每一名血瞳族之人，都拥有这种血脉。
<br/>　　换言之，这是一种遗传的血脉，世代传承。
<br/>　　苏莫沉吟了片刻，意念一动，立刻将吞噬的这种传承血脉，融进了身体之中。
<br/>　　随即，他全身血气沸腾，如同沸腾的开水。
<br/>　　而后，他感觉到磅礴的血脉之力，顺着全身的血液逆冲而上，疯狂的涌入他的眼瞳之中。
<br/>　　一瞬间，苏莫双眸腥红如血，比那些真正的血瞳族之人，还要腥红数倍不止。
<br/>　　实在是他吞噬的血瞳族血脉太多了，足足近百名血瞳族之人的血脉，血脉之力何等强大！
