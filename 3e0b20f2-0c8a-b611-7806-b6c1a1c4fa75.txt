　　　　盘坐在床榻上，慕清霄呼吸均匀，脑海中不断的寻找天书，可惜找了半天，半点痕迹都没找到，最终只能作罢，天书功法在强大，也不可能强过圣典。
<br/>　　心神退出脑海空间，万剑一已经回到祖先祠堂，在也没有多说什么，看来自己的一番话也是挺有效果的，希望不要在多生事端才好。
<br/>　　半个月的时间，一晃而过，面壁时间已过，慕清霄也不想在祖先祠堂多待。
<br/>　　“阁下，半月前的事情，我可以当作没有发生过，但也希望你不要做出危害青云们的事情。”
<br/>　　“如此甚好。”
<br/>　　慕清霄瞥了眼身后的万剑一，微不可查的点头，若是他将半月前的事情捅出去，肯定会招来很多不必要的麻烦，他这个人最讨厌的就是麻烦了。
<br/>　　穿过郁郁葱葱的小道，来到后山入口，慕清霄一愣，因为半个月前领他来此的道童，已经在入口处等候。
<br/>　　“师兄，张师兄几人已经在广场上等候，掌门真人已经通知过田师伯，师兄们直接下山即可。”
<br/>　　......
<br/>　　......
<br/>　　很快，来到云海广场，途中遇到几名通天峰弟子，无一不是对他恭恭敬敬，甚至眼中都有些惧意，慕清霄自然知道原因，微微摇头，不做理会。
<br/>　　放眼望去，慕清霄就锁定了悟道树下的四名身影，径直走了过去。
<br/>　　四人也看到了慕清霄，陆雪琪快速走了过来，在他身上看了看，语气温柔，道：“你没事吧？”
<br/>　　闻言，慕清霄一把将她的纤纤玉手握在手中，微笑道：“面壁思过而已，能有什么事，我这不是平安无事的站在你面前吗？”
<br/>　　“恩。”
<br/>　　由于心中的担忧，在这大庭广众之下，陆雪琪竟没有第一时间挣开慕清霄的手，脸色有些微红，看到张小凡几人围了过来，一鼓作气甩了开来。
<br/>　　“慕师弟，师傅他吩咐，让你面壁结束后，与我们一同下山，出门可不能给他丢脸，这些年存下来的银两我也给你带过来了，衣服去山下的河阳城就有的卖。”
<br/>　　张小凡走了上来，他知道两人是恋人关系，当下也见怪不怪了。
<br/>　　曾书书嬉笑的望着周围，眼底却是一些疑惑，没想到对他们不加以言辞的陆雪琪，对慕清霄如此热情，心下也猜到了些什么，而林惊羽的目光则一直放在张小凡身上。
<br/>　　简单的商议之下，决定先下山去河阳城里，于是一行人就直接上路，慕清霄和陆雪琪走在最后面，张小凡，曾书书和林惊羽走在最前面。
<br/>　　曾书书对张小凡笑道：“小凡，掌门师伯传给你的六合镜厉害么？”
<br/>　　闻言，张小凡看了一眼慕清霄，道：“听说这六合镜乃是我青云门至宝，虽然我没有试过，但肯定是极其厉害的，此处乃是山顶，除了七脉首座，其余弟子都不能御剑飞行，我们快离开云海，御剑离开吧。”
<br/>　　陆雪琪默默的走在身后，慕清霄去哪里，她就去哪里，曾书书笑容满面，张小凡和林惊羽也有些期待，他们自从拜入青云门后，这还是第一次下山呢。
<br/>　　......
<br/>　　......
<br/>　　从青云门到河阳城，这一路上，青云门最牛逼的五位弟子御剑而来，都显得极为轻松，宛如仙神。
<br/>　　张小凡手中也多出一柄较为普通的仙剑，陆雪琪祭起蓝色的‘天琊’神剑，曾书书则是一柄微带紫色的仙剑‘轩辕’，林惊羽自然是斩龙剑。
<br/>　　慕清霄也不藏着掖着，反正他们也看不出魂剑的品质，直接祭出魂剑，霸气无双外形，凌厉的气息不免让众人一阵惊叹，同时也震惊他的修为恐怖。
<br/>　　要知道，慕清霄在七脉会武上可没有使用自己的仙剑，每场比试都是一击废了对手，成就了他的凶名，这还是没有用全力的情况下，若是使用仙剑，那该有多强。
<br/>　　若是没有被剥夺比试资格，或许七脉会武桂冠就不是张小凡了，但他们也都没有多问，知道有些事情是个人隐私，毕竟谁心里没有点秘密。
<br/>　　穿云越山，由于众人心中都好奇，四处观景，所以路上也有些耽搁，这一段本是半日的路程，他们却直到太阳下山才抵达河阳城。
<br/>　　为了避嫌，慕清霄五人选择了个较为偏僻的地方落下，由于众人修为都不错，体内灵气也较为浑厚，长久御剑飞行，对他们影响也不大。
<br/>　　稍做休息，整理了下凌乱的白衫，在夕阳的余晖中，五人徒步向远处高大的河阳城走去。
<br/>　　慕清霄依旧与陆雪琪走在最后，曾书书也识趣的没有来打扰，而是滔滔不绝的向众人介绍河阳城，倒是让两人都清静了会。
<br/>　　“方圆百里之内，这里是最繁华的地方了，住在这城里的百姓，少说也有数十万，而且地理位置又好，主来商旅极多，城里更是热闹非凡。”
<br/>　　众人听着，心中着实佩服曾书书博学多识。
<br/>　　听众人嘀嘀咕咕，慕清霄望了一眼天色，道：“雪琪，天色不早了，今晚我们就在这里过夜吧，明日在继续赶路。”
<br/>　　陆雪琪微微一笑，慕清霄低声在她的耳边窃窃私语几句，也不知道说了些什么，令陆雪琪冷若寒霜的绝美脸颊涌上两朵不自然的红晕。
<br/>　　进入城中，因为众人都穿着白衫，倒也没有引起什么怀疑，倒是陆雪琪容貌绝美，却是在河阳城中引起了不小的轰动，惹得不少路人驻足观看。
<br/>　　慕清霄看了一眼陆雪琪，见她虽然面无表情，但一双美眸却满含怒意，拍了拍她的香肩，道：“爱美之心人皆有之，这不过是人之常情。”
<br/>　　听到他的话，陆雪琪眼中的怒意也逐渐消散，可见涵养是极其的不错。
<br/>　　很快，曾书书就负责在河阳城中找了一家有名的客栈后，由于他对此处比较熟悉，所以住客栈很多琐事都得由他处理，后者也显得乐此不疲。
<br/>　　曾书书也不负所望，与客栈老板商量过后，就有客栈的仆役领着他们来到了客栈最上等的后园居住。
