        &lt;h3&gt;第三卷第二十三章  为妹妹铺路&lt;/h3&gt;
<br/>
<br/>        在京城，陈子寒见到了他的妹妹，只是他不能相认，他是谷强，是谷老大的义子，他从小无父无母，是在孤儿院长大的。
<br/>
<br/>        他见到了他的小百灵，他的小百灵已经长大成人了，而且还是一个成功人士，看着她那么自信，那么美丽，他真是替她感到开心。
<br/>
<br/>        他的小百灵认出他来了吗？看着他的小百灵和他像普通人那样聊天，他有些心塞，他是她的哥哥啊。
<br/>
<br/>        小百灵已经认不出他来了，是忘记了吗？也难怪，他离家的时候他的小百灵才13岁，没有认出来他也好。
<br/>
<br/>        当时为了打击张函，他提出了让陈子昂到甸城发展，可是没有想到她那么快就答应了，而且真的准备要去甸城发展，那他就得回去为她铺路了。
<br/>
<br/>        他们分开后，他和祁铮聊了一下蔣靖的事情，可惜祁铮的身份和他一样，他们都是单线联系，祁铮是航城一个部门的，而且他做的事情祁铮也不了解。
<br/>
<br/>        他们当时是被挑选的精英，都归蔣靖，为了保护他们的安全和身份，他们的档案都是绝密的。目前他们两个人都不知道怎么办。
<br/>
<br/>        他们只能等，等着有人来和他们联系。
<br/>
<br/>        谷强这一次的出行唯一的收获就是见到了他日思夜想的妹妹，但是也给他带来了苦恼。
<br/>
<br/>        他调查了一下陈子昂这几年的发展历程和速度，发现她不但和落家有关系，还和京城六大世家中的陆家、张家俩个大少都有联系，张家还好，清清白白的，可是陆家和秦寿往来已久。
<br/>
<br/>        唯一的庆幸的是，她虽然和他们有过来往，但是没有任何瓜葛，他能想象得到，当初的陈子昂应该是很辛苦的。
<br/>
<br/>        他们兄妹面相都有些相似，如果子昂去了甸城会不会被发现或者被怀疑呢。
<br/>
<br/>        这是一个问题。
<br/>
<br/>        三部合纵的计划一直在进行中，目前他一直在拖延着时间，托一段时间是一段时间，如果那个计划真的实施，而没有任何措施的话，他将是千古罪人。
<br/>
<br/>        他已经做了最坏的打算，如果他找不到回家的路，那么他将和那些货同归于尽。
<br/>
<br/>        陈子昂如果去甸城也好，这样他可以将部分精力放在甸城的信息化项目上，以此来拖延三部合纵的计划进度，不过，这需要谷正娟的配合。
<br/>
<br/>        他去了m国，见到了托尼，但托尼不是最后做抉择的人，他刚好用这个为借口，回来给谷老大和秦寿汇报，m国的人不够诚心。
<br/>
<br/>        这批货可以说已经搭进去了秦寿和谷老大的三分之二的资产，他们也会倍加小心的。
<br/>
<br/>        春临山庄
<br/>
<br/>        “爸，我此次去m国只见到了托尼，而你知道，托尼做不了决定。我等了四天，桑斯都借口不见，我就回来了。”
<br/>
<br/>        真实的原因是，谷强知道桑斯去了外地，大概需要两个礼拜的时间。
<br/>
<br/>        “桑斯是什么意思？”谷老大皱着眉。
<br/>
<br/>        “我也不知道，是不是会有别的想法，或者是出了什么事情？我在和托尼聊天中，托尼一直在找借口推脱。爸，我建议三部合纵进度不要那么快。”
<br/>
<br/>        “给你秦叔叔说过这个问题吗？”
<br/>
<br/>        “还没有来得及呢。”
<br/>
<br/>        “我知道了，我和你秦叔叔商量一下再说。”
<br/>
<br/>        “好的，爸，还有一件事情，我去了京城，那边发展太快了，我住酒店的时候，参加了一个信息化项目。”
<br/>
<br/>        “信息化项目？”
<br/>
<br/>        “是的，通过互联网传输，这样和外界联系就方便了很多，比喻你要和托尼联系，只要用一台电脑就可以和他联系，比打电话还方便。还有这个一般的单位使用起来很方便，是将来发展的趋势，我想将这个项目拿下来，让正娟来做。”
<br/>
<br/>        “可以做？”
<br/>
<br/>        “如果拿下来，全面铺开得3年，这三年我们是能赚到不少的，而且还没有任何风险。”
<br/>
<br/>        谷老大是知道谷强的聪明和才能的，他还是很相信谷强的眼光。
<br/>
<br/>        “既然如此，那你就和正娟开始做吧。”
<br/>
<br/>        “爸，不过，这些都是和一些重要的人打交道的。”
<br/>
<br/>        “你爸爸我和你秦叔叔也算是甸城有名的人，谁能比得过我们？你只管做就好了。况且，少卿已经是副总了，有陆家在，这一两年少卿就会升职的。”
<br/>
<br/>        “爸，和我联系的是一个女孩子。”
<br/>
<br/>        “女孩子？喜欢上人家了？”谷老大一直在给谷强介绍对象，谷强都看不上。
<br/>
<br/>        “就怕你这样想，我和她只是想合作，她在京城发展的很迅速，而且生意做得很好。好像背景也挺复杂的。”
<br/>
<br/>        “都有些啥背景？”
<br/>
<br/>        “和落家、陆家、张家都似乎有联系。”
<br/>
<br/>        “那这个女孩是什么出身？”
<br/>
<br/>        “普通的北漂族，一个小城市出来打拼的女孩，家庭背景很简单，就是不知道怎么和这几个大家族联系上了。”
<br/>
<br/>        “呵，我倒是很好奇是什么样的女孩了。”
<br/>
<br/>        谷强不知道这样说好不好，但是，如果子昂真的要来的话，他必须要她有强大的背景才能在甸城这个吃肉不吐骨头的地方生存。
<br/>
<br/>        “我和她一见如故，似乎认识很久了。”谷强微微的有点苦闷。
<br/>
<br/>        “强子，你终于开窍了。”谷老大拍了拍谷强的肩膀。
<br/>
<br/>        “你好好的做你想做的事情吧，爸爸支持你。”
<br/>
<br/>        “谢谢爸！”
<br/>
<br/>        “三部合纵的事情等我和你秦叔叔商量好了再说，目前你先不动了。”
<br/>
<br/>        “好的，爸。”
<br/>
<br/>        谷强回到了甸城。
<br/>
<br/>        “强子，你回来了！”谷正娟推开谷强办公室的门，两眼微红，天知道她有多担心他。
<br/>
<br/>        “正娟。”谷强看着谷正娟，她浓黑的眉毛下，大大的眼睛温柔多情，急迫的看着他，眼睛里有担忧有惊喜，还有说不出来的一种忧伤。
<br/>
<br/>        他们俩个人面对面的站着，彼此眼里都看到了自己，却不肯向前走一步。
<br/>
<br/>        在春临村的哪一个夜晚，善良温柔的谷正娟温暖了谷强那颗漂泊的心，给了谷强爱的勇气，她和孙正楠一样活泼热情，散发着青春洋溢的气息。
<br/>
<br/>        当，谷强明白谷老大身份之后，一切都变了。
<br/>
<br/>        当，谷正娟急切的回到春临村时，使她动心的愿意为他付出一切的人已经变成了她的哥哥。
<br/>
<br/>        是的，变成了她的哥哥。
<br/>
<br/>        “你为什么会变成我的哥哥？”
<br/>
<br/>        “娟子，是你爸爸让我做他的儿子。”
<br/>
<br/>        “不，我不要你做我的哥哥，我已经，我已经是你的人了。”
<br/>
<br/>        “娟子，父亲告诉我，你以后和少卿的婚礼要我好好操办。”
<br/>
<br/>        很多年了，谷正娟没有嫁给秦少卿，她一直一个人，谷强也是一直一个人。
<br/>
<br/>        “强子，你回来了就好。”谷正娟眼含热泪，也许，他们是会有将来的，她一直等着。
<br/>
<br/>        “正娟，我刚好有事情和你商量。坐吧。”谷强客气的邀请谷正娟坐下，并给谷正娟拿出来他带给她的小礼物。
