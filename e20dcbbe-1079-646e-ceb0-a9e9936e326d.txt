　　涂小安放眼前方，广场地带有一块巨大的白色晶石，伫立在人群中间，不时冒出耀眼的光芒。
<br/>　　只见一个年轻人走过去，站在白色晶石的旁边，将手放上去。
<br/>　　“冯峰义，觉醒一阶七段，力量修武者”
<br/>　　白色晶石直接将这个人的实力数据显化了出来。
<br/>　　“哇，好厉害...这人恐怕妥妥的上等班级了”
<br/>　　人群中一片涌动，不少人议论起来，虽然力量修武者算是普通的，但是新生就有一阶七段的实力，也已经很厉害了。
<br/>　　“那晶石就能测量个人修为实力，看着还挺神奇。”
<br/>　　涂小安微微思忖，这东西能测量出自己的实力吗。
<br/>　　那年轻人，涂小安一个意识洞察丢过去，17级，在这里就是一阶七段。
<br/>　　他现在有点担心，这个晶石有点不凡，会不会将自己是条蛇精的身份给暴露。
<br/>　　要给暴露了，那就不好玩了，恐怕他也没办法在古学院待了。
<br/>　　“大神，其实个人实力测量不单单只有上中下三个班，还有一个凌驾之上的超等天才班”林凡想了想，忽然开口。
<br/>　　“超等天才班？”涂小安微微一愣：“你细说我听听”
<br/>　　“这个超等天才班，里面每一个人都是变态，实力起码要二阶以上，而且这仅仅是第一个小要求，还有你的觉醒能力也非常的关键，能被吸收超等天才班，那将是这个世界未来最粲然的星星”
<br/>　　“目前，整个古学院，不分时期进入学院，超等天才班是五十人，当你被测量你的实力有资格进入超等天才班，还要挑战里面最弱的存在，赢了他，你才可以进入”
<br/>　　涂小安露出了饶有兴趣的表情：“也就是说这个天才班，永远都只有五十人对吧，弱智淘汰，强者进入”
<br/>　　“你在里面吗”涂小安看着林凡一问。
<br/>　　其实这个林凡在跟自己的宠物鼠精合体后，实力还是非常的不错，按等级是二阶五段的实力了。
<br/>　　“我当初有资格挑战天才班的第五十人，可惜我去年挑战失败了”林凡苦笑的摇了摇头。
<br/>　　涂小安呀色了下，林凡连天才般排名的第五十人都打不过，看来还真的是超等天才班呀。
<br/>　　里面每个人都有一身可怕的实力，或许是可怕的异能手段。
<br/>　　“大神，已你的实力，想进天才班，肯定没一点问题的”林凡对他非常的有自信。
<br/>　　他的元素能力，在自己合体之后，还能一招冰封，绝对是变态中的变态了。
<br/>　　涂小安并不是真的来古学院修炼的，一是为了保护东方袁的安全，二是他对古学院的湖泊底下有点兴趣。
<br/>　　很想知道湖泊下面是不是真的被镇压的一头三阶妖灵。
<br/>　　另外，那笛声到底是怎么回事。
<br/>　　“听说你很强，进天才班，我等着你”
<br/>　　突然，一道冷冽的声音骤然传到了涂小安的脑海中。
<br/>　　有人对他千里传音？
<br/>　　什么人！
<br/>　　涂小安何等敏感，妖治的眸光朝着一个方向倏然看去，只见在广场外一颗茂密的大树下，站着一位剑眉星目，甚为仪表堂堂的年轻男子，他一袭白，露出古铜色的强壮肤色，浑身散发着一种骇人的气息，脸上噙着一根草，邪魅的望着他。
<br/>　　声音是他传给涂小安的。
<br/>　　这人貌似不认识呀。
<br/>　　“你是谁”
<br/>　　涂小安的声音同样出现在对方的脑海中。
<br/>　　古铜色男人笑了笑，倏然地，身形一动，恍若幻影，既然不到三个呼吸，就出现在了广场中涂小安的面前。
<br/>　　宛如鬼魅般的出现，将林凡跟东方袁吓了一大跳，
<br/>　　当林凡看到古铜色男子，脸色骤然大变，不禁的倒退了三步，可见他内心有多么的震撼。
<br/>　　“你夺了我的蛇王石，却不知道我是谁，未免有点可笑吧”他悠悠然的开口，带着不羁。
<br/>　　“白家少主，白君然...”林凡尖叫了出来。
<br/>　　蛇王石？白君然？
<br/>　　涂小安秒懂了，又是白家的人，只是眼前这位白家的人，总算有点不同凡响，浑身气息很强烈，就好像火山随时要爆发。
<br/>　　涂小安笃定，这是他来到温城，遇见最强的人类修武者了。
<br/>　　比妖精管理局组长，二十九级冰霜修武者冷霜还要强。
<br/>　　非常接近三阶生灵，几乎一脚迈过去了。
<br/>　　涂小安是很敏感的，这人纵然不是三阶生灵，但是恐怕战斗力要比三阶生灵还要可怕一点。
<br/>　　因为对方一出现，就有一股强大的气息压在涂小安的身上，犹如五指山，让人动弹不得，空间开始剧烈的波动起来。
<br/>　　退后，退后...
<br/>　　林凡大喊，立刻将东方袁拉飞了出去，晚一点，这胖子恐怕要身受重伤，对方这强大的气息，不是还没觉醒的胖子可以承受。
<br/>　　“原来蛇王石是你的”涂小安恍然大悟，话说着，眼睛浑然一眯，周身一震，一股气浪破体而出，溃散了对方的压力。
<br/>　　白君然纹丝不动，见此，反而笑了，笑的很冷，最后变成了狞笑。
<br/>　　“还敢反抗？”
<br/>　　霎时间，一波恐怖而磅礴的气息冲天而起，白君然的身体突然爆出一团青色的光芒，青色光芒如跳动的焰火一般笼罩在他的体外。
<br/>　　他上身的衣服在刹那间碎裂，粉碎的衣衫如尘沙一般在空中飘扬而下，露出了一身古铜色的皮肤，与此同时，青色光芒排山倒海的朝着涂小安压过去。
<br/>　　空间层层爆裂。
<br/>　　涂小安眼眸精光一闪，周身深蓝色气焰迸发，自称一股力量，一股冰雕战甲骤然笼罩在他的体外，抵挡住了对方的青色光芒力量。
<br/>　　嘭嘭嘭...
<br/>　　空间发出大方位的爆响声音，一股股狂风开始肆意的席卷出去，将不少路人直接给掀飞了出去。
<br/>　　“怎么回事...”
<br/>　　天空似乎要垮塌下来了，一股窒息感侵袭了整个广场，巨大的动静，将小广场上准备测量实力的新生都给吸引了，一个个撑着惊骇的目光看着不远处两个年轻男子，他们针尖对麦芒，散发出来的力量波动，简直让人不敢置信。
<br/>　　“不好，是超等天才班的白君然“
<br/>　　“听说这个人无限接近三阶生灵，在天才班是前五的高手，也是古学院站在巅峰的学员”
<br/>　　很多人一下子就认出了没了上衣露出一身古铜色皮肤的白居然，他浑身爆炸性的青色光芒，简直让人惊为天人。
<br/>　　另外一个是谁？
<br/>　　既然能跟白君然对峙起来，而且好像还完全不落下风。
<br/>　　很多人用更加惊讶的目光投放到涂小安的身上。
