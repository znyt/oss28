　　万寿山地界外，东皇太一率领妖族偷袭红云道人，正在红云岌岌可危之际，他的老友镇元子赶来了。
<br/>　　这一切只发生在短短片刻的时间内，这场战争却汇聚了洪荒中大多数人的眼睛，洪荒七圣，三清、女娲、西方二圣，甚至就连道祖都隐匿与九天之上，悄悄注视这这场争斗，不是因为他们之间的大战有多厉害，而是这场战争决定这圣道之基鸿蒙紫气的归属，甚至决定着洪荒未来的走势。
<br/>　　昆仑山，元始三人面目表情的看着这场战争，对于他们来说，谁胜谁负已经不是很重要了，因为不管哪方得到鸿蒙紫气，也威胁不到他们，不过，为了日后的利益，他们还是不希望被妖族得到它。若是妖族得到鸿蒙紫气，妖族再出一位圣人，将会打破洪荒平衡，这是圣人所不希望的。
<br/>　　圣人之所以为圣，是因为他们高高在上，享受着诸天的祝福，若是这种高高在上的人变多了，那么圣人也就变得没意义了。
<br/>　　西方灵山。准提圣人与接引圣人遥遥看着这场战争，他们面无表情，好似这些人的生死与他们无关似的。
<br/>　　事实的确如此，当初紫霄宫讲道之时，因为有着红云的让位，这才有了西方二圣的诞生，若是当初红云没有让位，西方便不会出现二圣，他们此时还不知道在哪里奋斗着呢？
<br/>　　这无形中让西方二圣与红云结下了大因果，这份因果关系着成圣，大到他们偿还不起，所以，他们如今也希望这红云道人就此消亡，也让他们了去这一段因果。
<br/>　　看着就快要被东皇太一灭杀的红云道人，两声双手合十，默念道：
<br/>　　“阿弥陀佛！”
<br/>　　却说大战之处，东皇太一正打算一鼓作气偷袭红云，将之灭杀，可就在此时，阵外传来镇元子的轰阵之声，只听“轰隆隆”一声巨响，足足六个妖神便被灵宝地书砸死，大阵因为死了六人，出现不完美，顿时有了一丝晦涩，红云抓住这个机会奋起反抗，朝着大阵纰漏之处轰了过去，东皇太一还没来得及阻止，大阵便被破了。
<br/>　　原本的周天星辰大阵本不该如此脆弱，但这因为这是简易版，比不上完整的大阵威力，而且主阵的东皇太一一心扑到红云身上，并没有如之前那般有两个人主阵，所以才给了镇元子可乘之机，一举破阵。
<br/>　　阵外，镇元子头顶一株参天大树，枝叶遮天，碧叶流翠，他脚踏一本古朴书籍，正是大地胎膜——地书。浑厚的厚土之力从地书上冲了上来，与人参果的乙木之力结合在一起，土生木，将他与红云牢牢的守护在其中。
<br/>　　“镇元子，识相的赶紧交出红云，否则今天你也要陨落于此。”
<br/>　　东皇太一有些羞恼，差点就得手了，却偏偏被镇元子破坏了，这让他如何不恼。同时，虚空之中再次出现一道霸气的身影，一身金色九龙帝袍，一柄金色帝剑，却是妖帝帝俊来了。
<br/>　　“镇元子，你确定要保红云？”
<br/>　　帝俊脸色冷峻，眼神深邃，谁也看不出他此刻在想些什么，一句轻飘飘的话被他问了出来，无形中却增添了几分帝王的霸气。
<br/>　　镇元子听此，不由为之一笑，接着朗声笑道：
<br/>　　“帝俊，太一，我镇元子是那般贪生弃友之人吗？”
<br/>　　“看来，你是不打算交出红云啰！既然如此，你就随着他一起去死吧！”
<br/>　　帝俊眼神一冷，语气淡淡道。他手持金色帝剑，毫不犹豫，直接开封，斩向地书中的红云道人。金色剑气破空，一股强烈的火红之气从帝俊身上升起，剑道带着无边霸气从虚空之中斩出，划破虚空禁锢，斩向镇元子。
<br/>　　这时，一旁的东皇太一也动了，他挥手间便召唤出无尽金色太阳之火，一股炙热、至阳、霸道的火焰之力，燃烧而来，火克木，对付镇元子的乙木之力，太阳真火无疑是最好的选择。
<br/>　　霸道而炙热的太阳真火就好像一团美丽而绚烂的烟花，划破长空，以熊熊之势，轰向镇元子，同时帝俊的剑锋冷厉霸道，破开结界，配合太一杀向镇元子，因为他们都知道，只有灭了镇元子，才有机会得到红云。
<br/>　　就在破开结界的那一瞬间，一道寒光闪烁，疾风闪现，电光火石之间，便卷向一旁的红云，这一幕谁也没有想到。
<br/>　　来人速度之快，把握时机的准确性，都是极为顶尖的，众人之中谁也没有想到会发生如此变故。但既然变故已经发生了，那便不能让它继续这么下去。
<br/>　　“尔敢！”
<br/>　　帝俊一声冷哼，剑锋转向，顺势斩向那道疾风。谁知那疾风速度之快出乎帝俊想象，剑光还没有斩到他，便被它将红云裹住。
<br/>　　“鲲鹏，你找死！”
<br/>　　帝俊一声大喝，却是认出了那疾风的身份，在这洪荒之中，要数速度之快，除了圣人之外就属妖师鲲鹏了。
<br/>　　“爆！”
<br/>　　一声冷喝从红云身上传出，紧接着，便听到一声剧烈的爆炸声。
<br/>　　轰隆隆！
<br/>　　神光闪耀，红光漫天，火红的爆炸之威席卷方圆数百丈，天地破碎，山河毁灭，却是红云自爆善尸。正处于爆炸中心的鲲鹏，无处可躲，被炸得血吐三升，血肉模糊，损失惨重，真是偷鸡不成蚀把米！
<br/>　　“红云！”
<br/>　　镇元子大恨，自己的老友就在自己眼前自爆了，自己却没有能力阻止，他恨不得杀了鲲鹏。
<br/>　　“鲲鹏，我与你势不两立！”
<br/>　　“嗖！”
<br/>　　这时，从那爆炸中心，飞出一道红色的流光，想要向着远处飞去。可惜，众人都将目光紧紧地盯着那里，他哪有那么容易跑得了啊。
<br/>　　流光刚一飞了出来，便被帝俊截胡了。镇元子乍一见自己老友并没有死，顿时大喜，但见到被帝俊截胡了，顿时大怒，拿起脚下的地书便向着帝俊挥力砸去。
<br/>　　地书光芒大涨，土黄色的厚土之力旋转而来，将帝俊包围，同时地书化为一座大山向着帝俊压了下来。这给了红云逃跑的时间。
<br/>　　“走！”
<br/>　　镇元子向着红云大喝一声。
<br/>　　“找死！”
<br/>　　这时，东皇太一冷哼一声，混元剑开锋，凶气纵横，戾气冲天，熊熊太阳真火纵横，一剑毫不保留地斩向会镇元子的头颅。
<br/>　　镇元子已经被帝俊缠住了，顿时脱不开身，眼见这一剑就快要将镇元子斩杀了，这时，红云眼中闪过坚定之色，看向东皇太一目光凶狠，竟然要以自己的身体前去阻挡东皇太一那必杀一剑。
<br/>　　“噗嗤——”
