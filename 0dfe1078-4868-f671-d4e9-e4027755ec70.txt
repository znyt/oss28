　　轰！
<br/>　　叶尘在痛苦之源的刺激下，理智已经几乎完全丧失，似乎不知道防御，竟然被其中一名高手偷袭得手背上中了一剑。

<br/>　　可饶是如此，也让众人全都精神一震，
<br/>　　“他已经是强弩之末，这种状态不可能持续太久，大家加把劲啊！”
<br/>　　云千秋更是忍不住兴奋的高呼起来。
<br/>　　然而，
<br/>　　“吼！”
<br/>　　身中一剑的叶尘，顿时发出一道猛兽般的咆哮，两眼之中的血光，几乎要喷射而出！
<br/>　　“杀！！”
<br/>　　一股浩瀚无比的杀意，充斥着惊人的毁灭气息，突然自叶尘的身上释放了出来。
<br/>　　轰！
<br/>　　叶尘的速度竟然再次暴涨，完全无视众位众人的攻击，直奔方才砍中他一剑的那名高手，不要命般冲了过去！
<br/>　　那人顿时吓了一跳，当即飞速暴退。
<br/>　　而其他高手们，被叶尘身上的惊人杀意所震惊，一时间竟然忘记了搭救。
<br/>　　唰！
<br/>　　一道璀璨的剑芒在空中划过，瞬间便将那人一剑斩成了两半！
<br/>　　“嘶！！”
<br/>　　其他人看到这一幕，顿时全都倒吸了一口凉气，同时也反应了过来，这才又开始纷纷发动攻击。
<br/>　　轰！
<br/>　　又有一名太乙门的长老，趁叶尘不备，一剑斩落在了他的臂膀之上，险些将他的整条胳膊斩断！
<br/>　　“吼！”
<br/>　　叶尘顿时再次发出一声怒吼，转而又朝那位太乙门长老冲了过去。
<br/>　　轰轰轰！
<br/>　　那太乙门长老火速暴退，周围其他高手也纷纷出手帮忙。
<br/>　　叶尘又硬抗了几名强者的攻击，同样将那名太乙门长老一剑斩杀。
<br/>　　轰隆隆！
<br/>　　天空上的战斗，很快便进入了白热化状态。
<br/>　　叶尘已经完全就像是疯了一般，一旦确定了目标，便非杀死不可。
<br/>　　渐渐地，众人也意识到，想要活捉眼前这个少年，几乎是不可能的，而且逃跑恐怕也不行，他们只能死战到底，要么他们被叶尘杀死，要么他们杀死叶尘。
<br/>　　很快，
<br/>　　随着这些强者们也开始拼命，叶尘身上的伤口也越来越多，气息有明显减弱的趋势。
<br/>　　轰！
<br/>　　叶尘又一剑斩杀了，一名元婴境后期强者，至此他已经杀了近十名强者，而他的身上，也早已经到处都是伤痕，鲜血已经将全身的衣服染成血红色，犹如地狱中爬出来的恶魔！
<br/>　　“哈哈哈！他已经快不行了！大家千万不要松懈啊！”
<br/>　　一直躲在人群后方的云千秋，再次开口高呼道。
<br/>　　众强者的脸上，也纷纷露出一抹笑容，似乎已经胜券在握。
<br/>　　“叶狂仙！来接我最强一击！”
<br/>　　“雷神之怒！”
<br/>　　天雷宗宗主上官云阙，忽然飞至九天高空，一声暴喝传来。
<br/>　　只见他，手握一把权杖，而权杖之上，则聚集了一团恐怖的雷光，直冲云霄！
<br/>　　上官云阙此刻沐浴在雷光之中，犹如九天之上的雷神下凡，似乎要将整个苍穹吞噬。
<br/>　　此人之前，显然一直在保留实力，直到看见叶尘实力衰落，这才使出了自己的最强一击。
<br/>　　“死吧！”
<br/>　　轰隆隆！
<br/>　　恐怖到令人心悸的雷光，犹如一头能毁灭万物的雷龙，朝下方的叶尘咆哮而来，速度更是快到可怕，远远超过了十倍音速。
<br/>　　彭！
<br/>　　没有任何悬念，叶尘瞬间便被这雷光所击中，整个人顿时淹没在那恐怖的雷光之中。
<br/>　　“打中了！这便是天雷宗的最强绝技，雷神之怒吗？”
<br/>　　“这小子已经是强弩之末，再中这么一记雷神之怒，就算不死，也得重伤吧？”
<br/>　　“他死定了！”
<br/>　　……
<br/>　　五大宗门，共三十多名元婴境强者，已经死了将近三分之一，剩下的这些人，个个对叶尘咬牙切齿，恨不得要将他挫骨扬灰。

<br/>　　众人却看到了无比震惊的一幕！
<br/>　　只见，
<br/>　　那团充满着毁灭气息的雷光，竟然在迅速的消散！
<br/>　　最终，竟然完全没入叶尘的身体之中，消失的无影无踪。

<br/>　　所有人全都看傻了眼，
<br/>　　他们怎么也没有想到，叶尘不但没事，竟然还把那股恐怖的能量，完全吞噬进入了自己体内！
<br/>　　“不！这怎么可能？如此狂暴的能量，竟然被他吸收了？他用的什么妖术！”
<br/>　　众人全都又惊又恐，个个满脸的不可思议之色。
<br/>　　而接下来的一幕，却更加让众人惊愕的说不出话来。
<br/>　　因为他们发现，之前气息已经开始急剧衰落的叶尘，不但开始逐渐恢复，竟然再次暴涨！

<br/>　　“这怎么可能？他明明已经中了痛苦之源，居然还能自行突破？”
<br/>　　“快！阻止他！绝对不能让他完成突破！”
<br/>　　……
<br/>　　众强者一下子全都慌了。
<br/>　　他们又哪里知道，叶尘之前虽然几乎丧失了理智，可是在生死存亡之际，终于恢复了几分意识，当即一连吞下了两枚扶摇仙果，压制住了痛苦之源。
<br/>　　同时运转吞天神功，将那团狂暴的雷光吞噬进入了体内，再加上吞天鼎的作用，叶尘一鼓作气，直接冲破了修为壁障，竟然破而后立，一举迈入了金丹境！
<br/>　　就连下方的祝明飞，看到这一幕，同样两眼一瞪，
<br/>　　“他竟然……在打斗之中强行突破！他这是不要命了吗？”

<br/>　　一直躲在人群后方的云千秋，此时已经绕到了叶尘身后，趁着他刚刚突破之际，准备对他发动致命的一击。

<br/>　　叶尘环视了一圈众人，缓缓开口，声音冰寒彻骨。
