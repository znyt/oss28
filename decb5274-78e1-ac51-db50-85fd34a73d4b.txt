其实夜老在设计让夜滢借腹上位，嫁入厉家大门时，他是十分紧张和忐忑的。
<br/>
<br/>        因为他的对手有两个，一是厉君墨，二是林水瑶。
<br/>
<br/>        这两个人一个是帝都之子，一个是绝艳风华的神秘少女，这两个人可以说共同开创了帝都医学的新纪元。
<br/>
<br/>        当夜老看着林水瑶怀孕，肚子一天又一天的大起来时，他很怕，怕林水瑶找到厉君墨，如果她去找了，那一切都泡汤了。
<br/>
<br/>        但是，林水瑶丝毫没有去找厉君墨的念头。
<br/>
<br/>        林水瑶很喜欢肚子里的这个女儿，也很享受当妈咪的这一段人生新旅程，那样绝艳的女孩一天天的变得温柔，她的手总是放在自己凸起的小腹上抚摸，她很爱自己的女儿，但是她不喜欢这个女儿的爹地，那晚过后，她就将厉君墨那个男人给彻底抛诸脑后了。
<br/>
<br/>        当然也有一次夜老曾经站在门口看到了柳璎珞，柳璎珞和林水瑶是最好的闺蜜，他听到林水瑶向柳璎珞抱怨了一句那晚那个炮.友真不是人，而是禽兽，我差一点就被他给折腾死了。
<br/>
<br/>        林水瑶对厉君墨就是这样准确定位的，炮.友！
<br/>
<br/>        夜老还十分担心厉君墨这里，因为当年的帝都城都知道这位帝都之子不近女色，他骨血里流淌着高贵而禁欲的血统，对女人这种生物毫无性趣，就算夜滢怀了他的孩子，他也不一定会娶夜滢的。
<br/>
<br/>        但是，厉君墨娶了。
<br/>
<br/>        夜老还记得那一天厉君墨站在落地窗前，任由外面璀璨的霓虹灯镀了他一身神秘尊贵的金芒，他看了夜滢很久，很久很久，然后点头，同意负责了。
<br/>
<br/>        夜老不知道厉君墨在想些什么，但是有一点是肯定的，那晚对于厉君墨而言，念念难忘。
<br/>
<br/>        夜老这些年运筹帷幄，自己成为了科学院的院长，女儿夜滢上位成了厉夫人，虽然厉君墨冷落了夜滢这么多年，但是厉君墨一手扶持夜家成为帝都四大豪门之一，成为了夜家的金库，取之不尽用之不竭。
<br/>
<br/>        现在，一切平衡都打乱了，夏夕绾竟然是林水瑶和厉君墨的亲生女儿！
<br/>
<br/>        夜老倏然眯起了那双浑浊的双眼，他想起自己第一眼看到夏夕绾时就有一种很熟悉的感觉，夏夕绾生的纤尘绝色，气质聪慧从容，一身大家风范，她身上有林水瑶的影子。
<br/>
<br/>        夜老这些年一直在寻找林水瑶和厉君墨的这个亲生女儿，想要斩草除根，因为他知道这个女儿就是一个定时炸弹，他会毁了夜家现在所拥有的一切。
<br/>
<br/>        夏夕绾！
<br/>
<br/>        夜老在心里反复咀嚼着这个名字，真不愧是林水瑶和厉君墨的亲生女儿，现在已经堪称风华。
<br/>
<br/>        再过个几年，不知道要出落成何等光芒万丈的模样。
<br/>
<br/>        他现在就要除了她！
<br/>
<br/>        “滢滢，夏夕绾的事情交给我，我会让无忧出手的，但是你记住了，嫣然的身世绝对不能暴露！”夜老叮嘱道。
<br/>
<br/>        夜滢又不傻，厉嫣然的身世是必须死守的秘密，她点头，“爸，我嘴巴里是透不出风的，你放心吧，但是夏夕绾的事情刻不容缓，陆寒霆陆子羡徐少南那些天之骄子都被她给迷晕了，越到后面越难对付。”
<br/>
<br/>        夜老并没有多说什么，只是应道，“我知道了。”
