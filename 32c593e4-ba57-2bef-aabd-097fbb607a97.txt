也挡不住请客吃饭的人，所以生意是一日胜似一日。<br/>　　<br/>　　沈博文看着这种情况，除了苦笑也就没有别的了。西市文哥，还真不是乱叫的，当时要不是自己的侄女，他绝对能以低价格控制陈默的蔬菜源头，而不是现在，无端端的多了竞争对手。<br/>　　<br/>　　好在沈赵两家人，虽然斗争的厉害，但是相互之间还是有着底线的，并且很多时候都是自家斗可以，外人参和进来绝对会死的很难看的那种关系，是既相互争斗，又相互扶持。<br/>　　<br/>　　赵家的高恒并没有将酒店业务开到西市来，就已经表明了很多东西。而且这个情，沈家还要承。<br/>　　<br/>　　沈博文内心的无语的，这都是什么事情么，将本来是属于自己的东西抢走不说，转身还要让自己承情，这真的是麻麦皮的！这里面要是没有自己的侄女，上次陈默能走出西市你来找我！<br/>　　<br/>　　这次，看到爽肤水这么热，他也非常的眼馋，所以就询问沈婷婷，但是却遭到拒绝。<br/>　　<br/>　　好在沈博文有个优点，就是顾家，对于家族也好对于自家也好，都非常的照顾，所以被拒绝后也就断了心思。<br/>　　<br/>　　不过还是调查了一下，可是这一调查，就让他后悔的想撞墙。<br/>　　<br/>　　爽肤水也是陈默研发的，这他么的都是什么事情啊，一次错误的决策，竟然落下了这么多的后遗症。<br/>　　<br/>　　现在要是折腾陈默，那么自己绝对绕不开自家侄女，所以沈博文除了叹气，什么也做不了。<br/>　　<br/>　　他看着席家的丫头和自家侄女，合开的专卖店，他都有心砸了说。如果爽肤水这种东西，放到自己手里头，那么一瓶的价格不买到个五万，那都不是做生意的。<br/>　　<br/>　　要知道这个爽肤水他可是亲自试验过，效果绝对是碾压所有化妆品的级别，实在是好上加好，非常的好。<br/>　　<br/>　　但是就是这么好的东西，却在席家丫头和自家侄女手里，像是过家家一样的再玩。<br/>　　<br/>　　是的，在他的眼中，两个丫头就是再玩，没有什么销售策划，也没有开拓市场，更不用说什么广告啊媒体的，就这么开了一家店，然后就开业售卖了。<br/>　　<br/>　　结果，就是这样，还买的比较红火，这也体现了爽肤水到底有多好！<br/>　　<br/>　　看着她们的售卖方式，真的是很无语，要是让他来的话，每个月不说大了，一万瓶都是小意思，而且要是铺货全国的话，那是个什么样的结果？<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
