　　悠悠岁月，不知千年，时光如水，一去不回。
<br/>　　在这千年的时间内，各族都有发展。巫族因为失去了四位祖巫，如今只有八位，却使得他在三族中处于弱势。
<br/>　　不过，千年之前后土身化轮回，巫族将一些族中成员送到地府，既解决了地府缺人的局势，又保留了巫族的一些族人，使得巫族不至于灭绝。所以巫族此时完全没有后顾之忧，战斗起来却是最为疯狂。
<br/>　　而且，后土建立轮回之前，还留下了自己的一部分精血，足够巫族再次造出一位祖巫。
<br/>　　人族之中，经过血与泪的清洗，经过仇恨的发酵，经过战斗的洗礼，战后迎来了一个突破的爆发时期，各个阶段的高手都有突破。最明显的便是，洞天武尊境与法相武相境大量增加，堪比准圣的转轮武皇境，虽然只增加一位，但已是转轮的人族高手却再有突破，突破到了转轮中期。
<br/>　　如今，人族的实力已经能够压倒巫、妖两族中的任意一族。洪荒大地之上的妖族，经过这些年人族强者与巫族强者的猎杀，已经变得很少了，要么直接撤入天界，要么被逼迫到西方大地。
<br/>　　于是，洪荒大地之上，人族与巫族两族形成对峙之势。妖族占据洪荒天界，目光却紧紧盯着地界。
<br/>　　如今，巫族在忙着创造祖巫，妖族再忙着恢复天界，人族也在秘密发展，所以，也没有谁有心思在这个时候与人族开战。
<br/>　　诡异的平衡，就仿佛是暴风雨来临之前的异象。
<br/>　　却说，妖帝帝俊与羲和联姻之后，生下十只小金乌，为妖族十大太子。妖族十大太子中，除了那第十只金乌陆压之外，其他各个都是顽劣之辈，经常在天宫之上大玩大闹，弄得天界乌烟瘴气，不得安宁。
<br/>　　是以，后来帝俊实在是没办法了，便将十位太子封印了起来。
<br/>　　在大海极东之地，有一座岛屿，岛的形状像一个山谷，名为汤谷。
<br/>　　汤谷的水常年都是热的，在汤谷的中央有一棵数，叫扶桑树。当然，这并不是太阳星上的十大先天灵根之一的扶桑树，而只是一株向着扶桑树进化的神树，其已经具备了扶桑神树的许多特征，十分不平凡，所以叫扶桑树。
<br/>　　汤谷中的的扶桑树的叶子通红，隐隐有神火火苗在上面不时冒出，远远看去，就像一团巨大的神火。
<br/>　　大树高达三百丈，树冠极大，把整个岛都遮蔽起来。整个岛就仿佛是一座飘在海面上的火岛。
<br/>　　这正是帝俊封印十大太子的地方，帝俊在岛的周围布下禁制，十大金乌都在其中，不得出岛，好在金乌们在这扶桑树上，也颇为舒适，倒也没有出什么事。
<br/>　　只是，这个岛太小了，十大金乌待久了，也就烦了。故此，每个金乌都想着想要出去玩。只是，他们的修为不够，打不破帝俊设下的封印，无奈只得困于岛中。
<br/>　　这一日，十大金乌正在玩耍，一个身穿黑色道袍的道人来到此处。
<br/>　　“大哥，你说我们何时才能够出得此岛，去洪荒大地之上玩啊！”
<br/>　　“是啊，是啊，大哥！”
<br/>　　“听说外面很好玩，好想出去啊！”
<br/>　　“……”
<br/>　　听到里面传来的声音，神秘道人神秘一笑，默默地将岛外的禁制全部破除，之后隐匿于虚空之中。
<br/>　　金乌们玩着玩着，一个意外，破除了帝俊设下的禁制，顿时，十大金乌纷纷大喜，想要飞出去玩。
<br/>　　“禁制怎么破了，太好了，终于可以出去玩了！”
<br/>　　“出去玩喽，出去喽！”
<br/>　　“走，出去了！”
<br/>　　“……”
<br/>　　“小十，走啊！”
<br/>　　“大哥，我怎么有些觉得不对劲？”
<br/>　　陆压看着大金乌有些疑惑道，按理来说，父皇不可能放他们出去的，眼前的情况却是让他非常疑惑。
<br/>　　“怎么不对劲？我们好久没有出去了，现在出去玩玩，若是不对劲，一会儿再回来不就是了。”大金乌毫不在意道。
<br/>　　“也是！走吧！”
<br/>　　陆压听此，放开心中疑惑，轻声道。终究是忍受不住心中的诱惑，想要飞出洪荒，去见识外面的精彩。
<br/>　　“走！”
<br/>　　十大金乌，终究是出了岛屿，来到精彩的洪荒大地，在天空中自由的飞翔。而他们的出现，也为天地间三族大战，拉开了序幕。
<br/>　　虚空之中，看着飞翔洪荒的十只金乌，神秘道人微微一笑，呢喃道：
<br/>　　“没有大战，本尊怎么提升实力？巫妖大战，本尊期待已久了，就让本尊来掀起这一场大战吧。桀桀桀……”
<br/>　　一阵嚣张的大笑，从虚空中传了出来，久久不散……
<br/>　　却说十只金乌出了岛屿，就仿佛是挣脱了笼子束缚的小鸟，在天空之上自由自在的飞翔，玩得不亦乐乎。但他们却丝毫不知，威胁正一步步的向他们靠近。
<br/>　　十只金乌，尽皆是太阳火精，妖帝后裔，飞翔在天上，顿时迎来无数太阳精气汇聚，再被他们周身的太阳真火点燃，在高空之上，燃烧成一个个被巨大的火球，就仿佛十个巨大的太阳。
<br/>　　十日当空，循着太阳的方向，十大金乌所过之处，河流干涸，生灵尽灭，草木干枯，随即大火点燃，大火燎原，灾难，从天而降，直接笼罩洪荒大地。
<br/>　　最先遭劫的地狱之中，有一处巫族部落。巫族本身就是妖族的死敌，十只金乌虽然不知道下面是哪个巫族的部落，但也知道巫族的可恨，他们毫不犹豫的便接引下无数太阳火精，降临到巫族部落之中。
<br/>　　巫族虽然分为十二个大部落，然而每个大部落中仍有无数中、小部落，每个部落都有自己的首领。部落首领一般都为大巫。中型部落更是有数位大巫，甚至有些大巫的实力直追祖巫。
<br/>　　夸父便是一个小部落的首领。他虽然身处小部落，但实力却无比强大，比起一些中部落的大巫也不差。他的个头在巫族之中是十分高大的，便是在巫族中也不常见。
<br/>　　这十只金乌所降临的便是夸父的部落。
<br/>　　这日，夸父打猎回归，入眼所见尽是一片火海，许多弱小的族人都在火灾之中丧生了，更有许多族人在大火之中挣扎哀嚎！
<br/>　　夸父见此，十分大怒，吼叫道：
<br/>　　“吼！妖族小二，我夸父不将你们碎尸万段，我就不叫夸父！吼！吼！……”
<br/>　　一声吼叫，夸父使出法天象地，身为顶尖大巫，夸父的修为自是无比强大，再使出法天象地之后，更是高达万丈。如同一个顶天的巨人一般。
<br/>　　一路之上，夸父追着十只金乌奔跑，只等有机会，便将十大金乌金乌除去。
