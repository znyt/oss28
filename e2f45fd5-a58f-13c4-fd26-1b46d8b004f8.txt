　　她这番话，反而更加刺激了叶少阳心中的豪情，笑着捏了一下她的脸，说道：“放心，我说了救你，就一定要做到的。”
<br/>　　跑是跑不掉了。叶少阳等毛小方跑到跟前，把小云推给他，说道：“你先带他走，我来对付这个五通神！”
<br/>　　毛小方愣了一下，转头看了一下那朵乌云，叹道：“跑不了，我要是带着她跑，五通神一定会来追我，一起营应敌吧。”
<br/>　　叶少阳转身看了云儿一眼，云儿已经吓得瘫在地上，魂不附体。
<br/>　　“别怕，一切有我！”
<br/>　　叶少阳说完，两腿分开，挡在她的身前，一脸的凌然无惧。云儿从后面看到他的脸，不知道为什么，有一种很安心的感觉。
<br/>　　乌云飞快地来到叶少三人头顶上空，然后迅速扩大，好像一个罩子，从上空罩了下来，瞬间挡住了阳光，叶少阳三人好像从白天一下子到了夜晚，还在抬头看天，突然听见脚步声从前方响起，低头一看，一个身穿绿袍的矮胖子走了过来，正是昨晚见过的那个五通神。
<br/>　　云儿一看到五通神，吓得浑身哆嗦，抱住了叶少阳的腿。
<br/>　　毛小方站在叶少阳身边，做好了斗法的准备。
<br/>　　五通神站住，看看毛小方，又看看叶少阳，两只手背在后面，看上去一副十分悠闲的样子，说道：“昨晚上的火，是你们放的吧。”
<br/>　　叶少阳道：“好可惜，没烧死你。”
<br/>　　五通神咧开大嘴，笑了笑，“你们见到我，并不惧怕，我也知道你们是法师，不过……我有点不敢相信，你们敢抢我女人？”
<br/>　　叶少阳耸了耸肩。
<br/>　　毛小方往地上吐了口唾沫，道：“平生不信帝释天。划个道吧。”
<br/>　　这句话是佛门中一个典故，帝释天得道，被奉为佛教护法，但是有几大宗派不同意，就喊出了这个口号，不承认帝释天在佛门中的地位，毛小方此刻喊出这句话，意思就是不把五通神当神的意思。
<br/>　　五通神笑了笑，“大水冲了龙王庙，你们两个法师，何故要抢我的女人？”
<br/>　　叶少阳道：“你这话刚好说反了，到底是我抢你女人，还是你抢良家妇女，哦不对，是少女。”
<br/>　　五通神不以为意，笑道：“我镇守一方古墓，享受当地百姓的血食，他们自愿服侍于我，有何不可？”
<br/>　　“自愿？”叶少阳转头看云儿，“你是自愿吗？”
<br/>　　“我……”云儿刚要回答，一抬头看到五通神凶狠的目光，立刻有些畏惧。叶少阳看出来了，对她说道：“你不要怕，实话实说。我一定带你走。”
<br/>　　云儿低着头不敢看五通神，细声说道：“是我父亲他们强迫我服侍他的……”
<br/>　　叶少阳对五通神道：“你听到了？”
<br/>　　五通神脸有点黑，其实云儿指认他，并不能代表什么，不过五通神毕竟不是一般的邪物，要真是一般邪物，在这时候根本就什么都不管了，他大小也是一尊神，面子也是要一点的。”
<br/>　　五通神冷哼一声，道：“这又能代表什么？我为这一方百姓，勤勉尽职，享受一个女子又算得了什么，你也听到了，是她的父亲和家人愿意把她送给我，她父母家人都没有话说，你们有什么话说？”
<br/>　　叶少阳道：“我们是讲人/权的，她父母也不能代表她，何况你何止是糟蹋一个姑娘，这些年来，你糟蹋了多少姑娘？”
<br/>　　五通神沉下脸来，说道：“若非是看在你二人是法师的份上，本王才不与你们分说这半天，你们亵渎神灵，又该当何罪？”
<br/>　　当下身上的绿袍鼓起，无风自动，一股灵力从袖管里汩汩涌出。
<br/>　　这是要动手的节奏。叶少阳和毛小方立刻把自己的天师牌亮出来，毛小方厉声说道：“我是天道宗一品天师毛小方，这位是茅山传人少阳子，我俩俱是天师，你敢杀我们？”
<br/>　　五通神看到他们手中的天师牌，也愣住了。人间天师，在地府都是挂过玄名的，一旦被杀，阴司必然追查，非同小可。若是邪修的邪物也无所谓了，反正手上都是人命，杀天师跟多杀一个普通人也没什么区别。但五通神不同，他是阴司册封的正神，若是杀害人间天师，阴司必然降罪，甚至会剥夺神位。
<br/>　　他自诩从未杀生，最多也就是糟蹋姑娘，自己的功劳能够压得住，这对他来说，也是一种平衡之术，他一直没有开杀戒，何况今天面对的还是两位天师，自然不想破这个戒。
<br/>　　最重要的是，这两位天师虽然烧了他的庙宇，亵渎神威，但是罪不至死。
<br/>　　叶少阳赌的就是这个。他虽然决意要跟这五通神过不去，但是今天来主要是救人的，得先把云儿带走再说，暂时还不能跟五通神撕破脸皮，而且看五通神这架势，好像也打不过……
<br/>　　五通神望着二人，内心琢磨了半天，最后将灵力收了起来，恨恨说道：“你们走吧。”
<br/>　　叶少阳和毛小方互相看了一眼，把云儿从地上拉起来，刚要走，五通神说道：“把她留下。”
<br/>　　叶少阳道：“我说了要带她走，就一定会带她走。”
<br/>　　五通神眼中露出凶光，说道：“一个女人，对我来说也什么，但俗话说佛争一炷香，我也是要脸面的，你当着我的面，把我女人带走，本王的脸面往哪搁？你若要带她走，今天连你也得留下！”
<br/>　　叶少阳感受到云儿抓着他的手在颤抖，拍了拍她的手背，表面上看风轻云淡，内心其实也在琢磨怎么处理……看五通神的架势，是非要留下云儿不可。叶少阳心念飞转，对五通神笑道：“你让我带她走，回头我给你重塑金身，给你装光。”
<br/>　　五通神冷笑，显然不屑一顾，抬起一只手，一抹流光从掌心浮现，面对叶少阳，一张脸越来越沉，冷冷说道：“本王不想与你纠缠，让你们走，已经是给足了面子，若再纠缠不休，莫怪本王对天师不敬！”
<br/>　　叶少阳深吸一口气，拔出了七星龙泉剑。
