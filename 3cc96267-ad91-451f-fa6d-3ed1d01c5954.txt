        &lt;h3&gt;第三卷第二十五章  回家&lt;/h3&gt;
<br/>
<br/>        陈子霖开着车载着陈子昂回去看父母。
<br/>
<br/>        陈子昂已经有很多年没有回到泉城这个小山村了，儿时的回忆像潮水一样涌来，于六哥、七哥一起上学，一起玩耍的情景历历在目。
<br/>
<br/>        记得小的时候，水果也很少，可是山里面的野生水果很多，野山桃，野杏子，六哥和七哥都会带着陈子昂一起去采摘。
<br/>
<br/>        “子昂，六哥给你们摘杏子吃，你给哥哥唱首歌。”
<br/>
<br/>        “六哥讨厌，每次都让子昂唱歌，子昂都不知道唱什么了。”
<br/>
<br/>        “你唱采蘑菇的小姑娘。”
<br/>
<br/>        “六哥，你能不能再多教教子昂一下新歌啊。”
<br/>
<br/>        “哟，子霖都听腻了？”
<br/>
<br/>        “子霖没听腻，就是六哥应该给我们教一些新的东西。”
<br/>
<br/>        六哥在城里读高中，子昂和子霖才上初中，他六哥每次回来都会给他们带回来一些新奇的东西，偶尔还会给他们俩带回来一颗水果糖。
<br/>
<br/>        “那你们听好了啊.....”六哥还没有开始唱，树枝丫被六哥踩断了，六哥从树上掉下来了。
<br/>
<br/>        六哥手中紧紧的抓着一个树枝，树枝上有两个微微泛黄的杏子。
<br/>
<br/>        “六哥。”子霖和子昂一同奔向六哥，六哥忍着疼。
<br/>
<br/>        “摘了两个黄的，一人一个。”
<br/>
<br/>        “六哥，你没事吧？”
<br/>
<br/>        “没事，六哥结实着呢。”六哥龇牙咧嘴的说。
<br/>
<br/>        过了几天，六哥说出去打工，便再也没有音讯。
<br/>
<br/>        “子昂，你是不是在想六哥？”陈子霖和陈子昂只相差四岁，陈子昂想什么他大概能猜到一点，尤其子昂之前有说过她见了六哥。
<br/>
<br/>        他没有读高中，早早回家就娶了媳妇，陈子昂上高中的时候他就开始娶妻生子了，开始出去打工赚钱养家。
<br/>
<br/>        “嗯，六哥走的前几天你还记得吗？”
<br/>
<br/>        “当然记得，我要上树，他不让我上，然后他摔下来了。”
<br/>
<br/>        “给我们两个一人摘了一颗杏。”
<br/>
<br/>        “子昂，你说要真是六哥，他为什么装着不认识你？我觉得这几年你除了穿的漂亮一点外，长高了，别的并没有多大变化，他不可能不认识你吧？”
<br/>
<br/>        “我是觉得特别像，而且他好像很维护我，但是他又好像第一次见我，所以我觉得很奇怪，我就想去甸城看看。”
<br/>
<br/>        “如果不是六哥呢？”
<br/>
<br/>        “如果不是六哥，那咱们就继续找。如果是六哥，他不认我们肯定是有理由的。只要我们知道他过得好就行。”陈子昂微不可察的叹了口气。
<br/>
<br/>        谷强，甸城第一企业家、第一慈善家的儿子，手中资产不可估量，谷家一儿一女，女儿浙大毕业，现任谷氏总裁，谷家涉及房产，建筑，公路、运输，甸城的经济命脉基本在他们手中。
<br/>
<br/>        谷强，年少不思进取，不好好读书，被谷老大赶出家门，从最基层做起，现任谷氏企业副总裁，虽说是副的，但谷家所有的事情都是他说了算。
<br/>
<br/>        她了解的信息就这么些。
<br/>
<br/>        她希望他是他的六哥，也许，六哥是为了更好的生活？
<br/>
<br/>        不，六哥不是那样只图钱财的人，不是只会为了自己而不顾家人的人。
<br/>
<br/>        但是，谷强真的是太像六哥了，天底下会有一模一样的俩个人吗？谷家是什么样的家庭呢？谷正娟也会和谷强很像吗？
<br/>
<br/>        “子昂，你知不知道，你和六哥也很像？”
<br/>
<br/>        “我们几个不是都很像吗？”
<br/>
<br/>        “我们家真是两级分化，几个老大长得像爸爸，我们几个小的长得像妈妈，爸和妈也真会生。”陈子霖笑着说。
<br/>
<br/>        陈子霖他们到家了，家里小卖部依然热火朝天的，乡亲们忙忙碌碌的。
<br/>
<br/>        父亲和母亲头发都白了，可是精神很好，耳不聋眼不花。
<br/>
<br/>        “爸、妈！”陈子昂哽咽着喊了一声爸妈，便说不出来话了。
<br/>
<br/>        之前西城开业的时候，爸妈去了西城，她便没有回来，这几年又是忙忙碌碌的，愧疚立马涌上心头，爸妈虽然说身体很好，可是毕竟年龄大了。
<br/>
<br/>        “老幺，你还知道回来？”爸爸语气责怪，可是满脸含笑，大眼睛眯成了缝。
<br/>
<br/>        “我的儿，你回来了？”妈妈拉着陈子昂的手，上上下下左左右右的看着。
<br/>
<br/>        “嗯，我回来了。”陈子昂流着泪笑着说。
<br/>
<br/>        妈妈舍不得放掉陈子昂的手，拉着坐到自己的身边，轻轻的抚摸着陈子昂的手，并且拿起了看了一看，可是看到陈子昂缺了一根指头。
<br/>
<br/>        陈子昂想抽回来手，可是妈妈紧紧的攒在手中，轻轻的抚摸着，并且拿起来放在嘴里含了好一会。
<br/>
<br/>        “我的儿，你这几年受苦了。”妈妈的泪水顺着脸颊流下来。
<br/>
<br/>        “妈，这个是不小心摔跤的，摔坏了，当时医生说只能截掉，要不然会影响整个胳膊的，妈，你看，我这不是好好的嘛。”陈子昂比妈妈高出好多，她将妈妈搂过来，妈妈只能靠在她的肩膀上。
<br/>
<br/>        “当时很疼吧？”
<br/>
<br/>        “不疼的，当时打了麻药，一点都不疼。”当时陈子昂确实不知道疼，因为她的内心充满了愤怒和恐惧。
<br/>
<br/>        “怎么会不疼呢，十指连心啊。”
<br/>
<br/>        “妈，我回来什么都没有给你和我爸带。”陈子昂心里有些难过，走的时候怎么就不想着带点东西好回来啊。
<br/>
<br/>        “你回来就行了，这个世界上有什么东西能比得上你回来呢。”
<br/>
<br/>        “妈，这些年你和我爸辛苦了。”
<br/>
<br/>        “年轻的时候是辛苦，不过现在政策好了，你看，我们家里现在的样子。”妈妈怒了一下嘴，让陈子昂看乡亲们。
<br/>
<br/>        “妈，你和我爸厉害的很。”
<br/>
<br/>        “我和你爸啊，这一生也没有什么值得骄傲的事情，最骄傲的事情就是下了七八个臭崽子。”妈妈用手拨开陈子昂脸上的头发，细细的看着陈子昂。
<br/>
<br/>        “妈，我回来没有带什么东西，这么多乡亲都在，我给他们一点钱？”
<br/>
<br/>        “行，一人给几百块钱吧，你们现在日子都好过了，给他们一点，你小时候可是吃了人家不少饭呢。”他们家里的孩子多，村子里每家陈子昂都去吃过饭。
<br/>
<br/>        “好。”
<br/>
<br/>        虽然每个月都有寄钱回家，但是她知道，父母亲现在不缺钱了。
<br/>
<br/>        最近这几年无论她有多忙，一个礼拜保证打上三次电话，打电话的时候妈妈和爸爸就是问一些家常话，反反复复的重复，她也反反复复的回答。
<br/>
<br/>        陈子昂并没有带东西回来，空着两只手，她觉得有些不好意思，于是给乡亲们每个人发五百块钱，让乡亲们自己买喜欢的东西。
<br/>
<br/>        她现在还算是创业阶段，手中需要资金的地方太多了，再说回来也是临时起意，并没有带太多的钱。
<br/>
<br/>        不过这样，乡亲们也很开心了，他们自己的子女也未必能给他们这么多钱。
<br/>
<br/>        乡亲们已经帮着妈妈把饭做好了，几十个人在道场上摆了几桌，大家像过年一样开心。
<br/>
<br/>        乡亲们七嘴八舌的聊起了陈家孩子的事情，陈子昂微笑的听着，家里真好！
