        明司寒听着她这自带软骨功能的声音，浑身酥软得抖了一下。
<br/>
<br/>        “怎么不说话呢？”连星儿仰起小脑袋，眨巴眼睛看着明司寒的侧脸。
<br/>
<br/>        明司寒回过神来后，睐了连星儿一眼，心里思忖着，她这个问题，简直就是他的“送命题”。
<br/>
<br/>        万一回答得不好，惹这小丫头不开心了怎么办？
<br/>
<br/>        明司寒想了一会儿，求生欲望极强地回答道：“你的什么我都喜欢。”
<br/>
<br/>        连星儿又故作委屈道：“可是……有的人说，是我耍心机、耍手段、还霸王硬上弓，强迫你喜欢我的怎么办？”
<br/>
<br/>        明司寒噗嗤一笑，抬手揉了揉连星儿的头顶，然后将这丫头的小脑袋按在自己的肩头与脸颊之间：“你什么时候对我耍心机、耍手段、还霸王硬上弓了？”
<br/>
<br/>        顿了一下，明司寒忽然又想到了什么，微微偏头，勾起连星儿的下巴，与她四目相对，声音突然暧昧起来：“不过，你打算什么时候对我霸王硬上弓？我倒是十分期待了。”
<br/>
<br/>        “唰”地一下，连星儿脸一热，一股酥麻的感觉，就像触电般从心头窜出。
<br/>
<br/>        此时此刻，后排那三个女生脸色是半紫半红，极其难看。
<br/>
<br/>        而坐在明司寒另一边的张子晋他们，看着这对掉进蜜罐里的小情侣，鸡皮疙瘩掉了一地。
<br/>
<br/>        明司寒嘴角微扬，细细地打量着连星儿那双颊上的红润，瞧着它向白玉般的耳垂蔓延，直至占领她整个精致的小耳朵。
<br/>
<br/>        这丫头此刻可爱到令他有种犯罪的冲动……
<br/>
<br/>        哎——
<br/>
<br/>        他怎么就早没发现，这丫头对自己有意思了呢？
<br/>
<br/>        若是再早点就好了。
<br/>
<br/>        明司寒突然起身，顺带着将连星儿也从椅子上拉了起来。
<br/>
<br/>        还有一分钟就要上课了，但是他却直接牵着她，在众目睽睽之下离开了教室。
<br/>
<br/>        “你要带我去哪儿？”连星儿一头雾水地跟在明司寒的身侧。
<br/>
<br/>        他牵着她的那只大手，好像跟她的脸颊一样在发烫。
<br/>
<br/>        上课铃声响起后，学生们全部回到了各自的教室里。
<br/>
<br/>        明司寒牵着连星儿去了教学楼旁的榕树林。
<br/>
<br/>        直到他突然停下脚步，她还像个愣头青般继续往前走。
<br/>
<br/>        明司寒会心一笑，直接将连星儿往回拉，揽入怀中的下一秒，他托住她的后脑勺，俯身吻了下去。
<br/>
<br/>        连星儿倏然瞪大了眼睛，瞳仁里剪影着明司寒的那张俊脸。
<br/>
<br/>        他闭着眼睛，睫毛长而翘，整张脸帅得令人惊艳。
<br/>
<br/>        就在她惊怔地微张开嘴唇的一瞬间，有个柔软的东西乘机而入。
<br/>
<br/>        这个热吻来得太突然，甚至令她防不胜防。
<br/>
<br/>        不知过了多久……
<br/>
<br/>        他和她耳鬓厮磨，呢喃细语地问道：“星儿，从宿舍搬出来，跟我同居，好不好？”
<br/>
<br/>        回教室前，连星儿将耳后根的长发拉了出来遮挡着自己的耳朵和脖子。
<br/>
<br/>        为了不引起大家的注意，他俩走的后门，坐在了阶梯教室里的最后一排。
<br/>
<br/>        同居的这件事情，连星儿没有立马给明司寒一个答复。
<br/>
<br/>        上完课，回到宿舍里后，连星儿把这事儿跟三个舍友一说，她们三个人则各有各的说法。
<br/>
<br/>        路娅玟第一个赞同：“为什么不同居？你收割了咱们学校的首席校草，成为全校女生的公敌。星儿，你得趁热打铁，把明司寒牢牢抓紧。”
<br/>
<br/>        “你们俩是打算在校外租房子吗？”赵安彤问道。
<br/>
<br/>        连星儿微微摇了摇头：“他在临海城有自己的公寓，不需要另外租房子。”
<br/>
<br/>        顾千凝忍不住插嘴道：“话虽如此，可我总感觉，男生提出想同居，十有八九是‘馋’你的身子。”
<br/>
<br/>        “我的身子？”连星儿下意识地看了看自己，没明白顾千凝话中的意思。
<br/>
<br/>        赵安彤噗嗤一笑，直接开门见山地解释：“千凝的意思是，明司寒是想跟你上床。”
<br/>
<br/>        连星儿顿时无语了。
<br/>
<br/>        顾千凝点头道：“星儿，你还是拒绝吧！”
<br/>
<br/>        路娅玟接着说道：“就算不同居，大学里谈恋爱的男女，也能找机会上床的。我倒是觉得，明大校草想跟星儿同居，只是单纯地想跟星儿多相处在一起。”
<br/>
<br/>        “你们有婚约在身的话，将来肯定是要步入婚姻的殿堂。现在就开始同居，岂不是失去了距离美？”赵安彤振振有词地说道。
<br/>
<br/>        顾千凝忽然一惊一乍起来：“莫非，明司寒是想跟你婚前试爱？”
<br/>
<br/>        “什么婚前试爱？”连星儿更加迷茫了。
<br/>
<br/>        顾千凝一脸严肃地回答道：“就是，在结婚前先跟你同居相处，如果发现彼此不合适，就会立马解除婚约。”
